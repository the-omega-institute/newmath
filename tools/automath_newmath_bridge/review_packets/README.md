# Bridge review packets

This directory is for durable Automath-to-NewMath bridge review packets that
have passed bridge gates and are ready to be cited from the BEDC BOARD.

Runtime material stays in `../inbox`, `../out`, `../state`, and `../logs`.
Those directories are ignored and must not be cited by durable BOARD entries.

Packets here are review inputs only. They do not mean a candidate has been
accepted by BEDC, written into a paper, checked by Lean, or published.
