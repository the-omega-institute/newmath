Your job: classify codex's attempt into one of four verdicts.

2. `reject_hygiene` ÔÇö content is mathematically correct but breaks one or

3. `escalate` ÔÇö proof is unsound, field citations point to non-existent

4. `accept_with_fixups` ÔÇö proof is correct, field citations check out,

   single typo). Provide cleaned content.

Review criteria ÔÇö apply rigorously:

A. **Field-citation existence**. For each entry in `field_citations`, grep
   the target chapter file under `papers/bedc/parts/`. The named field /
   lemma / definition MUST exist. If any citation points to a non-existent
   object, verdict is `escalate` with reason naming the missing citation.

B. **Proof correctness**. Walk through codex's LaTeX line by line. Does
   each step actually follow from the cited field? Is the chained
   reasoning sound? Are intermediate `$...$` claims supported? If you
   spot a step that does NOT follow from cited fields, verdict is
   `escalate` with reason naming the unsupported step.

C. **On-topic**. Codex's proof must address the target's stated Problem
   from the BOARD context. If codex proves something adjacent but not
   the target claim, verdict is `escalate` with reason "off-topic:
   <what was proved instead>".

D. **Hygiene** (these are corrective-retry fixable; verdict
   `reject_hygiene`):

     (not `$$...$$`)
   - bare typed-name macros (not `\mathsf{{X}}` when `\X` is in preamble)
   - no manual proof markers (`Q.E.D.`, `QED`, `\qed`, `\square`,

     `\leanstmt` macros

     `\leantarget` blocks

     `ŠťČŔŻ«`, `vNNN-`, etc.)
   - English LaTeX content only
