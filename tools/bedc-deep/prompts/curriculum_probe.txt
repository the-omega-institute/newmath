## Size discipline — STEPPING STONES BEFORE HEADLINES

Codex closes a target in 1-3 rounds only when ALL the lemmas its proof
needs ALREADY EXIST in the paper. If a candidate's proof requires
infrastructure that does not yet exist (a missing definition, missing
fold lemma, missing dual structure), codex's audit drops to ≤6 and the
target escalates to the oracle, which then takes 30-90 minutes per
target trying to write the entire infrastructure stack from scratch.

That outcome is the failure mode this prompt must avoid.

Before proposing a candidate, decompose it into its proof obligations:

1. List the named objects, lemmas, and constructors the proof would
   reference (you can sketch the proof briefly in your head).
2. For each, run a focused Grep across `papers/bedc/parts/` to verify
   it is already present.
3. If ANY required obligation is missing, DO NOT propose the headline
   theorem. Propose the missing obligation instead.

Examples of headline theorems that should be DEFERRED until their
prerequisites land (do not propose these directly):

  - **Determinant multiplicativity** det(AB) = det(A)·det(B)
    Requires: finite Sym(n) carrier, sign function, alternating
    multilinear pairing — currently only the 1×1 singleton case is
    formalized. Propose the prereqs first: "Sym(n) finite carrier",
    "Sym(n) sign function on transposition spine", "Alternating
    multilinear pairing on Fin(n) tuple".
  - **Lagrange's theorem** |G| = |H|·[G:H]
    Requires: right-coset representative spine, coset partition, finite
    quotient cardinality readback. Propose: "Right-coset partition
    closes G into disjoint H-orbits", "Coset cardinality equals |H|".
  - **Euclid's lemma** prime divides product → prime divides factor
    Requires: GCD existence + Bezout + divides-as-multiplication
    interface that the Nat chapters do not yet uniformly carry.
    Propose: "Bezout coefficients for two unary histories", "GCD
    uniqueness up to history-sameness".
  - **Polynomial factor theorem** root r implies (x − r) divides p
    Requires: polynomial evaluation predicate, division algorithm,
    factor characterization. Propose: "Polynomial evaluation is an
    additive homomorphism" (already in BOARD as B-128) before factor
    theorem.
  - **Rank-nullity for linear maps**
    Requires: dimension as cardinality of basis, finite basis spine,
    direct sum universal property. Propose those building blocks
    first.

A good curriculum candidate is a **stepping-stone lemma whose proof
uses only what is already in the paper**, even when the goal is
ultimately a famous textbook theorem several layers above.
