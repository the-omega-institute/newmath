1. Read each rejection reason. Map it to specific lines in the original
   content. The mapping rules:

   - "$...$ inline math" / "use \( \[" → replace `$...$` with `\(...\)`
     and `$$...$$` with `\[...\]` (display blocks on their own line).
   - "Q.E.D." / "\qed" / "\square" / "as required" / "as desired" inside
     proof body → delete the marker; the `proof` environment auto-emits
     the closing symbol.
   - "\mathsf{X}" where `\X` exists in `papers/bedc/preamble.tex` → replace
     `\mathsf{X}` with the bare macro `\X`. For names NOT in preamble,
     `\mathsf{X}` is acceptable.
   - "ad-hoc subscript on preamble macro" (e.g. `\hsame_R`, `+_R`) → either
     remove the subscript if context makes the structure obvious, or replace
     with a fully-qualified named operator (e.g. `\(\hsame[R]\)` is fine if
     `\hsame` accepts an optional argument; otherwise inline as `(R)`).
   - "section opener" / "\section / \chapter" → delete the sectioning command;
     the block is appended into an existing chapter file.
   - "iteration vocabulary" / "we extend" / "新增" / "patch" / "vNNN-" →
     rewrite the prose in current-state form: state the result as canonical
     fact, no reference to prior drafts.
   - "topic mismatch" / "off-topic" → STOP. This is not a hygiene fix; the
     target was answered with content for a different target. Set
     `verdict=escalate` with reason "topic mismatch — original content is
     about <X>, target is <{target_title}>".

     that does not collide. If the canonical statement is genuinely already
     in the paper, set `verdict=duplicate_of` with `duplicate_label` set.
   - "non-LaTeX trailing content" / "NEXT:" / scratchpad markers → strip every
     line outside theorem / proof / definition environments that is not body
     prose.
   - "Chinese characters in LaTeX" → translate to English.
