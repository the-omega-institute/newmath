\section{First derived interface seeds}

\begin{definition}[Unary history]
A unary history is a generated history built from \(\e\) using only \(\Eone\).  This is not yet a natural number; it is only a repetition history.
\end{definition}

\begin{corollary}[Natural-number-like interface seed]
The family of unary histories licenses a minimal \(\mathsf{Nat}^{\uparrow}\) seed once supplied with a name certificate whose pattern is repeated one-emission and whose stability certifies closure under one-extension.
\end{corollary}

\begin{corollary}[Addition-like interface seed]
Continuation of unary histories licenses a minimal \(\mathsf{Add}^{\uparrow}\) seed once supplied with a name certificate whose stability certifies unit and associativity behavior of continuation.
\end{corollary}

\begin{remark}
These are only interface seeds.  They do not yet amount to a full arithmetic reconstruction.
\end{remark}
