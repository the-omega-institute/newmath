\section{Thin completion interface}

\begin{definition}[Thin seal interface]
A seal-like interface records the shape of an eventual completion module: staged data, restriction events, compatible threads, a seal event, and a tail ledger.  It is not part of the finite kernel proof target.
\end{definition}

\begin{remark}
Real-like, limit-like, circle-like, and fold-like interfaces should not be expanded before the finite kernel is mechanically checked.  They remain downstream modules.
\end{remark}
