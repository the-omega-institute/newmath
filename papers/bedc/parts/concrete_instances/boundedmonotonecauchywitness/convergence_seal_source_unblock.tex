A global impossibility theorem for bounded monotone convergence would require a theorem row not indexed by the local packet $W$. A host theorem about real completion would require host equality, ambient completeness, or quotient data outside the displayed route. Hidden external validation would require an external certificate channel beyond $P$, $N$, and the displayed consumer rows. The source theorem and non-escape theorem exclude exactly those coordinates. Hence the obstruction is strict and local: it blocks only the named converse or reuse route and leaves every consumer route that remains inside the displayed packet available.
