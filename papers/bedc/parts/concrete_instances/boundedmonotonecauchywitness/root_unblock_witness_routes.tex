\begin{theorem}[Bounded monotone Cauchy witness root extraction]

$$

$$

\begin{proof}
Project the carrier of \autoref{def:bounded-monotone-cauchy-witness-carrier}. Use the window and finite-observation locks. Finally apply the convergence-seal input package of \autoref{thm:bounded-monotone-cauchy-witness-convergence-seal-input-package}. That theorem identifies the downstream seal inputs with the same rows of $W$, so the root extraction has no coordinate from which an excluded completion object or choice-like selector could be produced.
\end{proof}
