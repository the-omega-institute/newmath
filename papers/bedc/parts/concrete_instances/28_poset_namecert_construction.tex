\chapter{A Concrete Naming Certificate for \(\POSetUp\)}
\label{ch:concrete-instances-poset-namecert}

The partially ordered set interface \(\POSetUp\) extends \autoref{ch:concrete-instances-preorder-namecert} with the antisymmetry field: if \(a \le b\) and \(b \le a\) then \(a\) and \(b\) are classified the same by the carrier. The integer order from \autoref{ch:concrete-instances-int-namecert} is the canonical concrete instance.

\section{Carrier definition}

\begin{definition}[POSet carrier]
\label{def:poset-carrier}
Use a preorder setup extended with antisymmetry under the carrier classifier.
\end{definition}

\section{The certificate}

\begin{definition}[POSet source specification]
\label{def:poset-source-specification}
The source is a certified preorder carrier plus an antisymmetry field.
\end{definition}

\begin{definition}[POSet pattern specification]
\label{def:poset-pattern-specification}
The pattern is preorder relation formation with classifier collapse from bidirectional comparison.
\end{definition}

\begin{definition}[POSet classifier specification]
\label{def:poset-classifier-specification}
The classifier is the preorder carrier classifier.
\end{definition}

\begin{definition}[POSet stability certificate]
\label{def:poset-stability-certificate}
The stability certificate consists of:
\begin{enumerate}
  \item all preorder stability fields
  \item antisymmetry: \(a\le b\) and \(b\le a\) imply \(a\sim_C b\)
  \item compatibility of antisymmetry with relation congruence
\end{enumerate}
\end{definition}

\begin{definition}[POSet ledger policy]
\label{def:poset-ledger-policy}
Record both comparison derivations when antisymmetry is used.
\end{definition}

\begin{theorem}[Concrete \(\POSetUp\) naming certificate]
\label{thm:concrete-poset-namecert}
Any preorder setup with antisymmetry forms a valid NameCert for \(\POSetUp(C)\).
\end{theorem}

\begin{corollary}[\(\POSetUp\) is licensed, not primitive]
\label{cor:poset-licensed-not-primitive}
\(\POSetUp\) is licensed as the antisymmetric specialization of \(\PreorderUp\).
\end{corollary}
