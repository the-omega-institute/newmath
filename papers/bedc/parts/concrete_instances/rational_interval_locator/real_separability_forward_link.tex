\label{thm:rational-interval-locator-real-separability-forward-link}

Finally $H,C,P,N$ transport, replay, source, and name exactly those rows. Since no row stores a host interval, selected dense point, quotient representative, countable-choice schedule, or ambient completeness witness, the forward link is a finite locator-to-separability dependency and nothing beyond that boundary.
\end{proof}
