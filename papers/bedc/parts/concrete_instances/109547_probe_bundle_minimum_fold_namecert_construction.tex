Finally the handoff row $U$ gives the compact-uniform-continuity consumer a tolerance-only precision history. The structural rows $H,C,Q,N$ transport, replay, source, and name the same finite route, so no host finite minimum, escaped probe, open-cover compactness object, quotient point equality, external modulus selector, or ambient real-completeness object can enter the certificate.
\end{proof}
