\begin{theorem}[Attouch-Wets topology NameCert obligation surface]
\label{thm:attouch-wets-topology-namecert-obligation-surface}
For every accepted $\AttouchWetsTopologyUp$ carrier
$$
A_{\mathrm{aw}}=(M,H,D,W,R,E,Q,T,C,P,N),
$$
the local $\NameCert_{\AttouchWetsTopologyUp}$ surface consists exactly of the following obligations:
\begin{enumerate}
\item expose the $\MetricSpaceUp$ source row $M$ before any local Hausdorff comparison is read;
\item display the $\HausdorffDistanceUp$ window row $H$ only on bounded metric windows named by the packet;
\item bind all numerical precision to the dyadic tolerance ledger $D$, the $\StreamNameUp$ row $W$, the $\RegSeqRatUp$ readback $R$, and the terminal $\RealUp$ seal $E$;
\item preserve the same finite route by componentwise $\hsame$ transport $Q$ and displayed $\Cont$ replay $T$;
\item source and name the same packet through the bounded-window classifier $C$, $\Pkg$ provenance $P$, and the local $\NameCert$ row $N$.
\end{enumerate}
No obligation row is stored for a host hyperspace topology, arbitrary closed-set compactness, selected representatives, quotient closed-set equality, countable choice, or ambient $\RealUp$ completeness.
\end{theorem}
\begin{proof}
Project the accepted carrier of \autoref{def:attouch-wets-topology-carrier}. The metric row $M$ is the only source of ambient distance data, and the Hausdorff row $H$ is read only after the bounded window has been displayed.

The numerical part is downstream of that window. The tolerance ledger $D$, finite window row $W$, regular readback $R$, and real seal $E$ are the only precision and value rows available to an observation request.

Transport and replay keep the same route. Rows $Q$ and $T$ preserve the displayed metric, Hausdorff, tolerance, window, readback, and seal data, while $C,P,N$ classify, source, and name the finite packet.

Since the carrier contains none of the excluded topology, compactness, selection, quotient, choice, or ambient-completeness coordinates, the displayed obligations exhaust the local NameCert surface.
\end{proof}
