\item $D$ is the dyadic tolerance ledger used to compare dense approximants, finite windows, and the terminal Real seal.

The classifier compares accepted packets only through $Q,F,D,R,E,U,K,H,C,P,N$. It has no coordinate for ambient $\RealUp$ completeness, quotient streams, selected host limits, countable choice, host function equality, Lean verification, or bridge checking.
