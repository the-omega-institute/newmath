\chapter{A Concrete Naming Certificate for $\AdjointRepUp$}
\label{ch:concrete-instances-adjointrep-namecert}

The adjoint-representation interface $\AdjointRepUp$ packages the conjugation action $\mathrm{Ad}\colon G \to \mathrm{Aut}(\mathfrak{g})$ together with its differential $\mathrm{ad}$. It depends on $\NameCert_{\LieGroupUp}$ (\autoref{ch:concrete-instances-liegroup-namecert}) and $\NameCert_{\LieAlgebraUp}$ (\autoref{ch:concrete-instances-liealgebra-namecert}).

\section{The certificate}
\label{sec:adjointrep-certificate}

\begin{theorem}[AdjointRep dependency source obligation]
\label{thm:adjointrep-dependency-source-obligation}
The dependency source for $\AdjointRepUp$ is exactly the displayed
$\LieGroupUp$ surface of
\autoref{ch:concrete-instances-liegroup-namecert} together with the displayed
$\LieAlgebraUp$ surface of
\autoref{ch:concrete-instances-liealgebra-namecert}. Every
$\AdjointRepUp$ row is read from these two $\NameCert$ sources: the group
source supplies the carrier, classifier, identity, multiplication, inverse,
and conjugation surface, while the algebra source supplies the carrier,
classifier, bracket, and adjoint endomap surface.
\end{theorem}
\begin{proof}
Unpack the $\AdjointRepUp$ interface as a dependent certificate over the two
cited public surfaces. No third ambient carrier is named in the chapter, and
no host equality is used to identify the two dependency rows.

The BEDC package discipline therefore fixes the source boundary before the
action rows are interpreted. The group side contributes only
$\LieGroupUp$-classified histories, and the algebra side contributes only
$\LieAlgebraUp$-classified histories.

Thus every subsequent $\mathrm{Ad}$, $\mathrm{ad}$, transport, stability, and
target row has these two sources as its complete input surface.
\end{proof}

\begin{theorem}[AdjointRep conjugation carrier obligation]
\label{thm:adjointrep-conjugation-carrier-obligation}
The $\mathrm{Ad}$ row is carried by the $\LieGroupUp$ singleton conjugation
carrier. For each accepted group history and accepted algebra history, the
conjugation endpoint remains a $\mathsf{BHist}$ action endpoint rather than a
host function outside the BEDC ledger.
\end{theorem}
\leanchecked{BEDC.Derived.AdjointRepUp.AdjointRepConjugation\_carrier\_obligation}
\begin{proof}
Start with the $\LieGroupUp$ source surface fixed by
\autoref{thm:adjointrep-dependency-source-obligation}. Its singleton carrier
already contains the conjugation source row and the classifier that licenses
group histories.

Feed an accepted algebra history from the $\LieAlgebraUp$ source into that
row. The result is recorded as a BEDC history endpoint, so the visible action
does not leave the $\mathsf{BHist}$ carrier.

The obligation is exactly this carrier check: $\mathrm{Ad}$ must be displayed
as an action endpoint in the ledger, with all representative choices hidden
behind $\hsame$ transport.
\end{proof}
\leanchecked{BEDC.Derived.AdjointRepUp.AdjointRepConjugationCarrier\_obligation}

\begin{theorem}[AdjointRep differential action obligation]
\label{thm:adjointrep-differential-action-obligation}
The $\mathrm{ad}$ row is the $\LieAlgebraUp$ adjoint endomap package generated
by the bracket. Its input and output endpoints are classified by the
$\LieAlgebraUp$ surface, and the row is not a separate derivative object over
an unlicensed tangent carrier.
\end{theorem}
\leanchecked{BEDC.Derived.AdjointRepUp.AdjointRepDifferentialAction\_obligation}
\begin{proof}
Use the $\LieAlgebraUp$ dependency named in
\autoref{thm:adjointrep-dependency-source-obligation}. That source already
exposes the bracket classifier and the adjoint endomap package used by the
algebra carrier.

Reading the differential at the identity group endpoint can therefore only
select the displayed adjoint endomap row. It does not create a second tangent
carrier, because the source package has no public row for one.

The certificate obligation is to keep $\mathrm{ad}$ attached to this bracket
endomap package and to record its endpoints through the same $\mathsf{BHist}$
and $\hsame$ classifier discipline as the algebra source.
\end{proof}

\begin{theorem}[AdjointRep automorphism target obligation]
\label{thm:adjointrep-automorphism-target-obligation}
The target row for $\mathrm{Ad}$ is an automorphism-style carrier over the
$\LieAlgebraUp$ classifier. Its inverse and composition data are read from
the $\LieGroupUp$ inverse and multiplication rows together with the
$\LieAlgebraUp$ endomap rows.
\end{theorem}
\leanchecked{BEDC.Derived.AdjointRepUp.AdjointRepAutomorphism\_target\_obligation}
\begin{proof}
Apply the conjugation carrier obligation to obtain the visible
$\mathrm{Ad}$ endpoint. Apply the group inverse and multiplication rows from
$\LieGroupUp$ to read inverse and composite group histories inside the same
classifier boundary.

Next use the $\LieAlgebraUp$ endomap package. It supplies the target
classifier over which composition of algebra endpoints is interpreted.

The automorphism-style row is then a repacking of these two source surfaces:
group inverse gives the inverse action endpoint, group multiplication gives
composition of action endpoints, and algebra endomaps keep the target over
the Lie-algebra classifier.
\end{proof}

\begin{theorem}[AdjointRep classifier stability obligation]
\label{thm:adjointrep-classifier-stability-obligation}
Transported $\LieGroupUp$ and $\LieAlgebraUp$ inputs preserve the displayed
$\mathrm{Ad}$ and $\mathrm{ad}$ endpoints through $\hsame$, $\Cont$, and
NameCert classifier transport. Consequently the action row, differential row,
and automorphism-target row are stable under the two source classifiers.
\end{theorem}
\leanchecked{BEDC.Derived.AdjointRepUp.AdjointRepSingleton\_classifier\_stability\_obligation}
\leanvariant{BEDC.Derived.AdjointRepUp.AdjointRepClassifier\_stability\_obligation}
\begin{proof}
First transport the group input along the $\LieGroupUp$ classifier and
$\hsame$. The conjugation carrier obligation reads the resulting
$\mathrm{Ad}$ endpoint from the same singleton action row, so the endpoint is
unchanged up to the displayed BEDC classifier.

Next transport the algebra input along the $\LieAlgebraUp$ classifier,
$\hsame$, and the continuation ledger $\Cont$. The differential action
obligation keeps $\mathrm{ad}$ inside the bracket endomap package, and the
automorphism target obligation keeps $\mathrm{Ad}$ inside the algebra endomap
target.

These two transports are the whole stability check. No row is stabilized by
host equality; every preserved endpoint is read through the two NameCert
classifiers and the BEDC continuation relation.
\end{proof}
