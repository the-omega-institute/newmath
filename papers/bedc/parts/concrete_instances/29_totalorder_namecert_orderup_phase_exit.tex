\section{OrderUp Phase Exit}
\label{sec:totalorder-orderup-phase-exit}

\begin{theorem}[Unary prefix OrderUp phase exit package]
\label{thm:unary-prefix-orderup-phase-exit-package}
The concrete L7 $\OrderUp$ package over $\mathsf{BHist}$ is the unary-prefix
package with carrier $\mathsf{UnaryHistory}$, classifier $\rho_{\mathsf{u}}$,
and order relation $\le_{\mathsf{u}}$. Its preorder, antisymmetry, and
constructive comparison rows are inherited from the concrete $\TotalOrderUp$
surface exposed in
\autoref{thm:unary-prefix-totalorder-supplies-arithmetic-orderup}. Its finite
bounded-interval coverage is the interval-collapse statement of
\autoref{thm:unary-prefix-orderup-bounded-interval-collapse}. Its endpoint
classifier uniqueness is the two-endpoint readback of
\autoref{thm:totalorder-prefix-finite-extrema-interval-unique}.

Consequently, the scoped Full Nat Arithmetic $\OrderUp$ phase object can be
consumed as a single package consisting of the carrier
$\mathsf{UnaryHistory}$, classifier $\rho_{\mathsf{u}}$, relation
$\le_{\mathsf{u}}$, inherited preorder and antisymmetry rows, constructive
comparison rows, finite bounded-interval coverage for carried finite spines,
and classifier uniqueness of selected interval endpoints. This package
introduces no second order relation and asserts no classical trichotomy.
\end{theorem}
\begin{proof}
First expose the $\OrderUp$ surface. Apply
\autoref{thm:unary-prefix-totalorder-supplies-arithmetic-orderup}. The consumer
sees the same concrete $\mathsf{UnaryHistory}$ carrier, the same classifier
$\rho_{\mathsf{u}}$, and the same relation $\le_{\mathsf{u}}$ that were used
by the unary $\TotalOrderUp$ certificate.

The order-law rows are therefore projections of the existing certificate. The
reflexivity, transitivity, and antisymmetry rows come from the prefix
$\POSetUp$ part of the $\TotalOrderUp$ package, while the comparison row is
the constructive strict-left, same, or strict-right branch partition recorded
there.

Next project the finite bounds. For a nonempty finite carried spine, apply
\autoref{thm:unary-prefix-orderup-bounded-interval-collapse}. It supplies
selected endpoints $m$ and $M$ such that every member of the spine lies between
$m$ and $M$ under $\le_{\mathsf{u}}$.

Finally project endpoint uniqueness. Apply
\autoref{thm:totalorder-prefix-finite-extrema-interval-unique} to any two
finite extrema interval witness records over the same carried spine. The lower
endpoints are classified by $\rho_{\mathsf{u}}$, and the upper endpoints are
classified by $\rho_{\mathsf{u}}$.

These projected rows are exactly the phase-exit data needed by the Full Nat
Arithmetic consumer for $\OrderUp$: one concrete carrier, one classifier, one
order relation, finite interval coverage, and endpoint classifier uniqueness.
All rows are inherited from the unary prefix $\TotalOrderUp$ construction, so
the package does not create another order and does not replace constructive
comparison by classical trichotomy.
\end{proof}

\begin{theorem}[OrderUp carrier and classifier obligation]
\label{thm:orderup-carrier-classifier-obligation}
The concrete carrier for the unary-prefix $\OrderUp$ phase object is
$\mathsf{UnaryHistory}$. Its classifier is $\rho_{\mathsf{u}}$, and its order
relation is $\le_{\mathsf{u}}$.
\end{theorem}
\begin{proof}
Apply \autoref{thm:unary-prefix-orderup-phase-exit-package}. The phase-exit
package exposes the same carried objects as
\autoref{thm:unary-prefix-totalorder-supplies-arithmetic-orderup}: unary
histories are the carrier, $\rho_{\mathsf{u}}$ is the classifier used to read
them, and $\le_{\mathsf{u}}$ is the only order relation exported to
$\OrderUp$.
\end{proof}

\begin{theorem}[OrderUp preorder and antisymmetry obligation]
\label{thm:orderup-preorder-antisymmetry-obligation}
On carried unary histories, $\le_{\mathsf{u}}$ is reflexive and transitive.
If two carried unary histories lie below each other under $\le_{\mathsf{u}}$,
then their endpoints land in the same $\rho_{\mathsf{u}}$ classifier class.
\end{theorem}
\begin{proof}
First project the order rows from the concrete unary-prefix total-order
certificate through
\autoref{thm:unary-prefix-totalorder-supplies-arithmetic-orderup}. The
$\OrderUp$ phase package of
\autoref{thm:unary-prefix-orderup-phase-exit-package} exports those rows
without introducing another relation.

Reflexivity and transitivity are the preorder rows of the inherited
$\le_{\mathsf{u}}$ surface. Antisymmetry is read in the classifier used by the
same package: mutual $\le_{\mathsf{u}}$ comparison identifies the carried
unary histories up to $\rho_{\mathsf{u}}$, which is the equality notion exposed
to the phase consumer.
\end{proof}

\begin{theorem}[OrderUp constructive comparison obligation]
\label{thm:orderup-constructive-comparison-obligation}
For any two carried unary histories, the $\OrderUp$ comparison data gives one
of the strict-left, same, or strict-right branches from the unary-prefix
total-order package. The phase object exports this constructive branch data
and exports no classical trichotomy principle.
\end{theorem}
\begin{proof}
Use the comparison row supplied by
\autoref{thm:unary-prefix-totalorder-supplies-arithmetic-orderup} and
projected into $\OrderUp$ by
\autoref{thm:unary-prefix-orderup-phase-exit-package}. The branch names and
their witnesses are the concrete unary-prefix strict-left, same, and
strict-right alternatives.

Because the exported row is this constructive package row, the consumer
receives only the witnessed branch split. No additional excluded-middle or
classical trichotomy statement is part of the $\OrderUp$ phase object.
\end{proof}

\begin{theorem}[OrderUp finite-interval and endpoint obligation]
\label{thm:orderup-finite-interval-endpoint-obligation}
Every nonempty finite carried unary spine has lower and upper bounds under
$\le_{\mathsf{u}}$. Moreover, for any two endpoint-witness records for the
same carried spine, their lower endpoints are $\rho_{\mathsf{u}}$-equivalent
and their upper endpoints are $\rho_{\mathsf{u}}$-equivalent.
\end{theorem}
\begin{proof}
First obtain finite interval coverage from
\autoref{thm:unary-prefix-orderup-bounded-interval-collapse}. For a nonempty
finite carried spine, it supplies selected lower and upper unary endpoints
with every carried member lying between them under $\le_{\mathsf{u}}$.

Next apply
\autoref{thm:totalorder-prefix-finite-extrema-interval-unique} to two
endpoint-witness records over the same carried spine. The theorem identifies
the selected lower endpoints through $\rho_{\mathsf{u}}$ and likewise
identifies the selected upper endpoints through $\rho_{\mathsf{u}}$.

Together with the phase package of
\autoref{thm:unary-prefix-orderup-phase-exit-package}, these are precisely the
finite-interval and endpoint-uniqueness obligations exported by the concrete
unary-prefix $\OrderUp$ object.
\end{proof}
