\begin{definition}[Measure root downstream image surface]
\label{def:measure-root-downstream-image-surface}
The $\MeasureUp$ root downstream image surface is the ten-field public image
through which the direct analysis consumers read the root measure package:
\begin{enumerate}
  \item traditional target: a constructive measure root exposed as the common
  source for integration, probability, and functional-analysis measure rows;
  \item BEDC source: $\mathsf{BHist}$, componentwise $\hsame$, $\Cont$,
  $\NameCert_{\RealUp}$ endpoint comparison, $\StreamNameUp$ countable-family
  names, and the additive $\AbGroupUp$ real rows;
  \item carrier: the public $\mathsf{MeasCode}$ row carried by the displayed
  $\mathsf{BHist}$ ledger;
  \item classifier: the public measure classifier $\sim_M$, with endpoint
  motion read through $\NameCert_{\RealUp}$ and $\sim_R$;
  \item stability rows: carrier transport, classifier transport, visible
  event-row transport, finite and countable family transport,
  relative-difference transport, monotone-inclusion transport, nonnegative
  endpoint transport, and continuity-from-below transport;
  \item ledger: hidden membership, disjointness, endpoint-value,
  relative-difference, countable-union, and countable-sum witnesses attached
  only to displayed public $\MeasureUp$ rows;
  \item core theorem list:
  \autoref{thm:measure-root-downstream-image-carrier},
  \autoref{thm:measure-root-downstream-image-additivity}, and
  \autoref{thm:measure-root-downstream-image-consumer};
  \item exactness: every $\IntegralUp$, $\ProbSpaceUp$, and
  $\FunctionalAnalysisUp$ row displayed here is one of the public rows of
  \autoref{def:measure-root-public-obligation-basis},
  \autoref{def:measure-root-domain-ready-package}, and
  \autoref{def:measure-root-analysis-unblock-obligation-surface};
  \item standard bridge: no $\StdBridge$ row is asserted, imported, or needed
  by this image surface;
  \item status: paper proof target under
  $\mathsf{BEDC.Derived.MeasureUp}$.
\end{enumerate}
\end{definition}

\begin{theorem}[Measure root downstream image carrier]
\label{thm:measure-root-downstream-image-carrier}
The carrier and classifier rows of
\autoref{def:measure-root-downstream-image-surface} project to the
$\IntegralUp$, $\ProbSpaceUp$, and $\FunctionalAnalysisUp$ consumers through
the same public $\mathsf{MeasCode}$, $\sim_M$, $\mathsf{BHist}$, $\hsame$,
$\Cont$, $\NameCert_{\RealUp}$, $\StreamNameUp$, and $\AbGroupUp$ rows of the
root $\MeasureUp$ package.
\end{theorem}
\begin{proof}
Start from \autoref{def:measure-root-public-obligation-basis}. Its carrier
and classifier fields display $\mathsf{MeasCode}$, $\sim_M$, the visible
$\mathsf{BHist}$ ledger, endpoint comparison through $\NameCert_{\RealUp}$,
and countable-family transport through $\StreamNameUp$.

Apply \autoref{thm:measure-root-public-obligation-basis-exactness}. The
exactness theorem identifies the carrier face, additive face, and consumer
face as one public root package and excludes host set equality, quotient
event equality, analytic completion objects, and standard bridge rows.

Next read the same carrier-classifier package through
\autoref{def:measure-root-domain-ready-package}. Apply
\autoref{thm:measure-root-domain-downstream-ready}; it names the three
consumer fronts and says that their visible rows are projections of the
domain-ready carrier, classifier, endpoint, additive, and ledger rows.

Finally read the analysis front through
\autoref{def:measure-root-analysis-unblock-obligation-surface}. Apply
\autoref{thm:measure-root-analysis-unblock-downstream-package}. The
analysis-unblock theorem repacks the same public carrier and classifier rows
as the package consumed by $\IntegralUp$, $\ProbSpaceUp$, and
$\FunctionalAnalysisUp$, so the downstream image has no separate carrier or
classifier source.
\end{proof}

\begin{theorem}[Measure root downstream image additivity]
\label{thm:measure-root-downstream-image-additivity}
The finite additivity, countable additivity, relative-difference additivity,
monotone inclusion, nonnegative endpoint, and continuity-from-below rows of
\autoref{def:measure-root-downstream-image-surface} are the same public
$\MeasureUp$ rows seen by $\IntegralUp$, $\ProbSpaceUp$, and
$\FunctionalAnalysisUp$.
\end{theorem}
\begin{proof}
Project the additive and order rows from
\autoref{def:measure-root-public-obligation-basis}. Its exactness theorem,
\autoref{thm:measure-root-public-obligation-basis-exactness}, keeps
empty-zero, finite readback, relative-difference, countable disjointness,
countable-sum, ledger, and consumer rows inside one visible root package.

Use \autoref{def:measure-root-domain-ready-package} as the domain reading of
that package. Apply \autoref{thm:measure-root-domain-downstream-ready}. The
domain-ready theorem exposes the finite, countable, endpoint, ledger, and
consumer rows to the three analysis fronts without adding host or quotient
event equality.

Now read the same rows through
\autoref{def:measure-root-analysis-unblock-obligation-surface}. Apply
\autoref{thm:measure-root-analysis-unblock-downstream-package}. It identifies
finite readback, countable-sum stability, relative-difference additivity,
monotone inclusion, nonnegative endpoint values, and continuity from below as
the rows consumed by the direct analysis fronts.

Thus the additivity, order, endpoint, and continuity fields of the
downstream image are not parallel copies. They are the common public
$\MeasureUp$ rows already displayed by the root basis, the domain-ready
package, and the analysis-unblock surface.
\end{proof}

\begin{theorem}[Measure root downstream image consumer]
\label{thm:measure-root-downstream-image-consumer}
Every $\IntegralUp$, $\ProbSpaceUp$, or $\FunctionalAnalysisUp$ row exposed
by \autoref{def:measure-root-downstream-image-surface} factors through the
displayed public $\MeasureUp$ image. The image adds no host set equality, no
quotient event equality, no analytic completion object, and no standard
bridge row.
\end{theorem}
\begin{proof}
Begin with the consumer face of
\autoref{def:measure-root-public-obligation-basis}. Apply
\autoref{thm:measure-root-public-obligation-basis-exactness}; the theorem
exhausts the public carrier, additive, and consumer faces and leaves no slot
for host set equality, quotient event equality, analytic completion, or a
$\StdBridge$ row.

Next pass through \autoref{def:measure-root-domain-ready-package}. Apply
\autoref{thm:measure-root-domain-downstream-ready}. The theorem lists the
$\IntegralUp$, $\ProbSpaceUp$, and $\FunctionalAnalysisUp$ projections as the
downstream face of the domain-ready root package and keeps every consumed row
attached to the displayed carrier, classifier, endpoint, additive, ledger,
and exactness fields.

Finally pass through
\autoref{def:measure-root-analysis-unblock-obligation-surface}. Apply
\autoref{thm:measure-root-analysis-unblock-downstream-package}. This
analysis-unblock reading states that the three direct consumers use only the
carrier, classifier, stability, exactness, ledger, additivity, order,
endpoint, and continuity rows named in the public surface.

Combining the three readings gives the desired factorization: each direct
analysis consumer row exposed by the downstream image is already a row of
the public $\MeasureUp$ image. Since none of the cited packages contains the
excluded equality, completion, or bridge fields, the downstream image adds
none of them.
\end{proof}
