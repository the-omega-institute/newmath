\section{Measure root real-stream premeasure bridge}
\label{sec:measure-root-real-stream-premeasure-bridge}

The real-stream premeasure bridge is the concrete root surface that lets
downstream analysis chapters read countable premeasure data through the
displayed $\mathsf{BHist}$ event rows, $\StreamNameUp$ observation names, and
$\RealUp$ endpoint comparisons already exposed by the $\MeasureUp$ root. It is
a bridge between existing root rows, not a host measure-space construction.

\begin{definition}[Measure root real-stream premeasure surface]
\label{def:measure-root-real-stream-premeasure-surface}
The $\MeasureUp$ root real-stream premeasure surface is the ten-field surface
with the following data:
\begin{enumerate}
  \item traditional target: a countable premeasure surface whose measurable
  events and countable disjoint sums are visible before any analytic
  completion is admitted;
  \item BEDC source: $\mathsf{BHist}$, $\hsame$,
  $\NameCert_{\RealUp}$, $\StreamNameUp$, and the existing $\MeasureUp$ root
  rows of \autoref{def:measure-root-public-obligation-basis};
  \item carrier: displayed measurable-event $\mathsf{BHist}$ rows, the
  $\StreamNameUp$-indexed observation family on those rows, and the
  $\RealUp$ endpoint value assigned to each displayed event;
  \item classifier: componentwise $\hsame$ for event histories, StreamName
  classifier transport for observation names, and $\RealUp$ endpoint
  classifier transport for event values and countable sums;
  \item stability rows: carrier transport, classifier transport, countable
  disjoint additivity, and ledger exactness over the displayed root rows;
  \item ledger: measurable witnesses, disjointness witnesses,
  countable-family indices, countable-union witnesses, and countable-sum
  endpoints attached to the same displayed $\mathsf{BHist}$ rows;
  \item exactness coverage: every covered row of the surface is one of the
  displayed rows named by the public root basis, the countable-premeasure
  front, and the real-stream event obligation;
  \item exactness soundness: every accepted carrier, classifier, additivity,
  and ledger row has the corresponding visible witness in the same
  $\mathsf{BHist}$ ledger;
  \item exactness completeness: every displayed row with its StreamName
  observation, $\RealUp$ endpoint, disjointness data, union witness, and
  countable-sum endpoint is accepted by the surface;
  \item standard bridge boundary: no host set equality, quotient event
  equality, external sigma-algebra equality, analytic completion object, or
  $\StdBridge$ row is part of this surface.
\end{enumerate}
\end{definition}

\begin{theorem}[Measure root real-stream premeasure carrier obligation]
\label{thm:measure-root-real-stream-premeasure-carrier-obligation}
The surface of
\autoref{def:measure-root-real-stream-premeasure-surface} exposes exactly the
$\mathsf{BHist}$ event carrier, $\StreamNameUp$ observation family,
$\RealUp$ endpoint map, and visible ledger rows needed by the $\MeasureUp$
$\NameCert$.
\end{theorem}
\begin{proof}
Start with the public root basis of
\autoref{def:measure-root-public-obligation-basis}. Its carrier field gives
the displayed $\mathsf{BHist}$ event rows, while its ledger field attaches
measurable witnesses, endpoint witnesses, disjointness witnesses, and
countable-sum witnesses only to displayed rows.

Read the same event row through
\autoref{def:measure-root-real-stream-regularity-front}. The real-stream front
adds the $\StreamNameUp$ observation family and the attached $\RealUp$
endpoint comparison without changing the underlying event carrier.

Apply \autoref{thm:measure-root-real-stream-event-obligation}. It identifies
each carried root event with a displayed event history, a StreamName
observation name, and the real endpoint comparison on the same ledger row.

The carrier, observation, endpoint, and ledger clauses of
\autoref{def:measure-root-real-stream-premeasure-surface} are precisely this
combined reading. Since the boundary clause excludes host set equality and
quotient event equality, the exposed carrier is exactly the visible root
carrier required by the $\MeasureUp$ $\NameCert$.
\end{proof}

\begin{theorem}[Measure root real-stream premeasure classifier obligation]
\label{thm:measure-root-real-stream-premeasure-classifier-obligation}
For the surface of
\autoref{def:measure-root-real-stream-premeasure-surface}, componentwise
$\hsame$, StreamName classifier transport, and $\RealUp$ endpoint classifier
transport preserve the carried premeasure row and its ledger.
\end{theorem}
\begin{proof}
Begin with a displayed premeasure row and a componentwise $\hsame$ comparison
of its $\mathsf{BHist}$ event histories. The public root basis transports the
carried event code and its visible ledger witnesses along the same root
classifier row.

Apply \autoref{thm:measure-root-event-classifier-stability}. It keeps the
carried $\MeasureUp$ root row stable under componentwise $\hsame$ together
with StreamName observation transport and $\RealUp$ endpoint transport.

The classifier field of
\autoref{def:measure-root-real-stream-premeasure-surface} adds no independent
classifier. It records exactly the event-history transport, the StreamName
transport, and the endpoint transport already aligned by the event classifier
stability theorem.

The ledger is preserved because its measurable witnesses, disjointness
witnesses, countable-family indices, union witnesses, and countable-sum
endpoints remain attached to the transported displayed row. Thus the
premeasure row and its ledger are invariant under the stated classifier
motion.
\end{proof}

\begin{theorem}[Measure root real-stream premeasure sigma-additivity obligation]
\label{thm:measure-root-real-stream-premeasure-sigma-additivity-obligation}
The countable disjoint-additivity row of
\autoref{def:measure-root-real-stream-premeasure-surface} is read by combining
the existing root public obligation basis, the countable-premeasure front, the
real-stream event obligation, and countable-sum classifier stability over the
same displayed $\mathsf{BHist}$ ledger.
\end{theorem}
\begin{proof}
Project a countable disjoint family from the ledger of
\autoref{def:measure-root-real-stream-premeasure-surface}. Its component
events, disjointness witnesses, countable-family index, union witness, and
countable-sum endpoint are all displayed rows over the same
$\mathsf{BHist}$ carrier.

Apply \autoref{def:measure-root-public-obligation-basis} and
\autoref{thm:measure-root-countable-additivity-scope}. These keep the
countable additivity obligation scoped to visible event families, their union
row, and the public ledger, with no external sigma-algebra row.

Next apply \autoref{def:measure-root-countable-premeasure-front} and
\autoref{thm:measure-root-countable-premeasure-additivity-obligation}. They
read the same visible family as a countable premeasure row whose union value
is compared with the countable sum of its component $\RealUp$ endpoints.

Finally apply \autoref{thm:measure-root-real-stream-event-obligation} to read
the components through StreamName observations, and apply
\autoref{thm:measure-root-event-classifier-stability} together with
\autoref{thm:measure-obligation-countable-sum-classifier-stability} to keep
endpoint and countable-sum classifier transport aligned with the displayed
event histories.

Combining these rows gives the sigma-additivity obligation of the surface:
the value of the displayed union is the countable $\RealUp$ endpoint sum of
the displayed disjoint component events, with every witness remaining inside
the visible ledger. The conclusion does not assert host-level measure theory,
quotient event equality, or analytic completion.
\end{proof}
