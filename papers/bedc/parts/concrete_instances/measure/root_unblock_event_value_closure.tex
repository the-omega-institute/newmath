\section{Measure root event-value closure}
\label{sec:measure-root-unblock-event-value-closure}

The event-value closure package records the smallest root surface that
downstream analysis chapters may cite when they need the $\MeasureUp$ event
carrier, its public value rows, and its countable-family endpoint rows
together.

\begin{definition}[Measure root event-value closure package]
\label{def:measure-root-unblock-event-value-closure}
The $\MeasureUp$ root event-value closure package is the following
three-field public root package.
\begin{enumerate}
  \item event carrier and classifier field: the carried $\mathsf{BHist}$
  event rows, the $\mathsf{MeasCode}$ carrier predicate, componentwise
  $\hsame$ event transport, and the public $\sim_M$ classifier read through
  $\NameCert_{\RealUp}$ endpoint transport;
  \item countable-family field: each $\StreamNameUp$-indexed row names a
  visible family of carried $\mathsf{BHist}$ events, pairwise-disjoint
  witnesses, countable-union readback, and the attached $\RealUp$
  countable-sum endpoint;
  \item endpoint and consumer field: the value of each carried event and the
  value of each displayed countable union are compared only through
  $\RealUp$ endpoint rows, and the only downstream consumers are
  $\IntegralUp$, $\ProbSpaceUp$, and $\FunctionalAnalysisUp$ reading those
  displayed rows.
\end{enumerate}
The package contains no host set equality, no quotient event equality, no
analytic completion object, and no open-cover compactness row.
\end{definition}

\begin{theorem}[Measure root carrier-value event closure]
\label{thm:measure-root-unblock-carrier-event-value-closure}
The carrier and value face of
\autoref{def:measure-root-unblock-event-value-closure} is exactly the
carrier, classifier, event-row, stream-observation, and $\RealUp$ endpoint
face supplied by \autoref{def:measure-root-public-obligation-basis},
\autoref{thm:measure-root-real-stream-event-obligation}, and
\autoref{thm:measure-root-event-classifier-stability}.
\end{theorem}
\begin{proof}
Read the first and third fields of
\autoref{def:measure-root-unblock-event-value-closure}. They name a carried
$\mathsf{BHist}$ event row, the $\mathsf{MeasCode}$ carrier, the public
$\sim_M$ classifier, StreamName observation data, and $\RealUp$ endpoint
comparisons for the visible value rows.

Apply \autoref{def:measure-root-public-obligation-basis}. Its carrier,
classifier, stability, and ledger fields supply the same carried event rows,
endpoint witnesses, and public classifier transport, with every hidden
witness attached to a displayed row.

Apply \autoref{thm:measure-root-real-stream-event-obligation}. It reads the
same root row as a displayed event history, a StreamName observation name,
and the attached real endpoint comparison.

Finally apply \autoref{thm:measure-root-event-classifier-stability}. The
componentwise $\hsame$ event motion, StreamName observation motion, and
$\RealUp$ endpoint motion preserve the carried row and its visible ledger.
Thus the carrier-value face is precisely the displayed public face cited in
the statement.

None of the three cited surfaces introduces host set equality, quotient
event equality, analytic completion, or open-cover compactness. Therefore
the closure package exports no such row on its carrier-value face.
\end{proof}

\begin{theorem}[Measure root countable-additivity event-value closure]
\label{thm:measure-root-unblock-countable-additivity-event-value-closure}
The countable-additivity face of
\autoref{def:measure-root-unblock-event-value-closure} is exactly the
displayed $\mathsf{BHist}$ countable-family surface, StreamName observation
surface, disjoint-union readback, and $\RealUp$ endpoint-sum comparison
supplied by \autoref{thm:measure-root-countable-additivity-scope} and
\autoref{thm:measure-root-minimum-countable-additivity-obligation}.
\end{theorem}
\begin{proof}
Project the countable-family field of
\autoref{def:measure-root-unblock-event-value-closure}. It has only
StreamName-indexed carried event families, pairwise-disjoint witnesses,
countable-union readback, and the displayed $\RealUp$ countable-sum endpoint.

Apply \autoref{thm:measure-root-countable-additivity-scope}. The theorem
confines countable disjoint-additivity to displayed $\mathsf{BHist}$ event
families, StreamName observation names, real endpoint sums, and visible
ledger rows of the public basis.

Apply \autoref{thm:measure-root-minimum-countable-additivity-obligation}.
It supplies measurable-sequence coverage, disjoint-union readback, and
$\RealUp$ endpoint comparison through the same public ledger and
componentwise classifier motion.

The two source theorems account for every countable-family row named by the
closure package. They do not add an external sigma-algebra equality,
quotient event equality, analytic completion, or open-cover compactness
principle, so the additive face remains exactly the event-value surface
stated above.
\end{proof}

\begin{theorem}[Measure root analysis-consumer event-value closure]
\label{thm:measure-root-unblock-analysis-consumer-event-value-closure}
The downstream $\IntegralUp$, $\ProbSpaceUp$, and
$\FunctionalAnalysisUp$ consumers of
\autoref{def:measure-root-unblock-event-value-closure} consume only the
displayed event rows, $\RealUp$ endpoint rows, countable-sum rows,
finite-readback rows, normalization rows, monotone rows,
relative-difference rows, and visible ledger rows supplied by
\autoref{thm:measure-root-probspace-integral-consumer-surface} and
\autoref{thm:measure-root-minimum-downstream-consumer-obligation}.
\end{theorem}
\begin{proof}
Read the endpoint and consumer field of
\autoref{def:measure-root-unblock-event-value-closure}. It names exactly the
three outgoing consumers $\IntegralUp$, $\ProbSpaceUp$, and
$\FunctionalAnalysisUp$, and it restricts their input to displayed event
rows, endpoint rows, countable-family rows, and visible ledger rows.

Apply \autoref{thm:measure-root-probspace-integral-consumer-surface}. It
identifies the rows consumed by the three analysis chapters as displayed
event rows, endpoint rows, countable-sum rows, finite-readback rows,
normalization rows, monotone rows, and relative-difference rows carried by
the public root and real-stream fronts.

Apply \autoref{thm:measure-root-minimum-downstream-consumer-obligation}. It
restricts the same consumer triad to carrier/classifier rows,
countable-additivity rows, nonnegative endpoint rows, and ledger rows
displayed by the root minimum triad.

Intersecting these two readings leaves exactly the event-value rows listed
in the statement: no fourth analysis consumer, no host set equality, no
quotient event equality, no analytic completion, and no open-cover
compactness. Hence the downstream consumers can cite the closure package
without reconstructing any larger root interface.
\end{proof}
