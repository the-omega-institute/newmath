\begin{theorem}[$\ZNormalUp$ root total-host source coverage]
\label{thm:znormal-root-total-host-source-coverage}
For every accepted $\ZNormalUp$ packet
$$
Z=(T,F,V,K,R,H,C,P,N),
$$
the root total-host source is covered by the finite $\BHist$ display $T,F,V,K,R,H,C,P,N$. The source row $T$ supplies the typed endpoint, $F$ supplies bounded fuel replay, $V$ supplies the terminal normal-form row, $K$ and $R$ supply the finite sibling normal-word and continuation surfaces, $H$ supplies the displayed $\hsame$ transports, $C$ supplies the $\Cont$ consumer routes, $P$ supplies $\Pkg$ provenance, and $N$ supplies the local $\NameCert_{\ZNormalUp}$ row. No covered source requires host conversion, divergence detection, quotient normalization, unrestricted evaluation, substrate halting, or a global normalizer.
\end{theorem}
\leanchecked{BEDC.Derived.ZnormalUp.ZnormalPacket\_root\_total\_host\_source\_coverage}
\begin{proof}
Begin with \autoref{def:znormal-carrier} and \autoref{def:znormal-total-host-obligation-packet}. They name the accepted root source as the finite tuple $T,F,V,K,R,H,C,P,N$, so the source coverage question has no coordinate outside the displayed $\BHist$ packet.

Apply \autoref{thm:znormal-total-host-decision-surface}. It reads typed endpoint, bounded fuel, terminal normality, finite sibling evidence, continuation readback, transport, replay, provenance, and local naming from the same rows and refuses the omitted host-side objects.

Finally use \autoref{thm:znormal-root-obligation-downstream-handoff}. Downstream consumers can reach the local NameCert row only after the sibling typed endpoint, normal-word, carry, and continuation surfaces have entered the packet through $T,K,R$. Hence the root total-host source is fully covered by the finite display.
\end{proof}

\begin{theorem}[$\ZNormalUp$ root terminal-normality determinacy]
\label{thm:znormal-root-terminal-normality-determinacy}
Let $Z=(T,F,V,K,R,H,C,P,N)$ be an accepted $\ZNormalUp$ packet. A root consumer that asks for terminal normality is determined by the terminal normal-form row $V$ over the typed endpoint $T$, the bounded fuel replay row $F$, the continuation readback row $R$, and the structural rows $H,C,P,N$. The sibling row $K$ contributes only the finite Zeckendorf and carry-facing normal-word surface. The consumer cannot replace this determination by host conversion, divergence detection, quotient normalization, unrestricted evaluation, substrate halting, or a global normalizer.
\end{theorem}
\leanchecked{BEDC.Derived.ZnormalUp.ZnormalPacket\_root\_terminal\_normality\_determinacy}
\begin{proof}
Use \autoref{thm:znormal-terminal-normality-scope-package}. It confines every terminal-normality read to $T,F,V,R,H,C,P,N$, with $K$ available only as finite sibling evidence.

Next apply \autoref{thm:znormal-terminal-route-component-totality}. The public terminal route is exhausted by $T,F,V,K,R,H,C,P,N$, so there is no second route on which terminal normality could be chosen by a host evaluator or quotient normalizer.

The decision surface of \autoref{thm:znormal-total-host-decision-surface} then fixes the root consumer. Since $V$ is the only terminal normal-form row and the supporting rows are exactly $T,F,R,H,C,P,N$, terminal normality is determined inside the finite packet.
\end{proof}

\begin{theorem}[$\ZNormalUp$ root fuel-replay exhaustion]
\label{thm:znormal-root-fuel-replay-exhaustion}
For every accepted $\ZNormalUp$ packet $Z=(T,F,V,K,R,H,C,P,N)$, bounded fuel replay at the root is exhausted by $F$ together with the displayed $\Cont$ row $C$ and the transport, provenance, and naming rows $H,P,N$. The replay is used only to expose the terminal normal-form row $V$ over $T$ and the continuation row $R$, while $K$ remains finite sibling normal-word evidence. It does not export a divergence oracle, an unrestricted evaluator, substrate halting, host conversion, quotient normalization, or a global normalizer.
\end{theorem}
\leanchecked{BEDC.Derived.ZnormalUp.ZnormalPacket\_root\_fuel\_replay\_exhaustion}
\begin{proof}
Start with \autoref{thm:znormal-terminal-fuel-row-exhaustion}. It places every terminal-normality read behind the finite fuel row $F$ and the route $T,F,V,R,H,C,P,N$.

Now apply \autoref{thm:znormal-terminal-host-obligation-totality}. The total-host obligation packet presents replay, transport, provenance, and local naming as rows of the same finite packet, with the excluded host-side objects absent from the surface.

Finally use \autoref{thm:znormal-tastegate-field-readback-lock}. The field-readback lock interprets \texttt{FieldFaithful ZNormalUp} and \texttt{Nontrivial ZNormalUp} on the displayed packet, not on an evaluator. Thus bounded fuel replay is exhausted by $F,C,H,P,N$ as support for $V$ and $R$.
\end{proof}

\begin{theorem}[$\ZNormalUp$ root refusal ledger separation]
\label{thm:znormal-root-refusal-ledger-separation}
The root refusal ledger of an accepted $\ZNormalUp$ packet separates the admitted finite rows
$$
T,F,V,K,R,H,C,P,N
$$
from host conversion, divergence detection, quotient normalization, unrestricted evaluation, substrate halting, and global normalization. The admitted side contains terminal normality, bounded fuel replay, displayed $\hsame$ transport, $\Cont$ consumer routing, $\Pkg$ provenance, and local $\NameCert_{\ZNormalUp}$ naming; the refused side contains exactly the omitted host-facing objects and cannot be transported back into the packet.
\end{theorem}
\begin{proof}
Read the finite side through \autoref{thm:znormal-root-total-host-source-coverage}. The admitted ledger entries are precisely the rows $T,F,V,K,R,H,C,P,N$ and their carrier, terminal, replay, transport, provenance, and naming uses.

Read the refused side through \autoref{thm:znormal-terminal-normality-refusal-stability}. A request for divergence, host conversion, or global normalization remains refused under componentwise $\hsame$ transport, so transport cannot convert the refusal into a packet row.

Finally apply \autoref{thm:znormal-total-host-readback-separation}. Quotient computation, unrestricted evaluation, substrate halting, and the other host-side objects have no ledger coordinate. The ledger therefore separates admitted root evidence from refused host-facing objects.
\end{proof}

\begin{theorem}[$\ZNormalUp$ root NameCert unblock package]
\label{thm:znormal-root-namecert-unblock-package}
For every accepted $\ZNormalUp$ packet $Z=(T,F,V,K,R,H,C,P,N)$, the local $\NameCert_{\ZNormalUp}$ root-unblock package is the finite obligation packet consisting of source coverage, terminal-normality determinacy, fuel-replay exhaustion, and refusal-ledger separation. It binds the terminal normal-form row $V$, bounded fuel replay row $F$, displayed $\hsame$ transports $H$, $\Cont$ routes $C$, $\Pkg$ provenance $P$, and local naming row $N$ over the same finite $\BHist$ source $T,F,V,K,R,H,C,P,N$, and it exports no host conversion or divergence oracle.
\end{theorem}
\begin{proof}
First assemble the admitted source rows by \autoref{thm:znormal-root-total-host-source-coverage}. This gives the finite $\BHist$ packet on which the local NameCert row can operate.

Next combine \autoref{thm:znormal-root-terminal-normality-determinacy} with \autoref{thm:znormal-root-fuel-replay-exhaustion}. Terminal normality is determined by $V$ over $T,F,R,H,C,P,N$, and bounded fuel replay is exhausted by $F$ together with the displayed replay and structural rows.

Finally apply \autoref{thm:znormal-root-refusal-ledger-separation} and \autoref{thm:znormal-root-unblock-namecert-exhaustion}. The refusal ledger keeps host conversion, divergence detection, quotient normalization, unrestricted evaluation, substrate halting, and global normalization outside the packet, while root-unblock NameCert exhaustion leaves no remaining local clause beyond $T,F,V,K,R,H,C,P,N$. The four components therefore form the root NameCert unblock package.
\end{proof}
