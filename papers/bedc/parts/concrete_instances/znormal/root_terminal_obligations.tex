\begin{theorem}[$\ZNormalUp$ root terminal carrier admission]
\label{thm:znormal-root-terminal-carrier-admission}
Every accepted $\ZNormalUp$ packet admitted at the root terminal surface is a finite carrier
$$
Z=(T,F,V,K,R,H,C,P,N)
$$
whose rows expose typed endpoint evidence, finite fuel, terminal-normality, finite sibling normal-word evidence, terminal continuation readback, componentwise transport, displayed replay, provenance, and local naming. The admission route is the carrier and $\NameCert_{\ZNormalUp}$ route of \autoref{def:znormal-carrier} and \autoref{thm:znormal-namecert-obligations}; it introduces no host conversion row, quotient normalization row, unrestricted evaluator, divergence detector, or global normalizer.
\end{theorem}
\begin{proof}
Project the carrier using \autoref{def:znormal-carrier}. Its displayed rows are exactly $T,F,V,K,R,H,C,P,N$, so root admission has no extra coordinate from which a host conversion or quotient normalizer could be read.

Apply \autoref{thm:znormal-namecert-obligations}. The local certificate binds typed endpoint readback, finite fuel, terminal normality, sibling normal-word evidence, continuation readback, transport, replay, provenance, and naming to the same finite packet.

Finally use \autoref{thm:znormal-root-terminal-nonescape-interface}. The terminal root interface keeps every downstream terminal read inside the displayed rows and refuses host conversion, quotient computation, unrestricted evaluation, divergence detection, and global normalization.
\end{proof}

\begin{theorem}[$\ZNormalUp$ root normal-word route exposure]
\label{thm:znormal-root-normal-word-route-exposure}
For every accepted $\ZNormalUp$ packet
$$
Z=(T,F,V,K,R,H,C,P,N),
$$
the finite normal-word row $K$ is exposed at the root only through the sibling normal-word and continuation routes already certified by the packet. A consumer that reads $K$ must pass through the terminal rows $T,F,V$, the continuation read $R$, and the transport, replay, provenance, and naming rows $H,C,P,N$; it receives no quotient normalization, host conversion, unrestricted evaluation, divergence detector, or global normalizer.
\end{theorem}
\begin{proof}
Start with \autoref{thm:znormal-root-normal-word-route-totality}. It places the normal-word row on the displayed root route and treats $K$ as finite sibling evidence rather than as an evaluator.

Next apply \autoref{thm:znormal-continuation-normal-word-pullback}. The normal-word row and continuation read are pulled back to the same transported finite packet through $H,C,P,N$, so $K$ cannot be detached from $T,F,V,R$.

Use \autoref{thm:znormal-root-terminal-nonescape-interface} to close the escape boundary. Since all visible reads remain inside $T,F,V,K,R,H,C,P,N$, the root route exposes no host conversion, quotient normalization, unrestricted evaluation, divergence detection, or global normalization.
\end{proof}
