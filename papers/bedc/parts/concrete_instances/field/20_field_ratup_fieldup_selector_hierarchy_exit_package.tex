\begin{theorem}[RatUp FieldUp algebra hierarchy selector-chain exit]
\label{thm:ratup-fieldup-algebra-hierarchy-selector-chain-exit}
For $\mathsf{RatUpFieldSource}_F$, the concrete denominator tuple
$(C_D^u,R_D^u,\mu_D^u,e_D^u,N_D^u)$, histories
$h,k,\ell:\mathsf{BHist}$ with $C_D^u(h)$, $C_D^u(k)$, and $C_D^u(\ell)$,
selectors $s,t,u,v,w,z:\mathsf{BMark}$, and contexts
$L,R,r:\mathsf{BHist}$, if $\mathsf{ThreeSupp}_D^u(h,k,\ell)$ holds, then one
concrete $\mathsf{BHist}$ package contains all of the following rows:
\begin{enumerate}
  \item every row of
  \autoref{thm:ratup-fieldup-algebra-hierarchy-exit-witness};
  \item every shared certificate and operation-table row of
  \autoref{thm:ratup-fieldup-assembled-certificate-operation-table-coverage};
  \item every selector-chain final-boundary row of
  \autoref{thm:ratup-fieldup-selector-chain-final-boundary} for the selector
  pairs $(s,t)$, $(u,v)$, and $(w,z)$.
\end{enumerate}
Equivalently, the concrete source tuple, the concrete denominator tuple, and
the three certified selector pairs determine a single
$\FieldUp(\RatUp)$ exit package for the Phase Algebra Hierarchy interface.
\end{theorem}
\begin{proof}
First instantiate
\autoref{thm:ratup-fieldup-algebra-hierarchy-exit-witness} at the source tuple
$\mathsf{RatUpFieldSource}_F$. This supplies the algebra hierarchy exit rows
over the common carrier $\mathsf{BHist}$, the RatUp history classifier, and the
concrete denominator endpoints.

Next instantiate
\autoref{thm:ratup-fieldup-assembled-certificate-operation-table-coverage} at
the same source tuple, denominator tuple, and concrete operation table of
Definition~\ref{def:ratup-fieldup-concrete-operation-table}. This supplies the
shared certificate rows and the operation-table rows carried by the same
$\mathsf{BHist}$ package.

Finally instantiate \autoref{thm:ratup-fieldup-selector-chain-final-boundary}
with the histories, carrier witnesses, support witness, selectors
$s,t,u,v,w,z$, and contexts $L,R,r$. This contributes the three final-boundary
packages for $(s,t)$, $(u,v)$, and $(w,z)$ together with the chained
componentwise $R_D^u$ and $\mathsf{RatHistoryClassifier}$ rows.

The three cited packages name the same source
$\mathsf{RatUpFieldSource}_F$, the same carrier $\mathsf{BHist}$, the same
RatUp classifier, and the same concrete denominator tuple. Conjunction
introduction repacks the common carrier, classifier, denominator, certificate,
operation-table, and selector-chain boundary rows as one concrete
$\FieldUp(\RatUp)$ algebra hierarchy exit package.
\end{proof}
