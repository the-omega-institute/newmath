\begin{theorem}[RatUp FieldUp selector-complete strict exit package]
\label{thm:ratup-fieldup-selector-complete-strict-exit-package}
For the concrete denominator tuple $(C_D^u,R_D^u,\mu_D^u,e_D^u,N_D^u)$ and
histories $h,k,\ell:\mathsf{BHist}$ with $C_D^u(h)$, $C_D^u(k)$, and
$C_D^u(\ell)$, let $E_s(h,k,\ell)$ be the endpoint selector of
\autoref{def:ratup-fieldup-bracketing-endpoint-selector}. If
$\mathsf{ThreeSupp}_D^u(h,k,\ell)$ holds, then for every selector pair
$s,t:\mathsf{BMark}$ and every context pair $L,R:\mathsf{BHist}$ the
selector-complete denominator instance of
\autoref{def:ratup-fieldup-selector-complete-denominator-instance} supplies the
full strict concrete exit row
$$
C_D^u(E_s(h,k,\ell))
\land
C_D^u(E_t(h,k,\ell))
\land
R_D^u(E_s(h,k,\ell),E_t(h,k,\ell))
\land
\left(
\mathsf{fieldSingletonEmptyClassifier}
(\mathsf{append}(L,E_s(h,k,\ell)),\mathsf{append}(R,E_t(h,k,\ell)))\to\bot
\right)
\land
\mathsf{fieldSingletonEmptyNonZero}(E_s(h,k,\ell))
\land
\mathsf{fieldSingletonEmptyNonZero}(E_t(h,k,\ell)).
$$
\end{theorem}
\begin{proof}
Apply \autoref{thm:ratup-fieldup-selector-complete-exit-package} to the same
concrete denominator tuple, carrier hypotheses, support witness, selectors
$s,t:\mathsf{BMark}$, and contexts $L,R:\mathsf{BHist}$. It supplies the two
selected endpoint carrier rows, the denominator classifier row, and the
contextual singleton-classifier noncollapse row.

Apply
\autoref{thm:ratup-fieldup-selector-complete-denominator-strict-support-package}
to the same tuple, carrier hypotheses, support witness, and selector pair.
It supplies the same denominator classifier row together with the two strict
endpoint witnesses
$\mathsf{fieldSingletonEmptyNonZero}(E_s(h,k,\ell))$ and
$\mathsf{fieldSingletonEmptyNonZero}(E_t(h,k,\ell))$.

Use the classifier row only once, taking it from either package. Combining it
with the two carrier rows, the contextual noncollapse row, and the strict
endpoint witnesses gives exactly the displayed strict concrete exit row
consumed in the denominator coordinate of $\mathsf{RatUpFieldSource}_F$.
\end{proof}
