\begin{theorem}[RatUp FieldUp nondegenerate semantic exit source]
\label{thm:ratup-fieldup-nondegenerate-semantic-exit}
Let $d_1:=\Eone(\emp)$. For the concrete source tuple
$\mathsf{RatUpFieldSource}_F$ of
\autoref{def:ratup-fieldup-concrete-source-tuple}, the same $\mathsf{BHist}$
data simultaneously give the following four pieces of the
$\FieldUp(\mathsf{RatUp})$ exit source:
\begin{enumerate}
  \item the assembled semantic naming certificate rows for
  $\FieldUp(\mathsf{RatUp})$;
  \item source anchoring of all certificate rows at
  $\mathsf{RatHistoryCarrier}$ and $\mathsf{RatHistoryClassifier}$;
  \item the assembled support-neutrality row for the denominator contextual
  action $\Gamma$;
  \item the endpoint separation
  $$
  \mathsf{RatHistoryCarrier}(d_1)
  $$
  together with
  $$
  \mathsf{fieldSingletonEmptyCarrier}(d_1)\to\bot.
  $$
\end{enumerate}
Consequently the semantic exit source for $\FieldUp(\mathsf{RatUp})$ over
$\mathsf{RatUpFieldSource}_F$ is not collapsible to the singleton empty-history
$\FieldUp$ source.
\end{theorem}
\begin{proof}
The semantic certificate rows are those assembled in
\autoref{thm:ratup-fieldup-semantic-certificate-assembly}. They include the
carrier, source, classifier, stability, ledger, exactness, and affine rows of
the concrete $\FieldUp(\mathsf{RatUp})$ certificate.

The common source location is fixed by
\autoref{thm:ratup-fieldup-certificate-rows-source-anchored}. It identifies the
field carrier as $\mathsf{RatHistoryCarrier}$, the field classifier as
$\mathsf{RatHistoryClassifier}$, and the denominator, contextual-action,
context-pair, and affine rows as entries of the same
$\mathsf{RatUpFieldSource}_F$ tuple.

The denominator contextual action contributes the support-neutrality component
through \autoref{thm:ratup-fieldup-assembled-support-neutrality-certificate}.
Thus the assembled certificate contains the carried nested action, the
collected-action classifier comparison, strict-support neutrality, and the
second-endpoint classifier exactness row over the denominator tuple.

Finally,
\autoref{thm:ratup-fieldup-source-non-singleton-witness} supplies the displayed
endpoint $d_1=\Eone(\emp)$, its
$\mathsf{RatHistoryCarrier}(d_1)$ witness, and the rejection map
$\mathsf{fieldSingletonEmptyCarrier}(d_1)\to\bot$. Any collapse to the singleton
empty-history $\FieldUp$ source would have to send this carried endpoint to the
singleton carrier, contradicting that rejection map.
\end{proof}
