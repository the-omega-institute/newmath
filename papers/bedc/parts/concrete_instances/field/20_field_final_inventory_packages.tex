\begin{theorem}[RatUp FieldUp final row inventory semantic certificate]
\label{thm:ratup-fieldup-final-row-inventory-semantic-certificate}
For $\mathsf{RatUpFieldSource}_F$, let $\mathcal E_F$ be the concrete exit
object of Definition~\ref{def:ratup-fieldup-concrete-exit-object}. For the
concrete denominator tuple $(C_D^u,R_D^u,\mu_D^u,e_D^u,N_D^u)$, histories
$h,k,\ell:\mathsf{BHist}$ with carrier witnesses $C_D^u(h)$, $C_D^u(k)$, and
$C_D^u(\ell)$, selectors $s,t,u,v,w,z:\mathsf{BMark}$, contexts
$L,R,r:\mathsf{BHist}$, and a support witness
$\mathsf{ThreeSupp}_D^u(h,k,\ell)$, the semantic certificate boundary for
$\mathsf{RatUpFieldSource}_F$ contains, over the same concrete exit object, all
of the following rows:
\begin{enumerate}
  \item the paper certificate assembly of
  \autoref{thm:ratup-fieldup-semantic-namecert};
  \item the concrete schema row of
  \autoref{thm:ratup-fieldup-concrete-schema-instance};
  \item the proof-target exit package of
  \autoref{thm:ratup-fieldup-proof-target-exit-package};
  \item every row of
  \autoref{thm:ratup-fieldup-final-row-inventory-coverage}.
\end{enumerate}
Consequently the selector inverse-domain coverage, selector row transport,
complete obstruction, and selector-chain boundary rows are all anchored over
$\mathsf{RatHistoryCarrier}$, $\mathsf{RatHistoryClassifier}$,
$\mathsf{RatDenomUnitCarrier}$, $\mathsf{RatDenomUnitClassifier}$,
$\mathsf{append}$, $\emp$, $\Gamma$, and $\mathsf{AffAut}_F$ inside the
concrete $\FieldUp(\RatUp)$ semantic certificate.
\end{theorem}
\begin{proof}
Start with \autoref{thm:ratup-fieldup-semantic-namecert}. It fixes the
paper-side semantic boundary for $\mathsf{RatUpFieldSource}_F$ over
$\mathsf{RatHistoryCarrier}$, $\mathsf{RatHistoryClassifier}$, the displayed
denominator tuple, the contextual action $\Gamma$, and the affine source
$\mathsf{AffAut}_F$.

Next apply \autoref{thm:ratup-fieldup-concrete-schema-instance}. The schema
row identifies the same concrete source tuple and the same concrete exit object
$\mathcal E_F$ with the $\mathsf{BHist}$ rows used by the certificate boundary.

Then apply \autoref{thm:ratup-fieldup-proof-target-exit-package}. This places
the concrete schema row inside the proof-target exit package for
$\FieldUp(\RatUp)$, still over the same carrier, classifier, denominator
tuple, action source, and affine source.

Finally apply \autoref{thm:ratup-fieldup-final-row-inventory-coverage} to the
displayed histories, carrier witnesses, support witness, selectors, and
contexts. It supplies the selector inverse-domain, selector row transport,
complete obstruction, and selector-chain boundary rows for $\mathcal E_F$.

Conjunction introduction packages the semantic assembly, schema row,
proof-target exit package, and final row inventory as one consumed certificate
boundary for $\mathsf{RatUpFieldSource}_F$.
\end{proof}
