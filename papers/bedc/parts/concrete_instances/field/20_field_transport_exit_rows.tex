\begin{theorem}[RatUp FieldUp transported exit row contextual separation package]
\label{thm:ratup-fieldup-transported-exit-row-contextual-separation-package}
For the concrete source tuple $\mathsf{RatUpFieldSource}_F$, the concrete
denominator tuple $(C_D^u,R_D^u,\mu_D^u,e_D^u,N_D^u)$, and the contextual
action $\Gamma$ of
\autoref{def:ratup-fieldup-denominator-contextual-action-instance}, let
$x:=\Gamma_{p',q',s',t'}(\Gamma_{p,q,s,t}(h))$. Under the hypotheses of
\autoref{thm:ratup-fieldup-transported-support-neutrality-row}, suppose also
that $N_D^u(h)$ holds. Then the concrete $\FieldUp(\mathsf{RatUp})$ exit row
contains a single contextual separation package with:
\begin{enumerate}
  \item the exit-row closure recorded by
  \autoref{thm:ratup-fieldup-transported-exit-row-nondegenerate-closure};
  \item the direct endpoint separation $x\hsame\emp\to\bot$;
  \item for every $r:\mathsf{BHist}$, both contextual rejection maps
  $\mathsf{fieldSingletonEmptyCarrier}(\mathsf{append}(r,x))\to\bot$ and
  $\mathsf{fieldSingletonEmptyCarrier}(\mathsf{append}(x,r))\to\bot$.
\end{enumerate}
\end{theorem}
\begin{proof}
First apply
\autoref{thm:ratup-fieldup-transported-exit-row-nondegenerate-closure} with the
same source tuple, denominator tuple, contextual action, endpoint, and
strict-support hypothesis. This supplies the concrete exit-row closure consumed
by the $\FieldUp(\mathsf{RatUp})$ certificate boundary.

Next apply
\autoref{thm:ratup-fieldup-transported-strict-support-separation-package} at the
same endpoint. Its final clause is the direct endpoint separation
$x\hsame\emp\to\bot$.

Finally fix an arbitrary context history $r:\mathsf{BHist}$ and apply
\autoref{thm:ratup-fieldup-transported-strict-support-contextual-singleton-exclusion}.
The two clauses obtained there are exactly the rejection maps for
$\mathsf{append}(r,x)$ and $\mathsf{append}(x,r)$. Collecting these clauses gives
the displayed contextual separation package.
\end{proof}
