\begin{theorem}[RatUp FieldUp two-step algebra hierarchy exit certificate]
\label{thm:ratup-fieldup-two-step-algebra-hierarchy-exit-certificate}
For $\mathsf{RatUpFieldSource}_F$, the concrete exit object
$\mathsf{RatUpFieldExitObject}_F(\mathcal E_F)$, the concrete denominator tuple
$(C_D^u,R_D^u,\mu_D^u,e_D^u,N_D^u)$, histories
$h,k,\ell:\mathsf{BHist}$ with $C_D^u(h)$, $C_D^u(k)$, and $C_D^u(\ell)$,
selectors $s,t,u,v,w,z:\mathsf{BMark}$, contexts $L,R,r:\mathsf{BHist}$, and
the endpoint selector $E_q(h,k,\ell)$ of
Definition~\ref{def:ratup-fieldup-bracketing-endpoint-selector}, if
$\mathsf{ThreeSupp}_D^u(h,k,\ell)$ holds, then the one concrete exit object
contains every algebra-hierarchy exit row, every operation-table coverage row,
both adjacent complete selector-obstruction certificates for $(s,t,u,v)$ and
$(u,v,w,z)$, and the selector-chain final boundary rows, including the chained
$R_D^u$ rows and the chained $\mathsf{RatHistoryClassifier}$ endpoint rows.
\end{theorem}
\begin{proof}
First instantiate
\autoref{thm:ratup-fieldup-algebra-hierarchy-chain-stable-exit} with the
displayed source, denominator tuple, histories, selectors $s,t,u,v,w,z$,
contexts $L,R,r$, endpoint selector, and
$\mathsf{ThreeSupp}_D^u(h,k,\ell)$. This supplies the hierarchy rows, the
operation-table coverage rows, the first selector-obstruction certificate, and
the selector-chain boundary rows.

Next instantiate
\autoref{thm:ratup-fieldup-concrete-two-step-obstruction-chain-package} with
the same data. This supplies both adjacent complete selector-obstruction
certificates and the same selector-chain final boundary rows, including the
chained denominator and Rat-history endpoint rows.

Finally cite \autoref{thm:ratup-fieldup-concrete-exit-object-complete} to keep
the hierarchy package and the two-step obstruction package inside the same
concrete $\mathsf{RatUpFieldExitObject}_F(\mathcal E_F)$. Conjunction
introduction repacks those row families as one Phase Algebra Hierarchy
$\FieldUp(\RatUp)$ exit certificate.
\end{proof}
