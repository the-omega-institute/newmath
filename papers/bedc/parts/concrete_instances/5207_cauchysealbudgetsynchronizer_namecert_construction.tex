\begin{theorem}[Cauchy seal-budget synchronizer NameCert obligations]

\end{theorem}
\begin{proof}
Carrier admission is the displayed tuple $Z=(E,S,B,T,D,L,H,C,P,N)$. The request, seal, budget, tail schedule, selector, compatibility, transport, route, provenance, and naming rows are all explicit coordinates.

Selector, budget, and shared-tail exposure are projections from the same tuple. The selector row $D$ names the diagonal tail, the budget row $B$ supplies either the Cauchy diagonal budget or the observation-budget route, and $T$ supplies the shared tail threshold consumed by the same request.

The seal and compatibility clauses are also finite projections. The row $S$ is not read as an isolated seal; it is tied by $L$ to the selected diagonal row, budget row, and shared threshold. Componentwise transports in $H$ and continuation routes in $C$ can replay these displayed rows, but they cannot introduce a second selector, budget, threshold, or seal.

\begin{theorem}[Cauchy seal-budget synchronizer non-escape]
\label{thm:cauchy-seal-budget-synchronizer-nonescape}
Every accepted $\CauchySealBudgetSynchronizerUp$ packet read by a $\RealUp$ or $\CauchyLimitSealUp$ consumer factors through both the displayed seal row $S$ and a displayed budget row $B$ that is supplied by $\CauchyDiagonalBudgetUp$ or $\RealObservationBudgetUp$. If the read uses the tail schedule, it must also pass through the displayed selector $D$, shared threshold $T$, and compatibility row $L$. No accepted read can bypass both the budget row and the selector-threshold compatibility row while still exporting the seal.
\end{theorem}
\begin{proof}
Start from the carrier inventory. The only seal-facing coordinate is $S$, and the only budget-facing coordinate is $B$.

Apply the NameCert obligation theorem. It requires every admitted consumer route to expose $D$, $B$, $T$, $L$, and $S$ through the displayed transport and continuation rows before the local name can be consumed.

Thus a seal read that omits the budget row has no route to $B$, and a seal read that omits both the selector row and the shared-threshold compatibility row has no route to $D,T,L$. Since the carrier contains no quotient stream, host real equality, selected external limit, countable-choice row, or ambient completeness theorem, no omitted analytic object can be repacked as a second seal route.
\end{proof}

\end{proof}
