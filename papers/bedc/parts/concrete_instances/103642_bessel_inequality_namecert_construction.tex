\label{thm:bessel-inequality-namecert-obligations}

\item carrier exposure for the Hilbert, inner-product, norm, vector-space, orthogonal-family, coefficient-ledger, partial-square-sum, Hilbert-bound handoff, transport, replay, provenance, and local naming rows;
\item dependency coherence, requiring the $\HilbertUp$, $\InnerProductUp$, $\NormUp$, and $\VecSpaceUp$ rows to read the same carried vector endpoints and the same classifier route;
