Case-analyze the tagged classifier witness. In the first direction, the
absent-absent branch stores an endpoint witness from $k$ to $\emp$; composing it
with the displayed witness from $k$ to $\Eone(a)$ gives empty-versus-one
no-confusion. The present-present branch stores an endpoint witness from $h$ to
some $\Eone(b)$; composing it with the displayed witness from $h$ to $\emp$ again
gives empty-versus-one no-confusion. The second direction is symmetric: the
absent-absent branch contradicts the displayed present presentation of $h$, and
the present-present branch contradicts the displayed absent presentation of $k$.
\end{proof}
\leanchecked{BEDC.Derived.OptionUp.TaggedOptionHistoryClassifier\_carrier\_mixed\_tag\_separation}

\begin{theorem}[Tagged option history classifier visible-branch inversion]
\label{thm:tagged-option-history-classifier-visible-branch-inversion}
This theorem fits the per-object template as follows: the traditional target is
tagged option branch inversion; the BEDC source is $\mathsf{BHist}$ with
$\hsame$; the carrier is $\mathsf{TaggedOptionHistoryCarrier}$; the classifier is
$\mathsf{TaggedOptionHistoryClassifier}$; the stability obligation is branch
determinism under visible endpoint presentations; the ledger is the visible
absent/present tag plus payload; the core theorem is this inversion; exactness
is no mixed branch and payload exposure; the standard bridge is
$\mathsf{none}$/$\mathsf{some}$; and the status is $\mathsf{ProofTarget}$.
For every BEDC history source carrier $S:\mathsf{BHist}\to\mathsf{Prop}$,
source classifier $\sim_S$, and histories $h,k$, the classifier determines the
visible branch presented by the left endpoint:
$$
\mathsf{TaggedOptionHistoryClassifier}_{S,\sim_S}(h,k)\to
\hsame(h,\emp)\to
\hsame(k,\emp)
$$
and, for every payload history $a$,
$$
\mathsf{TaggedOptionHistoryClassifier}_{S,\sim_S}(h,k)\to
S(a)\to
\hsame(h,\Eone(a))\to
\exists a',b:\mathsf{BHist}.\;
S(a')\land S(b)\land \hsame(a,a')\land a'\sim_S b\land
\hsame(k,\Eone(b)).
$$
\end{theorem}
\leanchecked{BEDC.Derived.OptionUp.TaggedOptionHistoryClassifier\_visible\_branch\_inversion}
\begin{proof}
For the absent clause, case-analyze the tagged classifier witness. The
absent-absent branch already stores $\hsame(k,\emp)$. The present-present branch
stores $\hsame(h,\Eone(a))$ for some source payload; composing it with the
displayed witness $\hsame(h,\emp)$ gives empty-versus-one no-confusion.

For the present clause, case-analyze the classifier witness again. The
absent-absent branch stores $\hsame(h,\emp)$ and contradicts the displayed
present endpoint $\hsame(h,\Eone(a))$. In the present-present branch, obtain
payloads $a'$ and $b$ with $S(a')$, $S(b)$, $a'\sim_S b$,
$\hsame(h,\Eone(a'))$, and $\hsame(k,\Eone(b))$. Compose the two presentations
of $h$ and invert the one-constructor equality to get $\hsame(a,a')$; the
remaining fields are exactly the displayed witnesses.
\end{proof}

\begin{theorem}[Tagged option classifier branch occupancy equivalence]
\label{thm:tagged-option-history-classifier-branch-occupancy-equivalence}
This theorem fits the per-object template with traditional target tagged optional
history branch occupancy, BEDC source $\mathsf{BHist}$ with $\hsame$, carrier
$\mathsf{TaggedOptionHistoryCarrier}$, classifier
$\mathsf{TaggedOptionHistoryClassifier}$, stability obligation branch preservation,

constructor no-confusion, standard bridge $\mathsf{none}$/$\mathsf{some}$, and

classifier $\sim_S$, histories $h,k$, and witness
$\mathsf{TaggedOptionHistoryClassifier}_{S,\sim_S}(h,k)$, the visible branch

$$
(\hsame(h,\emp) \Longleftrightarrow \hsame(k,\emp))
\land
((\exists a.\; S(a)\land\hsame(h,\Eone(a))) \Longleftrightarrow
(\exists b.\; S(b)\land\hsame(k,\Eone(b)))).
$$
\end{theorem}
\begin{proof}
Case-analyze the tagged classifier witness. The absent-absent branch gives both
empty endpoint witnesses and rules out either present endpoint by
empty-versus-one no-confusion. The present-present branch projects the two source
payload witnesses and their visible endpoint witnesses, and rules out either
empty endpoint by the same no-confusion argument.
\end{proof}
