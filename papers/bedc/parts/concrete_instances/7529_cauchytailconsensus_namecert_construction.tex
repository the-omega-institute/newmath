\item $L$ is the $\CauchyLimitSealUp$ handoff row exposed only after $S$ and $A$ agree on the route;
\item $W$ records the $\StreamNameUp$ finite-window rows used by both $S$ and $A$;
\item $R$ records the $\RegSeqRatUp$ readbacks consumed by the agreement and limit-seal rows;

\label{thm:cauchy-tail-consensus-namecert-obligations}

\begin{proof}

\begin{theorem}[Cauchy tail consensus shared route determinacy]
\label{thm:cauchy-tail-consensus-shared-route-determinacy}
Every accepted $\CauchyTailConsensusUp$ packet exposes a $\RealUp$ seal only when the shared-tail row $S$ and the tail-agreement row $A$ use the same finite threshold, window, and readback route through $W,R,H,C,P,N$. A packet whose $S$ and $A$ rows disagree on that route has no accepted Real seal row $E$.
\end{theorem}
\begin{proof}
Start with the NameCert obligations. The $\CauchyLimitSealUp$ handoff $L$ and terminal seal $E$ occur after both $S$ and $A$ have consumed the displayed $W,R$ route.

Apply the sibling boundaries. The shared-tail modulus chapter supplies one shared finite threshold, and the Real tail-agreement seal chapter accepts the paired agreement only through its displayed window and readback rows.

If those routes disagree, there is no common continuation from $A$ to $L$ in the carrier. Thus the terminal seal row cannot be exposed; if they agree, the displayed $H,C,P,N$ rows replay the single accepted consensus route.
\end{proof}

\begin{theorem}[Cauchy tail consensus Real-completion handoff]
\label{thm:cauchy-tail-consensus-real-completion-handoff}
Every Real-completion consumer of an accepted $\CauchyTailConsensusUp$ packet reads only the finite route
$$
S,A,W,R,L,E,H,C,P,N.
$$
The route first consumes the shared-tail modulus, then the tail-agreement seal, then the Cauchy limit seal, and finally the Real seal; it exports no quotient stream, selected host limit, independent threshold, host real equality, or ambient completeness principle.
\end{theorem}
\begin{proof}
Use shared route determinacy. It makes $W,R$ the common route for the shared-tail row and the tail-agreement row before any seal is visible.

Next read the Cauchy-limit handoff. The row $L$ is reached only after the consensus route has passed through both $S$ and $A$.

Finally project the terminal seal row $E$ and the structural rows $H,C,P,N$. They transport, replay, package, and name the same finite route, so the consumer has no coordinate from which to import any excluded completion resource.
\end{proof}
