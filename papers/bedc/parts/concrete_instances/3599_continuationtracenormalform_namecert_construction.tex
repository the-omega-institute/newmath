\end{theorem}

\autoref{thm:continuation-trace-normal-form-normal-boundary-cut}. It states
that the normal-boundary consumer cut is exactly the displayed route through
$S,T,R,Q,V,H,C,P,N$.

Use root readback order exhaustion to keep the boundary on the root order
from source through trace, terminal, terminal-read, and typed-boundary rows.
Then use root terminal readback totality,
\autoref{thm:continuation-trace-normal-form-root-terminal-readback-totality},
to expose the terminal row before any consumer can detach it.

Finally apply root replay prefix totality,
\autoref{thm:continuation-trace-normal-form-root-replay-prefix-totality}. It
blocks off-packet replay at the prefix stage, while the normal-boundary cut
already blocks host reduction and hidden terminal data. Therefore the root
normal-form boundary is exhausted by the displayed packet.
\end{proof}
