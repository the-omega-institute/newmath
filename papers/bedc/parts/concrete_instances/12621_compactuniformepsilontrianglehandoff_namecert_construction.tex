\begin{theorem}[Compact-uniform epsilon-triangle handoff estimate]
\label{thm:compact-uniform-epsilon-triangle-handoff-estimate}
For every accepted $\CompactUniformEpsilonTriangleHandoffUp$ packet
$$
E=(X,Y,c,x,x',\varepsilon,\mu,\rho,d_x,d_{x'},d_y,d_{y'},H,C,P,N),

$$
d_X(x',c) \le d_X(x',x)+d_X(x,c)<\rho.
$$
The continuous-map pointwise-modulus row at $c$ then applies to both $x$ and $x'$, giving $d_y<\varepsilon/3$ and $d_{y'}<\varepsilon/3$. The target metric triangle row combines $d_y$ and $d_{y'}$ into the displayed $\varepsilon$-bound exported to $\UniformModulusUp$.
\end{theorem}
\begin{proof}
Start with the compact center row. The carrier preserves $c$ together with the coverage row $d_x$, so the input point $x$ is within the displayed half-radius around $c$.

Use the source triangle row in $X$. Since the uniform precision row $\mu$ bounds the distance between $x$ and $x'$ and is below the selected half-radius ledger, the source triangle gives the displayed bound on $d_X(x',c)$ and keeps $x'$ inside the same local radius $\rho$.

Now apply the pointwise-modulus rows at $c$. The radius bound $\rho$ is subordinate to the local modulus, so the same continuous-map packet supplies the target estimates $d_y<\varepsilon/3$ and $d_{y'}<\varepsilon/3$.

Finally read the target triangle certificate in $Y$. It joins the two target estimates through the same center image and exports the resulting $\varepsilon$-bound along the continuation route to $\UniformModulusUp$.
\end{proof}
