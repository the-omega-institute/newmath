\begin{theorem}[GaloisGroupUp identity closure]
\label{thm:galoisgroup-identity-closure}
For an automorphism-action packet $(E,G,b,a,m,i,c,\rho,\lambda)$ of
\autoref{def:galoisgroup-bhist-automorphism-action-packet}, the identity
automorphism row inherited from the $\GroupUp$ unit surface fixes every
displayed $\GaloisExtUp$ endpoint and is accepted by the same
automorphism-action packet. Its fixed-base witnesses are the $\hsame$ rows
obtained by reading the unit row through the Galois-extension endpoints
displayed by $E$ and the group unit face displayed by $G$.
\end{theorem}
\begin{proof}
Project the packet to $E$, $G$, $b$, $a$, $m$, $i$, $c$, $\rho$, and
$\lambda$. The row $E$ supplies the displayed extension endpoints and their
classifier rows, while $G$ supplies the $\GroupUp$ unit row together with the
operation classifier attached to that unit.

Read the unit row as the identity automorphism-action row. On each displayed
extension endpoint, the action endpoint is transported by the same
$\GaloisExtUp$ classifier row, and the base endpoint is compared with itself
by the fixed-base $\hsame$ witness already recorded in $b$ and $\lambda$.

Repack the displayed extension endpoints, the group unit row, the fixed-base
witnesses, the classifier row $c$, and the shared $\Pkg$ provenance $\rho$.
The resulting row has the packet shape of
\autoref{def:galoisgroup-bhist-automorphism-action-packet}, so the identity
automorphism is accepted inside the public $\GaloisGroupUp$ surface.
\end{proof}

\begin{theorem}[GaloisGroupUp associative composition ledger]
\label{thm:galoisgroup-associative-composition-ledger}
For three composable accepted automorphism-action rows of
\autoref{def:galoisgroup-bhist-automorphism-action-packet}, the two
parenthesisations of their composite are classified by the same finite
$\Cont$ reassociation ledger. The comparison is inherited from the
$\GroupUp$ associativity surface of $G$ and from the shared $\hsame$ endpoint
classifier and $\Pkg$ provenance of the Galois-group packet.
\end{theorem}
\begin{proof}
First compose the first two rows, then compose the result with the third row.
\autoref{thm:galoisgroup-composition-closure} gives an accepted composite
whose endpoints, fixed-base witnesses, $\Cont$ ledger, classifier rows, and
$\Pkg$ provenance are all projections of the same packet.

Next compose the last two rows, then compose the first row with that result.
The same closure theorem gives the other accepted composite, with the same
outer source and target endpoints and with the same dependency packets $E$
and $G$.

The $\GroupUp$ associativity surface in $G$ identifies the two multiplication
parenthesisations. Transport the middle endpoint comparisons through the
shared $\hsame$ rows, and concatenate the two displayed $\Cont$ ledgers in
the reassociated order named by $\lambda$.

Both composites now expose the same outer endpoints, fixed-base witnesses,
classifier row $c$, $\Pkg$ provenance $\rho$, and dependency packets. Hence
the public classifier accepts them as the same Galois-group composition row.
\end{proof}

\begin{theorem}[GaloisGroupUp inverse cancellation boundary]
\label{thm:galoisgroup-inverse-cancellation-boundary}
For any accepted automorphism-action row of
\autoref{def:galoisgroup-bhist-automorphism-action-packet}, composing the row
with its inverse on either side classifies to the identity row inside the same
packet. The cancellation uses only the inverse row $i$, the composition row
$m$, the $\GroupUp$ inverse-law surface of $G$, and the fixed-base transport
rows recorded by $b$ and $\lambda$.
\end{theorem}
\begin{proof}
Start with an accepted automorphism-action row and apply
\autoref{thm:galoisgroup-inverse-closure}. This supplies the reversed
automorphism row, its reversed endpoints, the inverse ledger, and the
fixed-base witness transported by $\hsame$ along the same base endpoint.

Compose the original row with the inverse row using
\autoref{thm:galoisgroup-composition-closure}. The GroupUp right inverse law
classifies the operation endpoint to the unit row, while the fixed-base
transport row identifies the acted-on base endpoint with the original base
endpoint.

For the opposite order, use the same inverse row and composition closure with
the reversed endpoints. The GroupUp left inverse law again classifies the
operation endpoint to the unit row, and the fixed-base witness is transported
through the same $\hsame$ comparison.

By \autoref{thm:galoisgroup-identity-closure}, the unit-classified action row
is the accepted identity row of the packet. Thus both left and right
cancellation composites classify to the public identity boundary without
adding any host automorphism equality.
\end{proof}

\begin{theorem}[GaloisGroupUp public group-law package]
\label{thm:galoisgroup-public-group-law-package}
For an automorphism-action packet $(E,G,b,a,m,i,c,\rho,\lambda)$ of
\autoref{def:galoisgroup-bhist-automorphism-action-packet}, the public
$\GaloisGroupUp$ group-law package consists exactly of the identity row, the
composition-closure row, the associative composition ledger, the inverse
closure row, the inverse cancellation boundary, the action-ledger exactness
row, and the public NameCert obligation boundary. Each row is read only
through the displayed $\mathsf{BHist}$, $\mathsf{Cont}$, $\hsame$,
$\mathsf{Pkg}$, $\NameCert_{\GaloisExtUp}$, and $\NameCert_{\GroupUp}$
surfaces of the packet, with no host automorphism equality and no quotient of
automorphisms.
\end{theorem}

\begin{proof}
Start by projecting the packet to its displayed dependency rows. The
$\GaloisExtUp$ side supplies the extension endpoints, classifier rows, and
exactness ledgers, while the $\GroupUp$ side supplies the unit, multiplication,
inverse, associativity, congruence, and inverse-law rows that can be read by
the automorphism-action packet.

The unit part of the package is exactly
\autoref{thm:galoisgroup-identity-closure}. The multiplication part is exactly
\autoref{thm:galoisgroup-composition-closure}, and its associativity control is
the finite reassociation ledger of
\autoref{thm:galoisgroup-associative-composition-ledger}.

For inverses, use \autoref{thm:galoisgroup-inverse-closure} to expose the
inverse row and \autoref{thm:galoisgroup-inverse-cancellation-boundary} to

$\mathsf{BHist}$, $\mathsf{Cont}$, $\hsame$, $\mathsf{Pkg}$,
$\NameCert_{\GaloisExtUp}$, and $\NameCert_{\GroupUp}$ data. No additional
ambient equality or quotient carrier is present in the packet shape, so the
listed rows are exactly the consumable public group-law package.
\end{proof}
