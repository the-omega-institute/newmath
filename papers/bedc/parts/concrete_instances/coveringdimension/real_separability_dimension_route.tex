\begin{theorem}[Covering dimension real-separability sampling]
\label{thm:covering-dimension-real-separability-sampling}
Every finite-cover dimension consumer may use $\RealSeparabilityUp$ only through displayed rational dense windows before compact metric cover checks reduce to finite sampled rows. The $\CoveringDimensionUp$ carrier exports only the finite cover row, refinement map, order-bound witness, compact Lebesgue ledger, $\hsame$, $\Cont$, $\Pkg$, and $\NameCert$ rows of \autoref{def:covering-dimension-carrier}.
\end{theorem}
\begin{proof}
Read the dense-window evidence from \autoref{ch:concrete-instances-real-separability-namecert}. It supplies sampled rational names for inspected balls and mesh witnesses before any compact metric cover check is evaluated.

Now project the $\CoveringDimensionUp$ carrier of \autoref{def:covering-dimension-carrier}. The only exported rows are the finite cover row, refinement map, order-bound witness, compact Lebesgue ledger, componentwise $\hsame$ transport, displayed $\Cont$ replay, $\Pkg$ provenance, and local $\NameCert$ row.

Thus the dependency is a finite-data sampling interface. It does not add arbitrary open-cover compactness, a selected infinite refinement, or any host dimension theorem.
\end{proof}
