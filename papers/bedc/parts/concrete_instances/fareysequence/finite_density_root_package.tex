\begin{theorem}[Farey sequence finite-density root package]
\label{thm:farey-sequence-finite-density-root-package}
For every accepted $\FareySequenceUp$ carrier
$$
F=(B,A,M,L,T,S,D,Q,W,R,G,E,H,C,P,N),
$$
each finite-density read exported to an L10 approximation or completion consumer factors through one displayed root package: the boundary row $B$, adjacency ledger $A$, mediant row $M$, denominator and tolerance rows $L,T$, $\SternBrocotUp$ handoff $S$, $\StreamNameUp$ window $W$, $\RegSeqRatUp$ readback $R$, $\RealRationalApproximationUp$ row $G$, and terminal $\RealUp$ seal $E$, with $H,C,P,N$ preserving the same route. The package exports no selected best approximant, host order, quotient real equality, trichotomy, countable choice, or ambient completeness principle.
\end{theorem}
\begin{proof}
Start with the Farey carrier and NameCert obligations. The carrier of \autoref{def:farey-sequence-carrier} and the obligations of \autoref{thm:farey-sequence-namecert-obligations} expose exactly the finite boundary, adjacency, mediant, denominator, tolerance, Stern--Brocot, rational-source, stream, readback, approximation, Real-seal, transport, replay, provenance, and naming rows.

The density route is then locked to finite denominator data. The denominator-bound window theorem \autoref{thm:farey-sequence-denominator-bound-window} and the denominator refinement induction of \autoref{thm:farey-sequence-denominator-refinement-induction} place each inspected density window behind $B,A,M,L,T$ before any real-facing row can be read.

Next apply the Stern--Brocot handoff lock of \autoref{thm:farey-sequence-stern-brocot-handoff-lock}. It keeps each mediant or successor-level read attached to $A,L,T,S$ rather than to a host-order branch choice.

Finally use the approximation-window factorization theorem \autoref{thm:farey-sequence-approximation-window-factorization}. It sends the resulting finite bracket through $W,R,G,E$, while $H,C,P,N$ transport, replay, source, and name the same packet. Hence every finite-density read factors through the displayed root package and cannot export any of the refused ambient objects.
\end{proof}
