\begin{theorem}[GapClosureBoundary certificate-gate row]
\label{thm:gap-closure-boundary-certificate-gate-row}
For every accepted $\GapClosureBoundaryUp$ carrier
$B=(G,S,R,H,C,P,N)$, a boundary read by $\AspectChainUp$,
$\CausalCommitmentUp$, $\FiniteReflectionTupleUp$,
$\FiniteWitnessedRefutationUp$, $\GroundCompilerRecognizerFlowUp$,
$\OtherMindsCommitmentUp$, or $\SolipsismGapRefusalUp$ must pass through the
displayed certificate-gate row before the source branch $G \longrightarrow S$
or refusal branch $G \longrightarrow R$ is public. The gate is carried only by
the $\BHist$ packet, componentwise $\hsame$ transport, displayed $\Cont$
replay, $\Pkg$ provenance, and the local $\NameCert_{\GapClosureBoundaryUp}$
row.
\end{theorem}
\begin{proof}
Start with recognizer source admission,
\autoref{thm:gap-closure-boundary-recognizer-source-admission}. It sends an
accepted source read through the $\GroundCompilerRecognizerFlowUp$ gate and
then through the displayed $\FiniteReflectionTupleUp$ image.

Next use the local NameCert obligation surface,
\autoref{thm:gap-closure-boundary-namecert-obligation-surface}. It records
carrier exposure, source-route admission, refusal-ledger exactness, consumer
boundary exhaustion, and non-escape as rows of $B$.

Finally combine the hard-problem root package with source/refusal consumer
totality. The listed consumers therefore receive a gated source or refusal
branch, and no private gate outside $G,S,R,H,C,P,N$ is exported.
\end{proof}

\begin{theorem}[GapClosureBoundary witnessed-refutation row]
\label{thm:gap-closure-boundary-witnessed-refutation-row}
For every accepted $\GapClosureBoundaryUp$ carrier $B=(G,S,R,H,C,P,N)$, any
$\FiniteWitnessedRefutationUp$ or $\FiniteReflectionTupleUp$ readback of the
boundary is forced onto the displayed source/refusal split. A supported read
uses $G \longrightarrow S$ under $H,C,P,N$, an unsupported read uses
$G \longrightarrow R$ under the same structural rows, and neither read imports
a host refutation witness, quotient datum, hidden completion source, or
consumer-specific continuation.
\end{theorem}
\begin{proof}
Project the recognizer consumer exhaustion theorem,
\autoref{thm:gap-closure-boundary-recognizer-consumer-exhaustion}. It exhausts
the recognizer-facing route by successful tuple landing or refusal landing.

For the positive side, use
\autoref{thm:gap-closure-boundary-recognizer-source-admission}. The witnessed
row reaches the boundary only after the gate and tuple readback, and then it
is consumed as $G \longrightarrow S$.

For the refusal side, use
\autoref{thm:gap-closure-boundary-recognizer-refusal-stability}. Failed or
unsupported landings remain on $G \longrightarrow R$ and are transported only
through $H,C,P,N$.
\end{proof}

\begin{theorem}[GapClosureBoundary solipsism-refusal row]
\label{thm:gap-closure-boundary-solipsism-refusal-row}
Every $\SolipsismGapRefusalUp$, $\OtherMindsCommitmentUp$,
$\AspectChainUp$, or $\CausalCommitmentUp$ consumer of an accepted
$\GapClosureBoundaryUp$ packet receives only the displayed source/refusal
boundary rows $G,S,R,H,C,P,N$. In particular, a private solipsism escape,
private other-minds access, direct aspect-reduction row, causal-necessity row,
or hidden $\mathsf{T}$ supply is refused unless it is already represented by
the public source branch or the refusal branch named by the local
$\NameCert_{\GapClosureBoundaryUp}$ row.
\end{theorem}
\begin{proof}
Begin with hard-problem consumer factorization and consumer exactness. These
theorems place the listed sibling consumers inside the common boundary packet
and divide their reads between the source branch and the refusal branch.

Use the certificate-gate row above together with the NameCert route completion
surface. The local certificate names only $G,S,R,H,C,P,N$, so a consumer has
no private row from which to read solipsism escape, other-minds access, aspect
reduction, or causal necessity.

Finally apply refusal route exactness and refusal non-escape. Unsupported
requests stay on the displayed refusal ledger $R$ under $\hsame$, $\Cont$,
$\Pkg$, and $\NameCert$ rows, and cannot become hidden closure supply.
\end{proof}
