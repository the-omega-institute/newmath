\begin{theorem}[Cellular trust substrate obligation surface]
\label{thm:cellular-trust-substrate-obligation-surface}

No consumer can read a cellular trust substrate, a Rule 110 substrate, a closed observation boundary, or a no-induction substrate premise as $\CellularTrustSubstrateUp$ evidence without passing through these rows.
\end{theorem}
\begin{proof}
Carrier exposure is projection from \autoref{def:cellular-trust-substrate-carrier}. The packet displays the rule table, orbit window, classifier, manifest, audit route, transports, continuation routes, provenance, and local naming row as its only coordinates.

Local-rule, orbit-window, classifier, manifest, and audit accountability are read from $R,W,K,M,A$. The rule row decides the next state locally; the orbit-window row states the finite observation domain; the classifier row names only windows generated inside that domain; the manifest row states the replay schema; and the audit row records the finite route by which each accepted read is recovered.

The remaining rows close the NameCert surface. The $H,C,P,N$ rows transport, replay, record, and name exactly the displayed packet. Since the carrier lacks coordinates for unbounded orbit access, hidden update tables, detached manifests, quotient streams, host evaluator identity, and external induction, the listed rows exhaust the public obligation surface.
\end{proof}

\begin{theorem}[Local-rule nonescape]
\label{thm:cellular-trust-substrate-local-rule-nonescape}
Every accepted classifier read exported by a $\CellularTrustSubstrateUp$ carrier factors through the displayed local rule row $R$ and the finite orbit-window row $W$. In particular, two packets whose accepted evidence differs only by a hidden update table, a host evaluator, or an unstated boundary convention cannot be classified as the same cellular trust substrate packet.
\end{theorem}
\begin{proof}
Begin with the accepted-read route. By \autoref{thm:cellular-trust-substrate-obligation-surface}, a classifier read reaches the local NameCert row only through $R,W,K,M,A,H,C,P,N$.

The row $R$ supplies the complete local update table and boundary convention, while $W$ supplies the initial configuration and finite observation domain. The classifier row $K$ has no independent transition semantics; it can only tag windows admitted by the route from $R$ through $W$.

Thus changing the update table, host evaluator, or boundary convention without changing the displayed $R$ row is not a move inside the carrier. Since the packet has no hidden coordinate from which such a change can be recovered, the accepted classifier cannot identify the two displays as the same substrate evidence.
\end{proof}

\label{thm:cellular-trust-substrate-orbit-observation-boundary}

Consequently, a consumer obtains a finite observation boundary for the displayed orbit window, not an unbounded orbit, a stream quotient, or an internal induction rule over all cellular times.

It follows that the exported boundary is finite and displayed. The carrier has no coordinate for an all-time orbit, quotient stream representative, or induction rule over cellular time, so no such object is obtained by consuming the packet.
