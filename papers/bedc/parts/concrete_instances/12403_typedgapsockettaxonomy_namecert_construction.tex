\label{thm:typed-gap-socket-taxonomy-noncollapse-prediction}

Any such reclassification loses the Type III gate row in $G$ and the corresponding non-collapse entry in $M$, so it is a failed taxonomy read rather than an accepted Type II packet.

\begin{proof}

\end{proof}
