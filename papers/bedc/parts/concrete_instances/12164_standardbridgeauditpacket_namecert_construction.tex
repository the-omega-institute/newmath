\begin{definition}[Standard bridge audit packet carrier]
\label{def:standard-bridge-audit-packet-carrier}
A $\StandardBridgeAuditPacketUp$ carrier is a finite $\BHist$ packet
$$

with the following displayed rows:
\begin{enumerate}
\item $N$ is the accepted BEDC object row, read through the named $\KernelAcceptanceWitnessUp$ surface for the object being bridged;
\item $T$ is the host-theory row, recording the host theory only as the right-hand side of the bridge and not as a BEDC primitive;
\item $E$ is the encoding row from the BEDC carrier into the host theory;
\item $D$ is the decoding row from the host theory back to the BEDC carrier;
\item $R$ is the paired round-trip row, containing both decode-after-encode and encode-after-decode witnesses under the licensed classifier;
\item $U$ is the finite theorem-preservation row, pairing each preserved host theorem with a BEDC counterpart and a displayed translation;
\item $P$ is the no-host-leak row, recording the host primitives, quotient principles, semantic equality assumptions, proof objects, and host compiler readings that do not appear on the BEDC public surface;

\end{enumerate}
The carrier has no coordinate for a host primitive promoted to BEDC input, an infinite theorem schema, an unpaired host theorem, a hidden decoder, a private compiler route, or a bridge assertion detached from the displayed no-host-leak row.
\end{definition}
