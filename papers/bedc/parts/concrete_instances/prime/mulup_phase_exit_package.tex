$\mathsf{SemanticNameCert}$, the two $\AddUp$ distributivity rows, the
$\OrderUp$ monotonicity row, and the unary divisibility reflexive, transitive,
and antisymmetric consumer rows.
\end{theorem}
\begin{proof}

the carrier, classifier, multiplication relation, unit rows, algebraic laws, and
result $\mathsf{SemanticNameCert}$ for unary multiplication over
$\mathsf{BHist}$.

Next apply \autoref{thm:mulup-semantic-exit-audit-row-coverage}. It adds the
$\AddUp$ distributivity rows and the $\OrderUp$ monotonicity row without leaving
the displayed unary-history carrier or the $\hsame$ classifier surface.

Finally apply \autoref{thm:mulup-divisibility-order-consumer-package}. Its
reflexive, transitive, and antisymmetric divisibility rows use the same
$\mathsf{UnaryHistory}$ carrier, the same $\hsame$ comparison, the same $\Cont$
transport ledgers, and the same $\mathsf{NatMul}$ relation. Therefore the three
packages assemble as one Full Nat Arithmetic consumption point for $\MulUp$.
\end{proof}
