\begin{theorem}[Symmetry restoration stabilizer NameCert obligations]
\label{thm:symmetry-restoration-stabilizer-namecert-obligations}
For every accepted $\SymmetryRestorationStabilizerUp$ carrier
$S=(Q,A,T,U,M,O,R,H,C,P,N)$, the local
$\NameCert_{\SymmetryRestorationStabilizerUp}$ surface consists exactly of
the following obligation rows:
\begin{enumerate}
\item carrier admission for the $Q_{6}$ codon window, active support,
hidden $C_{2}$ stabilizer, orbit-persistence row, median-closure stabilizer,
orbit ledger, refusal row, transport, continuation, provenance, and local
naming row;
\item support visibility, requiring every stabilizer or orbit read to expose
the active-support row $A$ before reading $T,U,M$, or $O$;
\item stabilizer exactness, requiring the hidden $C_{2}$ row $T$ to name only
the identity and $\tau:s_{1}\leftrightarrow f_{2}$ for the active-support
and first-median-layer reads;
\item median-closure compatibility, requiring the restored
$S_{3}\times C_{2}$ row $M$ to be read only after the median-closure surface
and its orbit ledger are displayed;
\item orbit non-escape, requiring any hidden $C_{2}$ orbit claim to pass
through $A,T,U,M,O,H,C,P,N$;
\item package provenance and local $\NameCert$ naming for exactly the
displayed packet.
\end{enumerate}
No NameCert obligation is stored outside these rows.
\end{theorem}
\begin{proof}
Project the carrier of
\autoref{def:symmetry-restoration-stabilizer-carrier}. The projection
exposes the codon window, active support, stabilizer row, orbit-persistence
row, median-closure stabilizer, orbit ledger, refusal row, and the structural
rows $H,C,P,N$.

Support visibility comes first. The row $A$ is the source for the stabilizer
calculation, so $T,U,M$, and $O$ cannot be read as free group-theoretic
objects detached from the displayed support.

The stabilizer exactness row is $T$. It records exactly the identity and the
transposition $\tau:s_{1}\leftrightarrow f_{2}$ for the active-support
surface, while $U$ records the same nonidentity element along the
inactive-anchor and first-median-layer rows.

The median-closure compatibility row is $M$, with orbit data stored in $O$.
Transport by $H$ and replay by $C$ preserve this order, and $P,N$ package and
name the same rows. Thus no hidden orbit claim escapes the displayed support,
stabilizer, median-closure, transport, replay, provenance, and local naming
surface.
\end{proof}

\label{thm:symmetry-restoration-orbit-nonescape}

\end{theorem}
