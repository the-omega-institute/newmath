\begin{theorem}[Category associative empty result components]
\label{thm:category-hom-carrier-comp-assoc-empty-result-components}
For composable category hom-carriers \(f,g,h\) with both associated composite
ledgers displayed as \(left\) and \(right\), if the same source and target
endpoints also carry the empty hom-carrier, then both associated composite
histories are \(\hsame\) to \(\emp\), and the source and target endpoints are
\(\hsame\)-equal.
\end{theorem}
\begin{proof}
Apply displayed associativity determinacy using the empty hom-carrier as the
displayed comparison target. This identifies both bracketed composite histories
with \(\emp\). Empty identity hom-carrier exactness reads the same displayed
empty carrier back as the endpoint equality \(a\hsame d\).
\end{proof}
