\section{Public NameCert export}
\label{sec:innerproduct-public-namecert-export}

\begin{theorem}[InnerProductUp public NameCert export]
\label{thm:innerproduct-public-namecert-export}
The carrier, classifier, scalar endpoint classifier, form row,
conjugate-symmetry row, sesquilinearity rows, positive-definite row,
orthogonality rows, and visible ledger assembled by
\autoref{thm:innerproduct-namecert-obligation-surface} form the public
$\NameCert_{\InnerProductUp}$ export consumed by $\NormUp$ and $\HilbertUp$.
Completeness, topology, Banach metric, Cauchy limits, and standard bridge
data are not fields of this export.
\end{theorem}
\begin{proof}
Start from the root dependency surface of
\autoref{def:innerproduct-root-dependency-surface}. It names the displayed
$\VecSpaceUp$ carrier and classifier, the retained scalar endpoint
classifier, the scalar-valued form row, its law rows, the positive-definite
exactness row, the orthogonality rows, and the visible ledger as the
available $\mathsf{BHist}$ rows.

The vector and scalar endpoints exported to consumers are exactly the rows
identified by
\autoref{thm:innerproduct-root-vecspace-scalar-exposure}. The form row,
conjugate-symmetry row, sesquilinearity rows, endpoint transport, and
positive-definite row are exactly the rows identified by
\autoref{thm:innerproduct-root-form-law-exposure}.

Now apply \autoref{thm:innerproduct-namecert-obligation-surface}. Its
obligation surface packages those carrier, classifier, law, exactness,
orthogonality, and ledger rows as the paper-side
$\NameCert_{\InnerProductUp}$ surface. Since every listed field is already
one of the displayed $\mathsf{BHist}$ rows, the export is a public
NameCert-facing object rather than an additional hidden dependency.

Finally use \autoref{thm:innerproduct-consumer-hilbert-threshold-package}
and \autoref{thm:innerproduct-root-norm-hilbert-consumer-frontier}. They
identify the $\NormUp$ and $\HilbertUp$ consumer boundary as the same
inner-product rows while excluding completeness, topology, Banach metric,
Cauchy-limit selectors, and standard bridge data. Thus the displayed rows
form the public export and its negative boundary.
\end{proof}

\begin{theorem}[InnerProduct singleton terminal standard bridge]
\label{thm:innerproduct-singleton-terminal-standard-bridge}
The public singleton $\InnerProductUp$ export of
\autoref{thm:innerproduct-public-namecert-export} supplies a paper-level
$\StdBridge(\InnerProductUp,T_{IP}^\ast)$ to the one-point inner-product host
theory. The encode field sends the unique host vector of $T_{IP}^\ast$ to
$\emp$. The decode field sends every carried
$\mathsf{VecSpaceSingletonCarrier}$ endpoint to the host point. The
decode-encode round trip is terminal equality, and the encode-decode round
trip is the singleton classifier witness for the carried endpoint. The
theorem-preservation field covers the carrier row, scalar endpoint row, form
row, zero-left orthogonality row, and public export rows. The no-host-leak
field keeps the BEDC public surface to $\mathsf{BHist}$,
$\mathsf{VecSpaceSingletonCarrier}$,
$\mathsf{RealConstantHistoryClassifier}$,
$\mathsf{InnerProductSingletonForm}$, and the displayed public rows.
\end{theorem}
\begin{proof}
We check the six fields of \autoref{sec:acceptance-bridge-six-fields}. The
encode field is the constant map from the unique host vector of
$T_{IP}^\ast$ to the empty BEDC endpoint $\emp$. The carrier and classifier
rows in \autoref{thm:innerproduct-public-namecert-export} place $\emp$ inside
the public singleton vector carrier.

The decode field sends every carried
$\mathsf{VecSpaceSingletonCarrier}$ endpoint to the unique host point. Since
the host theory has one vector, decoded endpoints are identified by terminal
equality.

For the decode-encode round trip, start with the unique host vector, encode it
as $\emp$, and decode back to the host point. This is terminal equality in
$T_{IP}^\ast$.

For encode-decode, start with a carried singleton endpoint. Decoding gives the
unique host vector and encoding returns $\emp$. The singleton classifier row
of the public export supplies the classifier witness identifying the original
carried endpoint with $\emp$.

The theorem-preservation field is finite and explicit. The carrier, scalar
endpoint, form, law, orthogonality, and ledger rows are those exposed by
\autoref{thm:innerproduct-public-namecert-export}. The zero-left
orthogonality row is \autoref{thm:innerproduct-zero-left-orthogonal-row},
backed by
$\mathsf{BEDC.Derived.InnerProductUp.InnerProductSingletonOrthogonal\_zero\_left}$.
No completeness, Banach, Hilbert, spectral, or non-terminal host theorem is
included in this list.

Finally, the no-host-leak field follows from the same public-object inventory.
The bridge mentions the one-point host theory only inside encode, decode, and
the finite preservation list. The BEDC public surface remains exactly
$\mathsf{BHist}$, $\mathsf{VecSpaceSingletonCarrier}$,
$\mathsf{RealConstantHistoryClassifier}$,
$\mathsf{InnerProductSingletonForm}$, and the displayed rows of
\autoref{thm:innerproduct-public-namecert-export}.
\end{proof}
