\begin{theorem}[Cauchy oscillation criterion NameCert obligations]
\label{thm:cauchy-oscillation-criterion-namecert-obligations}

\item carrier admission for the oscillation ledger, tail-diameter witness, single-sequence criterion consumer, finite-family criterion consumer, shared dyadic tolerance, Real seal, transport, replay, provenance, and local naming rows;
\item oscillation-to-diameter projection, requiring every accepted oscillation comparison used by $O$ to expose the same finite source tail, threshold discipline, and dyadic tolerance from which the two visible tail-index comparison in $K$ is read;
\item diameter-to-oscillation reconstruction, requiring the two-point bound recorded by $K$ to replay as a finite oscillation-ledger entry over the same displayed source tail and tolerance row;
\item consumer handoff through $\CauchyCriterionUp$ and $\UniformCauchyCriterionUp$, requiring both consumer rows $A$ and $U$ to read the shared $O,K,Q$ route before exposing the terminal $\RealUp$ seal row $R$;
\item non-escape from quotient streams, selected limits, host equality, choice, Real completeness, and detached completion data.
\end{enumerate}
No Cauchy oscillation criterion obligation is stored outside these rows.
\end{theorem}

\begin{proof}
Project the accepted packet $X=(O,K,A,U,Q,R,H,C,P,N)$. The projection exposes exactly one oscillation ledger, one tail-diameter witness, two criterion consumer rows, one shared dyadic tolerance, one Real seal handoff, and the structural transport, replay, provenance, and naming rows.

For the oscillation-to-diameter direction, read the ledger row $O$ through its finite tail window, threshold schedule, dyadic tolerance, and displayed comparisons. The tail-diameter row $K$ uses the same source tail and tolerance but focuses the read to two visible tail indices beyond the displayed threshold, so every two-point comparison consumed by the criterion is a projection of the finite oscillation route.

For the diameter-to-oscillation direction, replay the two visible indices and bound stored in $K$ as one ledger entry over the same source tail. The shared tolerance row $Q$ fixes the dyadic budget, so the reconstructed ledger entry cannot alter the radius, threshold discipline, or source observations.

The consumer rows then read the common route. The $\CauchyCriterionUp$ row $A$ and the $\UniformCauchyCriterionUp$ row $U$ both consume $O,K,Q$ before exposing $R$, and the replay row $C$ records this order as a displayed continuation.

Finally, $H,P,N$ transport, source, and name only the displayed finite rows. Since the carrier contains no coordinate for quotient streams, selected limits, host equality, choice, Real completeness, or detached completion data, none of those principles can be part of the local obligation surface.
\end{proof}

\begin{theorem}[Cauchy oscillation and tail-diameter equivalence]
\label{thm:cauchy-oscillation-criterion-tail-diameter-equivalence}

\begin{proof}

Carrier admission gives the consumer no analytic coordinates except the oscillation ledger $O$, the tail-diameter witness $K$, the shared tolerance row $Q$, and the terminal seal $R$.

\end{proof}
