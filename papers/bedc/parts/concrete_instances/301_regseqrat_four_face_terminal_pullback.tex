\begin{theorem}[RegSeqRat four-face terminal pullback]
\label{thm:regseqrat-four-face-terminal-pullback}
Every accepted Phase Real terminal read and every completion-facing terminal
read obtained from a $\RegSeqRatUp$ packet factors through one finite
$\mathsf{BHist}$ pullback with exactly four source faces:
\begin{enumerate}
\item the $\DyadicRatCoreUp$ endpoint, radius, and distance ledgers;
\item the $\StreamNameUp$ scheduled window rows and displayed bundle rows;
\item the $\RegSeqRatUp$ regularity rows, classifier rows, and provenance
rows;
\item the terminal $\RealUp$ seal row.
\end{enumerate}
If two terminal reads expose the same four faces, then they classify together
through the displayed $\hsame$ comparisons, $\Cont$ routes, $\mathsf{Pkg}$
provenance, and local $\NameCert$ rows of that pullback.

The terminal pullback exports no quotient stream equality, host sequence
equality, selected limit, host real equality, ambient completeness,
metric-completion point, or standard bridge.
\end{theorem}
\begin{proof}
Start with \autoref{thm:regseqrat-streamname-dyadic-window-lock}. It places
the $\StreamNameUp$ schedule rows and the $\DyadicRatCoreUp$ endpoint and
radius ledgers inside the same accepted finite $\RegSeqRatUp$ window.

Next apply \autoref{thm:regseqrat-seal-route-visible-source-normal-form}. The
terminal $\RealUp$ seal route can read only the visible finite source surface
formed from the schedule, dyadic ledgers, regularity rows, classifier rows,
$\mathsf{BHist}$ packet rows, $\hsame$, $\Cont$, $\mathsf{Pkg}$, and local
$\NameCert$ provenance.

Use \autoref{thm:regseqrat-real-seal-route-exclusion-package} to remove every
off-surface route component. The remaining route package is exactly the
finite inventory from which the four faces of the pullback are read.

Apply \autoref{thm:regseqrat-real-seal-classifier-factorization}. Agreement on
the four faces transports the two terminal reads through the displayed
$\RegSeqRatUp$ classifier surface together with the visible $\hsame$, $\Cont$,
$\mathsf{Pkg}$, and local $\NameCert$ rows.

Finally invoke \autoref{thm:regseqrat-real-seal-schedule-confluence}. The
common displayed inventory replays to one terminal finite $\RealUp$ seal
route, and the cited exclusion package leaves no coordinate for quotient
stream equality, host sequence equality, selected limits, host real equality,
ambient completeness, metric-completion points, or standard bridge data.
\end{proof}
