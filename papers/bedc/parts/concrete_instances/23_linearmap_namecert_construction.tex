The linear map interface \(\LinearMapUp\) is licensed between two vector spaces over the same scalar field from \autoref{ch:concrete-instances-vecspace-namecert}. The carrier is the predicate "\(f : V \to W\) preserves the additive structure and commutes with scalar multiplication"; the classifier is pointwise carrier sameness on the codomain restricted to maps satisfying linearity. It is not a primitive function space; it is licensed only as a derived predicate over function pairs.
