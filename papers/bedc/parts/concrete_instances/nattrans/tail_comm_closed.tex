\begin{theorem}[NatTrans prefix component vertical composite tail comm closure]
\label{thm:nattrans-prefix-component-vert-comp-tail-comm-closed}
For prefix components over unary histories, if \(\eta\) is a component from
\(\mathsf{append}(p,a)\) to \(\mathsf{append}(q,a)\), \(\theta\) is a component
from \(\mathsf{append}(q,a)\) to \(\mathsf{append}(r,a)\),
\(\ContR(\eta,\theta,\eta\theta)\), and
\(\ContR(\theta,\eta,\theta\eta)\), then \(\eta\theta\) is a component from
\(\mathsf{append}(p,a)\) to \(\mathsf{append}(r,a)\), and
\(\HSame(\eta\theta,\theta\eta)\).
\end{theorem}
\begin{proof}
Vertical composition closure supplies the component carrier for the forward
composite. Tail commutativity of the same two component carriers identifies
the forward and reversed continuation results.
\end{proof}
