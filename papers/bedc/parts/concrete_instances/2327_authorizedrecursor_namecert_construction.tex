\chapter{A Concrete Naming Certificate for $\AuthorizedRecursorUp$}
\origin{human}

The $\AuthorizedRecursorUp$ packet records the finite certificate surface by which a kernel-emitted eliminator becomes an authorized BEDC generator. It sits between the accepted recursor authorization row and downstream recursor-generator consumers: the constructor signature, emitted eliminator, branch package, structural descent ledger, audit gate, generated output, and local \NameCert{} row are all visible packet coordinates.

\begin{definition}[AuthorizedRecursor certificate carrier]
\label{def:recursor-generator-certificate-carrier}
An $\AuthorizedRecursorUp$ certificate carrier is a finite $\BHist$ packet
$$
Q=(I,\Sigma,R,M,B,D,A,O,H,C,P,N)
$$
with the following displayed rows:
\begin{enumerate}
\item $I$ is the accepted inductive-name row and $\Sigma$ is its closed constructor-signature row;
\item $R$ is the kernel-emitted recursor row for $\Sigma$;
\item $M$ is the motive row accepted for the recursor call;
\item $B$ is the finite branch package indexed by the constructors of $\Sigma$;
\item $D$ is the structural descent ledger for every recursive occurrence exposed to $B$;
\item $A$ is the audit-gate row recording kernel acceptance and source-dependency refusals for unlisted proof principles;
\item $O$ is the generated output row handed to the downstream certificate surface;
\item $H$ stores componentwise $\hsame$ transports for $\Sigma,R,M,B,D,A,O$;
\item $C$ stores displayed $\Cont$ routes for reading the signature, recursor, motive, branch, descent, audit, and output rows;
\item $P$ is the $\Pkg$ provenance over $\BHist$, constructor signatures, emitted eliminators, branch packages, structural descent, audit gates, $\hsame$, $\Cont$, and local naming;
\item $N$ is the local $\NameCert_{\AuthorizedRecursorUp}$ row.
\end{enumerate}
The carrier has no coordinate for an unlisted constructor, an extra branch, a non-structural recursive call, a hidden generator, or an external proof-trust channel.
\end{definition}

\begin{theorem}[AuthorizedRecursor branch coverage]
\label{thm:recursor-generator-branch-coverage}
For every accepted $\AuthorizedRecursorUp$ carrier $Q=(I,\Sigma,R,M,B,D,A,O,H,C,P,N)$, the branch package $B$ contains exactly the branch rows indexed by the constructor signature $\Sigma$ read by the emitted recursor $R$. A downstream consumer of $O$ has no route to an omitted constructor case and no route to an additional branch outside $\Sigma$.
\end{theorem}
\begin{proof}
Project the constructor signature $\Sigma$, emitted recursor $R$, and branch package $B$ from the carrier. The recursor row is emitted for that signature, so its branch inventory is read against the constructors named by $\Sigma$.

The branch package $B$ is a finite coordinate of the same packet. Its rows are therefore exactly the rows exposed for the constructor cases of $\Sigma$.

The transport and continuation rows $H$ and $C$ preserve the same displayed indexing for downstream reads of $O$. Since no additional branch coordinate is present in $Q$, branch coverage is exhausted by the displayed constructor signature.
\end{proof}

\begin{theorem}[AuthorizedRecursor non-escape ledger]
\label{thm:recursor-generator-non-escape-ledger}
Every generated output row $O$ of an accepted $\AuthorizedRecursorUp$ carrier factors through the emitted recursor $R$, the accepted motive $M$, the constructor-indexed branch package $B$, the structural descent ledger $D$, and the audit-gate row $A$. The packet exports no host generator, non-structural recursive call, unlisted constructor, unchecked proof principle, or channel around the local $\NameCert_{\AuthorizedRecursorUp}$ row.
\end{theorem}
\begin{proof}
Start with the carrier projection. The only generator row available to a consumer is the emitted recursor $R$, and the only accepted call data are $M$, $B$, and $D$.

Apply \autoref{thm:recursor-generator-branch-coverage}. Every branch read used to form $O$ is one of the constructor-indexed rows in $B$, and every recursive occurrence is recorded in the structural descent ledger $D$.

The audit row $A$ is part of the same finite certificate surface. It records the source and dependency boundary before the generated row is exported.

Finally, $H,C,P,N$ transport, route, package, and name the displayed rows. They add no independent generator coordinate, so every public read of $O$ remains inside the authorized recursor ledger.
\end{proof}

\begin{theorem}[AuthorizedRecursor certificate surface]
\label{thm:authorized-recursor-certificate-surface}
The carrier of \autoref{def:recursor-generator-certificate-carrier}, the branch coverage theorem of \autoref{thm:recursor-generator-branch-coverage}, and the non-escape ledger of \autoref{thm:recursor-generator-non-escape-ledger} assemble the finite $\AuthorizedRecursorUp$ NameCert surface for kernel-emitted eliminators. The surface is available to recursor-generator, TasteGate, and self-compilation consumers without treating host recursion as an untracked primitive.
\end{theorem}
\begin{proof}
The carrier names every row that may be inspected by a downstream consumer: signature, emitted recursor, motive, branch package, descent ledger, audit gate, output row, transports, routes, provenance, and local naming.

Branch coverage binds the branch package to the accepted constructor signature. The non-escape ledger binds generated outputs to the recursor, motive, branch, descent, and audit rows.

Repacking these rows gives the advertised finite \NameCert{} surface. Since every consumer read factors through the displayed packet, host recursion never appears as an untracked generator.
\end{proof}
