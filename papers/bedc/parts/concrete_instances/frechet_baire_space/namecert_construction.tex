\begin{theorem}[Frechet-space Baire completeness route]
\label{thm:frechet-space-baire-completeness-route}

The topological-vector and locally convex rows $V,L$ remain visible throughout the route.
\end{theorem}
\begin{proof}

Use the Frechet dependency route of \autoref{thm:frechet-space-baire-dependency-route}. It sends every Baire request through the complete metric row and the metric-completion handoff before the request reaches the Baire surfaces.

The row $U$ supplies the complete-metric Baire inspection surface, and $T$ supplies the theorem-level dense-open package. The structural rows $H,C,P,N$ transport, replay, source, and name only this route, so the listed rows exhaust the completeness-to-Baire handoff.
\end{proof}
