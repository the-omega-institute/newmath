\chapter{A Concrete Naming Certificate for \(\MetricSpaceUp\)}
\label{ch:concrete-instances-metric-namecert}

The metric space interface \(\MetricSpaceUp\) is licensed over a carrier together with a distance function \(d : C \times C \to \RealUp\) satisfying non-negativity, identity-of-indiscernibles (with apartness in place of decidable equality), symmetry, and the triangle inequality, all expressed in Bishop-style with explicit real moduli. It depends on \autoref{ch:concrete-instances-real-namecert}.

\begin{remark}[Roadmap chapter---formalization pending]
\label{rem:concrete-instances-metric-namecert-roadmap}
This chapter sketches the licensing target. Lean implementations of the
listed specifications and certificate are not yet present; the chapter
is a spec roadmap, not a formalized license.
\end{remark}

\section{Carrier definition}

\begin{definition}[MetricSpace carrier]
\label{def:metric-carrier}
Use an abstract carrier \(C\), classifier, apartness, and distance \(d:C\times C\to\RealUp\).
\end{definition}

\section{The certificate}

\begin{definition}[MetricSpace source specification]
\label{def:metric-source-specification}
The source is \(C\) together with the Bishop real distance codomain.
\end{definition}

\begin{definition}[MetricSpace pattern specification]
\label{def:metric-pattern-specification}
The pattern is metric judgment formation from carrier pairs to real distance witnesses.
\end{definition}

\begin{definition}[MetricSpace classifier specification]
\label{def:metric-classifier-specification}
The classifier is the supplied carrier sameness, with distance stable under classifier replacement.
\end{definition}

\begin{definition}[MetricSpace stability certificate]
\label{def:metric-stability-certificate}
The stability certificate consists of:
\begin{enumerate}
  \item distance non-negativity
  \item identity of indiscernibles using classifier and apartness
  \item symmetry
  \item triangle inequality
  \item distance congruence in both arguments
  \item explicit real moduli for all inequality fields
\end{enumerate}
\end{definition}

\begin{definition}[MetricSpace ledger policy]
\label{def:metric-ledger-policy}
Record carrier witnesses and the real ledger of each distance value.
\end{definition}

\begin{theorem}[Concrete \(\MetricSpaceUp\) naming certificate]
\label{thm:concrete-metric-namecert}
Any carrier with constructive metric law fields forms a valid NameCert for \(\MetricSpaceUp(C)\).
\end{theorem}

\begin{corollary}[\(\MetricSpaceUp\) is licensed, not primitive]
\label{cor:metric-licensed-not-primitive}
\(\MetricSpaceUp\) is licensed by explicit Bishop-real metric data.
\end{corollary}

\begin{remark}[Zero-axiom encoding note]
\label{rem:metric-zero-axiom-note}
Replaces decidable equality with apartness and explicit real-modulus inequalities.
\end{remark}
