\begin{definition}[GoldenMeanShift carrier]
\label{def:golden-mean-shift-carrier}
A $\GoldenMeanShiftUp$ carrier is a finite $\mathsf{BHist}$ packet
$$
G=(W,Z,O,P,L)
$$
with the following visible rows:
\begin{enumerate}
\item $W$ is a finite list of binary bit rows, each row displayed as either a zero bit row or a one bit row inside $\mathsf{BHist}$;
\item $Z$ is the all-zero witness row for the same window shape;
\item $O$ is the adjacency ledger whose every neighboring pair in $W$ is one of zero-zero, zero-one, or one-zero;
\item $P$ is the $\mathsf{Pkg}$ provenance row naming the prefix and fold readbacks used to inspect the window;
\item $L$ is the $\NameCert$ ledger recording that every accepted classifier row is a componentwise $\hsame$ comparison between displayed bit rows.
\end{enumerate}
The carrier accepts exactly finite binary windows whose visible bit rows contain no adjacent one rows. It contains no infinite sequence, no shift-invariant quotient, no entropy row, and no host symbolic-dynamics theorem.
\end{definition}
