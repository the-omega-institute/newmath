$$

\item $T$ is the $\BoundedLinearOperatorUp$ row naming the operator whose calculus is being inspected.

\item $\Omega$ is the holomorphic scalar-window row. It records the finite function values and local compatibility checks visible on the contour windows.

\begin{theorem}[Dunford functional calculus NameCert obligations]
\label{thm:dunford-functional-calculus-namecert-obligations}
For every accepted $\DunfordFunctionalCalculusUp$ carrier
$$
F=(A,T,\Gamma,\Omega,R,I,Q,S,H,C,P,N),
$$
the local $\NameCert_{\DunfordFunctionalCalculusUp}$ surface consists exactly of carrier admission for the displayed rows, Banach-algebra source compatibility through $A$, bounded-operator source compatibility through $T$, contour-window admission through $\Gamma$, holomorphic scalar-window admission through $\Omega$, resolvent identity exactness through $R$, finite contour-ledger exactness through $I$, spectral-radius and compact-operator boundary visibility through $Q$, terminal calculus handoff through $S$, componentwise transport through $H$, displayed replay through $C$, package provenance through $P$, and local naming through $N$.

Every calculus-facing consumer must read the same rows before using the contour value. It receives no spectral projection, no spectral measure, no selected spectrum, no quotient equality of operators, no countable-choice schedule, and no ambient host functional calculus.
\end{theorem}
\begin{proof}
Carrier admission is projection from \autoref{def:dunford-functional-calculus-carrier}. The accepted packet displays exactly $A,T,\Gamma,\Omega,R,I,Q,S,H,C,P,N$.

The source rows $A$ and $T$ fix the Banach-algebra and bounded-operator surfaces. The contour and scalar-window rows $\Gamma,\Omega$ then provide the finite requests on which the calculus read is allowed to inspect function values.

The resolvent row $R$ supplies only the displayed inverse checks and two-shift identity ledger. The contour ledger $I$ reads those rows through finite Riemann-sum and dyadic-budget windows, so it cannot import a host integral or an unrecorded inverse.

The spectral-boundary row $Q$ and handoff row $S$ are downstream of the same route. Transport, replay, provenance, and local naming in $H,C,P,N$ preserve that finite packet. Since the carrier has no coordinate for spectral projections, spectral measures, selected spectra, quotient equality, countable choice, or a host calculus, the listed rows exhaust the local obligations.
\end{proof}
