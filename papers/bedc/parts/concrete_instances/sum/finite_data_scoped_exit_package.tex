\begin{theorem}[Sum finite-data scoped-exit package]
\label{thm:sum-finite-data-scoped-exit-package}
The scoped finite-data exit surface of $\SumUp$ is exactly the package consisting of the tagged semantic certificate of \autoref{thm:concrete-sum-namecert}, the selected visible injection tag, left and right branch exactness, the source ledger rows, visible-history descent through endpoint readback, and the mature finite consumer boundary of \autoref{thm:sum-mature-finite-consumer-boundary}. The selected injection tag is the two-branch finite-choice row already licensed by \autoref{ch:concrete-instances-bool-namecert}; $\SumUp$ consumes that sibling only as visible tag discipline, while the left and right payload rows remain governed by the local sum carrier and classifier. A consumer exiting the $\SumUp$ chapter through this surface may inspect the selected tag, recover the corresponding source payload and source ledger evidence, compare same-tag endpoints through the displayed branch classifier, and replay the visible history descent. It receives no hidden coproduct eliminator, no dependent target recursion, and no classifier beyond the tagged finite-data rows already exported by the semantic certificate.
\end{theorem}
\begin{proof}
Start with the semantic certificate in \autoref{thm:concrete-sum-namecert}. Its carrier is the tagged finite packet, so each public endpoint enters through an $\Ezero$ or $\Eone$ injection rather than through a host coproduct object.

The selected tag is governed by \autoref{thm:sum-ledger-visible-tag-separation}. The $\BoolUp$ reference supplies only the finite $b0/b1$ tag reading for this visible branch choice; it does not introduce a host coproduct, an eliminator, or an external Boolean equality. The same-tag clauses are governed by \autoref{thm:sum-ledger-branch-exactness-iff}, while visible payload recovery is governed by \autoref{thm:sum-ledger-visible-payload-readback} and endpoint descent by \autoref{thm:sum-ledger-chain-endpoint-exactness}.

Finally apply the consumer boundary in \autoref{thm:sum-mature-finite-consumer-boundary}. It states that every finite consumer factors through exactly these tagged payload, branch exactness, source ledger, and visible-history rows. Therefore the package is a scoped exit surface for finite data rather than a generic coproduct interface.
\end{proof}
