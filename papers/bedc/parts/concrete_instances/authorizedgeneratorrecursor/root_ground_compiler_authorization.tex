\begin{theorem}[AuthorizedGeneratorRecursor finite route exposure]
\label{thm:authorized-generator-recursor-finite-route-exposure}

$$
I,E,M,B,D,O,A,H,C,P,G,N.
$$
Consequently the public surface cannot introduce an unregistered branch, a hidden induction channel, or an off-packet generator path beyond the accepted signature, eliminator, motive, branch, descent, output, audit, transport, continuation, package, boundary, and local certificate rows.

\begin{proof}

Branch exposure is exhausted by \autoref{thm:authorized-generator-recursor-branch-exhaustion}. A public route can name only the branch package associated with the accepted constructors and the emitted eliminator, so an unregistered branch has no row from which to enter.

The remaining escape routes are closed by descent locality and ledger descent. Use \autoref{thm:authorized-generator-recursor-structural-descent-locality}, \autoref{thm:authorized-generator-recursor-audit-ledger-nonescape}, and \autoref{thm:authorized-generator-recursor-ground-compiler-handoff}; together they keep recursive use, audit provenance, and ground-compiler consumption inside the same finite packet.

\begin{theorem}[AuthorizedGeneratorRecursor output audit nonescape]
\label{thm:authorized-generator-recursor-output-audit-nonescape}
For every accepted $\AuthorizedGeneratorRecursorUp$ root packet, the output row $O$ cannot be consumed apart from the audit provenance row $A$, the ground-compiler boundary row $G$, and the local certificate row $N$. Any consumer of $O$ must preserve the same displayed route through audit provenance, boundary exposure, and $\NameCert_{\AuthorizedGeneratorRecursorUp}$ certification.
\end{theorem}
\begin{proof}
Begin with \autoref{thm:authorized-generator-recursor-audit-provenance-factorization}. It attaches public exposure of $O$ to the audit and provenance rows, so the output cannot be read as an isolated generator result.

The boundary side is supplied by \autoref{thm:authorized-generator-recursor-root-host-boundary-namecert-exhaustion}. The row $G$ is exhausted by the displayed packet and cannot export a hidden proof generator, external axiom, or detached induction channel.

Finally apply \autoref{thm:authorized-generator-recursor-ground-compiler-authorization}. Since every ground-compiler consumer reads $O$ only after the carrier, audit, boundary, package, and local $\NameCert$ rows are retained, the output row has no nonescaping consumer route apart from $A,G,N$ and the finite packet that carries them.
\end{proof}
