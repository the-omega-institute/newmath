\begin{theorem}[Authorized generator recursor branch descent audit totality]
\label{thm:authorized-generator-recursor-branch-descent-audit-totality}
Every accepted $\AuthorizedGeneratorRecursorUp$ packet reads the branch package $B$, descent row $D$, output row $O$, and audit row $A$ only through the displayed carrier
$$
I,E,M,B,D,O,A,H,C,P,G,N.
$$
The branch, descent, output, and audit coordinates therefore form one root boundary: a consumer cannot read a branch package without the matching descent row, cannot read an output row outside the audit row, and cannot replace the displayed carrier by host induction, an external axiom route, an unaudited dependency route, or an off-packet substitution.
\end{theorem}
\leanchecked{BEDC.Derived.AuthorizedGeneratorRecursorUp.AuthorizedGeneratorRecursor\_branch\_descent\_audit\_totality}
\begin{proof}
Begin with the carrier, \autoref{def:authorized-generator-recursor-carrier}. It displays exactly the input, authorization, measure, branch, descent, output, audit, transport, continuation, provenance, ground-compiler, and local naming rows.

Apply carrier admission, \autoref{thm:authorized-generator-recursor-carrier-admission}. Admission exposes $B,D,O,A$ as rows of the same accepted packet, not as detachable host data.

Now use root boundary ledger totality, \autoref{thm:authorized-generator-recursor-root-boundary-ledger-totality}. It forces every public read of the root boundary through the audit ledger attached to the displayed route.

Finally apply the ground-compiler handoff, \autoref{thm:authorized-generator-recursor-ground-compiler-handoff}. The handoff keeps branch choice, descent evidence, output naming, and audit replay inside $I,E,M,B,D,O,A,H,C,P,G,N$, excluding host induction, external axioms, unaudited dependency routes, and off-packet substitutions.
\end{proof}

\begin{theorem}[Authorized generator recursor output NameCert non-escape]
\label{thm:authorized-generator-recursor-output-namecert-nonescape}
For every accepted $\AuthorizedGeneratorRecursorUp$ packet, public output from the displayed output row $O$ remains inside the local $\NameCert$ row $N$ and the ground-compiler boundary $G$. It cannot escape into host induction, external axioms, unaudited dependency routes, detached branch packages, off-packet substitutions, or an output certificate not carried by
$$
I,E,M,B,D,O,A,H,C,P,G,N.
$$
\end{theorem}
\leanchecked{BEDC.Derived.AuthorizedGeneratorRecursorUp.AuthorizedGeneratorRecursor\_output\_namecert\_nonescape}
\begin{proof}
Project the accepted carrier of \autoref{def:authorized-generator-recursor-carrier}. The only public output coordinate is $O$, and it is packaged together with the audit row $A$, ground-compiler row $G$, and local $\NameCert$ row $N$.

Use carrier admission, \autoref{thm:authorized-generator-recursor-carrier-admission}, to identify the admitted output as a row of the same finite packet. Then apply root boundary ledger totality, \autoref{thm:authorized-generator-recursor-root-boundary-ledger-totality}, so every public output read is replayed through the audit boundary rather than through an off-packet dependency route.

The ground-compiler handoff, \autoref{thm:authorized-generator-recursor-ground-compiler-handoff}, supplies the final boundary. It sends the output read through $G$ and $N$, so public output cannot be reinterpreted as host induction, external axioms, a detached branch package, an unaudited substitution, or any output certificate outside the displayed carrier.
\end{proof}

\begin{corollary}[Authorized generator recursor branch descent local obstruction]
\label{cor:authorized-generator-recursor-branch-descent-local-obstruction}
For every accepted $\AuthorizedGeneratorRecursorUp$ packet, any displayed failure, refusal, separation, or countermodel row attached to branch descent blocks only the local route that tries to read one of the following coordinates outside the displayed carrier:
$$
B\text{ without }D,\quad O\text{ outside }A,\quad O\text{ outside }G\text{ and }N,\quad \text{or a certificate not carried by }I,E,M,B,D,O,A,H,C,P,G,N.
$$
The obstruction cannot be reinterpreted as a global impossibility theorem for recursors, as a host induction theorem, as hidden external validation, or as a rejection of any consumer route that still factors through the displayed branch, descent, output, audit, ground-compiler, and local $\NameCert$ rows.
\end{corollary}
\leanchecked{BEDC.Derived.AuthorizedGeneratorRecursorUp.AuthorizedGeneratorRecursorBranchDescentLocalObstruction}
\begin{proof}
Fix an accepted packet. By \autoref{thm:authorized-generator-recursor-branch-descent-audit-totality}, every consumer-facing read of branch, descent, output, and audit data is a read through the single displayed carrier
$$
I,E,M,B,D,O,A,H,C,P,G,N.
$$
In particular, $B$ is not available as a detached branch package, $D$ is the matching descent row for that branch package, $O$ is not available outside $A$, and the carrier cannot be replaced by host induction, an external axiom route, an unaudited dependency route, or an off-packet substitution.

Apply \autoref{thm:authorized-generator-recursor-output-namecert-nonescape} to the output side of the same read. Public output from $O$ remains inside the local $\NameCert$ row $N$ and the ground-compiler boundary $G$, and it has no escape route to a detached branch package, an external certificate, host induction, external axioms, unaudited dependencies, or an off-packet substitution.

Therefore the only coordinates on which a displayed obstruction can act are the named local coordinates of the carrier: branch detachment from $D$, output detachment from $A$, output escape from $G$ and $N$, or certificate reuse outside $I,E,M,B,D,O,A,H,C,P,G,N$. Since both cited theorems exhaust consumer-facing reads by those rows, the obstruction has no additional coordinate from which to assert a global impossibility, a host theorem, or hidden external validation. Conversely, a consumer route that still factors through the displayed rows is not blocked by this corollary; the corollary blocks only the named converse or reuse route that leaves the local carrier.
\end{proof}
