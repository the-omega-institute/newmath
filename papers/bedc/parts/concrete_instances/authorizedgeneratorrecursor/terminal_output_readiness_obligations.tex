\begin{theorem}[AuthorizedGeneratorRecursor terminal-output readiness]
\label{thm:authorized-generator-recursor-terminal-output-readiness}
For an accepted $\AuthorizedGeneratorRecursorUp$ packet
$$
R=(I,E,M,B,D,O,A,H,C,P,G,N),
$$
the terminal output row $O$ is dependency-ready only as a read of the same displayed carrier rows. \end{theorem}
\begin{proof}
Project the carrier of \autoref{def:authorized-generator-recursor-carrier}. The output row $O$ is one coordinate of $I,E,M,B,D,O,A,H,C,P,G,N$, not a second carrier.

Use audit provenance factorization and audit boundary. Finally attach output-name stability and ground-compiler handoff. Theorems~\ref{thm:authorized-generator-recursor-output-name-stability} and \ref{thm:authorized-generator-recursor-ground-compiler-handoff} preserve the name of $O$ only on the same finite carrier, so no hidden generator or host proof channel can replace the displayed packet.
\end{proof}
\leanchecked{BEDC.Derived.AuthorizedGeneratorRecursorUp.AuthorizedGeneratorRecursorTerminalOutputReadiness}
