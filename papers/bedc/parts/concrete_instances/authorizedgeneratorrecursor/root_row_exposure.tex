\begin{theorem}[AuthorizedGeneratorRecursor root branch row exposure]
\label{thm:authorized-generator-recursor-root-branch-row-exposure}
For every accepted $\AuthorizedGeneratorRecursorUp$ packet with displayed carrier
$$
R=(I,E,M,B,D,O,A,H,C,P,G,N),
$$
the root branch row $B$ is exposed before any downstream closed-normal or generator-audit consumer may read the output row $O$. The exposure is a projection of the carrier of \autoref{def:authorized-generator-recursor-carrier} through the local NameCert surface of \autoref{thm:authorized-generator-recursor-namecert-surface}, and it is exhaustive by \autoref{thm:authorized-generator-recursor-branch-exhaustion}.
\end{theorem}
\begin{proof}
Project the displayed packet $R=(I,E,M,B,D,O,A,H,C,P,G,N)$ from \autoref{def:authorized-generator-recursor-carrier}. The branch row $B$ is one of its public coordinates, not a hidden generator source.

Apply the local NameCert surface. It exposes the output only together with the accepted signature, emitted eliminator, checked motive, branch package, structural descent, audit, transport, route, provenance, boundary, and local name rows.

Finally use branch exhaustion. The branch row contains exactly the public branch surface for the constructors displayed in $I$, so a downstream consumer cannot read $O$ while bypassing $B$ or inventing an additional branch row.
\end{proof}
\leanchecked{BEDC.Derived.AuthorizedGeneratorRecursorUp.AuthorizedGeneratorRecursorRootBranchRowExposure}

$$
R=(I,E,M,B,D,O,A,H,C,P,G,N),

$$

They place public output behind the accepted signature, emitted eliminator, checked motive, complete branch package, structural descent, audit provenance, continuation route, package row, boundary row, and local certificate, so no root consumer receives an unaudited recursor result.
\end{proof}
