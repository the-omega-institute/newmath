\label{thm:authorized-generator-recursor-root-output-audit-totality}

emitted eliminator $E$, checked motive $M$, branch package $B$,
structural-descent row $D$, audit row $A$, transport row $H$, continuation

row $N$. Thus the public output is total only as the displayed NameCert packet
$$
(I,E,M,B,D,O,A,H,C,P,G,N),
$$
not as a detached recursor result.
\end{theorem}
\begin{proof}
Start with the carrier and branch surface. The carrier of
\autoref{def:authorized-generator-recursor-carrier} displays all rows named in
the conclusion, and
\autoref{thm:authorized-generator-recursor-branch-exhaustion} forces each
branch read to stay on the constructor signature $I$, eliminator $E$, checked
motive $M$, and branch package $B$.

Next add the recursive and audit rows.
\autoref{thm:authorized-generator-recursor-structural-descent-locality}
confines public recursive reads to the displayed descent row $D$, while
\autoref{thm:authorized-generator-recursor-audit-boundary} places public
exposure of $O$ behind the audit row $A$.

It remains to keep the public output attached to the downstream rows.
\autoref{thm:authorized-generator-recursor-audit-provenance-factorization}
retains $H,C,P,G,N$ at the output boundary, and
\autoref{thm:authorized-generator-recursor-output-name-stability} prevents a
stable public output from dropping the signature, motive, branch, descent, or
audit rows. Therefore every public read of $O$ consumes the whole displayed
packet.
\end{proof}

\begin{proof}

recursor row.

TasteGate marker only as the single-carrier alignment witness for the finite
packet, so $G$ does not export a second induction principle or a new proof
generator.

Finally apply the audit refusal surface.
\autoref{thm:authorized-generator-recursor-audit-dependency-refusal} blocks
external axiom channels and hidden dependency routes before $O$ becomes
public. Hence a packet that has reached $G$ has no non-displayed route by
which generator authority, axiom authority, or audit authority can escape the
carrier.
\end{proof}

The second component is descent locality. Apply
\autoref{thm:authorized-generator-recursor-structural-descent-locality} to

The third component is audit provenance. Use

so obligation closure cannot expose $O$.
\end{proof}
