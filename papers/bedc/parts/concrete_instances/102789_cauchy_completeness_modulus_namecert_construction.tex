\begin{theorem}[CauchyCompletenessModulus NameCert obligations]
\label{thm:cauchy-completeness-modulus-namecert-obligations}

\item threshold exposure, requiring each completion request to display its $\CauchyModulusUp$ threshold row before any $\RealUp$ seal is read;
