\begin{theorem}[ContinuationMonad root public consumer totality]
\label{thm:continuation-monad-root-public-consumer-totality}
Every public $\ContinuationMonadUp$ consumer of an accepted packet $Z=(A,B,C,f,g,u,H,K,L,N)$ must pass through the $\CategoryUp$ hom row, the $\GeneratorClosureUp$ source row, the unit row, the bind row, the Kleisli composite row, the componentwise $\hsame$ transport, the $\Cont$ replay, the $\Pkg$ provenance, and the local $\NameCert$ row before it receives the public continuation-monad surface. It exports no host evaluator, quotient Kleisli category, ambient category completion, unlisted continuation generator, or monad law outside those displayed rows.
\end{theorem}
\leanchecked{BEDC.Derived.ContinuationMonadUp.ContinuationMonadCarrier\_root\_public\_consumer\_totality}
\begin{proof}
Begin with the root public obligation triad \autoref{thm:continuation-monad-root-public-obligation-triad}. It fixes the public source histories, continuation tails, and NameCert rows of $Z$ before any downstream consumer reads the monad-facing surface.

Use the root public generator boundary \autoref{thm:continuation-monad-root-public-generator-boundary} and the root category-generator factorization \autoref{thm:continuation-monad-root-category-generator-factorization}. These place the $\CategoryUp$ hom row and $\GeneratorClosureUp$ source row on the same displayed route as the source histories, routes, provenance, and local name.

Apply the root public Kleisli readback \autoref{thm:continuation-monad-root-public-kleisli-readback}. It routes the unit row, bind row, Kleisli composite row, componentwise $\hsame$ transport, $\Cont$ replay, $\Pkg$ provenance, and local $\NameCert$ row through the same finite packet.

The exclusions follow by row exhaustion. Since the public consumer has only the displayed rows of $Z$, host evaluators, quotient Kleisli categories, ambient completions, unlisted generators, and external monad-law objects remain outside the carrier.
\end{proof}

\begin{theorem}[ContinuationMonad root public source lock]
\label{thm:continuation-monad-root-public-source-lock}
Any public $\ContinuationMonadUp$ consumer that preserves the $\CategoryUp$ hom row and the $\GeneratorClosureUp$ source row must preserve the same packet
$$
Z=(A,B,C,f,g,u,H,K,L,N)
$$
before reading the unit, bind, or Kleisli composite rows. Changing either source lock invalidates the public $\ContinuationMonadUp$ consumer route and cannot be repaired by host function composition, quotient Kleisli arrows, or an ambient category completion.
\end{theorem}
\leanchecked{BEDC.Derived.ContinuationMonadUp.ContinuationMonadCarrier\_root\_public\_source\_lock}
\begin{proof}
Begin with root public consumer totality. It places every public consumer of $Z$ behind the Category hom row, GeneratorClosure source row, unit row, bind row, Kleisli row, componentwise $\hsame$ transport, $\Cont$ replay, $\Pkg$ provenance, and local $\NameCert$ row.

Apply the category-generator NameCert source lock. The Category and generator faces are source locks for the same $A,B,C,f,g,u,H,K,L,N$ packet, so preserving both rows also preserves the source histories, continuation tails, transport, route, provenance, and local name before any monad-facing read.

Finally use the root Kleisli formal target package. The formal target has no coordinate for host composition, quotient Kleisli arrows, ambient completion, external evaluation, or an unlisted generator. Therefore changing either source lock leaves the displayed package and is not a public $\ContinuationMonadUp$ consumer route.
\end{proof}

\label{thm:continuation-monad-root-category-generator-namecert-source-lock}

\end{theorem}
\leanchecked{BEDC.Derived.ContinuationMonadUp.ContinuationMonadCarrier\_root\_category\_generator\_namecert\_source\_lock}

\begin{proof}

Row exhaustion gives the negative clauses. There is no displayed coordinate for host composition, quotient Kleisli arrows, ambient completion, external evaluation, or an unlisted generator.
\end{proof}
