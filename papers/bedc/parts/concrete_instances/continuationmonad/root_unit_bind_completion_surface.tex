$$

\end{theorem}

Use the root Kleisli source totality of \autoref{thm:continuation-monad-root-kleisli-source-totality}. That source exposes the histories and continuation tails before any root consumer can read unit or bind data.

Finally apply the root bind-unit boundary of \autoref{thm:continuation-monad-root-bind-unit-boundary}. It forces public unit and bind reads through $u,f,g,H,K,L,N$ over the same histories $A,B,C$, leaving no coordinate for a host or quotient-level operation.
\end{proof}
