\end{proof}

\begin{theorem}[ContinuationMonad root consumer non-escape]
\label{thm:continuation-monad-root-consumer-nonescape}
Every root consumer of an accepted $\ContinuationMonadUp$ packet $Z=(A,B,C,f,g,u,H,K,L,N)$ reads through the same finite generator-category route, the same displayed bind readback, and the same $\Cont$, $\hsame$, $\Pkg$, and $\NameCert$ support rows. \end{theorem}
\begin{proof}
First combine \autoref{thm:continuation-monad-generator-category-route-totality} with \autoref{thm:continuation-monad-displayed-bind-readback-exhaustion}. The first theorem fixes the generator-category route, and the second fixes the bind readback.

Next use the root downstream unblock package and root Kleisli source totality. They identify downstream and Kleisli-facing reads with the carrier rows $A,B,C,f,g,u,H,K,L,N$ and the structural rows already displayed by the packet.

Finally apply the generator boundary readback and CategoryHomCarrier consumer totality surfaces. Since both root consumer faces are already routed through the accepted packet, there is no remaining route to host composition, quotient Kleisli arrows, host equality, ambient category completion, independent hom carriers, or unlisted generators.
\end{proof}
