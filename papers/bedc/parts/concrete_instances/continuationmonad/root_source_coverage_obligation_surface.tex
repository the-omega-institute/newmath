\begin{theorem}[ContinuationMonad root source coverage]
\label{thm:continuation-monad-root-source-coverage}
For every accepted $\ContinuationMonadUp$ packet $Z=(A,B,C,f,g,u,H,K,L,N)$, every root consumer must read the displayed source histories $A,B,C$ and the displayed continuation tails $f,g,u$ before any unit or bind law can be consumed. The source coverage is exactly the finite $\BHist$, $\hsame$, $\Cont$, $\Pkg$, and $\NameCert$ surface of $Z$; it does not pass through host functions, quotient Kleisli arrows, ambient hom sets, or unlisted generators.
\end{theorem}
\leanchecked{BEDC.Derived.ContinuationMonadUp.ContinuationMonadCarrier\_root\_source\_coverage}
\begin{proof}
Start with \autoref{thm:continuation-monad-root-route-source-exposure}. It identifies the source of a root read with $A,B,C,f,g,u,H,K,L,N$ and leaves no coordinate for an external source object.

Next use \autoref{thm:continuation-monad-root-continuation-rule-coverage}. Unit and bind reads are covered only after the displayed histories, tails, route row, transport row, ledger row, and local name have been read.

Finally apply \autoref{thm:continuation-monad-root-downstream-consumer-separation}. This excludes host functions, quotient Kleisli arrows, ambient hom sets, independent category carriers, and unlisted continuation generators from the root source route, so coverage is exactly the displayed packet.
\end{proof}
