\begin{theorem}[RandomVar ProbSpace Distribution sibling route]
\label{thm:random-var-probspace-distribution-sibling-route}
Every accepted $\RandomVarUp$ preimage packet that is later consumed by $\DistributionUp$ first reads the source probability carrier from \autoref{ch:concrete-instances-probspace-namecert} and the target measurable-event surface from \autoref{ch:concrete-instances-measure-namecert}. It then exports only the finite preimage rows, the source and target event classifiers, $\hsame$, $\Cont$, $\Pkg$, and $\NameCert$ rows to the distribution readback surface of \autoref{ch:concrete-instances-distribution-namecert}. No host function equality, independent probability-space bridge, target event outside the displayed $\MeasureUp$ surface, or distribution row outside the carried preimage readback is exported.
\end{theorem}
\begin{proof}
Start with the preimage exactness surface of \autoref{thm:randomvar-preimage-disjoint-binary-union-exactness}, \autoref{thm:randomvar-preimage-relative-difference-exactness}, \autoref{thm:randomvar-preimage-complement-difference-exactness}, and \autoref{thm:randomvar-preimage-empty-event-exactness}. Each row uses the same carried map $X:S\to T$, reads the source carrier through $\ProbSpaceUp$, and reads the target event family through $\MeasureUp$.

Next use the scoped carried preimage package of \autoref{thm:randomvar-scoped-closure-package}. It collects the finite, total, terminal, and countable preimage rows over the same source and target classifiers, so the downstream consumer receives a single carried package rather than a second event lattice.

For a $\DistributionUp$ readback, retain only the rows that the distribution surface can consume: the displayed finite preimages, the classifiers that compare source and target events, and the structural $\hsame$, $\Cont$, $\Pkg$, and $\NameCert$ support. The source probability ledger remains the $\ProbSpaceUp$ input, while the measurable target surface remains the $\MeasureUp$ input.

Thus the sibling route runs from the probability carrier through the carried random-variable preimage packet to the distribution readback surface. Since every exported row has already appeared on that route, no host function equality, independent probability-space bridge, target event outside the displayed measurable surface, or off-route distribution row is available.
\end{proof}
