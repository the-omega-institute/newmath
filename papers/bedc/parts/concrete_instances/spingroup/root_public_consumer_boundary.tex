\begin{definition}[SpinGroup root public consumer boundary packet]
\label{def:spingroup-root-public-consumer-boundary-packet}
The $\SpinGroupUp$ root public consumer boundary packet is the finite
$\mathsf{BHist}$ packet whose visible rows are exactly the Clifford source row
from $\NameCert_{\CliffordUp}$, the singleton $\GroupUp$ row from
$\NameCert_{\GroupUp}$, the unit endpoint, product endpoint, and inverse
endpoint rows, the even-parity $\mathsf{UnaryHistory}$ ledger, the displayed
$\Cont$ transport rows, the displayed $\hsame$ classifier rows, and the
$\Pkg$ provenance rows tying these entries to the root package.

The packet has no field for a host orthogonal group, a Clifford quotient, a
matrix model, or a standard bridge row.
\end{definition}

\begin{theorem}[SpinGroup root public consumer boundary coverage]
\label{thm:spingroup-root-public-consumer-boundary-coverage}
Every public $\SpinGroupUp$ consumer row at the root boundary projects from
\autoref{def:spingroup-root-public-consumer-boundary-packet}. In particular,
the readable rows are the Clifford source row, singleton $\GroupUp$ row, unit,
product, inverse, even-parity ledger, $\Cont$, $\hsame$, and $\Pkg$ rows
displayed by that packet.
\end{theorem}
\leanchecked{BEDC.Derived.SpinGroupUp.SpinGroupRootCarrier\_public\_consumer\_boundary\_coverage}
\begin{proof}
Start with the root threshold package of
\autoref{thm:spingroup-root-namecert-threshold-package}. It confines the root
certificate to the Clifford source, singleton group-law source, transport
rows, classifier rows, unit/product/inverse endpoints, and finite ledger
rows.

Project the consumer read through
\autoref{thm:spingroup-root-ledger-exhaustion}. The ledger inversion leaves
only displayed $\mathsf{BHist}$ rows, $\Cont$ rows, $\hsame$ rows,
$\NameCert_{\CliffordUp}$ source rows, $\NameCert_{\GroupUp}$ source rows,
singleton $\mathsf{UnaryHistory}$ rows, and $\Pkg$ provenance.

Finally repack those projected rows into
\autoref{def:spingroup-root-public-consumer-boundary-packet}. The packet
contains exactly the row kinds left by the threshold package and ledger
exhaustion, so every public root consumer row is covered by it.
\end{proof}

\begin{theorem}[SpinGroup root public consumer boundary exhaustion]
\label{thm:spingroup-root-public-consumer-boundary-exhaustion}
Every row displayed by
\autoref{def:spingroup-root-public-consumer-boundary-packet} repacks as a
public $\SpinGroupUp$ root consumer row, and every public root consumer row is
one of those displayed rows. Thus the root public boundary is exhausted by the
Clifford source row, singleton $\GroupUp$ row, unit/product/inverse endpoints,
even-parity $\mathsf{UnaryHistory}$ ledger, $\Cont$, $\hsame$, and $\Pkg$
provenance.

No host orthogonal group row, Clifford quotient row, matrix-model row, or
standard bridge row is in scope for this public root boundary.
\end{theorem}
\leanchecked{BEDC.Derived.SpinGroupUp.SpinGroupRootCarrier\_public\_consumer\_boundary\_exhaustion}
\begin{proof}
For coverage, use
\autoref{thm:spingroup-root-public-consumer-boundary-coverage}. It projects
each public root consumer row to one coordinate of the boundary packet.

For repacking, read the coordinates of
\autoref{def:spingroup-root-public-consumer-boundary-packet}. The Clifford
source and singleton group-law rows repack through
\autoref{thm:spingroup-root-carrier-source-scope}; the unit endpoint repacks
through \autoref{thm:spingroup-root-clifford-unit-lift}; and the product and
inverse endpoints repack through
\autoref{thm:spingroup-root-group-law-transport}.

The even-parity ledger, $\Cont$, $\hsame$, and $\Pkg$ rows repack through
\autoref{thm:spingroup-root-ledger-exhaustion} and then through the threshold
package of \autoref{thm:spingroup-root-namecert-threshold-package}. The
definition of the boundary packet provides no coordinate for host orthogonal
groups, Clifford quotients, matrix models, or standard bridges, so the
exhausted public surface cannot read any such row.
\end{proof}

\begin{theorem}[SpinGroup root public boundary transport stability]
\label{thm:spingroup-root-public-boundary-transport-stability}
Let $r$ be a public $\SpinGroupUp$ root consumer row displayed by
\autoref{def:spingroup-root-public-consumer-boundary-packet}. If $r$ is
transported by the displayed $\hsame$ and $\Cont$ rows while keeping the same
Clifford source row, singleton $\GroupUp$ row, even-parity
$\mathsf{UnaryHistory}$ ledger, and $\Pkg$ provenance, then the transported
row is again a public $\SpinGroupUp$ root consumer row.

No host orthogonal group, matrix model, Clifford quotient, or standard bridge
row enters this public boundary under the transport.
\end{theorem}
\leanchecked{BEDC.Derived.SpinGroupUp.SpinGroupRootCarrier\_public\_boundary\_transport\_stability}
\begin{proof}
Project the original public row through
\autoref{thm:spingroup-root-public-consumer-boundary-coverage}. The projected
coordinates are exactly the Clifford source row, singleton $\GroupUp$ row,
unit, product, inverse, even-parity ledger, displayed $\Cont/\hsame$ rows,
and $\Pkg$ provenance.

Transport only the displayed classifier and continuation coordinates. The
source coordinates remain fixed by
\autoref{thm:spingroup-root-carrier-source-scope}, and the product and
inverse coordinates remain admitted by
\autoref{thm:spingroup-root-group-law-transport} over the same singleton
group-law row and unary ledger.

Repack the transported coordinates by
\autoref{thm:spingroup-root-public-consumer-boundary-exhaustion}. Since the
boundary packet has no coordinate for host orthogonal groups, matrix models,
Clifford quotients, or standard bridges, the transported row is still a
public root consumer row and no external row has entered the boundary.
\end{proof}

\begin{theorem}[SpinGroup root public consumer NameCert boundary]
\label{thm:spingroup-root-public-consumer-namecert-boundary}
The root public consumer surface for $\SpinGroupUp$ is exactly the finite
$\mathsf{BHist}$ $\NameCert$ boundary consisting of the Clifford source row,
the singleton $\GroupUp$ row, unit, product, and inverse endpoints, the
even-parity $\mathsf{UnaryHistory}$ ledger, the displayed $\Cont$ rows, the
displayed $\hsame$ rows, and the $\Pkg$ provenance rows of
\autoref{def:spingroup-root-public-consumer-boundary-packet}. Coverage,
exhaustion, and transport stability are precisely the three public boundary
fields for this surface.

No host orthogonal group, matrix model, Clifford quotient, or standard
bridge row is exported by this public $\NameCert$ boundary.
\end{theorem}
\leanchecked{BEDC.Derived.SpinGroupUp.SpinGroupRootCarrier\_public\_consumer\_namecert\_boundary}
\begin{proof}
The coverage field is
\autoref{thm:spingroup-root-public-consumer-boundary-coverage}. It sends any
public root consumer read to one of the displayed Clifford, singleton
$\GroupUp$, endpoint, parity-ledger, $\Cont$, $\hsame$, or $\Pkg$ rows.

The exhaustion field is
\autoref{thm:spingroup-root-public-consumer-boundary-exhaustion}. It repacks
each displayed coordinate as a public root consumer row and proves that no
other coordinate is available at the root boundary.

The stability field is
\autoref{thm:spingroup-root-public-boundary-transport-stability}. It shows
that transport along the displayed $\Cont$ and $\hsame$ rows preserves the
same finite public packet while keeping the Clifford source, singleton
$\GroupUp$ source, parity ledger, and $\Pkg$ provenance fixed.

Together these three fields are exactly the public consumer
$\NameCert$ boundary for the finite packet of
\autoref{def:spingroup-root-public-consumer-boundary-packet}. Since each
field is formulated only in terms of that packet, the public boundary exports
no host orthogonal group, matrix model, Clifford quotient, or standard bridge
row.
\end{proof}

\begin{theorem}[SpinGroup root public boundary no extra row]
\label{thm:spingroup-root-public-boundary-no-extra-row}
Every public $\SpinGroupUp$ root consumer row that is stable under the
displayed $\Cont$ and $\hsame$ transports is one of the rows of
\autoref{def:spingroup-root-public-consumer-boundary-packet}. In particular,
no host orthogonal group row, matrix-model row, Clifford quotient row, or
standard bridge row can be recovered by transport from this public boundary.
\end{theorem}
\begin{proof}
Project the stable public consumer row through
\autoref{thm:spingroup-root-public-consumer-boundary-coverage}. The projection
lands in the finite packet containing only the Clifford source, singleton
$\GroupUp$ row, unit, product, inverse, even-parity ledger, $\Cont$, $\hsame$,
and $\Pkg$ coordinates.

Use
\autoref{thm:spingroup-root-public-boundary-transport-stability} for the
transport step. Stability keeps the row inside the same displayed packet and
does not create a coordinate outside the packet.

Finally apply
\autoref{thm:spingroup-root-public-consumer-boundary-exhaustion}. It says that
the packet rows exhaust the public root boundary, so every transported stable
row is one of the displayed coordinates. Because the packet has no coordinate
for host orthogonal groups, matrix models, Clifford quotients, or standard
bridges, none of those rows can be recovered by transport.
\end{proof}
