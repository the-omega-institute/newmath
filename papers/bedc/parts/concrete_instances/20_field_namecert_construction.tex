\begin{theorem}[RatUp FieldUp normalized contextual full noncollapse package]
\label{thm:ratup-fieldup-normalized-contextual-full-noncollapse-package}
For the concrete source tuple $\mathsf{RatUpFieldSource}_F$, the concrete
denominator tuple $(C_D^u,R_D^u,\mu_D^u,e_D^u,N_D^u)$, and the contextual
action $\Gamma$ of
\autoref{def:ratup-fieldup-denominator-contextual-action-instance}, suppose
$C_D^u(h)$, $C_D^u(k)$, $C_D^u(l)$, $C_D^u(r)$, $C_D^u(l')$, and
$C_D^u(r')$. Suppose also that $p\hsame\emp$, $q\hsame\emp$, $p'\hsame\emp$,
$q'\hsame\emp$, $\neg N_D^u(l)$, $\neg N_D^u(r)$, $\neg N_D^u(l')$,
$\neg N_D^u(r')$, and $N_D^u(h)$. Let
$x:=\Gamma_{p',q',l',r'}(\Gamma_{p,q,l,r}(h))$ and let
$y:=\Gamma_{p',q',\mu_D^u(l',l),\mu_D^u(r,r')}(h)$. Then the concrete
$\FieldUp(\mathsf{RatUp})$ exit certificate simultaneously contains:
\begin{enumerate}
  \item the assembled certificate boundary of
  \autoref{thm:ratup-fieldup-assembled-certificate-boundary};
  \item the carried nested endpoint $C_D^u(x)$;
  \item the normalized classifier row $R_D^u(x,y)$;
  \item the strict-support equivalence $N_D^u(x)\Longleftrightarrow N_D^u(h)$;
  \item the exit-row closure recorded by
  \autoref{thm:ratup-fieldup-transported-exit-row-nondegenerate-closure};
  \item the direct endpoint separation $x\hsame\emp\to\bot$;
  \item for every context history $r:\mathsf{BHist}$, both singleton-context
  rejection maps
  $\mathsf{fieldSingletonEmptyCarrier}(\mathsf{append}(r,x))\to\bot$ and
  $\mathsf{fieldSingletonEmptyCarrier}(\mathsf{append}(x,r))\to\bot$;
  \item for all histories $L,R:\mathsf{BHist}$,
  $\mathsf{fieldSingletonEmptyClassifier}(\mathsf{append}(L,x),\mathsf{append}(R,y))
  \to\bot$.
\end{enumerate}
\end{theorem}
\begin{proof}
First apply
\autoref{thm:ratup-fieldup-normalized-contextual-separation-package} to the
same source tuple, denominator tuple, contextual action, carried histories,
empty endpoint hypotheses, empty-support context hypotheses, and strict-support
witness. This supplies the assembled certificate boundary, the carried nested
endpoint, the normalized classifier row, the strict-support equivalence, the
exit-row closure, the direct endpoint separation, and the singleton-context
carrier rejections.

Next apply
\autoref{thm:ratup-fieldup-normalized-contextual-singleton-classifier-separation}
under the same hypotheses and with the same definitions of $x$ and $y$. It
supplies the uniform singleton-classifier obstruction for arbitrary
$L,R:\mathsf{BHist}$.

Collect the seven certificate clauses from the normalized contextual separation
package together with the singleton-classifier obstruction. The resulting row
is one concrete $\mathsf{BHist}$ package for the normalized
$\FieldUp(\mathsf{RatUp})$ exit criterion.
\end{proof}

Per-object template: traditional target normalized contextual full noncollapse
for $\FieldUp(\mathsf{RatUp})$; BEDC source $\mathsf{RatUpFieldSource}_F$,
$\mathsf{BHist}$, $\mathsf{append}$, $\Gamma$, $R_D^u$, and $N_D^u$; carrier
$\mathsf{RatHistoryCarrier}$; classifier $\mathsf{RatHistoryClassifier}$ plus
singleton-empty classifier; stability obligations normalized contextual
separation and singleton-classifier obstruction; ledger contextual-pair
denominator action; core theorem
\autoref{thm:ratup-fieldup-normalized-contextual-full-noncollapse-package};
exactness predicate carrier and classifier noncollapse for the normalized
contextual pair; standard bridge deferred; status $\mathsf{ProofTarget}$.

\begin{theorem}[RatUp FieldUp proof-target exit package]
\label{thm:ratup-fieldup-proof-target-exit-package}
For the concrete source tuple $\mathsf{RatUpFieldSource}_F$, the
$\FieldUp(\RatUp)$ proof target is exhausted by concrete $\mathsf{BHist}$ rows.
Its exit package is the constructive composition of the paper certificate
assembly of \autoref{thm:ratup-fieldup-semantic-namecert}, the source anchoring
of \autoref{thm:ratup-fieldup-certificate-rows-source-anchored}, the concrete
schema instance of \autoref{thm:ratup-fieldup-concrete-schema-instance}, the
concrete exit certificate package of
\autoref{thm:ratup-fieldup-concrete-exit-certificate-package}, the transported
exit-row contextual separation of
\autoref{thm:ratup-fieldup-transported-exit-row-contextual-separation-package},
and the normalized contextual separation of
\autoref{thm:ratup-fieldup-normalized-contextual-separation-package}, together
with the normalized semantic exit certificate of
\autoref{thm:ratup-fieldup-normalized-semantic-exit-certificate} and the
normalized singleton-classifier separation of
\autoref{thm:ratup-fieldup-normalized-contextual-singleton-classifier-separation}.
Thus the displayed $\mathsf{BHist}$ source tuple carries the exit package for
$\FieldUp(\RatUp)$: it contains the normalized semantic nondegenerate witness
$d_1:=\Eone(\emp)$, the normalized classifier row, and the all-context
singleton-classifier rejection for $\mathsf{append}(L,x)$ and
$\mathsf{append}(R,y)$, while no exit row remains as a schematic
$\mathsf{mul}$, $\mathsf{add}$, or $\mathsf{neg}$ operator parameter.
\end{theorem}
\begin{proof}
Start with \autoref{thm:ratup-fieldup-semantic-namecert}. It supplies the
paper-side carrier, source, classifier, stability, ledger, and exactness rows
for the concrete source tuple $\mathsf{RatUpFieldSource}_F$ over
$\mathsf{BHist}$.

Next apply \autoref{thm:ratup-fieldup-certificate-rows-source-anchored}. It
identifies every row consumed by the concrete certificate with the same source
tuple: the rational history carrier and classifier, the denominator tuple, the
contextual action $\Gamma$, the context-pair monoid, and the affine coordinate.

The schema and exit boundary are then fixed by
\autoref{thm:ratup-fieldup-concrete-schema-instance} and
\autoref{thm:ratup-fieldup-concrete-exit-certificate-package}. Together these
theorem sites repack the source-anchored rows as the public
$\FieldUp(\RatUp)$ certificate instance and its concrete exit certificate.

Finally use
\autoref{thm:ratup-fieldup-transported-exit-row-contextual-separation-package}
and \autoref{thm:ratup-fieldup-normalized-contextual-separation-package}. The
first theorem supplies the transported noncollapse and contextual rejection
rows, while the second records the normalized row package in the same concrete
source. Combining these clauses leaves only rows carried by
$\mathsf{RatUpFieldSource}_F$.

Apply \autoref{thm:ratup-fieldup-normalized-semantic-exit-certificate}. It
adds the nondegenerate RatUp witness $d_1:=\Eone(\emp)$, its source anchoring in
$\mathsf{RatHistoryCarrier}$ and $\mathsf{RatHistoryClassifier}$, and the same
normalized contextual rows to the concrete exit certificate.

Apply
\autoref{thm:ratup-fieldup-normalized-contextual-singleton-classifier-separation}
for arbitrary $L,R:\mathsf{BHist}$. The resulting row rejects every
$\mathsf{fieldSingletonEmptyClassifier}(\mathsf{append}(L,x),\mathsf{append}(R,y))$
claim for the normalized pair.

These semantic and classifier-separation rows live in the same concrete
$\mathsf{RatUpFieldSource}_F$ boundary, so the exit package has no schematic
$\mathsf{mul}$, $\mathsf{add}$, or $\mathsf{neg}$ operator coordinate.
\end{proof}

\begin{theorem}[RatUp FieldUp normalized semantic classifier exit certificate]
\label{thm:ratup-fieldup-normalized-semantic-classifier-exit-certificate}
Under the hypotheses and notation of
\autoref{thm:ratup-fieldup-normalized-contextual-separation-package}, the same
concrete $\FieldUp(\mathsf{RatUp})$ certificate contains the normalized
semantic exit package of
\autoref{thm:ratup-fieldup-normalized-semantic-exit-certificate} and, for all
$L,R:\mathsf{BHist}$, the classifier obstruction
$$
\mathsf{fieldSingletonEmptyClassifier}(\mathsf{append}(L,x),\mathsf{append}(R,y))
\to\bot
$$
supplied by
\autoref{thm:ratup-fieldup-normalized-contextual-singleton-classifier-separation}.
Equivalently, the normalized semantic rows for the carried pair $x,y$ and the
singleton-classifier noncollapse rows for every contextual pair $L,R$ occupy
one $\mathsf{BHist}$-anchored $\FieldUp(\mathsf{RatUp})$ exit certificate over
$\mathsf{RatUpFieldSource}_F$.
\end{theorem}
\begin{proof}
First invoke
\autoref{thm:ratup-fieldup-normalized-semantic-exit-certificate} under the
displayed carried endpoint hypotheses, empty endpoint hypotheses,
empty-support context hypotheses, and strict-support witness. It gives the
assembled certificate boundary, the normalized row $R_D^u(x,y)$, the
strict-support equivalence $N_D^u(x)\Longleftrightarrow N_D^u(h)$, the
contextual rejection maps, source anchoring of the concrete rows, and the
nondegenerate witness $d_1:=\Eone(\emp)$ with
$\mathsf{RatHistoryCarrier}(d_1)$ and
$\mathsf{fieldSingletonEmptyCarrier}(d_1)\to\bot$.

Next invoke
\autoref{thm:ratup-fieldup-normalized-contextual-singleton-classifier-separation}
with the same carried endpoints, empty endpoint hypotheses, empty-support
context hypotheses, contextual action $\Gamma$, and strict-support witness.
This yields, for arbitrary $L,R:\mathsf{BHist}$, the obstruction
$$
\mathsf{fieldSingletonEmptyClassifier}(\mathsf{append}(L,x),\mathsf{append}(R,y))
\to\bot.
$$

These two applications share the same concrete source tuple, denominator
tuple, contextual action, normalized endpoints, and support witness. Therefore
the semantic exit rows and the singleton-classifier obstruction are not
separate targets: they are clauses of one normalized concrete
$\FieldUp(\mathsf{RatUp})$ exit certificate.
\end{proof}

Per-object template: traditional target concrete normalized
$\FieldUp(\mathsf{RatUp})$ classifier exit; BEDC source
$\mathsf{RatUpFieldSource}_F$, $\mathsf{BHist}$, $\mathsf{append}$, $\Gamma$,
$R_D^u$, and $N_D^u$; carrier $\mathsf{RatHistoryCarrier}$; classifier
$\mathsf{RatHistoryClassifier}$ plus
$\mathsf{fieldSingletonEmptyClassifier}$ obstruction; stability obligations
normalized semantic exit and singleton-classifier noncollapse; ledger
contextual-pair denominator action and nondegenerate witness; core theorem this
block; exactness predicate semantic exit plus classifier noncollapse for the
normalized pair; standard bridge deferred; status $\mathsf{ProofTarget}$.

\begin{theorem}[RatUp FieldUp right-nested threefold contextual boundary]
\label{thm:ratup-fieldup-right-nested-threefold-contextual-boundary}
For the concrete denominator tuple $(C_D^u,R_D^u,\mu_D^u,e_D^u,N_D^u)$ and
histories $h,k,\ell:\mathsf{BHist}$ with $C_D^u(h)$, $C_D^u(k)$, and
$C_D^u(\ell)$, let $y:=\mu_D^u(h,\mu_D^u(k,\ell))$. If
$\mathsf{ThreeSupp}_D^u(h,k,\ell)$ holds, then the right-nested
$\FieldUp(\mathsf{RatUp})$ denominator endpoint carries the following
contextual boundary:
\begin{enumerate}
  \item for every context history $r:\mathsf{BHist}$,
  $\mathsf{fieldSingletonEmptyCarrier}(\mathsf{append}(r,y))\to\bot$;
  \item for every context history $r:\mathsf{BHist}$,
  $\mathsf{fieldSingletonEmptyCarrier}(\mathsf{append}(y,r))\to\bot$;
  \item for every context history $r:\mathsf{BHist}$,
  $\mathsf{fieldSingletonEmptyNonZero}(\mathsf{append}(r,y))$;
  \item for every context history $r:\mathsf{BHist}$,
  $\mathsf{fieldSingletonEmptyNonZero}(\mathsf{append}(y,r))$;
  \item for all $L,R:\mathsf{BHist}$,
  $$
  \mathsf{fieldSingletonEmptyClassifier}(\mathsf{append}(L,y),\mathsf{append}(R,y))
  \to\bot.
  $$
\end{enumerate}
\end{theorem}
\begin{proof}
Apply
\autoref{thm:ratup-fieldup-right-nested-threefold-support-non-singleton-exactness}
to the same concrete denominator tuple, carried histories, and support witness.
The right-nested endpoint data recovers $\mathsf{RatHistoryCarrier}(y)$ and the
singleton rejection map $\mathsf{fieldSingletonEmptyCarrier}(y)\to\bot$.

Fix a context history $r:\mathsf{BHist}$. Apply
\autoref{thm:field-singleton-empty-append-rat-carrier-absurd} with context $r$
and Rat carrier endpoint $y$ to reject
$\mathsf{fieldSingletonEmptyCarrier}(\mathsf{append}(r,y))$. Apply
\autoref{thm:field-singleton-empty-append-left-rat-carrier-absurd} with the same
two histories in the opposite append order to reject
$\mathsf{fieldSingletonEmptyCarrier}(\mathsf{append}(y,r))$.

For the right-append inverse-domain witness, unfold
$\mathsf{fieldSingletonEmptyNonZero}(\mathsf{append}(r,y))$ as rejection of the
singleton classifier against $\emp$. Any such classifier witness contains the
source-carrier component
$\mathsf{fieldSingletonEmptyCarrier}(\mathsf{append}(r,y))$, contradicting the
right-append rejection.

For the left-append inverse-domain witness, unfold
$\mathsf{fieldSingletonEmptyNonZero}(\mathsf{append}(y,r))$ as rejection of the
singleton classifier against $\emp$. Its source-carrier component is
$\mathsf{fieldSingletonEmptyCarrier}(\mathsf{append}(y,r))$, contradicting the
left-append rejection.

Finally package $\mathsf{RatHistoryClassifier}(y,y)$ from the recovered Rat
carrier witness on both sides and reflexivity of $\hsame$. For arbitrary
$L,R:\mathsf{BHist}$, apply
\autoref{thm:field-singleton-empty-classifier-append-rat-classifier-absurd}.
It rejects
$\mathsf{fieldSingletonEmptyClassifier}(\mathsf{append}(L,y),\mathsf{append}(R,y))$.
\end{proof}

Per-object template: traditional target bracketing-independent concrete
$\FieldUp(\mathsf{RatUp})$ denominator exit support; BEDC source
$\mathsf{BHist}$ with the concrete denominator tuple
$(C_D^u,R_D^u,\mu_D^u,e_D^u,N_D^u)$ and $\mathsf{RatUpFieldSource}_F$; carrier
$\mathsf{ThreeSupp}_D^u$; classifier $R_D^u$; stability obligations
right-nested support exactness plus contextual append separation; ledger the
right-nested endpoint $y$; core theorem
\autoref{thm:ratup-fieldup-right-nested-threefold-contextual-boundary};
exactness predicate contextual singleton and inverse-domain exclusion; standard
bridge deferred; status $\mathsf{ProofTarget}$.
