\begin{theorem}[RatUp FieldUp four endpoint product nonempty]
\label{thm:ratup-fieldup-four-endpoint-product-nonempty}
For a BEDC history-level $\FieldUp(\RatUp)$ candidate satisfying the hypotheses
of \autoref{thm:ratup-fieldup-rat-classified-factors-exclude-empty-product},
assume the same field-operation data, the same identification of the field-zero
endpoint with $\emp$, the same transport for
$\mathsf{ApartZero}_F(h):=(h\hsame0\to\bot)$, and the same contradiction of
$\mathsf{ApartZero}_F(0)$. Then for all histories
$a,a',b,b':\mathsf{BHist}$,
if
$$
\mathsf{RatHistoryClassifier}(a,a')
\land
\mathsf{RatHistoryClassifier}(b,b'),
$$
then the same concrete $\FieldUp(\RatUp)$ operation package contains all four
nonempty product rows:
\begin{enumerate}
  \item $\mathsf{mul}(a,b)\hsame\emp\to\bot$;
  \item $\mathsf{mul}(a,b')\hsame\emp\to\bot$;
  \item $\mathsf{mul}(a',b)\hsame\emp\to\bot$;
  \item $\mathsf{mul}(a',b')\hsame\emp\to\bot$.
\end{enumerate}
\end{theorem}
\begin{proof}
Apply
\autoref{thm:ratup-fieldup-rat-classified-factors-exclude-empty-product} to the
displayed classifier pair. This gives the two diagonal exclusions for
$\mathsf{mul}(a,b)$ and $\mathsf{mul}(a',b')$, transported through the fixed
identification of the field-zero endpoint with $\emp$.

Next build the left mixed row. The first classifier supplies the endpoint
carrier fields for $a$ and $a'$, while the second classifier supplies the
carrier field for $b$. Reflexivity of $\hsame$ at $b$ forms the reflexive Rat
classifier on the second endpoint pair $(b,b)$. Applying the same two-diagonal
product theorem to $\mathsf{RatHistoryClassifier}(a,a')$ and this reflexive
classifier gives the product exclusion for $\mathsf{mul}(a',b)$.

Now build the right mixed row. Reflexivity of $\hsame$ at $a$, together with
the endpoint carrier field supplied by the first classifier, forms the reflexive
Rat classifier on $(a,a)$. Applying the same two-diagonal product theorem to
this reflexive classifier and $\mathsf{RatHistoryClassifier}(b,b')$ gives the
product exclusion for $\mathsf{mul}(a,b')$.

Conjunction introduction packages the four exclusions in the displayed order.
All four applications use the same field-operation hypotheses, the same
zero-empty identification, the same apart-zero transport, and the same
$\mathsf{BHist}$ carrier and classifier data.
\end{proof}
