\begin{theorem}[Cauchy completion apartness NameCert obligations]
\label{thm:cauchy-completion-apartness-namecert-obligations}

$$

\item completion-source exposure, requiring an apartness request to pass through the displayed $\CauchyCompletionUniversalPropertyUp$ row $U$ before any completed comparison is read;

Finally $H,C,P,N$ transport, replay, source, and name the same carrier. Since the carrier has no coordinate for quotient representatives, host equality, trichotomy, choice, ambient completeness, Lean verification, or bridge checking, the listed rows exhaust the local NameCert surface.
