\begin{theorem}[CriticalLineWitness Zeta sibling route]
\label{thm:critical-line-witness-zeta-sibling-route}
Every $\CriticalLineWitnessUp$ source-modulus read factors through the displayed $\ZetaUp$ and zero-strip sibling rows before local $\CriticalLineWitnessUp$ naming is consumed. In particular, the source route must expose the zeta source surface of \autoref{ch:concrete-instances-zeta-basic-namecert}, the continuation handoff of \autoref{ch:concrete-instances-zeta-continuation}, the zero-facing witness boundary of \autoref{ch:concrete-instances-zeta-continuation-witness-namecert}, and the fixed-strip rows already displayed by the local witness packet. The route exports no $\mathsf{RH}$ proof, global zero classifier, analytic-continuation primitive, host equality, quotient representative, or ambient completeness theorem.
\end{theorem}
\begin{proof}
Project an accepted $\CriticalLineWitnessUp$ packet as $W=(Z,S,M,R,Q,H,C,P,N)$. The source row $Z$ is the only zeta-facing source coordinate, and the strip row $S$ is the only local zero-strip coordinate available before a modulus read reaches $M$.

The sibling zeta rows are read before the local name is consumed. The basic zeta source, continuation handoff, and zero-facing witness boundary supply the upstream row data, while the local packet records only the displayed $Z,S,M,R,Q$ route and its structural support.

The local naming row $N$ is terminal for this route. Transport through $H$, replay through $C$, and provenance through $P$ preserve the same source-modulus path, so no consumer can replace it by an $\mathsf{RH}$ proof, global zero classifier, analytic primitive, host equality, quotient representative, or ambient completeness datum.
\end{proof}
