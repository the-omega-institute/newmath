\begin{theorem}[CriticalLineWitness root source obligation triad]
\label{thm:critical-line-witness-root-source-obligation-triad}
Let $W=(Z,S,M,R,Q,H,C,P,N)$ be an accepted $\CriticalLineWitnessUp$ packet. Every root consumer reads the supplied $\BHist$ source through the zero-strip row $Z$ or the critical-strip row $S$, then through the modulus row $M$, the $\hsame$ transport row $H$, the $\Cont$ replay row $C$, the $\Pkg$ provenance row $P$, and the local $\NameCert_{\CriticalLineWitnessUp}$ row $N$. The consumer receives no root-facing source outside this finite triad and no $\mathsf{RH}$-facing conclusion before those rows have been displayed.
\end{theorem}
\leanchecked{BEDC.Derived.CriticalLineWitnessUp.CriticalLineWitnessCarrier\_root\_source\_obligation\_triad}
\begin{proof}
Project the accepted packet through \autoref{def:critical-line-witness-carrier}. The only source-facing coordinates are the zero row $Z$ and the critical-strip row $S$, and the only modulus-facing coordinate is $M$.

Apply the root source triad of \autoref{thm:critical-line-witness-root-source-triad}. It binds the source side to the displayed packet route before the comparison and refusal rows can be consumed.

Use the local transport obligation \autoref{thm:critical-line-witness-hsame-cont-ledger-obligation}. It carries the same source and modulus rows through $H,C,P,N$, so the root consumer must traverse the finite $\BHist$, $\hsame$, $\Cont$, $\Pkg$, and $\NameCert$ surface.

The $\mathsf{RH}$ boundary is then only a consumer boundary. Since no coordinate of $W$ names an RH proof, analytic-continuation theorem, quotient representative, host equality, or standard-model replacement, the displayed triad exhausts the root source obligation.
\end{proof}

\begin{theorem}[CriticalLineWitness root modulus classifier exhaustion]
\label{thm:critical-line-witness-root-modulus-classifier-exhaustion}
For every accepted $\CriticalLineWitnessUp$ packet $W=(Z,S,M,R,Q,H,C,P,N)$, the root modulus classifier is exhausted by the modulus row $M$, the real-part row $R$, the rational comparison ledger $Q$, the $\hsame$ transport row $H$, the $\Cont$ replay row $C$, the $\Pkg$ provenance row $P$, and the local $\NameCert$ row $N$, with $Z$ and $S$ retained as the zero-strip or critical-strip source rows. No modulus consumer can inspect an unlisted classifier, hidden numeric oracle, quotient equality, or analytic source theorem.
\end{theorem}
\begin{proof}
Start from the root comparison ledger \autoref{thm:critical-line-witness-root-comparison-ledger}. It routes comparison reads through the rational ledger $Q$ while keeping the source rows $Z,S$ visible.

Next apply the root modulus ledger \autoref{thm:critical-line-witness-root-modulus-ledger}. It forces depth and modulus data to pass through $M$ and the same structural rows rather than through a detached numeric oracle.

Transport and replay are fixed by \autoref{thm:critical-line-witness-hsame-cont-ledger-obligation}. The componentwise $\hsame$ row, the displayed $\Cont$ route, the provenance row, and the local name preserve the classifier route as one finite packet.

Thus the root modulus classifier has no remaining branch. Every accepted read is one of the displayed $M,R,Q,H,C,P,N$ reads over the retained $Z,S$ source, and every unlisted analytic or host-level classifier is outside the packet.
\end{proof}

\begin{theorem}[CriticalLineWitness root RH consumer boundary]
\label{thm:critical-line-witness-root-rh-consumer-boundary}
Every $\mathsf{RH}$-facing consumer of an accepted $\CriticalLineWitnessUp$ packet $W=(Z,S,M,R,Q,H,C,P,N)$ is bounded by the source obligation triad of \autoref{thm:critical-line-witness-root-source-obligation-triad} and the modulus classifier exhaustion of \autoref{thm:critical-line-witness-root-modulus-classifier-exhaustion}. The consumer may cite the supplied $\BHist$ source, zero-strip or critical-strip row, modulus row, $\hsame$ transport, $\Cont$ replay, $\Pkg$ provenance, and $\NameCert$ row, but receives no RH proof, zero search, global zero classifier, analytic-continuation theorem, quotient stream, host equality, or ambient completeness datum.
\end{theorem}
\begin{proof}
First use \autoref{thm:critical-line-witness-root-source-obligation-triad}. It places every root source read behind $Z,S,M,H,C,P,N$ before any RH-facing consumer can inspect the packet.

Then use \autoref{thm:critical-line-witness-root-modulus-classifier-exhaustion}. It exhausts modulus and comparison reads by $M,R,Q,H,C,P,N$ while retaining the same zero-strip and critical-strip source rows.

Finally apply the RH-refusal ledger completeness theorem \autoref{thm:critical-line-witness-rh-refusal-ledger-completeness} and the RH-boundary refusal totality row \autoref{thm:critical-line-witness-rh-boundary-refusal-totality}. These rows classify requests for proof content, zero search, analytic continuation, quotient streams, host equality, and ambient completeness as outside the accepted packet.

The RH-facing consumer boundary is therefore a finite BEDC readback boundary. It exports the source, modulus, transport, replay, provenance, and local naming rows, not an analytic theorem and not a truth value for $\mathsf{RH}$.
\end{proof}
