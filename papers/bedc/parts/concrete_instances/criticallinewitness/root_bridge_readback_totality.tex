\begin{theorem}[CriticalLineWitness root bridge readback totality]
\label{thm:critical-line-witness-root-bridge-readback-totality}
Let $W=(Z,S,M,R,Q,H,C,P,N)$ be an accepted $\CriticalLineWitnessUp$ packet.
Every root bridge readback of $W$ is exhausted by the displayed finite route
$$
Z,S,M,R,Q,H,C,P,N.
$$
The readback first consumes the root-source rows $Z,S,Q$, then the
comparison-ledger rows $M,R,Q$, and finally the fixed-tuple transport, replay,
provenance, and local naming rows $H,C,P,N$. It exports no $\mathsf{RH}$ proof,
host zero classifier, analytic-continuation theorem, quotient representative,
host equality, or ambient completeness coordinate.
\end{theorem}
\begin{proof}
Begin with the root source triad
\autoref{thm:critical-line-witness-root-source-triad}. It places the supplied
zero row, critical-strip row, and source-facing comparison row inside the same
finite packet before bridge readback begins.

Next use the root comparison ledger
\autoref{thm:critical-line-witness-root-comparison-ledger}. The modulus,
real-part, and rational-comparison reads are forced through $M,R,Q,H,C,P,N$
rather than through an analytic host object.

Finally apply the fixed tuple image and readback rows,
\autoref{thm:critical-line-witness-fixed-tuple-image-row} and
\autoref{thm:critical-line-witness-fixed-tuple-readback-row}. These rows return
each bridge-visible coordinate to the corresponding carrier coordinate, while
the root refusal route removes the forbidden analytic and host-level exits.
\end{proof}

\begin{theorem}[CriticalLineWitness root NameCert consumer totality]
\label{thm:critical-line-witness-root-namecert-consumer-totality}
For an accepted $\CriticalLineWitnessUp$ packet
$W=(Z,S,M,R,Q,H,C,P,N)$, every root-facing $\NameCert$ consumer is total on the
same finite packet. The consumer may read carrier admission, source exposure,
comparison-ledger exactness, componentwise $\hsame$ transport, displayed
$\Cont$ replay, $\Pkg$ provenance, and local naming; it has no route to a
second source packet, external comparison ledger, analytic oracle, quotient
representative, host equality, or ambient completeness witness.
\end{theorem}
\begin{proof}
Project the carrier with \autoref{def:critical-line-witness-carrier}. The local
packet exposes exactly $Z,S,M,R,Q,H,C,P,N$, so a root-facing $\NameCert$
consumer has a finite coordinate for every accepted read.

Apply \autoref{thm:critical-line-witness-namecert-obligations}. Carrier
admission, source compatibility, modulus discipline, ledger exactness, and
non-escape are all obligations of the same local naming row $N$.

The root-source triad and root comparison ledger then account for the source
and comparison sides. The fixed-tuple premise surface repacks those rows as the
bridge-readable packet, so the consumer is total exactly on the displayed route
and nowhere else.
\end{proof}

\begin{theorem}[CriticalLineWitness root analytic-refusal exhaustion]
\label{thm:critical-line-witness-root-analytic-refusal-exhaustion}
Let $W=(Z,S,M,R,Q,H,C,P,N)$ be an accepted $\CriticalLineWitnessUp$ packet. Any
root-facing request for an $\mathsf{RH}$ proof, zero search, global zero-set
classifier, analytic-continuation certificate, quotient representative, host
equality, or ambient completeness row is exhausted by the root refusal route of
$W$. The request is recorded as a refusal over $Z,S,M,R,Q,H,C,P,N$ and is not a
bridge readback, NameCert consumer, or analytic witness outside the packet.
\end{theorem}
\begin{proof}
Start from \autoref{thm:critical-line-witness-root-refusal-route}. It sends
off-surface root demands to the displayed finite refusal boundary instead of to
host analytic data.

Use the root-source triad and root comparison ledger to locate the positive
rows that remain readable after refusal. They identify the supplied source and
comparison data with $Z,S,M,R,Q,H,C,P,N$.

Finally use the fixed-tuple premise surface. Since every bridge-facing readback
must return to the fixed tuple and every local $\NameCert$ consumer reads the
same tuple, the analytic demand is fully exhausted by refusal rather than by a
hidden analytic coordinate.
\end{proof}
