\begin{theorem}[CriticalLineWitness root StdBridge refusal source]
\label{thm:critical-line-witness-root-stdbridge-refusal-source}
Let $W=(Z,S,M,R,Q,H,C,P,N)$ be an accepted $\CriticalLineWitnessUp$ packet rooted in $\BHist$. Before any $\StdBridge$-facing consumer reads $W$ as a critical-line witness, the displayed zero-strip and refusal source is fixed by the supplied $\ZetaZerosUp$ row $Z$, critical-strip row $S$, depth row $M$, real-part row $R$, rational-comparison row $Q$, componentwise $\hsame$ row $H$, displayed $\Cont$ row $C$, $\Pkg$ provenance row $P$, and local $\NameCert_{\CriticalLineWitnessUp}$ row $N$.

The fixed source is a finite refusal surface. It is not an analytic continuation theorem, host zero classifier, quotient-stream representative, ambient completeness principle, or proof of $\mathsf{RH}$.
\end{theorem}
\leanchecked{BEDC.Derived.CriticalLineWitnessUp.CriticalLineWitnessCarrier\_root\_stdbridge\_refusal\_source}
\begin{proof}
Project the accepted carrier to the displayed tuple $Z,S,M,R,Q,H,C,P,N$. These are the only $\BHist$, transport, continuation, provenance, and local naming rows available before a bridge-facing consumer can read the witness.

Use \autoref{thm:critical-line-witness-root-finite-strip-route} for the zero and strip side. It forces every root zero read and critical-strip read through the finite route before any downstream $\mathsf{RH}$-facing consumer sees the packet.

Apply \autoref{thm:critical-line-witness-root-refusal-ledger-scope} to the off-route demands. The refusal ledger records the absent analytic and host-level objects inside the same finite tuple, so the $\StdBridge$-facing source is fixed as the displayed refusal surface.
\end{proof}

\begin{theorem}[CriticalLineWitness root zero-strip readback]
\label{thm:critical-line-witness-root-zero-strip-readback}
For an accepted $\CriticalLineWitnessUp$ packet $W=(Z,S,M,R,Q,H,C,P,N)$ rooted in $\BHist$, every root zero-strip readback is exhausted by the supplied zero row $Z$, strip row $S$, modulus or depth row $M$, real-part row $R$, rational-comparison row $Q$, and the local $H,C,P,N$ structural rows. No additional analytic continuation row, global zero classifier, host equality, quotient representative, countable selector, or ambient completeness datum can extend the readback.
\end{theorem}
\begin{proof}
Read the source side through \autoref{thm:criticalline-witness-root-strip-source-totality}. The theorem lists $Z,S,M,R,Q,H,C,P,N$ as the full root source route for zero-strip consumers.

Next use \autoref{thm:critical-line-witness-root-modulus-read-totality}. It keeps modulus, depth, real-part, and rational-comparison reads inside the displayed numeric rows and the same structural route.

The remaining possible extensions are exactly the objects named in the refusal boundary. By \autoref{thm:criticalline-witness-root-refusal-ledger-exactness}, analytic continuation export, global zero classification, host equality, quotient representatives, countable selector data, and ambient completeness stay absent as exported readback coordinates.
\end{proof}

\begin{theorem}[CriticalLineWitness root RH refusal exhaustion]
\label{thm:critical-line-witness-root-rh-refusal-exhaustion}
Every $\mathsf{RH}$-facing consumer of an accepted $\CriticalLineWitnessUp$ packet receives only the refused finite witness surface over $Z,S,M,R,Q,H,C,P,N$. The consumer may inspect the supplied zero-strip source, modulus and comparison rows, real-part row, componentwise transport, continuation, provenance, and local $\NameCert$ rows, but receives no analytic continuation theorem, host critical-line fact, global zero classifier, quotient-stream equality, ambient completeness principle, or proof or refutation of $\mathsf{RH}$.
\end{theorem}
\begin{proof}
Begin with \autoref{thm:critical-line-witness-rh-refusal-readback-totality}. It states that every $\mathsf{RH}$-refusal readback factors through the displayed zero-strip, modulus, source, transport, route, provenance, and naming rows.

Then apply \autoref{thm:critical-line-witness-downstream-counter-surface}. Downstream consumers receive the finite refusal ledger over the same packet, not proof content or host zero classification.

Finally use \autoref{thm:critical-line-witness-root-stdbridge-refusal-source}. The bridge-facing source has already been fixed as a finite refusal surface, so the $\mathsf{RH}$ consumer cannot strengthen it into analytic continuation, a host critical-line fact, a quotient-stream equality, ambient completeness, or an $\mathsf{RH}$ theorem.
\end{proof}
