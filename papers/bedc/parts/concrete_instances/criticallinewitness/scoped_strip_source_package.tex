\begin{theorem}[Critical-line witness scoped strip source]
\label{thm:critical-line-witness-scoped-strip-source}
Let $W=(Z,S,M,R,Q,H,C,P,N)$ be an accepted $\CriticalLineWitnessUp$ packet. The
scoped strip-source route binds the supplied $\ZetaZerosUp$ row $Z$, the
$\CritStripUp$ row $S$, the $\NatUp$ depth requests for $M$, the $\RatUp$
comparison ledger $Q$, the $\RealUp$ real-part row $R$, and the local
$\BHist$, $\hsame$, $\Cont$, $\Pkg$, and $\NameCert$ rows into one finite source
surface. The route asserts only that the supplied zero-in-strip input is the
source of the displayed witness packet; it does not assert $\mathsf{RH}$,
analytic continuation, or a completed zero classification.
\end{theorem}
\begin{proof}
Begin with the supplied zero-strip transport lock. It keeps the $\ZetaZerosUp$
row $Z$ and the $\CritStripUp$ row $S$ attached to the same displayed history
through $H$ and $C$.

Next attach the depth and comparison route. The modulus comparison route and
root comparison ledger force each $\NatUp$ request for $M$ to read the $\RealUp$
row $R$ and the $\RatUp$ ledger $Q$ inside the same finite packet.

Finally apply the downstream refusal ledger. The package row $P$ and local
certificate $N$ name the source surface, while the refusal rows exclude
$\mathsf{RH}$ proof data, analytic continuation, host equality, quotient
representatives, and global zero classification.
\end{proof}

\begin{theorem}[Critical-line witness rational height ledger]
\label{thm:critical-line-witness-rational-height-ledger}
For an accepted $\CriticalLineWitnessUp$ packet
$W=(Z,S,M,R,Q,H,C,P,N)$, every displayed height read of the witness factors
through a finite natural depth request and the rational comparison ledger. The
read uses only the $\NatUp$ request for $M$, the $\RatUp$ rows inside $Q$, the
$\RealUp$ real-part row $R$, and the displayed $\BHist$, $\hsame$, $\Cont$,
$\Pkg$, and $\NameCert$ rows; it does not use a host real equality, quotient
representative, hidden numeric oracle, analytic-continuation certificate, or
completed zero-set classifier.
\end{theorem}
\begin{proof}
Project the modulus and ledger coordinates from the packet. A height request is
first a finite $\NatUp$ depth request to $M$, not an infinite analytic search.

Read the answer through the rational comparison ledger. The row $Q$ contains the
half-line, dyadic tolerance, and strict comparison rows used by the modulus
route, and the real-part row $R$ is the only $\RealUp$ source visible to that
comparison.

Use transport replay and ledger exactness to close the route. The rows $H,C,P,N$
carry the natural request and rational ledger inside the same accepted packet,
so no host equality, quotient representative, hidden numeric oracle, analytic
continuation, or global zero classification enters the height read.
\end{proof}

\begin{theorem}[Critical-line witness NameCert scope package]
\label{thm:critical-line-witness-namecert-scope-package}
The scoped strip-source theorem
\autoref{thm:critical-line-witness-scoped-strip-source} and the rational-height
ledger theorem \autoref{thm:critical-line-witness-rational-height-ledger}
assemble the local $\NameCert_{\CriticalLineWitnessUp}$ scope for an accepted
packet $W=(Z,S,M,R,Q,H,C,P,N)$. The scope consists exactly of the supplied
$\ZetaZerosUp$ and $\CritStripUp$ rows, the $\NatUp$ depth route, the $\RatUp$
comparison ledger, the $\RealUp$ real-part row, and the displayed $\BHist$,
$\hsame$, $\Cont$, $\Pkg$, and $\NameCert$ evidence.
\end{theorem}
\begin{proof}
First use scoped strip source. It binds $Z$ and $S$ as supplied source rows and
keeps the zero-strip route inside the displayed transport and continuation
surface.

Next use the rational height ledger. It identifies each witness-height read with
the finite natural request, the real-part row, and the rational ledger $Q$.

Finally repack the two rows into the local naming surface. The packet's
$\BHist$, $\hsame$, $\Cont$, $\Pkg$, and $\NameCert$ coordinates record exactly
those source and ledger routes, while the existing refusal boundary excludes
$\mathsf{RH}$, analytic continuation, host equality, quotient representatives,
and completed zero classification.
\end{proof}
