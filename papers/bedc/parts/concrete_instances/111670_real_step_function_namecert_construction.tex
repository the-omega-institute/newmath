\begin{definition}[Real step function carrier]
\label{def:real-step-function-carrier}
A $\RealStepFunctionUp$ carrier is a finite $\BHist$ packet
$$
E=(I,T,V,S,R,G,H,C,P,N)
$$
with the following displayed rows:
\begin{enumerate}
\item $I$ is the $\RealUp$ interval-domain row, carrying the located real endpoints and interval-side admission data on which the function is read.
\item $T$ is the $\TaggedRealPartitionUp$ cell row, listing finitely many ordered tagged real subintervals, endpoint incidence, tag containment, and coverage over the row $I$.
\item $V$ is the $\StepFunctionUp$ value row, assigning one displayed value row to each tagged cell of $T$ and exposing constant-on-cell admission only through that finite cell list.
\item $S$ is the $\RiemannSumUp$ readback row computed from the tagged cells, cell widths, tags, and displayed values.
\item $R$ is the $\RiemannIntegrableUp$ handoff row, exposing the Darboux/Riemann consumer route that may use $I,T,V,S$ as finite integrability evidence.
\item $G$ is the $\RegulatedFunctionUp$ comparison row, recording that the same finite cell-value surface can be read as a regulated-function approximation boundary when a downstream chapter asks for that interface.
\item $H$ records componentwise $\hsame$ transport for the interval-domain row, tagged partition, step-value row, Riemann-sum readback, integrability handoff, regulated comparison, continuation replay, provenance, and local naming.
\item $C$ records displayed $\Cont$ replay from $I$ to $T$, from $T,V$ to $S$, from $I,T,V,S$ to $R$, and from the same finite rows to $G$.
\item $P$ is the $\Pkg$ provenance over $\RealStepFunctionUp$, $\RealUp$, $\TaggedRealPartitionUp$, $\StepFunctionUp$, $\RiemannSumUp$, $\RiemannIntegrableUp$, $\RegulatedFunctionUp$, $\BHist$, $\hsame$, $\Cont$, and $\NameCert$.
\item $N$ is the local $\NameCert_{\RealStepFunctionUp}$ row.
\end{enumerate}
The classifier compares accepted packets only through these rows. It has no coordinate for a host-level function, quotient equality of integrands, a selected refining partition sequence, a measure-space completion, open-cover compactness, or ambient $\RealUp$ completeness.
\end{definition}

\begin{theorem}[Real step function NameCert obligations]
\label{thm:real-step-function-namecert-obligations}
For every accepted $\RealStepFunctionUp$ carrier $E=(I,T,V,S,R,G,H,C,P,N)$, the local $\NameCert_{\RealStepFunctionUp}$ surface consists exactly of real interval-domain admission, tagged real partition exposure, finite step-value exposure, Riemann-sum readback, Riemann-integrable handoff, regulated-function comparison, componentwise transport, displayed replay, provenance, local naming, and non-escape from host functions, quotient integrand equality, selected refinement limits, measure completion, open-cover compactness, and ambient real completeness.
\end{theorem}
\begin{proof}
Project the carrier of \autoref{def:real-step-function-carrier}. The interval-domain row $I$, tagged partition row $T$, and value row $V$ are the only rows from which a consumer can read the domain cells and their constant values.

The integration-facing rows are downstream of that finite data. The row $S$ is computed from the tagged cells, widths, tags, and displayed values, while $R$ admits a $\RiemannIntegrableUp$ consumer only after $I,T,V,S$ have been displayed.

The regulated-function comparison uses the same rows. The row $G$ does not introduce another function object; it records only the regulated approximation boundary supplied by the finite cell-value surface.

Finally $H,C,P,N$ preserve, replay, source, and name exactly this packet. Since the excluded host objects are not carrier coordinates, the listed obligations exhaust the local NameCert surface.
\end{proof}
