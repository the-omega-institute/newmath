\begin{definition}[Universal closed substrate carrier]
\label{def:universal-closed-substrate-carrier}
A $\UniversalClosedSubstrateUp$ carrier is a finite $\BHist$ packet
$$
U=(R,A,E,S,T,B,H,C,P,N)
$$
with the following displayed rows:
\begin{enumerate}
\item $R$ is the finite rule-table row for the closed substrate, naming the local state alphabet, neighborhood window, update rule, and boundary convention.
\item $A$ is the acceptance row saying that observations are finite substrate histories generated only by repeated use of $R$.
\item $E$ is the universality encoding row, recording the finite compiler surface from a program row and input row into an initial substrate history.
\item $S$ is the simulator-step row, relating one displayed program transition to a bounded block of rule-table steps.
\item $T$ is the finite simulator trace row, listing the bounded blocks served by $S$ for the accepted program prefix.
\item $B$ is the boundary-ledger row, recording where the packet stops: no unbounded halting decision, no oracle call, and no host semantic equality is exported.
\item $H$ records componentwise $\hsame$ transports for the rule table, closed-observation row, encoding row, simulator-step row, trace row, boundary row, package row, and local naming row.
\item $C$ records displayed $\Cont$ routes that replay the rule-table evolution, the encoding read, each bounded simulator block, and the boundary-ledger read.
\item $P$ is the $\Pkg$ provenance over $\UniversalClosedSubstrateUp$, \autoref{ch:visions-closure-universality-quadrant}, \autoref{ch:concrete-instances-cellularautomaton-namecert}, $\BHist$, $\hsame$, $\Cont$, $\Pkg$, and $\NameCert$ rows.
\item $N$ is the local $\NameCert_{\UniversalClosedSubstrateUp}$ row.
\end{enumerate}
The local classifier compares accepted packets only through $R,A,E,S,T,B,H,C,P,N$. It contains no coordinate for replacing the finite rule table by a semantic oracle, no coordinate for deciding all program behaviour, and no quotient identifying the universal row with the closure row.
\end{definition}

\begin{theorem}[Universal closed substrate NameCert obligations]
\label{thm:universal-closed-substrate-namecert-obligations}
For the carrier of \autoref{def:universal-closed-substrate-carrier}, the local $\NameCert_{\UniversalClosedSubstrateUp}$ surface consists exactly of these finite obligation rows:
\begin{enumerate}
\item carrier admission for $U=(R,A,E,S,T,B,H,C,P,N)$;
\item closed-substrate accountability through the finite rule-table row $R$ and the closed-observation row $A$;
\item universality accountability through the encoding row $E$ and the bounded simulator-step row $S$;
\item trace exposure through $T$, with every simulator block backed by the displayed rule table;
\item boundary discipline through $B$, excluding halting decision, oracle access, hidden host equality, and unbounded semantic closure;
\item componentwise transport, replay, provenance, and local naming through $H,C,P,N$.
\end{enumerate}
Thus a consumer may cite $\UniversalClosedSubstrateUp$ only when the finite rule table, closure row, universality encoding, bounded simulator trace, and boundary ledger appear in the same packet.
\end{theorem}
\begin{proof}
Project the carrier of \autoref{def:universal-closed-substrate-carrier}. Its coordinates are exactly $R,A,E,S,T,B,H,C,P,N$, so the local naming row has no other source for substrate closure, universality, simulation, or boundary data.

The closed side is read through $R$ and $A$. The rule table supplies the finite local update surface, and the acceptance row restricts observations to histories generated by that table.

The universal side is read through $E,S,T$. The encoding row names the finite program-input translation, the simulator-step row bounds each transition block, and the trace row lists only the blocks whose rule-table support is displayed.

Finally, $B,H,C,P,N$ bind the boundary and naming surface. Since the carrier has no coordinate for an oracle, a total halting predicate, host semantic equality, or a derivation collapsing the two axes, the listed rows exhaust the local NameCert obligations.
\end{proof}

\begin{theorem}[Universal closed substrate finite simulator boundary]
\label{thm:universal-closed-substrate-finite-simulator-boundary}
Every public read exported by an accepted $\UniversalClosedSubstrateUp$ packet factors through the finite route
$$
R,A,E,S,T,B,H,C,P,N .
$$
The route may certify that a displayed closed rule table carries a bounded simulator trace for an encoded program prefix, but it exports no proof that an arbitrary program halts, no oracle computation, no host semantic equality, no infinite closure over all traces, and no identification of universality with closure.
\end{theorem}
\begin{proof}
Begin with carrier projection. Any exported read must name the rule table $R$, closed-observation row $A$, encoding row $E$, simulator-step row $S$, trace row $T$, boundary ledger $B$, and the structural rows $H,C,P,N$.

The simulator route is finite. Each program-prefix read enters through $E$, advances only by the bounded blocks recorded by $S$, and is exposed only where $T$ lists the corresponding rule-table blocks.

The boundary row $B$ records the stopping point of the packet. A halting decision for arbitrary programs, an oracle computation, host equality, and infinite closure over all traces would require coordinates absent from $U$.

Therefore $\UniversalClosedSubstrateUp$ realizes the universal-and-closed quadrant as a finite NameCert substrate packet. It is strong enough to carry the quadrant witness of \autoref{ch:visions-closure-universality-quadrant} and the cellular automaton sibling surface of \autoref{ch:concrete-instances-cellularautomaton-namecert}, but not strong enough to export the excluded semantic objects.
\end{proof}
