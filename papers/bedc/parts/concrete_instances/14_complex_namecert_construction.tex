\chapter{A Concrete Naming Certificate for \(\ComplexUp\)}
\label{ch:concrete-instances-complex-namecert}

The complex number interface \(\ComplexUp\) has certificate obligations over ordered pairs of Bishop reals from \autoref{ch:concrete-instances-real-namecert}, read as Cartesian real and imaginary components. The classifier is componentwise real sameness. It is not a primitive number system; it is admitted only via the real source and the pair constructor of \autoref{ch:concrete-instances-prod-namecert}.

\section{Carrier definition}

\begin{definition}[Complex carrier]
\label{def:complex-carrier}
Use ordered pairs of Bishop reals \((x,y)\) from \(\RealUp\).
\end{definition}

\section{The certificate}

\begin{definition}[Complex source specification]
\label{def:complex-source-specification}
The source is \(\ProdUp(\RealUp,\RealUp)\).
\end{definition}

\begin{definition}[Complex pattern specification]
\label{def:complex-pattern-specification}
The pattern is Cartesian pairing of real and imaginary components.
\end{definition}

\begin{definition}[Complex classifier specification]
\label{def:complex-classifier-specification}
The classifier is componentwise \(\RealUp\) sameness.
\end{definition}

\begin{definition}[Complex stability certificate]
\label{def:complex-stability-certificate}
The stability certificate consists of:
\begin{enumerate}
  \item componentwise reflexivity
  \item componentwise transitivity
  \item closure under pair construction
  \item closure of complex arithmetic through real arithmetic
\end{enumerate}
\end{definition}

\begin{definition}[Complex ledger policy]
\label{def:complex-ledger-policy}
Record both real component ledgers.
\end{definition}

\begin{definition}[\(\ComplexUp\) certificate obligations]
\label{def:complex-certificate-obligations}
The carrier, source, pattern, classifier, stability, and ledger data listed above are the obligations required for the interface name \(\ComplexUp\).
\end{definition}

\paragraph{Primitive boundary.}
\(\ComplexUp\) is not a BEDC primitive. It is admitted only when those obligations are supplied.
