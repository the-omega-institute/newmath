\label{thm:apophatic-gate-question-auditgate-composition-boundary-lock}

\end{theorem}

Use refusal ledger routing and the provenance route exhaustion theorem. They force the remaining rows $H,C,P,N$ to transport, replay, package, and name the same route before any downstream audit-composition consumer can cite the packet.

Finally apply the root unblock package and refusal nonescape coverage. Since every handoff coordinate is already one of $S,Q,R,A,H,C,P,N$, $\AuditGateCompositionUp$ receives no coordinate for host gate answers, quotient collapse, proposition-extensionality collapse, unmarked selectors, outside-supply witnesses, or hidden source predicates.
\end{proof}
