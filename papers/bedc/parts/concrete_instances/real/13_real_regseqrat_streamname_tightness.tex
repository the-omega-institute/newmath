\begin{theorem}[Real finite-window seal tightness]
\label{thm:real-regseqrat-streamname-tightness}
A $\RealUp$ finite-window seal is admissible exactly when its $\RegSeqRatUp$ tail budget, $\StreamNameUp$ finite-window section, and $\DyadicRatCoreUp$ denominator ledger agree on a common $\hsame$-compatible finite request. The common request carries the displayed $\BHist$, $\Cont$, $\Pkg$, and $\NameCert$ rows used by the public seal.

Equivalently, a proposed seal whose RegSeqRat tail selector, StreamName request section, and dyadic denominator ledger do not factor through one common finite request is not an accepted $\RealUp$ public seal. The statement uses only the displayed BEDC route and does not invoke host real equality, quotient streams, selected limits, or ambient completeness.
\end{theorem}
\begin{proof}
For the forward direction, start with the Real dyadic RegSeq cross-reference lattice of \autoref{thm:real-dyadic-regseq-crossref-lattice}. It places every accepted Real current-phase name read on the DyadicRatCore, StreamName, RegSeqRat, and local Real faces.

Next apply the finite request fusion and finite-window seal exhaustion results, \autoref{thm:real-regseqrat-finite-request-fusion} and \autoref{thm:real-finite-window-seal-exhaustion}. These identify the public seal with a displayed finite request whose StreamName section, RegSeqRat readback, dyadic tolerance ledger, and support rows are componentwise compatible.

For the reverse direction, assemble the common request through the sibling route. The RegSeqRat terminal budget exhaustion supplies the tail-budget side, the StreamName finite request section supplies the public window, and the DyadicRatCore ledger supplies the denominator data for the same displayed request.

Finally repack the route as a RealUp public seal by the RegSeqRat-to-StreamName handoff and the dyadic Stream-RegSeq route. Since each cited route refuses quotient streams, host equality, selected limits, and ambient completeness, the common finite request is the whole admissibility witness.
\end{proof}
