\begin{theorem}[Constructive uniform-limit NameCert obligations]
\label{thm:constructive-uniform-limit-namecert-obligations}

\end{theorem}
\begin{proof}
Project the carrier of \autoref{def:constructive-uniform-limit-carrier}. The rows $A,L,W,R,E,H,C,P,N$ are the complete visible coordinates of one finite packet.

Read the analytic rows in dependency order. The continuous approximants provide local modulus ledgers, the uniform-limit row provides the common tail threshold, the window row supplies the shared finite schedule, and the regular-rational readback row exposes the endpoint observations used by both routes.

The seal and replay rows then close the local obligation surface. The $\RealUp$ endpoint $E$ is available only after the shared windows, uniform threshold ledger, and regular readback row have been consumed; $H$ and $C$ preserve exactly those displayed rows.

Finally, $P$ and $N$ source the same finite packet as provenance and local naming. Since no carrier coordinate stores a selected host limit, quotient function equality, pointwise-only convergence, open-cover compactness, or ambient real completeness, no further obligation is exported.
\end{proof}

\begin{proof}

Next use the uniform-limit row $L$. It supplies the common threshold ledger that lets a single window and tolerance surface control the family uniformly, so the consumer cannot replace it with a pointwise convergence witness.
