\chapter{A Concrete Naming Certificate for \(\GroupUp\)}
\label{ch:concrete-instances-group-namecert}

The group interface \(\GroupUp\) extends \autoref{ch:concrete-instances-monoid-namecert} with a unary inverse operation \({}^{-1} : C \to C\) satisfying left and right inverse laws under the carrier's classifier. It is not a primitive structure; it is admitted only by composing the monoid certificate with the inverse field.

\section{Carrier definition}

\begin{definition}[Group carrier]
\label{def:group-carrier}
Use a monoid setup extended with inverse operation \(a^{-1}\).
\end{definition}

\section{The certificate}

\begin{definition}[Group source specification]
\label{def:group-source-specification}
The source is a certified monoid carrier plus inverse data.
\end{definition}

\begin{definition}[Group pattern specification]
\label{def:group-pattern-specification}
The pattern is monoid expression formation extended by inverse formation.
\end{definition}

\begin{definition}[Group classifier specification]
\label{def:group-classifier-specification}
The classifier is the monoid carrier classifier, with inverse congruence.
\end{definition}

\begin{definition}[Group stability certificate]
\label{def:group-stability-certificate}
The stability certificate consists of:
\begin{enumerate}
  \item all monoid stability fields
  \item inverse congruence
  \item left inverse law
  \item right inverse law
  \item closure under inverse
\end{enumerate}
\end{definition}
\leanchecked{BEDC.Derived.GroupUp.group\_stability\_certificate\_fields}

\begin{definition}[Group ledger policy]
\label{def:group-ledger-policy}
Record the monoid ledger plus every inverse operation applied.
\end{definition}

\begin{definition}[\(\GroupUp\) certificate obligations]
\label{def:group-certificate-obligations}
The carrier, source, pattern, classifier, stability, and ledger data listed above are the obligations required for the interface name \(\GroupUp\).
\end{definition}

\paragraph{Primitive boundary.}
\(\GroupUp\) is not a BEDC primitive. It is admitted only when those obligations are supplied.
