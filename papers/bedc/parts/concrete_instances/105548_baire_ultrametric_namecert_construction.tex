\begin{definition}[Baire ultrametric carrier]
\label{def:baire-ultrametric-carrier}
A $\BaireUltrametricUp$ carrier is a finite $\BHist$ packet
$$
U=(B,M,V,S,H,C,P,N)
$$
with the following displayed rows:
\begin{enumerate}
\item $B$ is the $\BaireSpaceUp$ finite prefix row, naming the two inspected Baire prefixes and their common-prefix length.
\item $M$ is the $\BaireMetricUp$ row, recording the dyadic distance value attached to the first displayed disagreement or to full agreement on the inspected window.
\item $V$ is the $\UltrametricSpaceUp$ comparison row, exposing the strong-triangle comparison only through finite common-prefix witnesses.
\item $S$ is the $\StreamNameUp$ finite-window row, listing exactly the stream coordinates read before the prefix distance is declared.
\item $H$ records componentwise $\hsame$ transport for prefix rows, metric rows, ultrametric comparisons, finite windows, replay rows, provenance, and local naming.
\item $C$ records displayed $\Cont$ replay of the same finite prefix windows and distance rows.
\item $P$ is the $\Pkg$ provenance over $\BaireUltrametricUp$, $\BaireSpaceUp$, $\BaireMetricUp$, $\UltrametricSpaceUp$, $\StreamNameUp$, $\BHist$, $\hsame$, $\Cont$, and $\NameCert$.
\item $N$ is the local $\NameCert_{\BaireUltrametricUp}$ row.
\end{enumerate}
The classifier compares accepted packets only through $B,M,V,S,H,C,P,N$. It has no coordinate for host stream equality, quotient stream equality, selected branches, countable choice, ambient Real completeness, Lean verification, or bridge checking.
\end{definition}

\begin{theorem}[Baire ultrametric NameCert obligations]
\label{thm:baire-ultrametric-namecert-obligations}
For every accepted $\BaireUltrametricUp$ carrier $U=(B,M,V,S,H,C,P,N)$, the local $\NameCert_{\BaireUltrametricUp}$ surface consists exactly of the following five obligation rows:
\begin{enumerate}
\item carrier admission for the displayed Baire prefix row, BaireMetric distance row, UltrametricSpace comparison row, StreamName window row, transport, replay, provenance, and local naming;
\item common-prefix distance exactness, requiring each distance read to be the dyadic value named by the first displayed disagreement, or the zero-distance window when the inspected prefixes agree throughout the displayed window;
\item finite-cylinder classifier stability, requiring every cylinder comparison to transport only the displayed prefix rows through componentwise $\hsame$ and displayed $\Cont$ replay;
\item StreamName readback, requiring every coordinate read to pass through the finite window row $S$ before the metric row $M$ or ultrametric row $V$ is consulted;
\item nonescape, excluding selected branches, quotient stream equality, countable choice, ambient Real completeness, host stream equality, Lean verification, and bridge checking.
\end{enumerate}
No BaireUltrametric obligation is stored outside these rows.
\end{theorem}

Common-prefix distance exactness is read from $B$ and $M$. The prefix row records the inspected coordinates and the first displayed disagreement if one exists; the metric row records the corresponding dyadic distance, with the zero-distance window available only for agreement across the displayed finite window.

\label{thm:baire-ultrametric-finite-cylinder-separation}
