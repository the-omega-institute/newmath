\begin{definition}[BEDC image finite-net ledger]
\label{def:compact-image-finite-net-ledger}
Let $source$, $\mathsf{map}$, $target$, $\delta$, and $\varepsilon$ be histories
belonging to the metric and continuous-map naming certificates. A BEDC image
finite-net ledger is the history relation
$$
\mathsf{ImageFiniteNetLedger}(source,\mathsf{map},target,\delta,\varepsilon,
\mathcal{A},\mathcal{B},\mathsf{ledger})
$$
whose accepted row contains a source $\mathsf{ProbeBundle}$ $\mathcal{A}$, a
target $\mathsf{ProbeBundle}$ $\mathcal{B}$, and only BEDC rows over histories.
For each bundled source center row $\mathsf{InBundle}(p,\mathcal{A})$ there is
a center history $a$ admitted by the source finite-net ledger at precision
$\delta$. For each such center, the graph ledger supplies a bundled target row
$\mathsf{InBundle}(q,\mathcal{B})$ carrying a history $b$ and a row
$$
\mathsf{ContinuousMapCarrier}(a,\mathsf{map},b,\mathsf{modulus}_a,
\mathsf{cert}_a,\mathsf{distance}_a).
$$

The target-distance component is recorded by $\mathsf{MetricDistanceWitness}$:
for every image witness row
$$
\mathsf{ContinuousMapCarrier}(x,\mathsf{map},y,\mathsf{modulus}_x,
\mathsf{cert}_x,\mathsf{distance}_x),
$$
the source finite-net coverage row gives a bundled center $a$ and a source
distance witness below $\delta$, and the image ledger returns a bundled target
center $b$ together with a target row
$$
\mathsf{MetricDistanceWitness}(y,b,d_y)
$$
certified below $\varepsilon$. This coverage row is stable under $\hsame$ in
the source point, the image point, the bundled center, and the displayed
distance histories; graph single-valuedness and metric-distance classifier
transport are part of the ledger data. No external finite list, image predicate,
or host function is a field of the relation.
\end{definition}

\begin{proposition}[Uniformly continuous image of a BEDC finite-net source]
\label{prop:uniformly-continuous-image-totally-bounded-source}
Let $source$ and $target$ be metric BEDC carriers whose distance rows are
encoded by $\mathsf{MetricDistanceWitness}$, and let
$$
\mathsf{ContinuousMapCarrier}(source,\mathsf{map},target,\mathsf{modulus},
\mathsf{cert},\mathsf{distance})
$$
be a $\ContinuousMapUp$ carrier with graph totality, graph single-valuedness,
modulus validity, and classifier transport. For a target precision history
$\varepsilon$, suppose an accepted modulus-validity row of this carrier selects
a source precision history $\delta$. If the $\TotallyBoundedUp$ source
certificate supplies a finite-net ledger at precision $\delta$, with source
bundle $\mathcal{A}$ and classifier-stable source coverage, then there exist a
target $\mathsf{ProbeBundle}$ $\mathcal{B}$ and a history
$\mathsf{ledger}$ such that
$$
\mathsf{ImageFiniteNetLedger}(source,\mathsf{map},target,\delta,\varepsilon,
\mathcal{A},\mathcal{B},\mathsf{ledger})
$$
holds for the image carrier determined by the displayed
$\mathsf{ContinuousMapCarrier}$ rows.

The ten-field certificate template is as follows.
\begin{enumerate}
  \item traditional target: a uniformly continuous image of a totally bounded
  metric source is totally bounded;
  \item BEDC source: $\MetricSpaceUp$, $\TotallyBoundedUp$, and
  $\ContinuousMapUp$ histories over $\mathsf{BHist}$;
  \item carrier: the image finite-net relation of
  \autoref{def:compact-image-finite-net-ledger};
  \item classifier: $\hsame$ on source histories, target histories, graph rows,
  and distance rows;
  \item stability obligations: graph single-valuedness, modulus-validity
  transport, and $\mathsf{MetricDistanceWitness}$ classifier transport;
  \item ledger: the source $\mathsf{ProbeBundle}$, the constructed target
  $\mathsf{ProbeBundle}$, graph rows at bundled source centers, and target
  distance rows;
  \item core theorem list: this proposition together with the image finite-net
  ledger definition;
  \item exactness predicate: classifier-stable target coverage for every
  displayed image witness row;
  \item standard bridge: the ordinary metric theorem that uniformly continuous
  maps preserve total boundedness;
  \item status: $\mathsf{ProofTarget}$.
\end{enumerate}
\end{proposition}
\begin{proof}
Construct the target bundle by induction on the source $\mathsf{ProbeBundle}$
constructors. In the $\Bnil$ case, use $\Bnil$ as the target bundle; the source
coverage row has no admitted center to enumerate, and any image witness is
handled by the source coverage contradiction supplied by the finite-net ledger.

For a cons source bundle $\Bcons(p,\mathcal{A}_0)$, first apply the induction
hypothesis to the tail. The head row $\mathsf{InBundle}(p,\Bcons(p,\mathcal{A}_0))$
carries a source center history $a$. Graph totality from the displayed
$\mathsf{ContinuousMapCarrier}$ supplies a target history $b$ with
$$
\mathsf{ContinuousMapCarrier}(a,\mathsf{map},b,\mathsf{modulus}_a,
\mathsf{cert}_a,\mathsf{distance}_a).
$$
Adjoin the corresponding target probe to the recursively built target bundle,
and record this graph row in the image ledger.

Now take an arbitrary image witness
$$
\mathsf{ContinuousMapCarrier}(x,\mathsf{map},y,\mathsf{modulus}_x,
\mathsf{cert}_x,\mathsf{distance}_x).
$$
The source finite-net coverage row at precision $\delta$ gives a bundled
center $a$ and a source distance row
$\mathsf{MetricDistanceWitness}(x,a,d_x)$ certified below $\delta$. If this
center lies in the tail bundle, use the recursive ledger. If it is the head
center, use the graph row for $a$ and the modulus-validity row selected by
$\varepsilon$ to transport the source distance witness to a target distance
row $\mathsf{MetricDistanceWitness}(y,b,d_y)$ certified below $\varepsilon$.

It remains to show the row is classifier-stable. If the image witness is
presented with an $\hsame$ source representative, source coverage stability
transports the chosen center and source distance witness. If the graph row at a
center is presented with another target output, graph single-valuedness gives
the corresponding $\hsame$ target history. The metric-distance classifier
transport then carries $\mathsf{MetricDistanceWitness}(y,b,d_y)$ across the
target representative and the displayed distance history, so the same target
bundle certifies coverage for every equivalent image row.
\end{proof}
