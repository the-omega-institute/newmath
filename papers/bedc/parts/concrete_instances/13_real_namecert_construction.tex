\chapter{A Concrete Naming Certificate for \(\RealUp\)}
\label{ch:concrete-instances-real-namecert}

The real number interface \(\RealUp\) is licensed by Bishop-style Cauchy sequences of rationals, each carrying an explicit convergence modulus \(\omega : \mathbb{N} \to \mathbb{N}\) such that any two indices beyond \(\omega(k)\) yield rationals within \(1/(k+1)\). The classifier is the equivalence by which two such sequences converge to the same point, again expressed with an explicit modulus rather than as a quotient. It is not the classical Dedekind / Cauchy completion (which requires LEM and is not licensable in BEDC); it is licensed only via the rational source \autoref{ch:concrete-instances-rat-namecert} and the explicit modulus discipline.
