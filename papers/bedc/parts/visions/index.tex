\chapter{Capstone Results}
\label{ch:capstones}

Visions chapters collect exploratory bridge narratives that span
multiple BEDC concrete instances. Each vision is inspiration-only:
AI revision rounds consume these files as context for proposing new
\texttt{concrete\_instances/} chapters but never write back to
\texttt{parts/visions/}. When a vision's content is fully realized by
one or more concrete-instance NameCert constructions, the vision chapter
carries \verb|\concretizedIn| markers pointing to the realizations.
