\section{Lean-Style Scaffold Contract}
\label{sec:po-lean-scaffold-contract}

This chapter records the base-reflection scaffold file in the Lean repository. The file is a checked theorem-shape scaffold: abstract carriers and relations are represented as fields of a setup structure, the base relation is an inductive definition, and the local reflection theorems are proved in a form that future implementations can instantiate with concrete objects.

\section{Scaffold excerpt}
\begin{lstlisting}[language={}]
structure ExactGlobalizeBase (s : BaseReflectionSetup) (P : s.Pi) (D : s.Domain) : Prop where
  coverage : forall h, s.InDom D h -> exists p, s.InGapSig P D p h
  soundness : forall h k p q,
    s.InGapSig P D p h -> s.InGapSig P D q k ->
    GeneratedSameSig s P h k -> PsameBase s P p q
  completeness : forall h k p q,
    s.InGapSig P D p h -> s.InGapSig P D q k ->
    PsameBase s P p q -> Nonempty (GeneratedSameSig s P h k)

theorem ExactGlobalizeBase.classify_iff
  (ex : ExactGlobalizeBase s P D)
  (hp : s.InGapSig P D p h) (hq : s.InGapSig P D q k) :
  PsameBase s P p q <-> Nonempty (GeneratedSameSig s P h k) := by
  constructor
  - intro hbase
    exact ex.completeness h k p q hp hq hbase
  - intro hgen
    cases hgen with
    | intro hsig =>
        exact ex.soundness h k p q hp hq hsig
\end{lstlisting}

\section{Interpretation}
The scaffold separates three layers:
\begin{enumerate}
  \item \textbf{abstract carriers}: histories, signatures, packages, domains, bundles, and evidence, packaged by \texttt{BaseReflectionSetup};
  \item \textbf{relational object layer}: setup fields \texttt{TokIntro}, \texttt{hsame}, \texttt{SigGen}, and \texttt{InGapSig};
  \item \textbf{checked-shape target}: \texttt{PsameBase}, \texttt{PackageReflection.base}, and \texttt{ExactGlobalizeBase.classify\_iff}.
\end{enumerate}

\begin{principle}[No scaffold laundering]
\label{prin:no-scaffold-laundering}
A scaffold theorem may be cited only as a theorem-shape target. It may not be cited as a verified BEDC theorem until a build log records a successful checker run and discloses all axioms or postulates.
\end{principle}
