\section{Sprint order}
The next proof sprint should run in this order:
\begin{enumerate}
  \item define \(\psame_{\Pi}^{\mathrm{base}}\) with one public constructor;
  \item prove base inversion;
  \item prove token replacement from \(\TokUnique(\Pi)\);
  \item prove base package reflection;
  \item replay gap coverage and gap separation;
  \item prove \(\Eone\)-congruence and unary right-shift;
  \item prove unary continuation commutativity;
  \item only then activate \(\AddUp\) in the public name-certificate table.
\end{enumerate}

\section{Checker hygiene}
The implementation must not use host equality of histories as a substitute for \(\hsame\), host functions as a substitute for \(\TokIntro\), or host natural numbers as a substitute for \(\NatUp\). If such objects appear in the implementation language, they must remain representation devices rather than public BEDC primitives.
