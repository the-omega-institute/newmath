\subsection{Glossary}\label{subsec:reconstruction-glossary}

The following terms name the local reconstruction conventions used in this
section:
\begin{itemize}
  \item A shift-aware binder context is a lambda or Pi binder slot whose stored
    domain is $\mathrm{shift}\ 0\ 1\ \mathrm{dom}$ rather than
    $\mathrm{dom}$; this choice prevents capture by $\mathrm{var}\ 0$.
  \item A closed binder domain is the domain of that lambda or Pi satisfying
    $\mathrm{ClosedAt}\ 0$.
  \item $\mathrm{BetaStarStep}$ is the reflexive-transitive closure of one-step
    $\mathrm{BetaStep}$.
  \item $\mathrm{BetaParallel}$ is Tait--Martin-L\"of style parallel reduction.
  \item An atom shape is a term whose shape is either $\mathrm{sort}$ or
    $\mathrm{var}\ i$.
\end{itemize}
