\paragraph{Conflict rule.}
If a frozen statement and a later hardening module appear to differ, the frozen statement remains historical, while the later module governs current active development if and only if a migration note records the relation.

\paragraph{Current operational priority.}
The current priority is to make the finite kernel proof target mechanically checkable: first marks and histories, then extension and continuation, then signatures, then the base signature-token Globalize instance, its gap ledger, and unary continuation commutativity.
