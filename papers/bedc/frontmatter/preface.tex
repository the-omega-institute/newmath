
\chapter*{Project Preface}
\addcontentsline{toc}{chapter}{Project Preface}
This manuscript presents the Binary Emission Discovery Calculus as one maintained LaTeX source tree with a single compiled document.

The organizing principle is:
\[
\text{finite-kernel theory} + \text{proof obligations} + \text{separate implementation contracts}.
\]

\paragraph{Layer discipline.}
The theoretical core uses BEDC terminology and derived-interface notation. Implementation-facing text is placed in a separate part. This prevents the host language of a later proof assistant from contaminating the object theory.

\paragraph{Proof boundary.}
The proof boundary is the finite-kernel proof path:
\[
\msame\to \hsame\to \ExtR/\ContR\to \Sig\to \sameSig\to \PkgPol/\GapPol\to \CompGap\to \NameCert.
\]
Large derived interfaces such as \(\RealUp\), \(\CircleUp\), and \(\FoldUp\) remain thin interfaces or future extension modules until the finite kernel is mechanically checked.

\paragraph{External model posture.}
External semantic models remain explanatory artifacts. They do not alter the BEDC object theory and they do not replace the internal proof obligations recorded in the active chapters.
