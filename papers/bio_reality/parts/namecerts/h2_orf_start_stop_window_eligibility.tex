\subsection{Start--Stop Window Eligibility as a Contact-Delimited ORF Test}
\label{sec:bioreality-orff-start-stop-window-eligibility}
\origin{ai}

\paragraph{Finite question.}
The record considered here concerns a narrow biological predicate: whether a finite nucleotide interval is eligible under an explicit start/stop-codon window enumeration.  The predicate is not a claim that translation occurred.  It is not a claim about a folded protein, thermodynamic feasibility, organism-level function, regulatory mechanism, or a universal law over genomes.  It is only the statement that an externally supplied nucleotide window has been processed by a fixed start/stop classifier and has received an eligibility label within that classifier's declared scope.

The directly checked finite run for this predicate used experiment run $\mathrm{c26a0d6656f7}$.  Its input consisted of one curated nucleotide-window record, loaded from a manifest dated $2026$-$05$-$25$.  The run reports
$$
\begin{aligned}
\mathrm{input\ windows} &= 1,\\
\mathrm{classified\ windows} &= 1,\\
\mathrm{window\ count} &= 1,\\
\mathrm{external\ window\ dataset\ loaded} &= 1.
\end{aligned}
$$
The recorded scope is external-window start/stop enumeration only.  The run also records the local promotion guard as $\mathrm{orf\_eligibility\_only}$, so the output is deliberately confined to the eligibility layer.

Separately, the coordinate quotient has $|\pi_M|=10$, with $M$-level face vector $(20,38,26,8,1)$ and quotient face vector $(10,14,6,1)$.  These numbers are useful for describing the finite combinatorial environment, but they are not treated as observed transcription, translation, protein production, or biological activity.

\paragraph{Observed window enumeration.}
The actual eligibility run is therefore a two-part object.  First, a finite outside annotation supplies the nucleotide window.  Second, the frozen start/stop rule classifies that supplied window.  The direct run contains one supplied window and one classification, so its empirical content is simply that the classifier covered the manifest row.  In symbols, the checked coverage statement is
$$
\begin{aligned}
\mathrm{coverage} &= \frac{\mathrm{classified\ windows}}{\mathrm{input\ windows}}\\
&=\frac{1}{1}=1.
\end{aligned}
$$
This is a complete finite-row result for the manifest used in the run.  It is not a statistical population statement; there is only one row in this particular window manifest.  The biological force of the statement comes from the presence of the outside record and from the limited nature of the classifier, not from any autonomous codon geometry.

\paragraph{Curated interval contact.}
A stronger linked contact run, experiment $\mathrm{7c9b30b02d46}$, supplies the finite interval evidence that the seed-level boundary run by itself did not contain.  That run loaded $168$ rows from a curated interval dataset dated $2026$-$05$-$30$, sourced from NCBI RefSeq GenBank XML CDS intervals for the Escherichia coli K-12 MG1655 complete genome accession $\mathrm{NC\_000913.3}$.  Its input classes were present as positive curated ORF rows, matched non-ORF decoys, and boundary cases.  The classifier identifier was $\mathrm{orf\_start\_stop\_window\_eligibility}$, and the run records that the classifier rules were not modified after the protocol was fixed.

The curated contact has three held-out splits.  Each split contains $24$ positives and $24$ decoys.  In all three splits the positive accept rate is $1.0$ and the decoy reject rate is $1.0$.  Aggregated over the matched positives, the run reports $72$ accepted positives out of $72$, with one-sided binomial upper-tail value $2.117582368135751\times 10^{-22}$.  Aggregated over matched decoys, it reports $72$ rejected decoys out of $72$, with the same one-sided binomial value.  The predeclared paired baseline comparison reports $72$ wins in $72$ pairs, again with $p=2.117582368135751\times 10^{-22}$.  The boundary-control block has $24$ boundary cases and $0$ promotions, so the promoted fraction is $0.0$.

These values support a sharply delimited empirical statement: within this curated finite dataset and under the fixed classifier, the start/stop eligibility rule separates the declared positives from the matched decoys and does not promote boundary cases.  The result remains local to the finite contact and to the stated classifier.

\paragraph{Layer boundary.}
The present chapter therefore separates three things.  The codon-coordinate lists $M$ and $R$ give finite symbolic structure.  The one-row window manifest shows that a supplied window can be classified by the start/stop enumeration.  The $168$-row curated interval contact supplies the stronger biological interval test, with $72/72$ positive acceptance, $72/72$ decoy rejection, and $0/24$ boundary promotion.  None of these statements licenses a jump to translation, folding, physical admissibility, molecular function, mechanism, or universality.  Such claims would require separate observations capable of testing those layers.

The local negative clause is part of the result rather than an afterthought.  Start and stop codons can mark an eligible interval under a declared classifier, but eligibility is not realization.  A row may pass the ORF-window predicate while still saying nothing about ribosome engagement, expression level, peptide existence, secondary or tertiary structure, cellular role, evolutionary conservation, or causal mechanism.  Thus the finite conclusion is exactly this: the checked data establish start/stop-window ORF eligibility for the supplied finite contacts, and no higher biological layer follows from that enumeration alone.

The self-check run $\mathrm{0f47d2f0cf41}$ records three checked gates and states that no codon geometry is promoted across biological layers.  This agrees with the finite evidence above.  The one-row enumeration run is an eligibility classification; the curated interval contact is an external biological interval test; and the codon-coordinate audits remain finite symbolic checks.  The chapter is ready only because these roles are kept distinct.
