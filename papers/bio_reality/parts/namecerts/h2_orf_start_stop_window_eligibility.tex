\subsection{Start--Stop Window Eligibility as a Finite Boundary Test}
\label{subsec:bior_h2_orf_start_stop_window_eligibility}

\paragraph{Object under test.}
This record concerns the narrow biological layer called ORF eligibility: whether an externally supplied nucleotide window is classified by start- and stop-codon enumeration.  The object is not a translated product, not a folded molecular structure, and not a functional gene claim.  It is only the finite question of whether the observed window passes the declared boundary rule for open-reading-frame eligibility.  In the primary finite run, the data source was a curated nucleotide-window manifest for ORF start/stop enumeration, dated 2026-05-25.  The manifest contained exactly one input window, and the classifier returned exactly one classified window:
$$
\begin{aligned}
\mathrm{input\_windows} &= 1,\\
\mathrm{classified} &= 1,\\
\mathrm{window\_count} &= 1.
\end{aligned}
$$
The scope recorded for this run is \texttt{external\_window\_start\_stop\_enumeration\_only}.  Thus the finite row supports only a start/stop-window eligibility judgment for the supplied window.  The run explicitly records \texttt{orf\_eligibility\_only}, so no step in this chapter converts codon-window geometry into translation, protein structure, physical realizability, biological function, mechanism, or a universal biological law.

\paragraph{Independent interval contact.}
A separate curated interval run, dated 2026-05-30, loaded $168$ rows from NCBI RefSeq GenBank XML CDS intervals for \emph{Escherichia coli} K-12 MG1655, accession \texttt{NC\_000913.3}.  That run used the classifier \texttt{orf\_start\_stop\_window\_eligibility}; the record states that the classifier rules were not modified.  The required input classes were present: boundary cases, matched non-ORF decoys, and positive curated ORFs.  The positive class contained $72$ intervals and accepted all $72$, giving accept rate $1.0$ with one-sided binomial upper-tail value
$$
\begin{aligned}
P &= 2.117582368135751\times 10^{-22}.
\end{aligned}
$$
The matched decoy class contained $72$ decoys and rejected all $72$, giving reject rate $1.0$ with the same recorded one-sided binomial value
$$
\begin{aligned}
P &= 2.117582368135751\times 10^{-22}.
\end{aligned}
$$
The predeclared paired comparison records $72$ wins out of $72$ pairs, win rate $1.0$, again with upper-tail value $2.117582368135751\times 10^{-22}$.  The boundary controls contain $24$ cases and promote none of them:
$$
\begin{aligned}
\mathrm{promoted} &= 0,\\
\mathrm{promoted\_fraction} &= 0.0.
\end{aligned}
$$
The three held-out splits each report $24$ positives accepted at rate $1.0$ and $24$ decoys rejected at rate $1.0$: the left-locus fold, middle-locus fold, and right-locus fold all passed.  This contact strengthens the empirical status of the declared eligibility rule on the stated curated dataset, but it still confines the conclusion to interval-level ORF eligibility for that dataset.

\paragraph{Local boundary.}
The claim certified here is deliberately small.  From the one-window start/stop enumeration run, together with the finite codon and quotient audits, one may say that the supplied window was classified within the declared ORF-eligibility scope.  From the curated interval contact, one may additionally report that the same frozen classifier separated $72$ curated positive intervals from $72$ matched decoys, left $24$ boundary cases unpromoted, and remained stable across three held-out splits.  Nothing in these rows establishes translation realization, folded structure, physical admissibility, biological function, mechanism, or universality.  Any such assertion would require a further observation whose data type is capable of testing that higher biological layer.  The self-check row records the same guardrail: three gates were checked, and no codon geometry was promoted across biological layers.
