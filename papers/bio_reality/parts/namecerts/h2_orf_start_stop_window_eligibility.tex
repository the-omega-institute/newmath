\subsection{Start--Stop Window Eligibility as a Bounded ORF Contact Test}
\label{sec:bioreality-start-stop-window-eligibility}

\paragraph{Finite object.}
The record studied here is the layer named $\mathsf{orf\_eligibility}$: a finite test saying whether an externally supplied nucleotide interval passes a start--stop enumeration rule.  The biological object is therefore not translation and not protein formation.  It is only the eligibility status of a window after the window has already been supplied by a separate biological annotation or sequence contact.  In the finite row used for this check, the external window manifest was loaded with $1$ input window, and the enumeration classified $1$ of $1$ windows.  The scope field is $\mathsf{external\_window\_start\_stop\_enumeration\_only}$, the snapshot date is $2026$-$05$-$25$, and the source is a curated finite nucleotide-window manifest whose records are annotation inputs rather than facts derived from codon geometry alone.

\paragraph{Enumeration rule and local readout.}
Let $W$ be the finite set of windows presented to the classifier, and let $E(w)$ be the predicate that the window $w$ has the required start and stop boundary pattern under the frozen start--stop enumeration.  The observed run has
$$
\begin{aligned}
|W| &= 1,\\
|\{w\in W : E(w)\}| &= 1,\\
\frac{|\{w\in W : E(w)\}|}{|W|} &= 1.
\end{aligned}
$$
This is a complete classification of the manifest supplied to this run.  It is not a claim that every nucleotide interval in a genome has been classified.  It is also not a claim that the accepted window is translated, folded, physically realizable as a molecule, functional in an organism, mechanistically explained, or universal across organisms.

\paragraph{Local verdict and boundary.}
The finite result supports the following local statement.  When an external biological interval is supplied, the frozen start--stop enumeration can classify the supplied window at the $\mathsf{orf\_eligibility}$ layer; in the one-window manifest, $1$ of $1$ windows is classified, and in the curated interval contact the positive, decoy, boundary, and held-out controls all report the finite counts above.
