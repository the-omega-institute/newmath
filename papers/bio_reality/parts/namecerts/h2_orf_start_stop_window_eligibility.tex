\paragraph{Finite object under test.}
The record considered here is the layer named \emph{orf eligibility}: a finite start--stop window is tested only for the syntactic condition that a biological interval has an admissible start boundary and an admissible stop boundary in the supplied nucleotide data.  The biological object is not a protein, a transcript, a folding state, or a cellular role.  It is a bounded interval question: given an externally supplied finite nucleotide window, does the predeclared start--stop enumeration classify the window as eligible for the ORF layer?  The seed packet itself records that, before such an interval contact is supplied, the coordinate data alone remain only a boundary test.  In the finite run for claim \texttt{h2.orf.start\_stop.window\_eligibility}, the external window manifest was loaded with $1$ window, and the classifier assigned a class to all $1$ of $1$ input windows.  The scope field is explicitly \texttt{external\_window\_start\_stop\_enumeration\_only}, and the promotion field is \texttt{orf\_eligibility\_only}.  Thus the observed finite assertion is the exact equality
$$
\begin{aligned}
\#\{\text{classified windows}\} &= 1,\\
\#\{\text{input windows}\} &= 1,
\end{aligned}
$$
not a statement about translation, protein existence, structure, mechanism, or biological function.

\paragraph{Local negative boundary.}
The chapter therefore separates three levels.  First, the seed record alone is only a request for contact: without an interval supplied by sequence, transcript, or curated annotation data, no ORF-eligibility assertion is made.  Second, the one-window run verifies that the finite start--stop enumerator classified all supplied windows, namely $1$ of $1$, under an enumeration-only scope.  Third, the curated interval contact gives a finite external test with $168$ rows, perfect positive acceptance on $72$ positives, perfect decoy rejection on $72$ decoys, three stable held-out splits, and no promotion of $24$ boundary cases.  None of these facts licenses a stronger conclusion.  No translation event, folded conformation, physical admissibility, biological function, biochemical mechanism, organism-wide law, or universal genomic principle follows from the start--stop-window layer unless another independently specified contact is attached for that stronger layer.  The self-check row states the same guard in compact form: there are $3$ checked gates, and no codon geometry is promoted across biological layers.
