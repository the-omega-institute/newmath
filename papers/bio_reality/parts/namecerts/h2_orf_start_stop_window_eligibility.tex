\paragraph{Object of the record.}
This record concerns the narrow biological question whether a finite nucleotide window is eligible, at the start--stop boundary level, to be counted as an open-reading-frame candidate.  The biological object is not a translated protein and not a realized gene product.  It is the weaker finite event that a window supplied by an annotation source can be classified by an explicitly stated start/stop enumeration rule.  The verified run for the window-enumeration packet loaded one curated finite nucleotide-window manifest, classified all loaded windows, and reported
$$
\begin{aligned}
  \#\mathrm{windows} &= 1,\\
  \#\mathrm{classified} &= 1,\\
  \#\mathrm{input\ windows} &= 1.
\end{aligned}
$$
The run therefore establishes only that the single externally supplied window in the manifest lies inside the predeclared start/stop enumeration scope.  Its declared scope is external-window start/stop enumeration only, with snapshot date 2026-05-25.  The manifest is an annotation input to the experiment and is not produced by the codon geometry itself.

\paragraph{Window-enumeration verdict.}
For claim \texttt{h2.orf.start\_stop.window\_eligibility}, the direct run reports that the external dataset was loaded and that every input window was classified by the start/stop enumeration.  The run also stores the local promotion guard as \texttt{orf\_eligibility\_only}.  Hence the mathematical content of the record is the finite implication
$$
\begin{aligned}
  \mathrm{loaded}(W) \wedge \mathrm{start/stop\ classified}(W)
  \;\Longrightarrow\;
  \mathrm{eligible}_{\mathrm{boundary}}(W),
\end{aligned}
$$
where $W$ ranges only over the one window present in the loaded manifest.  The implication is not quantified over all genomes, all transcripts, all organisms, or all possible nucleotide strings.  It is a statement about the particular finite manifest read by experiment run \texttt{c26a0d6656f7}.
