\paragraph{Local interpretation.}
The result supports only the following bounded statement: given an externally supplied nucleotide interval, the start--stop classifier can evaluate ORF-window eligibility on that finite row, and the larger curated contact test shows perfect finite separation on the recorded positive, decoy, boundary, and held-out rows.  This is an eligibility-layer result.  It is not a statement that the window is translated in vivo, that a protein folds, that a molecular structure is physically realized, that a function is present, that a mechanism has been identified, or that a universal biological law has been obtained.  Those stronger claims require separate empirical contacts of the appropriate kind.
