\paragraph{Finite record.}
This record concerns only the eligibility layer attached to a finite nucleotide window whose boundary symbols are tested by start and stop codon enumeration.  The observed row for the window-enumeration experiment reports one loaded external window dataset, one input window, and one classified window.  Its declared scope is \texttt{external\_window\_start\_stop\_enumeration\_only}; its snapshot date is \texttt{2026-05-25}.  The source description says that the records are curated finite nucleotide-window manifest entries for BioReality ORF start/stop enumeration, and that these entries are annotation inputs to the experiment rather than facts generated by the mathematical kernel.  Thus the finite classification fact is exactly that the available window row was classified by the start/stop enumeration rule, not that translation, expression, folding, or biological use occurred.

\paragraph{Local exclusion boundary.}
The chapter's claim is therefore narrow.  The start/stop-window row is an ORF-eligibility classification at the finite enumeration layer, and the curated interval row supplies an external annotation contact for that layer.  Neither row authorizes a claim that a protein was translated, that a folded structure exists, that a physical state is admissible, that a molecular function is present, that a mechanism has been identified, or that the rule is universal.  Those conclusions would require separate contacts designed for those layers.  The self-check row agrees with this boundary: it records three checked gates and states that no codon geometry is promoted across biological layers.
