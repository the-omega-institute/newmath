\paragraph{ORF-eligibility interpretation.}
The biological object named by the packet is open-reading-frame eligibility from start/stop codon enumeration.  The row data can only express a conditional boundary: if a separate sequence, transcript, or curated-annotation contact supplies an observed biological interval, then this finite geometry can help specify which start/stop-window predicates must be tested on that interval.  In the present packet there are no reality contacts.  Therefore the coordinate data remain a boundary record for the layer $orf\_eligibility$, not an ORF-eligibility claim.
