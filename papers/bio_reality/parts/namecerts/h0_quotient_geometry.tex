\paragraph{Finite codon support.} The record begins with a finite codon-level calculation, not with a biological interval.  The verified codon set $M$ has cardinality $20$ and is exactly

\begin{aligned}

&\mathrm{CUG},\mathrm{CUU},\mathrm{UAA},\mathrm{UAG},\mathrm{UCA},\mathrm{UCG},\mathrm{UGA},\mathrm{UGG},\mathrm{UUA},\mathrm{UUG}\}.
\end{aligned}
$$
The motif-reduced support $R$ is the $13$-element set
$$
\begin{aligned}
R=\{&\mathrm{AAA},\mathrm{AGA},\mathrm{AGG},\mathrm{AUA},\mathrm{CUA},\mathrm{CUC},\mathrm{CUG},\mathrm{CUU},\\
&\mathrm{UAA},\mathrm{UAG},\mathrm{UCA},\mathrm{UGA},\mathrm{UUA}\}.
\end{aligned}
$$
The audit reports no missing and no extra elements in this motif union.  A separate finite check records that $M$ agrees with the enumerated $WNR\cup CUN$ support, again with size $20$.  These are row-level identities among codon strings; they do not say that any genomic interval has been observed.

The same run records an empty failure list for the product-decomposition check.  Thus the mathematical content available at this site is a finite quotient signature over enumerated codon rows.  It supplies a compressed coordinate system in which start and stop patterns can be discussed, but it is still only a calculation on strings and quotient rows.

\paragraph{Saturation boundary.} The saturation audit for $f_3$ verifies that
$$
\begin{aligned}
\operatorname{Sat}_{f_3}(R)=\{&\mathrm{AAA},\mathrm{AAG},\mathrm{AGA},\mathrm{AGG},\mathrm{AUA},\mathrm{AUG},\mathrm{CUA},\mathrm{CUC},\mathrm{CUG},\\
&\mathrm{CUU},\mathrm{UAA},\mathrm{UAG},\mathrm{UCA},\mathrm{UCG},\mathrm{UGA},\mathrm{UGG},\mathrm{UUA},\mathrm{UUG}\},
\end{aligned}
$$
so the saturated set is $M$ with $\mathrm{ACA}$ and $\mathrm{ACG}$ absent.  The run also reports the $f_3$ boundary as empty.  This is a useful internal check on the finite row geometry: the codon support, the motif support, and the quotient support fit together without an unaccounted finite boundary in that computation.  It is not, by itself, evidence that a cell uses any particular interval as an open reading frame.

\paragraph{ORF-eligibility contact.} The biological layer is therefore introduced only through an explicitly separate contact.  A minimal start--stop enumeration run loaded one external nucleotide-window manifest with $1$ input window, classified $1$ of $1$ windows, and recorded the scope as external window start--stop enumeration only.  A stronger curated interval contact then loaded $168$ rows from a RefSeq-derived interval dataset for \emph{Escherichia coli} K-12 MG1655 accession $\mathrm{NC\_000913.3}$, with snapshot date $2026$-$05$-$30$.  Its declared classifier was $\mathrm{orf\_start\_stop\_window\_eligibility}$, and the run states that classifier rules were not modified after the protocol was fixed.  The required input classes were all present: boundary cases, matched non-ORF decoys, and positive curated ORFs.

\paragraph{Finite statistical checks.} Within that external contact, $72$ positive intervals were accepted, giving accept rate $1.0$ and one-sided binomial upper-tail value $2.117582368135751\times 10^{-22}$.  The $72$ matched decoys were all rejected, giving rejection rate $1.0$ with the same reported one-sided binomial value $2.117582368135751\times 10^{-22}$.  The predeclared paired comparison had $72$ wins in $72$ pairs, win rate $1.0$, again with binomial upper-tail value $2.117582368135751\times 10^{-22}$.  Held-out stability was checked on three splits: the left, middle, and right locus folds each had $24$ positives and $24$ decoys, each passed, each had positive accept rate $1.0$, and each had decoy reject rate $1.0$.  Boundary controls counted $24$ cases with $0$ promoted, so the promoted fraction was $0.0$.

\paragraph{Local claim boundary.} The quotient computation and the curated interval contact jointly justify only an ORF-eligibility statement at the stated finite scope: a fixed start--stop-window classifier agrees with the curated interval rows and rejects the matched decoy rows in this dataset.  The record does not establish translation, folded protein structure, physical admissibility, molecular mechanism, biological function, or a universal genomic law.  None of those higher assertions follows from the codon quotient, from the start--stop enumeration, or from the matched interval statistics unless another contact supplies the corresponding observation.  The self-check records the same separation: three gates were checked, and no codon geometry was promoted across biological layers.
