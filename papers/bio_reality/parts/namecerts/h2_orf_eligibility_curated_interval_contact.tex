\paragraph{Claim.}
This record concerns only open-reading-frame eligibility as tested by finite start/stop codon enumeration against a curated interval annotation. The biological object is not translation, protein production, folded molecular structure, physical admissibility, function, mechanism, or a universal rule about genomes. It is the narrower question of whether a fixed start/stop-window classifier, with its rules fixed before observing the curated interval evidence, separates annotated coding intervals from matched non-ORF decoys inside one finite contact dataset. The carrier before contact is intentionally empty of biological assertion: it records that a start/stop-window pattern can pose a test, not that it has already found an ORF.

\paragraph{Local boundary.}
The result is a finite contact between a pre-fixed geometry-only ORF-eligibility classifier and a curated interval table. It supports the local statement that, within this dataset and protocol, the classifier separates annotated positive intervals from matched non-ORF decoys and refuses the listed boundary cases.
