\subsection{Leave-One-Code-Out Recovery in the Window-Six Codon Cube}
\label{sec:bioreality-h1-leave-one-code-out}

\paragraph{Finite codon display.}
The claim concerns a six-coordinate codon display whose carrier is the codon cube $Q_6$.  A codon is treated here only as a six-bit coordinate object equipped with a curated code label.  The biological contact is therefore the standard codon-to-label table: it says which codons read as amino-acid labels or stop labels, and it does not by itself say that translation dynamics, molecular folding, energetic admissibility, function, or a universal biological mechanism has been observed.  The finite recovery test asks whether leaving one genetic-code table out still leaves the variable codon targets in a locally recognizable region of the window-six geometry.

\paragraph{Residual coordinate region.}
The distinguished residual set $R$ has cardinality $13$.  In the finite row it is exactly
$\{AAA, AGA, AGG, AUA, CUA, CUC, CUG, CUU, UAA, UAG, UCA, UGA, UUA\}$.
The same row records that this set realizes a punctured-cube combinatorics with face vector $(13,16,4)$.  Thus the observed finite object is not merely a count: it is a named list of codons together with a local face structure.  The row-level module partition assigns all $13$ codons with no missing or extra elements: $AGR=\{AGA,AGG\}$, $AR/Lys=\{AAA\}$, $CUN=\{CUA,CUC,CUG,CUU\}$, $Ile/Met=\{AUA\}$, $SSN=\{UCA,UUA\}$, and $Stop/Trp=\{UAA,UAG,UGA\}$.  The module audit reports $R\_size=13$, $assigned\_count=13$, $missing\_from\_modules=\emptyset$, $extra\_module\_codons=\emptyset$, and $unassigned\_codons=\emptyset$.

The recovery rate inside $M\setminus T$ was $0.9365079365079365$, while allowing the recorded boundary raised the rate to $0.9841269841269841$.  The hit rate above baseline was also $0.9841269841269841$.  In finite terms, almost all left-out target labels remain in the predicted local region once the boundary part of the tile is admitted:
$$
\begin{aligned}
\text{baseline} &= 0.19821739130434785,\\

\text{inside } (M\setminus T)\cup \text{boundary} &= 0.9841269841269841,\\

\paragraph{Interpretation at the code layer.}
The mathematical content of the row is that the standard-code label table, when displayed in $Q_6$, exposes a small local tile around the residual codons listed above.  The strongest finite statement is a code-read statement: the curated labels line up with a local split whose residual set has size $13$, whose face vector is $(13,16,4)$, whose spectral values are the recorded $\lambda_M$, $\lambda_R$, and $\mu$, and whose leave-one-code-out recovery is $0.9841269841269841$ above baseline over $63$ targets from $23$ tables.  The result is therefore a finite contact between codon-coordinate geometry and curated code labels.

\paragraph{Local boundary of the claim.}
No stronger biological conclusion follows from this row alone.  The codon table supplies label readback, not a translation process.  A label sequence is not a folded protein structure.  The local tile is not evidence of physical admissibility, organismal function, or a universal biological law.  Crossing any of those boundaries would require a separate contact that directly observes the relevant layer, such as translation realization, structural order, energetic or physical feasibility, or biological function.  Until such a contact is present, the row should be read as a finite, code-layer recovery result for the standard-code label table and its related code variants.
