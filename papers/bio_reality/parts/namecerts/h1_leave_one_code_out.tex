\subsection{Leave-One-Code-Out Boundary Readback}\label{subsec:h1-leave-one-code-out}
\origin{ai}

\paragraph{Finite carrier and external contact.}
This record treats the standard-code codon table as a finite code-read contact, not as a mechanism.  The carrier is the six-bit codon cube $Q_6$, read through the four BEDC distinctions supplied by the row: six-bit codon display, local tile face, singleton label corner, and three-point label class.  The only external contact used here is the curated standard-code label table, whose observed fact is that each codon is assigned an amino-acid or stop label.  This contact has resolution at codon-label readback.  It does not observe translation realization, structural order, physical admissibility, folded protein structure, function realization, a universal mechanism, or any claim that DNA must obey this geometry as a biological law.

\paragraph{The finite boundary set.}
The row identifies the exceptional boundary set $R$ as a finite thirteen-codon subset of the codon cube.  The cardinality check gives $|R|=13$, and the punctured-cube check returns the same explicit list:
$$
\begin{aligned}
R=\{&\mathrm{AAA},\mathrm{AGA},\mathrm{AGG},\mathrm{AUA},\mathrm{CUA},\mathrm{CUC},\mathrm{CUG},\mathrm{CUU},\\
&\mathrm{UAA},\mathrm{UAG},\mathrm{UCA},\mathrm{UGA},\mathrm{UUA}\}.
\end{aligned}
$$
The associated face-count vector is
$$
\begin{aligned}
f(R)=(13,16,4),
\end{aligned}
$$
so the finite object being tested is not merely a count of codons but a shaped local tile inside the codon cube.  The shape-significance run used $N=2000$ sampled comparators with seed $42$.  It found one exact shape match among the sampled rows, giving $p_{\mathrm{exact}}=0.0005$, and twenty-five high-overlap rows at Jaccard threshold $0.9$, giving $p_{J\geq 0.9}=0.0125$.  The run required $57377$ distinct-sample attempts and encountered $55377$ duplicate retries, which is consistent with a constrained finite sampling space rather than an unconstrained continuum of alternatives.

It should be read as a codon-cube invariant attached to the local tile, not as a claim about molecular dynamics or protein structure.

\paragraph{Leave-one-code-out performance.}
For the primary claim $\mathrm{h1.leave.one.code.out}$, the verified run evaluated $23$ code tables and $63$ total targets.  The baseline hit rate was $0.19821739130434785$.  The row reports
$$
\begin{aligned}
\mathrm{hit\_rate\_in\_}M\setminus T&=0.9365079365079365,\\
\mathrm{hit\_rate\_in\_}M\setminus T\mathrm{\ plus\ boundary}&=0.9841269841269841,\\
\mathrm{hit\_rate\_above\_baseline}&=0.9841269841269841.
\end{aligned}
$$
This is the core finite-row evidence for the claim: when one code is held out, the local tile and its boundary nearly recover the observed target labels at the code-read layer, far above the reported baseline rate.  The result is not phrased as a biological necessity.  It says that the window-six coordinate system exposes a strong local boundary-tile regularity in curated code labels.

\paragraph{Null comparators.}
The amino-acid-class-preserving null model used $N=2000$ samples with seed $42$.  It found $0$ exact shape matches, so the reported exact-shape null value is $p=0.0$ at the displayed sampling resolution.  The lower-tail count at or below the observed $M$ size was $7$, with observed $|M|=20$, giving $p_{\mathrm{lower\ tail}}=0.0035$.  These null rows support the claim that the observed tile is not recovered by the tested class-preserving randomization at ordinary frequency.  They do not by themselves identify a causal biochemical mechanism.

\paragraph{Boundary of the claim.}
The admissible conclusion is narrow.  Window-six codon coordinates, together with the curated standard-code table, expose a local boundary-tile structure that predicts or classifies many code-label departures under the leave-one-code-out and leave-one-codon-out probes.  The forbidden promotions are part of the result: no translation realization, folding claim, physical admissibility claim, biological function claim, universal mechanism, or global biological law follows from this row.  Amino-acid label-sequence readback remains a code-layer observation.  Any move from this local codon tile to translation, structure, dynamics, admissibility, or function requires a separate reality contact with its own finite rows.
