\subsection{Leave-one-code-out stability of a window-six codon tile}
\label{sec:h1-leave-one-code-out}

\paragraph{Claim.}
The claim identified as \texttt{h1.leave.one.code.out} concerns a finite code-read experiment on the six-coordinate codon cube $Q_6$.  Its object is not translation as a biochemical process, but the much narrower question of whether a local codon-tile split, computed from window-six coordinates and then read against a curated standard-code label table, continues to recover held-out reassignment targets across alternative code tables.  The relevant carrier is the $64$-vertex codon cube with a six-bit display; the distinctions used here are local tile faces, singleton label corners, three-point label classes, and the code-label readback supplied by the curated table.  The table contact says only that each codon has an amino-acid or stop label in the standard code.  It does not by itself observe ribosomal realization, protein folding, physical admissibility, biological function, a mechanism of code evolution, or a universal law.

\paragraph{Finite residue set.}
The row set singled out by the window-six computation has cardinality $13$.  The verified codons are
$$\begin{aligned}
R=\{&\mathrm{AAA},\mathrm{AGA},\mathrm{AGG},\mathrm{AUA},\mathrm{CUA},\mathrm{CUC},\mathrm{CUG},\mathrm{CUU},\\
&\mathrm{UAA},\mathrm{UAG},\mathrm{UCA},\mathrm{UGA},\mathrm{UUA}\}.
\end{aligned}$$
This is also the verified punctured-cube row set.  Its finite face data are
$$\begin{aligned}
f(R)=(13,16,4),
\end{aligned}$$
so the local object has $13$ vertices, $16$ edges, and $4$ two-faces in the checked tile representation.  The row set is not being asserted as a global law of molecular biology.  It is a finite combinatorial locus where the standard-code labels can be inspected.

\paragraph{Spectral check.}
The same finite object carries a verified spectral summary.  The reported values are
$$\begin{aligned}

\lambda_R&=0.4758352843197002,\\

This near-zero residual is useful as an internal numerical check on the finite computation, but it is not evidence for folding, catalytic activity, or organism-level selection.  Its role is to confirm that the finite window-six object used by the leave-one-code-out test is the same numerical object across the coordinate, closure, spectrum, and relation views.

\paragraph{Shape significance.}
Against the reported finite null sample, the observed shape was rare.  In $N=2000$ sampled trials with seed $42$, there was $1$ exact shape match, giving
$$\begin{aligned}
p_{\mathrm{exact}}=0.0005.
\end{aligned}$$
There were $25$ trials with Jaccard similarity at least $0.9$, giving
$$\begin{aligned}
p_{J\geq 0.9}=0.0125.
\end{aligned}$$
The sampling record also reports $57377$ distinct-sample attempts and $55377$ duplicate retries.  A second amino-acid-class-preserving null sample, again with $N=2000$ and seed $42$, found $0$ exact matches and $7$ trials in the lower tail at or below the observed module size.  Thus
$$\begin{aligned}
p_{\mathrm{aa\ class\ exact}}&=0.0,\\
p_{\mathrm{aa\ class\ lower\ tail}}&=0.0035,
\end{aligned}$$
with observed module size $20$.  These numbers support the statement that the finite tile is not a typical draw under the tested null procedures.  They do not identify the biological cause of that atypicality.

\end{aligned}$$
The verified hit rate above baseline was
$$\begin{aligned}

\end{aligned}$$
while the stricter $M\setminus T$ row gave
$$\begin{aligned}

\end{aligned}$$
Adding the boundary rows returned the value to
$$\begin{aligned}
0.9841269841269841.

The interpretation is deliberately local: when one code table is held out, the finite tile and its boundary recover nearly all of the observed reassignment targets relative to the supplied code-label tables.  The measured event is a label-level recovery event inside curated genetic-code rows, not a proof that the tile generates translation or protein structure.

\paragraph{Scope boundary.}
The contact used here is a curated standard genetic-code table.  It supports codon-to-label readback and local split checks.  It does not supply an observation of translation dynamics, folded structure, physical admissibility, molecular function, causal mechanism, or biological universality.  Therefore the chapter records a positive finite code-read result and a negative promotion rule at the same time.  The row set $R$, the face vector $(13,16,4)$, the spectral values, the null-model $p$-values, the $23$-table leave-one-code-out score, and the complete $13$-codon module partition together make a concrete finite case that the local window-six tile is aligned with known code-label variation.  Any statement beyond code labels would require an additional contact that actually measures that higher layer.
