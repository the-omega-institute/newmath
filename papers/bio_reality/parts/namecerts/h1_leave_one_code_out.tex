\subsection{Leave-One-Code-Out Recovery in the Window-Six Codon Cube}\label{subsec:h1-leave-one-code-out}\origin{ai}

\paragraph{Finite coordinate record.} This record treats the standard-code codon table as a finite label-readback contact on the six-bit codon cube $Q_6$. The coordinate object is not taken as a biological mechanism. It is a bounded display in which codons are represented as cube vertices, local faces are inspected, and the resulting split is compared with curated codon labels. The distinguished recovery set $R$ has cardinality $13$ and is exactly
$$
\begin{aligned}
R=\{&\texttt{AAA},\texttt{AGA},\texttt{AGG},\texttt{AUA},\texttt{CUA},\texttt{CUC},\texttt{CUG},\texttt{CUU},\\
&\texttt{UAA},\texttt{UAG},\texttt{UCA},\texttt{UGA},\texttt{UUA}\}.
\end{aligned}
$$
The same row reports that $R$ is a punctured-cube pattern with face-vector $(13,16,4)$. Thus the finite datum is not merely a list of exceptions: it carries a local cubical boundary profile. The rows do not supply an explicit ordered list for the ambient $M$ set, but they do give its relevant finite size in the amino-acid-class-preserving comparison as $|M|=20$.

\paragraph{Spectral and shape checks.} The associated spectral row gives $\lambda_M=0.6752479308830213$, $\lambda_R=0.4758352843197002$, and $\mu=3.051487585298128$. The reported cubic residual is $-6.224354365258478\times 10^{-12}$, the same value recorded for the radial polynomial residual, so the numerical relation is satisfied to the displayed tolerance. In the shape-significance experiment with seed $42$, $N=2000$ distinct samples were obtained after $57377$ sampling attempts and $55377$ duplicate retries. Exactly one sample matched the observed shape, giving $p_{\mathrm{exact}}=0.0005$. There were $25$ samples with Jaccard similarity at least $0.9$, giving $p_{J\geq 0.9}=0.0125$. These numbers support the claim that the observed local tile is not a routine outcome under that finite null model.

\paragraph{Class-preserving null comparison.} The amino-acid-class-preserving null row gives a stricter comparison against randomizations that retain label-class information. With $N=2000$ and seed $42$, the exact-shape count is $0$, so the displayed exact-shape value is $0.0$. The lower-tail count at or below the observed $M$ size is $7$, with observed $M$ size $20$, giving lower-tail value $0.0035$. This is important because the comparison does not merely ask whether an arbitrary relabeling can imitate the pattern. It asks whether a class-respecting finite perturbation still recovers the same local structure. In the displayed finite run, it does not.

\paragraph{Leave-one-code-out recovery.} The direct leave-one-code-out experiment evaluated $23$ code tables and $63$ total targets. The baseline hit rate is $0.19821739130434785$. The observed hit rate above baseline is $0.9841269841269841$, the hit rate inside $M\setminus T$ is $0.9365079365079365$, and the hit rate inside $M\setminus T$ together with the boundary allowance is $0.9841269841269841$. Thus, once one code table is held out, the window-six local tile almost always places the target in the predicted region or in the explicitly bounded boundary region, while the baseline rate remains near one fifth.

\paragraph{Family and codon-module resolution.} The family-level row divides the leave-one-code-out data into $5$ families. The hit-rate-above-baseline values are $0.8333333333333334$ for alternative yeast, $1.0$ for bacterial plastid, $1.0$ for candidate other, $1.0$ for mitochondrial, and $1.0$ for nuclear ciliate. The codon-module row gives a complete assignment of the $13$ elements of $R$: $\texttt{AGR}=\{\texttt{AGA},\texttt{AGG}\}$, $\texttt{ar\_lys}=\{\texttt{AAA}\}$, $\texttt{CUN}=\{\texttt{CUA},\texttt{CUC},\texttt{CUG},\texttt{CUU}\}$, $\texttt{ile\_met}=\{\texttt{AUA}\}$, $\texttt{SSN}=\{\texttt{UCA},\texttt{UUA}\}$, and $\texttt{stop\_trp}=\{\texttt{UAA},\texttt{UAG},\texttt{UGA}\}$. The module-classification row reports $R_{\mathrm{size}}=13$, assigned count $13$, no extra module codons, no missing codons, and no unassigned codons. Separately, the leave-one-codon-out row reports anchor count $0$, boundary count $5$, predictable count $8$, high closure coverage $8$, and recovery rate $1.0$. Hence each module has either a predictable member or a boundary member, and the finite module partition covers the entire recovery set.

\paragraph{Scientific reading.} The finite evidence supports a code-layer statement: window-six codon coordinates expose a local boundary-tile organization that is highly recoverable under leave-one-code-out testing and complete at the displayed module level. The curated standard-code table supplies the observed codon-to-label contact. It does not supply translation dynamics, folded structure, physical admissibility, realized function, a biochemical mechanism, or a universal law. Amino-acid label readback remains label readback. Any statement about protein folding, structural order, organism-level translation, tRNA-mediated mechanism, physical viability, or biological function would require an additional contact designed for that layer. The present record therefore licenses a finite codon-label recovery claim and blocks promotion beyond the code-read layer.
