\subsection{Leave-One-Code-Out Recovery in the Window-Six Codon Tile}
\label{sec:h1-leave-one-code-out}

\paragraph{Finite carrier and code-layer contact.}
The carrier is the six-bit codon cube $Q_6$ equipped with the standard-code label table as an external code-layer contact.  A point of $Q_6$ is read as a codon display, while the amino-acid or stop label attached to that codon is not generated by the cube itself.  The local distinction under test is therefore not a law of translation, but a finite readback question: after withholding one code table from the comparison family, does the window-six tile still identify the codon-label changes that lie near the distinguished local boundary?  The contact used here is the curated standard genetic-code bridge, whose observable content is only that codons receive amino-acid or stop labels.  It does not observe translation realization, folded protein structure, physical admissibility, biological function, mechanism, or universality.

\paragraph{Distinguished row set.}
The finite row set $R$ has cardinality $13$.  In codon notation it is
$$
\begin{aligned}
R=\{&AAA,AGA,AGG,AUA,CUA,CUC,CUG,CUU,\\
&UAA,UAG,UCA,UGA,UUA\}.
\end{aligned}
$$
The punctured-cube check gives the same row list and reports the face-vector
$$
\begin{aligned}
f(R)=(13,16,4).
\end{aligned}
$$
Thus the local object is a small punctured tile inside the codon cube: thirteen labelled codon corners, sixteen local one-dimensional adjacencies, and four local two-dimensional faces.  This is the level at which the evidence lives.  The row set is not a complete biological mechanism and is not a proof that DNA or translation must obey this geometry.

\paragraph{Spectral finite-row signature.}
The spectral audit attaches three scalar values to the local split:
$$
\begin{aligned}
\lambda_M&=0.6752479308830213,\\
\lambda_R&=0.4758352843197002,\\
\mu&=3.051487585298128.
\end{aligned}
$$
The reported radial or cubic polynomial residual is $-6.224354365258478\times 10^{-12}$, which is numerically at zero for the supplied tolerance.  The role of these numbers is diagnostic: they show that the observed row split has a stable finite spectral signature.  They do not by themselves identify a biochemical pathway, a folding order, or an admissible physical dynamics.

\paragraph{Shape significance against finite nulls.}
The shape significance run used $N=2000$ sampled null tables with seed $42$.  It required $57377$ distinct-sample attempts and encountered $55377$ duplicate retries.  Exactly one sampled table matched the observed shape, giving
$$
\begin{aligned}
p_{\mathrm{exact}}&=0.0005.
\end{aligned}
$$
There were $25$ samples with Jaccard similarity at least $0.9$, giving
$$
\begin{aligned}
p_{J\geq 0.9}&=0.0125.
\end{aligned}
$$
A second amino-acid-class-preserving null used the same sample count $N=2000$ and seed $42$.  In that null, the observed marked set size was $|M|=20$, the exact-match count was $0$, and the lower-tail count at or below the observed marked size was $7$.  Hence
$$
\begin{aligned}
p_{\mathrm{aa\ class\ exact}}&=0,\\
p_{\mathrm{aa\ class\ lower\ tail}}&=0.0035.
\end{aligned}
$$
These values support the claim that the observed local tile is not a typical draw under the stated finite nulls.  They still remain code-read statistics; they do not promote the tile to a translation law or a protein-structure claim.

\paragraph{Leave-one-code-out recovery.}
For the leave-one-code-out experiment, $23$ code tables were evaluated and $63$ target changes were counted.  The baseline hit rate was $0.19821739130434785$.  The local tile boundary achieved
$$
\begin{aligned}
h_{\mathrm{above\ baseline}}&=0.9841269841269841,\\
h_{M\setminus T}&=0.9365079365079365,\\
h_{M\setminus T+\partial}&=0.9841269841269841.
\end{aligned}
$$
Thus almost every withheld-code target lies above the baseline when the local boundary information is admitted, while the stricter interior marked set $M\setminus T$ already captures most of the targets.  In ordinary prose, the window-six coordinates make the changing codons highly visible within this finite family of code tables.  The conclusion is deliberately local: it is a recovery result for code-label differences, not an explanation of how cells implement translation or why a variant code is biologically viable.

\paragraph{Family-level distribution.}
The family audit partitions the evaluated code tables into five families.  The hit rates above baseline are
$$
\begin{aligned}
h_{\mathrm{alternative\ yeast}}&=0.8333333333333334,\\
h_{\mathrm{bacterial\ plastid}}&=1.0,\\
h_{\mathrm{candidate\ other}}&=1.0,\\
h_{\mathrm{mitochondrial}}&=1.0,\\
h_{\mathrm{nuclear\ ciliate}}&=1.0.
\end{aligned}
$$
The lower value for the alternative-yeast family is still far above the baseline rate $0.19821739130434785$, while the other four families are saturated at the finite-row level.  This distribution is useful because it prevents the result from being read as a single pooled accident: the effect persists across the named code-table families supplied in the row data.

\paragraph{Leave-one-codon-out module check.}
The companion leave-one-codon-out run reports no anchor codons, $5$ boundary codons, $8$ high-coverage codons, and $8$ predictable codons, with recovery rate
$$
\begin{aligned}
r_{\mathrm{recovery}}=1.0.
\end{aligned}
$$
The module classification is complete for the thirteen distinguished rows: $R$ has size $13$, $13$ codons are assigned, there are no extra module codons, no missing rows, and no unassigned codons.  The modules are
$$
\begin{aligned}
\mathrm{agr}&=\{AGA,AGG\},\\
\mathrm{ar\_lys}&=\{AAA\},\\
\mathrm{cun}&=\{CUA,CUC,CUG,CUU\},\\
\mathrm{ile\_met}&=\{AUA\},\\
\mathrm{ssn}&=\{UCA,UUA\},\\
\mathrm{stop\_trp}&=\{UAA,UAG,UGA\}.
\end{aligned}
$$
Every listed module has a predictable or boundary status.  The module table is therefore not a loose annotation on top of $R$; it exhausts the thirteen-row punctured tile and records how each local codon group enters the recovery test.

\paragraph{Boundary verdict.}
The finite-row verdict is that the window-six codon display exposes a stable local boundary-tile structure for the standard-code label table and for the evaluated code-table family.  The evidence consists of the thirteen-row punctured cube, the spectral signature, the significant finite-null shape statistics, the $23$-table leave-one-code-out recovery, and the complete thirteen-row module classification.  The cannot-claim boundary is part of the result: no translation realization, folded structure, physical admissibility, biological function, universal mechanism, or necessity claim follows from these rows.  Any such stronger claim would require a separate reality contact that observes the higher layer directly.
