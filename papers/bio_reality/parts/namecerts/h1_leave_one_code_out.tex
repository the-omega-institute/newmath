\subsection{Leave-one-code-out readback for the window-six codon tile}
\label{sec:h1-leave-one-code-out}

\paragraph{Finite carrier and readback object.}
The carrier for this record is the six-bit codon cube $Q_6$, read through a curated standard-code label table.  The BEDC object under test is not translation, folding, function, or a physical model of the ribosome.  It is a finite coordinate display of codon labels, with distinctions for the six-bit codon display, local tile faces, singleton label corners, and three-point label classes.  The reality contact is the standard-code codon-to-label table: it supplies amino-acid and stop labels for codons, and nothing above the code-read layer.  The leave-one-code-out probe asks whether the local tile obtained from the standard code continues to locate code-table changes when one compares against other finite code tables.

\paragraph{The punctured-cube row set.}
The distinguished row set $R$ has cardinality $13$.  In the supplied finite run it is exactly
$$
\begin{aligned}
R=\{&\mathrm{AAA},\mathrm{AGA},\mathrm{AGG},\mathrm{AUA},\mathrm{CUA},\mathrm{CUC},\mathrm{CUG},\mathrm{CUU},\\
&\mathrm{UAA},\mathrm{UAG},\mathrm{UCA},\mathrm{UGA},\mathrm{UUA}\}.
\end{aligned}
$$
The same run identifies this set as a punctured-cube boundary pattern with face-vector
$$
\begin{aligned}
(f_0,f_1,f_2)=(13,16,4).
\end{aligned}
$$
Thus the local object is not a vague enrichment of the code table.  It is a finite row set with $13$ vertices, $16$ one-dimensional incidences, and $4$ two-dimensional faces inside the six-bit codon cube display.  This is the geometric part of the record; the labels attached to those rows still come from the external curated code table.

\paragraph{Shape significance against finite null rows.}
The shape-significance run used $N=2000$ sampled tables, with $57377$ distinct-sample attempts and $55377$ duplicate retries.  It found $1$ exact shape match, giving
$$
\begin{aligned}
p_{\mathrm{exact}}=0.0005.
\end{aligned}
$$
It also found $25$ cases with Jaccard score at least $0.9$, giving
$$
\begin{aligned}
p_{\mathrm{Jaccard}\ge 0.9}=0.0125.
\end{aligned}
$$
A second amino-acid-class-preserving null model again used $N=2000$.  In that run the observed $M$ size was $20$, the exact-match count was $0$, and the lower-tail count at or below the observed $M$ size was $7$.  Hence the recorded finite-row probabilities were
$$
\begin{aligned}
p_{\mathrm{aa\ class\ exact}}&=0.0,\\
p_{\mathrm{aa\ class\ lower\ tail}}&=0.0035.
\end{aligned}
$$
These numbers support the claim that the local tile is not a routine artifact under the tested finite null models.  They do not establish universality, mechanism, or functional necessity.

\paragraph{Leave-one-code-out recovery.}
The leave-one-code-out row evaluated $23$ code tables and $63$ total targets.  The baseline hit rate was
$$
\begin{aligned}
0.19821739130434785.
\end{aligned}
$$
Against that baseline, the tile-supported recovery rates were much larger:
$$
\begin{aligned}
\mathrm{hit\ rate\ above\ baseline} &= 0.9841269841269841,\\
\mathrm{hit\ rate\ in}\ M\setminus T &= 0.9365079365079365,\\
\mathrm{hit\ rate\ in}\ (M\setminus T)\cup \partial M &= 0.9841269841269841.
\end{aligned}
$$
The interpretation is deliberately narrow.  When alternative code-table labels are held out, the local standard-code tile and its boundary recover nearly all of the finite code-read changes in this run.  The claim is not that the tile causes the changes, nor that it predicts a protein product.  It is a code-layer readback statement: the coordinate pattern is informative about finite codon-label reassignment rows.

\paragraph{Cannot-claim boundary.}
The curated standard-code table is a label contact, not a mechanism contact.  Therefore this record cannot claim translation realization, folded structure, structural order, physical admissibility, biological function, universal mechanism, or biological necessity.  Amino-acid label-sequence readback is not folded protein structure.  A local codon tile is not a global biological law.  Window-six geometry alone is not translation, structure, physical admissibility, or function.  Any promotion beyond code-read would require a separate contact that directly observes the higher layer being claimed.

\paragraph{Local verdict.}
For the finite rows supplied here, the leave-one-code-out claim is ready at the code-read layer.  The evidence is the conjunction of the $13$-codon punctured-cube row set, the $(13,16,4)$ face-vector, the spectral consistency values, the finite null-model probabilities, the $23$-table and $63$-target leave-one-code-out recovery row, the five-family split, and the complete $13$-codon module classification.  The result should be read as a bounded BioReality record: window-six coordinates expose a local boundary-tile structure in the standard-code label table, and curated labels supply the external reality bridge, but no higher biological layer follows from this contact alone.
