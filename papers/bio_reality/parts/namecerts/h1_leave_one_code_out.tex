\subsection{Leave-One-Code-Out Stability of the Window-Six Codon Tile}\label{sec:h1-leave-one-code-out}\origin{ai}

\paragraph{Finite codon display.} The record uses the six-coordinate codon cube $Q_6$ as a finite display for standard-code label readback. The biological contact is only the curated standard-code table: each codon is assigned an amino-acid or stop label, and the test asks whether a local tile extracted from this table remains informative when one code table is held out. The relevant exceptional row set is
$$
\begin{aligned}
R = \{&\mathrm{AAA},\mathrm{AGA},\mathrm{AGG},\mathrm{AUA},\mathrm{CUA},\mathrm{CUC},\mathrm{CUG},\mathrm{CUU},\\
&\mathrm{UAA},\mathrm{UAG},\mathrm{UCA},\mathrm{UGA},\mathrm{UUA}\}.
\end{aligned}
$$
The finite audit gives $|R|=13$, and the same thirteen codons appear in the punctured-cube computation. Its face data are summarized by the vector
$$
\begin{aligned}
(f_0,f_1,f_2)=(13,16,4),
\end{aligned}
$$
so the observed row is not being treated as a free-form biological narrative: it is a specific finite subcomplex inside the codon cube.

\paragraph{Spectral and shape checks.} The associated spectral record reports $\lambda_M=0.6752479308830213$, $\lambda_R=0.4758352843197002$, and $\mu=3.051487585298128$. The radial cubic residual is $-6.224354365258478\times 10^{-12}$, the same value reported for the polynomial check, so the numerical row is stable at the stated tolerance. Against $N=2000$ sampled controls with seed $42$, the exact shape appeared once, giving $p_{\mathrm{exact}}=0.0005$. High-overlap matches with Jaccard threshold $0.9$ appeared $25$ times, giving $p_{\mathrm{Jac}\ge 0.9}=0.0125$. The sampling log records $57377$ distinct-sample attempts and $55377$ duplicate retries, which matters because the null space is visibly constrained rather than continuously resampled without collision. In the amino-acid-class-preserving control, the observed size was $|M|=20$, the exact-match count was $0$ over $2000$ trials, and the lower-tail count at or below the observed $M$ size was $7$, giving lower-tail probability $0.0035$.

\paragraph{Leave-one-code-out result.} The leave-one-code-out row evaluates $23$ code tables and $63$ held-out targets. The baseline hit rate is $0.19821739130434785$. The local row without the held-out table has hit rate $0.9365079365079365$ inside $M\setminus T$, and the same test after adjoining the boundary gives
$$
\begin{aligned}
\mathrm{hit}(M\setminus T+\partial) = 0.9841269841269841.
\end{aligned}
$$
The reported rate above baseline is also $0.9841269841269841$. Thus the finite evidence is not that codon geometry dictates biology, but that the observed code-label table contains a highly repeatable local boundary pattern under this particular leave-one-code-out test.

\paragraph{Family resolution.} The family split contains $5$ families. The family-level rates above baseline are $0.8333333333333334$ for alternative yeast, $1.0$ for bacterial and plastid tables, $1.0$ for the candidate-other family, $1.0$ for mitochondrial tables, and $1.0$ for nuclear ciliate tables. This supports the narrower reading that the boundary tile is robust across the evaluated table families, not the broader reading that all codon reassignment mechanisms have been explained.

\paragraph{Local scope.} The conclusion is deliberately confined to code-label readback. The curated table gives codon-to-label assignments, and the finite probes show a local tile with strong leave-one-code-out recovery. Nothing in this row observes translation dynamics, folded structure, physical admissibility, biochemical function, a generative mechanism, or a universal biological law. Amino-acid label sequences are still labels; they are not protein structures. Any claim about translation realization, folding, physical feasibility, organism-level function, or universality would require a separate measurement channel with its own observed rows.
