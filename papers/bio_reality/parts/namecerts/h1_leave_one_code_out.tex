\subsection{Leave-One-Code-Out Recovery on the Window-Six Codon Tile}
\label{subsec:h1-leave-one-code-out}
\origin{ai}

\paragraph{Claim.}
The claim named \texttt{h1.leave.one.code.out} records a code-read phenomenon for the standard genetic code under the six-coordinate codon display. The carrier is the codon cube $Q_6$, where each codon is represented by six binary coordinates. The finite distinctions used in this record are the six-bit display, a local tile face, a singleton label corner, and a three-point label class. The readout is not a biochemical mechanism; it is the standard codon-to-label table used as a curated contact for amino-acid and stop labels. The observed pattern is that window-six coordinates expose a sharply localized boundary structure, while the labels themselves remain supplied by the curated table.

\paragraph{Finite support set.}
The distinguished residual set $R$ has cardinality $13$. In codon notation the supplied row is
$$
\begin{aligned}

The same row is also reported as a punctured-cube shape with face vector

Thus the finite object is not merely a list of exceptional codons: it has a four-face local surface carried by thirteen vertices and sixteen edges inside the six-coordinate cube. No explicit enumerated list for the complementary $M$ row is present in the supplied data; the available finite facts record its size and its recovery behavior rather than a full codon enumeration.

The finite claim is therefore a strong code-read regularity: the coordinate tile has predictive contact with the curated label table across the evaluated code variants.

\paragraph{Null checks.}
The shape-significance run used $N=2000$ samples with seed $42$. It required $57377$ distinct-sample attempts and incurred $55377$ duplicate retries. Exactly one sampled shape matched the observed shape, giving
$$
\begin{aligned}
p_{\mathrm{exact}}=0.0005.
\end{aligned}
$$
The high-Jaccard count at threshold $0.9$ was $25$, giving
$$
\begin{aligned}
p_{J\geq 0.9}=0.0125.
\end{aligned}
$$
A second null that preserves amino-acid class also used $N=2000$ and seed $42$. It reported observed $M$-size $20$, exact-match count $0$, lower-tail count $7$, exact-shape $p$-value $0.0$, and lower-tail $p$-value $0.0035$. These null rows make the finite tile hard to dismiss as a generic random reshuffling artifact, although they still test only the codon-label layer.

\paragraph{Leave-one-codon-out companion row.}
The companion codon-removal row reports anchor count $0$, boundary count $5$, predictable count $8$, high closure coverage $8$, and recovery rate $1.0$. This supports the same local reading from the opposite direction: individual codon removal does not require a separate anchoring class in the supplied run, because all recoveries are accounted for by predictable or boundary positions.

\paragraph{Local boundary.}
The curated standard-code table is a contact for codon-to-amino-acid or stop-label assignment. It is not a contact for translation realization, folding, physical admissibility, biological function, mechanism, or universality. Consequently the present result says that the six-coordinate codon tile has a finite and unusually stable relationship to label readback. It does not say that DNA must obey this geometry as a biological necessity, that an amino-acid label sequence determines a folded protein, or that a local codon surface is a global law of organisms. Any such stronger assertion would require a separate measurement or curated contact at the relevant biological layer.
