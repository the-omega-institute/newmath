\subsection{Leave-one-code-out recovery inside a window-six codon tile}\label{sec:h1-leave-one-code-out}\origin{ai}

\paragraph{Finite codon display.}
The packet studies the standard genetic-code table only at the level of codon-to-label readback.  The finite carrier is the six-coordinate codon cube $Q_6$, with each codon treated as a six-bit display of a three-letter RNA word.  The observed contact is a curated standard-code table assigning each codon to an amino-acid label or to stop.  This chapter does not use that table as a mechanism for translation; it uses the table as a finite label source against which a local tile can be tested.

The distinguished codon set $R$ has cardinality $13$ and is exactly
$$
\begin{aligned}
R=\{&AAA,AGA,AGG,AUA,CUA,CUC,CUG,CUU,\\
&UAA,UAG,UCA,UGA,UUA\}.
\end{aligned}
$$
As a punctured-cube object its recorded face-vector is
$$
\begin{aligned}
(f_0,f_1,f_2)=(13,16,4).
\end{aligned}
$$
The same finite record gives an observed $M$-size of $20$ in the amino-acid-class preserving null experiment.  No larger biological conclusion is being read from this size: it is a combinatorial statistic of the codon-label table under the stated coordinate display.

\paragraph{Scope boundary.}
The supported conclusion is a code-layer statement.  Window-six coordinates, the punctured-cube row, the spectrum, the shape-null values, and the leave-one-code-out rates jointly support a local boundary-tile pattern for codon-label readback in the curated standard-code contact.  They do not show that translation is realized, that a protein folds, that any physical state is admissible, that a biological function is present, that a mechanism has been identified, or that the geometry is universal across biology.  Any such promotion would require a separate contact carrying the relevant observation.  In particular, an amino-acid label sequence read from the table remains a label sequence; it is not a folded structure and not an experimental function assay.
