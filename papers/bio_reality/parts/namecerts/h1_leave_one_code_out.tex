\subsection{Leave-One-Code-Out Recovery in the Window-Six Codon Cube}\label{subsec:h1-leave-one-code-out}
\origin{ai}

\paragraph{Finite display and claim.}
This record treats the standard-code codon table as a finite label readout over the six-bit codon cube $Q_6$.  The internal display is not a biochemical mechanism: it is a coordinate-level test in which codons are represented by six binary choices, local faces and corners are inspected, and amino-acid or stop labels are supplied by the curated standard-code table.  The claim tested here is that the window-six coordinate system exposes a stable local boundary pattern for code-layer label recovery.  It is therefore a statement about code readback only.  Nothing in this chapter asserts translation realization, folded structure, physical admissibility, biological function, a universal mechanism, or necessity of the genetic code.  Those would require independent contacts not present in this finite row.

\paragraph{Distinguished residual set.}
The punctured-cube residual set $R$ has cardinality $13$ and is exactly
$$
\begin{aligned}
R=\{&\mathrm{AAA},\mathrm{AGA},\mathrm{AGG},\mathrm{AUA},\mathrm{CUA},\mathrm{CUC},\mathrm{CUG},\mathrm{CUU},\\
&\mathrm{UAA},\mathrm{UAG},\mathrm{UCA},\mathrm{UGA},\mathrm{UUA}\}.
\end{aligned}
$$
The same run reports the face-vector signature
$$
\begin{aligned}
f(R)=(13,16,4),
\end{aligned}
$$
so the residual pattern is not merely a list of exceptional codons; it has a recorded local cubical profile.  The module decomposition covers all $13$ elements of $R$, with no missing codons, no extra module codons, and no unassigned entries.  The six modules are
$$
\begin{aligned}
\mathrm{agr}&=\{\mathrm{AGA},\mathrm{AGG}\},&
\mathrm{ar\_lys}&=\{\mathrm{AAA}\},&
\mathrm{cun}&=\{\mathrm{CUA},\mathrm{CUC},\mathrm{CUG},\mathrm{CUU}\},\\
\mathrm{ile\_met}&=\{\mathrm{AUA}\},&
\mathrm{ssn}&=\{\mathrm{UCA},\mathrm{UUA}\},&
\mathrm{stop\_trp}&=\{\mathrm{UAA},\mathrm{UAG},\mathrm{UGA}\}.
\end{aligned}
$$
Each module contains only predictable or boundary codons in the leave-one-codon classification, which is the local reason the row is interpreted as a boundary-tile phenomenon rather than as a free-form biological story.

\paragraph{Spectral and shape checks.}
The coordinate data also carry a finite spectral summary.  The reported values are
$$
\begin{aligned}
\lambda_M&=0.6752479308830213,\\
\lambda_R&=0.4758352843197002,\\
\mu&=3.051487585298128.
\end{aligned}
$$
The radial cubic check evaluates to $-6.224354365258478\times 10^{-12}$, the same numerical residual recorded for the cubic polynomial test, so the spectral constraint is satisfied to the stated numerical tolerance.  The shape-randomization run used $N=2000$ sampled tables with seed $42$.  Among those samples there was $1$ exact shape match, giving $p_{\mathrm{exact}}=0.0005$, and $25$ samples with Jaccard overlap at least $0.9$, giving $p_{\mathrm{Jaccard}\ge 0.9}=0.0125$.  The sampler attempted $57377$ distinct draws and recorded $55377$ duplicate retries, so the finite comparison is explicitly conditioned on a difficult discrete sampling space rather than on a smooth asymptotic approximation.

\paragraph{Class-preserving null comparison.}
A second null model preserves amino-acid class information.  In that run, again with $N=2000$ and seed $42$, the observed tile size was $20$.  The exact-shape count was $0$, so the recorded exact-shape probability is $0.0$.  The lower-tail count at or below the observed $M$ size was $7$, giving $p_{\mathrm{lower\ tail}}=0.0035$.  Thus the local tile is not explained away by simply preserving broad label classes.  This does not make the tile a law of biology; it only says that, inside the tested finite coordinate system and against the supplied null family, the observed arrangement is unusually structured.

\paragraph{Leave-one-code-out recovery.}
The main leave-one-code-out row evaluates $23$ code tables and $63$ targets.  The baseline hit rate is $0.19821739130434785$.  By contrast, the hit rate above baseline is $0.9841269841269841$, the hit rate inside $M\setminus T$ is $0.9365079365079365$, and the hit rate inside $M\setminus T$ together with the boundary allowance is $0.9841269841269841$.  In finite-count terms, the boundary-augmented predictor recovers all but one of the $63$ targets, while the baseline is near one fifth.  The family split records $5$ families: bacterial/plastid, candidate-other, mitochondrial, and nuclear-ciliate families each have hit rate $1.0$ above baseline, while the alternative-yeast family has hit rate $0.8333333333333334$.  The result is therefore not confined to a single table name, although the alternative-yeast row shows that the support is not literally uniform across every family.

\paragraph{Leave-one-codon-out compatibility.}
The codon-level holdout row is consistent with the table-level result.  It records anchor count $0$, boundary count $5$, predictable count $8$, high-closure coverage $8$, and recovery rate $1.0$.  This means that the codons singled out by the residual modules are accounted for by predictable or boundary status rather than by hidden anchor exceptions.  The local interpretation is narrow: the six-bit coordinate display supplies a finite pattern that can be used to recover withheld code labels with high success under the reported leave-one procedures.  It does not observe ribosomal translation, molecular folding, thermodynamic admissibility, organismal function, or evolutionary universality.

\paragraph{Reality contact and stopping rule.}
The only external contact used here is the curated standard genetic-code table, understood as a codon-to-label bridge at codon-label resolution.  That contact supports statements such as ``this codon is read as this amino-acid or stop label in the supplied table.''  It does not contain protein structures, kinetic data, cellular context, or functional assays.  Consequently, the finite row is accepted as a code-layer recovery record and rejected as evidence for any higher-layer biological claim.  The proper conclusion is that window-six codon coordinates expose a statistically sharp and module-complete local boundary pattern for standard-code label readback, with $R$ of size $13$, a $20$-element observed tile size in the class-preserving null row, spectral values $\lambda_M=0.6752479308830213$ and $\lambda_R=0.4758352843197002$, and leave-one-code-out recovery $0.9841269841269841$ after the boundary allowance.  Any promotion beyond label readback must be supplied by a separate, named empirical contact.
