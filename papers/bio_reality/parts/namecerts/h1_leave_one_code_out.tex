\subsection{Leave-One-Code-Out Readback for a Local Codon Tile}\label{sec:h1-leave-one-code-out}\origin{ai}

\paragraph{Carrier and contact.} The carrier for this finite-row claim is the six-bit codon cube $Q_6$, read as the $64$ codon displays over $\{\mathrm{A},\mathrm{C},\mathrm{G},\mathrm{U}\}^3$. The data used here do not treat the cube as a biological law. They treat it as a coordinate display on which local tile faces, singleton label corners, and three-point label classes can be inspected against one external contact: a curated standard-code table assigning each codon to an amino-acid or stop label. The claim is therefore a code-read claim. It is not a translation claim, not a folding claim, not a physical-admissibility claim, not a function claim, not a mechanism claim, and not a universality claim.

\paragraph{Finite boundary set.} The distinguished boundary row set has cardinality $13$. In codon notation it is
$$
\begin{aligned}
R=\{&\mathrm{AAA},\mathrm{AGA},\mathrm{AGG},\mathrm{AUA},\mathrm{CUA},\mathrm{CUC},\mathrm{CUG},\mathrm{CUU},\\
&\mathrm{UAA},\mathrm{UAG},\mathrm{UCA},\mathrm{UGA},\mathrm{UUA}\}.
\end{aligned}
$$
The same finite row is reported as a punctured-cube pattern with face vector
$$
\begin{aligned}
(f_0,f_1,f_2)=(13,16,4).
\end{aligned}
$$
Thus the row is not merely a list of exceptional labels: it carries a finite local shape in the codon-cube display. The internal structure used in this chapter consists of the coordinate presentation, the closure row, the spectral row, and the relation between code labels and the curated table. The readback is only the standard-code label table.

\paragraph{Spectral row.} The same finite construction has spectral scalars
$$
\begin{aligned}
\lambda_M&=0.6752479308830213,\\
\lambda_R&=0.4758352843197002,\\
\mu&=3.051487585298128.
\end{aligned}
$$
The radial polynomial residual reported for the row is $-6.224354365258478\times 10^{-12}$, the same number reported as the cubic-polynomial residual. Within the finite numerical tolerance of the run, this places the observed radius at $0.6752479308830213$. These numbers are used only as row diagnostics for the local display; they do not supply any biochemical mechanism.

\paragraph{Shape significance.} Against the null row sampling reported for the shape test, $N=2000$ samples required $57377$ distinct-sample attempts and produced $55377$ duplicate retries. There was $1$ exact shape match and $25$ matches with Jaccard score at least $0.9$. Hence the finite-row probabilities are
$$
\begin{aligned}
p_{\mathrm{exact}}&=0.0005,\\
p_{\mathrm{Jaccard}\geq 0.9}&=0.0125.
\end{aligned}
$$
Under the amino-acid-class-preserving null model, again with $N=2000$, the observed module size was $20$, the exact shape count was $0$, and the lower-tail count at or below the observed $M$ size was $7$. The corresponding finite-row values are
$$
\begin{aligned}
p_{\mathrm{aa\ class\ exact}}&=0.0,\\
p_{\mathrm{aa\ class\ lower\ tail}}&=0.0035.
\end{aligned}
$$
No explicit list of the $20$ elements of $M$ is part of the supplied row data, so this chapter uses the reported size and the two null-model probabilities rather than inventing an $M$ list.

\paragraph{Leave-one-code-out readback.} The leave-one-code-out test evaluates whether label changes in alternate code tables fall inside the local closure row after one code table is withheld. The run evaluated $23$ tables and $63$ total targets. The baseline hit rate was $0.19821739130434785$. The observed hit rate above baseline was $0.9841269841269841$. The hit rate inside $M\setminus T$ was $0.9365079365079365$, and the hit rate inside $M\setminus T$ together with the boundary row was $0.9841269841269841$:
$$
\begin{aligned}
\mathrm{baseline}&=0.19821739130434785,\\
\mathrm{hit}(>\mathrm{baseline})&=0.9841269841269841,\\
\mathrm{hit}(M\setminus T)&=0.9365079365079365,\\
\mathrm{hit}((M\setminus T)\cup R)&=0.9841269841269841.
\end{aligned}
$$
Thus the finite-row conclusion is narrow but strong: for this row set and this curated code-label contact, nearly all withheld-code targets are recovered by the local closure-plus-boundary readback, while a non-geometric baseline recovers about one fifth of them.

\paragraph{Family-level split.} The leave-one-code-out family row separates the $23$ evaluated tables into $5$ reported families. The hit rates above baseline are

\mathrm{alternative\ yeast}&=0.8333333333333334,\\
\mathrm{bacterial\ plastid}&=1.0,\\
\mathrm{candidate\ other}&=1.0,\\

The lowest family value is the alternative-yeast row, still above $0.8$. The other four reported families are complete at the finite resolution of the run. This supports the local-tile readback claim across the observed code-family partition, but it does not turn the tile into an evolutionary law.

\paragraph{Codon-withholding control.} The leave-one-codon-out row supplies a separate local control. It reports anchor count $0$, boundary count $5$, high closure coverage $8$, predictable count $8$, and recovery rate $1.0$. In symbols,
$$
\begin{aligned}
\mathrm{anchors}&=0,\\
\mathrm{boundary}&=5,\\
\mathrm{high\ closure\ coverage}&=8,\\
\mathrm{predictable}&=8,\\
\mathrm{recovery\ rate}&=1.0.
\end{aligned}
$$
This says that the codon-level withholding control is recovered by predictable or boundary positions, not by an additional hidden structural contact.

\paragraph{Module classification.} The module classification covers all $13$ elements of $R$. The six reported modules are
$$
\begin{aligned}
\mathrm{AGR}&=\{\mathrm{AGA},\mathrm{AGG}\},\\
\mathrm{AR/Lys}&=\{\mathrm{AAA}\},\\
\mathrm{CUN}&=\{\mathrm{CUA},\mathrm{CUC},\mathrm{CUG},\mathrm{CUU}\},\\
\mathrm{Ile/Met}&=\{\mathrm{AUA}\},\\
\mathrm{SSN}&=\{\mathrm{UCA},\mathrm{UUA}\},\\
\mathrm{Stop/Trp}&=\{\mathrm{UAA},\mathrm{UAG},\mathrm{UGA}\}.
\end{aligned}
$$
The completeness row reports $R$ size $13$, assigned count $13$, no extra module codons, no codons missing from modules, and no unassigned codons. Every module has a predictable-or-boundary witness: $\mathrm{AGR}$ contains $\mathrm{AGA},\mathrm{AGG}$; $\mathrm{AR/Lys}$ contains $\mathrm{AAA}$; $\mathrm{CUN}$ contains $\mathrm{CUA},\mathrm{CUC},\mathrm{CUG},\mathrm{CUU}$; $\mathrm{Ile/Met}$ contains $\mathrm{AUA}$; $\mathrm{SSN}$ contains $\mathrm{UCA},\mathrm{UUA}$; and $\mathrm{Stop/Trp}$ contains $\mathrm{UAA},\mathrm{UAG},\mathrm{UGA}$. This is the finite reason the boundary row can be read as a local label-split row rather than as an unstructured exception list.

\paragraph{Boundary of the claim.} The contact supplies codon-to-label assignments and nothing higher. Amino-acid label-sequence readback is not folded protein structure. A local codon-tile split is not translation realization, structural order, physical admissibility, biological function, universal mechanism, or a statement that DNA obeys this geometry by necessity. Any claim beyond code-layer readback would require a separate contact with its own observed facts. The present chapter therefore records only the finite fact that the six-bit codon display, the $13$-element boundary row, the reported spectral diagnostics, and the curated standard-code labels align with the leave-one-code-out and leave-one-codon-out recovery rows.
