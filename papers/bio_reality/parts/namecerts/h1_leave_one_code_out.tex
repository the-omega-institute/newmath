\paragraph{Carrier and contact.}
The finite carrier is the codon cube $Q_6$, read as a six-bit codon display with local tile faces, singleton label corners, and three-point label classes. The only biological contact used here is the curated standard-code table: it assigns each codon to an amino-acid or stop label and therefore supports code-layer readback. It does not observe translation realization, folded structure, physical admissibility, biological function, mechanism, or universality. All assertions in this row stay inside the code\_read layer unless a separate higher-layer contact is supplied.

Every listed module has a predictable or boundary status under the local codon-tile test. This is a finite-row statement about the display and the curated labels, not a claim that these modules implement a biochemical mechanism.

\paragraph{Local verdict.}
The row supports the following finite claim: window-six codon coordinates expose a stable local boundary-tile pattern whose curated code-label readback is strong under leave-one-code-out and leave-one-codon-out probes. The row does not claim that DNA obeys this geometry as biological necessity. Amino-acid label-sequence readback remains a code-layer observation; folded structure would require a separate structure contact, and function would require a separate functional contact.
