\paragraph{Finite carrier and observed boundary row.}
The record is a finite code-read experiment on the six-bit codon cube $Q_6$.  Its carrier is not a biochemical mechanism; it is the coordinate display in which each codon is read as a finite point, each local tile face is a finite relation, each singleton label corner is a finite label contact, and each three-point label class is a finite readback class.  The external contact is the curated standard-code table: it supplies codon-to-label assignments and nothing above that layer.  In this row the distinguished residual set $R$ has cardinality $13$ and is exactly
$$
\begin{aligned}

&\mathrm{UAA},\mathrm{UAG},\mathrm{UCA},\mathrm{UGA},\mathrm{UUA}\}.

$$
The same finite row reports the punctured-cube face count
$$
\begin{aligned}
(f_0,f_1,f_2)=(13,16,4),

$$
so the shape being tested is a small local boundary object inside $Q_6$, not the full biological space of translation or protein realization.

\paragraph{Boundary of the claim.}
The admissible conclusion is that window-six codon coordinates expose a local boundary-tile pattern in the standard-code label table.  The claim remains at the $\mathrm{code\_read}$ layer.  No translation process, folded structure, structural order, physical admissibility, biological function, universal mechanism, or biological necessity follows from these rows unless a separate reality contact supplies that higher-layer information.  In particular, amino-acid label-sequence readback is not folded protein structure, and a local codon tile is not a global biological law.  The curated table is a bridge for labels; it is not evidence for the mechanics of translation, for protein folding, for viable phenotype, or for universality across biological contexts.
