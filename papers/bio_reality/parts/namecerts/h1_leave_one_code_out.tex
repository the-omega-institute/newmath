\paragraph{Finite object.}
This BioReality record treats the standard-code codon table as a finite external contact against a six-bit codon display.  The carrier is the codon cube $Q_6$, and the only biological readback used here is the curated assignment of codons to amino-acid or stop labels.  The internal BEDC-side distinctions are coordinate, closure, spectrum, and relation data; the reality-side distinction is the standard-code label table.  The chapter tests a code-read claim: whether local window-six tile structure helps recover label changes seen across finite genetic-code tables when one code table is held out.  It does not test translation, folding, physical admissibility, function, mechanism, or universality.

$$
\begin{aligned}
\mathrm{hit\_rate\_above\_baseline} &= 0.9841269841269841,\\
\mathrm{hit\_rate\_in\_M\setminus T} &= 0.9365079365079365,\\
\mathrm{hit\_rate\_in\_M\setminus T\ plus\ boundary} &= 0.9841269841269841.
\end{aligned}
$$
Thus, for this finite code-read task, almost all held-out targets are recovered by the local $M\setminus T$ row together with the boundary allowance.  The result is a finite table-read success statement: when the curated code labels are supplied as the external contact, the window-six local tile is highly predictive for the held-out code-label targets.

\paragraph{Reality contact and boundary.}
The only external contact used here is the curated standard-code table assigning codons to amino-acid or stop labels.  That contact can test code readback and local codon-tile label splits.  It cannot test translation realization, folded structure, structural order, physical admissibility, biological function, mechanism, or universality.  The observed alignment is therefore narrow: the local tile supports code-layer readback, while the curated table blocks any higher-layer promotion.  In particular, amino-acid label-sequence readback is not folded protein structure, and a local codon tile is not a global biological law.  Any claim beyond the code layer would require a separate reality contact that directly observes the higher layer being claimed.
