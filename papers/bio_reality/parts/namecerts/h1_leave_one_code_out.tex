\paragraph{Finite carrier and reality contact.}
This record treats the standard-code codon table as a finite code-read contact on the six-bit codon cube $Q_6$.  The carrier is not a biochemical mechanism; it is the finite coordinate display in which each codon is read as a vertex and each curated label is read back from an external standard-code table.  The contact used here is the curated codon-to-label table: it observes that each codon has an amino-acid or stop label, and it has resolution only at codon-label readback.  Consequently the admissible claim is local.  Window-six coordinates may expose a boundary-tile pattern in the code layer, but the curated label table by itself supplies no translation realization, no folded structure, no physical admissibility criterion, no biological function, no mechanism, and no universality theorem.

\paragraph{Boundary verdict.}
The finite-row verdict is ready at the code-read layer.  The data support a local codon-tile boundary phenomenon on $Q_6$ with strong leave-one-code-out recovery, a thirteen-codon punctured-cube exceptional row, stable module coverage, and nontrivial sampled-null separation.  The same data also enforce a hard boundary.  A curated standard-code table is a label bridge, not a translation experiment, not a folding assay, not a physical admissibility filter, not a functional assay, not a mechanism, and not a universality proof.  Any promotion beyond codon-label readback would require a separate higher-layer reality contact.
