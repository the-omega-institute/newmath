\subsection{Leave-One-Code-Out Boundary Readback}
\label{sec:h1-leave-one-code-out}

\paragraph{Finite carrier and claim.}
The record \texttt{h1.leave.one.code.out} is a code-read claim over the six-bit codon cube $Q_6$.  The carrier is not a biological mechanism; it is a finite display in which codons are read as six binary coordinates, local tile faces, singleton label corners, and three-point label classes.  The biological contact is the curated standard-code table, used only as a codon-to-label readback bridge.  In this chapter the claim is that window-six coordinates expose a local boundary-tile structure that remains visible when whole code tables are held out, while the amino-acid and stop labels themselves remain external reality contacts.

\paragraph{Observed residual row set.}
The residual row set has cardinality $13$.  The verified row list is
$R=\{\mathrm{AAA},\mathrm{AGA},\mathrm{AGG},\mathrm{AUA},\mathrm{CUA},\mathrm{CUC},\mathrm{CUG},\mathrm{CUU},\mathrm{UAA},\mathrm{UAG},\mathrm{UCA},\mathrm{UGA},\mathrm{UUA}\}$.  The same set is certified as a punctured-cube row set with finite face-vector $(13,16,4)$.  Thus the local object has thirteen vertices, sixteen one-dimensional incidences, and four two-dimensional faces in the recorded tile complex.  This is the finite geometry being tested; it is not a translation apparatus and it is not a protein structure.

\paragraph{Leave-one-code-out probe.}
The leave-one-code-out run evaluated $23$ code tables and $63$ target rows.  The baseline hit rate is $0.19821739130434785$.  Against that baseline, the finite window recovers a hit rate of $0.9365079365079365$ inside $M\setminus T$, and $0.9841269841269841$ inside $M\setminus T$ plus the boundary.  The reported hit rate above baseline is therefore
$$\begin{aligned}
\mathrm{baseline}&=0.19821739130434785,\\
\mathrm{hit}(M\setminus T)&=0.9365079365079365,\\
\mathrm{hit}(M\setminus T\cup\partial)&=0.9841269841269841,\\
\mathrm{hit\ above\ baseline}&=0.9841269841269841.
\end{aligned}$$
The important point is not that the codon cube predicts biology by itself.  The point is narrower: after one code table is withheld, the same finite coordinate tile still recovers nearly all held-out code-read targets once the boundary rows are allowed.

\paragraph{Null comparisons.}
The shape-significance run used $N=2000$ sampled tables with seed $42$.  It required $57377$ distinct-sample attempts and incurred $55377$ duplicate retries.  The exact shape appeared once, giving $p_{\mathrm{exact}}=0.0005$.  A high-overlap threshold of Jaccard at least $0.9$ appeared $25$ times, giving $p_{J\geq 0.9}=0.0125$.  A separate amino-acid-class-preserving null with $N=2000$ and seed $42$ gave zero exact shape matches, lower-tail count $7$, observed $M$ size $20$, exact-shape value $0.0$, and lower-tail value $0.0035$.  Thus the local tile is not merely a frequent artifact of the tested randomizations.  The supplied finite record does not include the explicit ordered list of all $M$ rows, so the chapter uses the reported $M$-size and $M\setminus T$ hit-rate statistics rather than reconstructing an absent list.

\paragraph{Family readback.}
The family-level leave-one-code-out record has five families.  The hit rates above baseline are $0.8333333333333334$ for alternative yeast, $1.0$ for bacterial and plastid codes, $1.0$ for candidate other codes, $1.0$ for mitochondrial codes, and $1.0$ for nuclear ciliate codes.  This family split supports the same local reading: most families sit at perfect above-baseline recovery in the recorded probe, with alternative yeast still substantially above the baseline value $0.19821739130434785$.

\paragraph{Boundary of the claim.}
The curated standard-code table supplies codon-label readback and local label-split checks.  It does not supply translation realization, folded structure, structural order, physical admissibility, biological function, a universal mechanism, or a law saying that DNA must obey this geometry.  Amino-acid label-sequence readback is still a code-layer observation, not a folded protein.  Any promotion beyond code-read requires a separate higher-layer contact: a translation contact for translation, a structure contact for folding or structural order, a physical contact for admissibility, and a functional contact for biological function.  Within that boundary, the finite-row verdict is sharp: the window-six codon display has a thirteen-row punctured-cube residual, significant null-model separation, full module assignment, and leave-one-code-out recovery $0.9841269841269841$ when the boundary is included.
