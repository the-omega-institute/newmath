\subsection{Leave-one-code-out recovery at the code-read boundary}
\label{subsec:h1-leave-one-code-out}

\paragraph{Finite carrier and readback contact.}
The finite carrier is the six-bit codon cube $Q_6$ with a curated standard-code label table used only as a code-read contact.  The coordinate display supplies codon positions, local tile faces, singleton label corners, and three-point label classes; the external table supplies the amino-acid or stop label assigned to each codon.  The contact is therefore a label readback bridge, not a biological mechanism.  In this row the distinguished residue set $R$ has cardinality $13$ and is exactly
$$
\begin{aligned}
R = \{&AAA, AGA, AGG, AUA, CUA, CUC, CUG, CUU,\\
     &UAA, UAG, UCA, UGA, UUA\}.
\end{aligned}
$$
The same row presents $R$ as a punctured-cube object with $f$-vector $(13,16,4)$.  Thus the object being tested is not the whole genetic code as a law of nature; it is a finite local pattern inside the code table, with thirteen named codons, sixteen one-dimensional incidences, and four two-dimensional faces in the induced tile geometry.

\paragraph{Spectral and shape evidence.}
The associated spectral row records $\lambda_M = 0.6752479308830213$, $\lambda_R = 0.4758352843197002$, and $\mu = 3.051487585298128$.  The reported cubic, or radial, polynomial residual is $-6.224354365258478\times 10^{-12}$, so the numerical spectral certificate is at the displayed tolerance scale rather than at an exact symbolic identity scale.  The shape-significance row used $N=2000$ null samples with seed $42$; because distinct samples were required, it made $57377$ distinct-sample attempts and incurred $55377$ duplicate retries.  It found $1$ exact shape match and $25$ matches with Jaccard score at least $0.9$, giving
$$
\begin{aligned}
p_{\mathrm{exact}} &= 0.0005,\\
p_{\mathrm{Jaccard}\geq 0.9} &= 0.0125.
\end{aligned}
$$
The amino-acid-class-preserving null row also used $N=2000$ samples and seed $42$.  It reports observed $|M|=20$, exact-match count $0$, lower-tail count $7$, exact-shape $p$-value $0.0$, and lower-tail $p$-value $0.0035$.  These rows support the claim that the local shape is not easily reproduced under the displayed finite null models, while still leaving the biological interpretation attached to the curated code table rather than to the cube geometry alone.

\paragraph{Leave-one-code-out test.}
For the claim $h1.leave.one.code.out$, the finite evaluation covered $23$ code tables and $63$ total targets.  The baseline hit rate was $0.19821739130434785$.  The observed hit rate above baseline was $0.9841269841269841$; the hit rate inside $M\setminus T$ was $0.9365079365079365$; and the hit rate inside $M\setminus T$ plus boundary was again $0.9841269841269841$.  In finite-row terms, nearly all held-out code-table targets are recovered once the local $M\setminus T$ region and its boundary are allowed, while the baseline alone recovers about one fifth of targets.  The evidence is therefore a code-read recovery statement: after a code table is held out, its reassignment targets usually land back in the same local tile-and-boundary region determined from the six-bit codon display and the curated labels.

\paragraph{Family stratification.}
The family-stratified row divides the same test into $5$ families.  The hit rates above baseline are $1.0$ for bacterial/plastid tables, $1.0$ for candidate-other tables, $1.0$ for mitochondrial tables, $1.0$ for nuclear-ciliate tables, and $0.8333333333333334$ for alternative-yeast tables.  This stratification matters because it prevents the global rate $0.9841269841269841$ from hiding a single exceptional family.  Four families are saturated at the displayed precision, and the remaining family still exceeds the baseline by a large margin.  The row therefore supports a robust local-readback pattern across the displayed family partition, not merely an aggregate accident.

\paragraph{Codon-level companion row.}
The leave-one-codon-out companion row reports recovery rate $1.0$, predictable count $8$, boundary count $5$, anchor count $0$, and high closure coverage $8$.  These numbers match the thirteen-codon residue set: eight codons are predictable by closure coverage and five are boundary cases.  No codon is classified as an anchor in this row.  The complete module classification assigns every member of $R$ with no missing, extra, or unassigned codons: $agr=\{AGA,AGG\}$, $ar\_lys=\{AAA\}$, $cun=\{CUA,CUC,CUG,CUU\}$, $ile\_met=\{AUA\}$, $ssn=\{UCA,UUA\}$, and $stop\_trp=\{UAA,UAG,UGA\}$.  The module row reports $R$-size $13$, assigned count $13$, empty extra-module codons, empty missing-from-modules list, and empty unassigned-codons list.  Thus the finite classification is closed on the displayed residue set.

\paragraph{Boundary of the claim.}
The contact supports only codon-label readback and local split checks.  It does not test translation realization, folded protein structure, structural order, physical admissibility, biological function, universal mechanism, or biological necessity.  Amino-acid label-sequence readback is not folded protein structure.  A local codon tile is not a global biological law.  Window-six geometry alone is not translation, structure, physical admissibility, or function.  Any claim crossing from this code-read layer into translation, folding, admissibility, function, mechanism, or universality would require a separate higher-layer reality contact.  With that boundary enforced, the finite row is ready: it records a strong local code-table recovery phenomenon for the standard-code readback bridge, with explicit null-model pressure and complete module accounting on the thirteen-codon punctured cube.
