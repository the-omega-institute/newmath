\subsection{Leave-One-Code-Out Stability of the Window-Six Codon Tile}
\label{sec:bioreality-h1-leave-one-code-out}

\paragraph{Finite object under test.}
The record tests a code-layer claim about the standard genetic-code table, represented as a six-coordinate codon cube $Q_6$ with local face information and amino-acid or stop labels supplied by a curated codon table. The object of interest is not translation itself, but the finite label geometry obtained after the codons have been displayed in window-six coordinates. In that display, the distinguished residual set $R$ has cardinality $13$ and is exactly
$\{\mathrm{AAA},\mathrm{AGA},\mathrm{AGG},\mathrm{AUA},\mathrm{CUA},\mathrm{CUC},\mathrm{CUG},\mathrm{CUU},\mathrm{UAA},\mathrm{UAG},\mathrm{UCA},\mathrm{UGA},\mathrm{UUA}\}$. The same finite set is reported as a punctured-cube pattern with face-vector $(13,16,4)$, so the row evidence is not merely a count: it also fixes the observed local cubical shape.

\paragraph{Spectral and shape data.}
The associated spectral row gives $\lambda_M=0.6752479308830213$, $\lambda_R=0.4758352843197002$, and $\mu=3.051487585298128$. The recorded radial polynomial residual is $-6.224354365258478\times 10^{-12}$, matching the separately reported cubic-polynomial residual at the same numerical tolerance. In compact form, the finite spectral certificate is
$$
\begin{aligned}
\lambda_M&=0.6752479308830213,\\
\lambda_R&=0.4758352843197002,\\
\mu&=3.051487585298128,\\
P_{\mathrm{rad}}(\mu)&=-6.224354365258478\times 10^{-12}.
\end{aligned}
$$
A permutation-style comparison with $N=2000$ sampled alternatives found one exact shape match, giving $p_{\mathrm{exact}}=0.0005$, and $25$ samples with Jaccard score at least $0.9$, giving $p_{\mathrm{Jaccard}\geq 0.9}=0.0125$. The sampling record also reports $57377$ distinct-sample attempts and $55377$ duplicate retries at seed $42$, so the small exact-shape count is tied to an explicit finite sampling run rather than to an informal visual comparison.

\paragraph{Leave-one-code-out result.}
The central leave-one-code-out row evaluates $23$ code tables and $63$ total targets. Against a baseline hit rate of $0.19821739130434785$, the reported hit rate above baseline is $0.9841269841269841$. The stricter interior test $M-T$ gives hit rate $0.9365079365079365$, while adding the recorded boundary recovers $0.9841269841269841$. Thus the row-level result is that nearly all tested code changes falling into the target set are recovered by the local tile plus its boundary, while the baseline alone is far below that rate:
$$
\begin{aligned}
\mathrm{baseline}&=0.19821739130434785,\\
\mathrm{hit}(M-T)&=0.9365079365079365,\\
\mathrm{hit}(M-T+\partial)&=0.9841269841269841,\\
\mathrm{tables}&=23,\\
\mathrm{targets}&=63.
\end{aligned}
$$
The row therefore supports a code-read statement: the six-bit codon display detects a stable local boundary signal for the curated label table. It does not by itself assert that the organismal translation apparatus is generated by this geometry.

\paragraph{Family and module checks.}
The family-stratified row contains five families. The hit rates above baseline are $0.8333333333333334$ for the alternative-yeast family, $1.0$ for bacterial-plastid, $1.0$ for candidate-other, $1.0$ for mitochondrial, and $1.0$ for nuclear-ciliate. The codon-module row assigns all $13$ residual codons with no extras, no omissions, and no unassigned codons. The reported modules are
$\mathrm{agr}=\{\mathrm{AGA},\mathrm{AGG}\}$,
$\mathrm{ar\_lys}=\{\mathrm{AAA}\}$,
$\mathrm{cun}=\{\mathrm{CUA},\mathrm{CUC},\mathrm{CUG},\mathrm{CUU}\}$,
$\mathrm{ile\_met}=\{\mathrm{AUA}\}$,
$\mathrm{ssn}=\{\mathrm{UCA},\mathrm{UUA}\}$, and
$\mathrm{stop\_trp}=\{\mathrm{UAA},\mathrm{UAG},\mathrm{UGA}\}$.
This classification accounts for $13$ assigned codons out of $R$ of size $13$, so the residual set is partitioned by the recorded biological-label modules without leftover rows.

\paragraph{Single-codon removal and class-preserving null.}
The leave-one-codon-out row reports $8$ predictable codons, $5$ boundary codons, $0$ anchors, and recovery rate $1.0$; the same row records $8$ cases above the high-coverage threshold. This is compatible with the leave-one-code-out result: the predictable part and the boundary part together cover the finite residual behavior. A separate amino-acid-class-preserving null run used $N=2000$ samples and observed $M$-size $20$. It found $0$ exact matches and $7$ samples in the lower tail at or below the observed $M$-size, yielding $p_{\mathrm{class\ exact}}=0.0$ and $p_{\mathrm{class\ lower\ tail}}=0.0035$. This null row keeps the test at the codon-label layer: it shows that the recorded geometry is not easily reproduced by class-preserving randomization, but it does not convert the code table into a mechanistic or structural experiment.

\paragraph{Local boundary of the claim.}
The contact used here is a curated standard-code table assigning codons to amino-acid or stop labels. That contact is adequate for checking codon-label readback and local split behavior in $Q_6$. It is not evidence for translation realization, folded protein structure, physical admissibility, biological function, a universal mechanism, or a global biological law. Any such stronger claim would require a separate contact of the appropriate kind, such as an explicit translation-system observation, a structure experiment, an admissibility model with empirical calibration, or a functional assay. The present finite rows therefore justify only the narrower statement: window-six codon coordinates expose a statistically sharp and leave-one-out-stable local tile around the standard-code label table, with the residual set $R$ and its module partition recorded explicitly.
