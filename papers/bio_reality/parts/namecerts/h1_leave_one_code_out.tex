\subsection{Leave-One-Code-Out Boundary Readback for the Standard Code}

\paragraph{Finite carrier.}
The finite carrier is the six-bit codon cube $Q_6$, read through a standard-code label table.  A codon is treated here only as a finite row with a displayed coordinate and an externally supplied label.  The relevant distinction is therefore not translation as a cellular process, but code-layer readback: a curated table assigns each codon to an amino-acid or stop label, and the coordinate system is tested against that assignment.  The BEDC-minimal form has carrier $Q_6$, distinctions given by the six-bit codon display, local tile face, singleton label corner, and three-point label class, and internal structure given by coordinate, closure, spectrum, and relation.  The biological object is the standard-code codon table, but the claimed layer is only $\mathrm{code\_read}$.

$$
\begin{aligned}

The numerical content is therefore a finite recovery statement: when one code table is withheld, the local coordinate-and-boundary row structure recovers nearly all target labels above the baseline rate.  It is not a claim that the coordinate geometry implements translation.

\paragraph{Reality contact.}
The only reality contact used here is the curated standard-code table.  Its observed fact is that a curated standard-code table assigns codons to amino-acid or stop labels.  Its resolution is codon-label readback.  The known bias is exactly the scope condition: this table is a bridge for code labels, not a mechanism or universality proof.  Consequently the support condition is narrow.
