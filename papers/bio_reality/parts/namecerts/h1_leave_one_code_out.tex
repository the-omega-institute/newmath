\paragraph{Finite carrier and contact.}
This record treats the standard genetic-code table only as a finite code-layer contact. The carrier is the six-bit codon cube $Q_6$, read as a coordinate display for the sixty-four codons. The external contact is a curated standard-code table assigning each codon to an amino-acid or stop label. The contact is therefore a label readback bridge: it can test code labels and local codon-tile splits, but it is not a translation experiment, not a folding experiment, not a physical-admissibility test, not a function assay, not a mechanism, and not a universality theorem. The finite row under audit is the leave-one-code-out claim $h1.leave.one.code.out$: after withholding code-table rows, the window-six coordinate construction recovers most withheld label targets from the local tile and its boundary.
