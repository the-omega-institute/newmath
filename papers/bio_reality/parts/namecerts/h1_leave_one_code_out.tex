\subsection{Leave-one-code-out recovery on the codon tile}
\label{subsec:h1-leave-one-code-out}

\paragraph{Claim.}
The finite row for claim $\mathrm{h1.leave.one.code.out}$ tests whether a window-six codon coordinate model retains local predictive force when one curated genetic-code table is withheld. The object under study is not translation as a biochemical process. It is the code-level readback problem in which codons are represented in a six-bit cube $Q_6$ and then compared with a curated standard-code label table. The tested layer is therefore the codon-label layer: the data ask whether local tile structure helps recover label changes across known code tables, while the amino-acid and stop labels themselves remain supplied by the curated table contact.

\paragraph{Scope boundary.}
The curated standard-code table supplies codon-to-label assignments and nothing beyond that layer. Consequently this chapter does not assert translation realization, folded structure, physical admissibility, biological function, mechanism, or universality. An amino-acid label sequence is not a protein fold, and a local codon tile is not a global law of biology. Any passage from code labels to translation dynamics, structural order, function, or physical admissibility would require a separate empirical contact designed for that layer. The present finite result is therefore sharply delimited: window-six codon geometry exposes a strong local code-read pattern in the supplied finite rows, while higher biological claims remain outside the data recorded here.
