\paragraph{Finite carrier.}
The finite carrier is the six-bit codon cube $Q_6$, read only as a codon-coordinate display.  A codon is a finite row in this display, and the reality contact is the curated standard-code label table assigning each codon to an amino-acid or stop label.  The contact therefore supports code-layer readback and local label-split checks, but it is not a translation apparatus, a folding experiment, a physical-admissibility test, a function assay, a mechanism, or a universality theorem.

$$
\begin{aligned}
R=\{&\mathrm{AAA},\mathrm{AGA},\mathrm{AGG},\mathrm{AUA},\mathrm{CUA},\mathrm{CUC},\mathrm{CUG},\mathrm{CUU},\\
&\mathrm{UAA},\mathrm{UAG},\mathrm{UCA},\mathrm{UGA},\mathrm{UUA}\}.

$$
The same row is reported as a punctured-cube pattern with face-vector
$$
\begin{aligned}
(f_0,f_1,f_2)=(13,16,4).

$$
This is a finite combinatorial statement about the local codon tile.  It says that the selected label-sensitive codons form a structured row in the coordinate display; it does not say that the biological code is forced by this geometry.

\paragraph{Leave-one-code-out row.}
The leave-one-code-out experiment evaluates $23$ tables and $63$ targets.  Its baseline hit rate is $0.19821739130434785$.  The hit rate above baseline is $0.9841269841269841$, the hit rate inside $M-T$ is $0.9365079365079365$, and the hit rate inside $M-T$ plus boundary is $0.9841269841269841$.  In finite-row form,
$$
\begin{aligned}
\mathrm{tables}&=23,\\
\mathrm{targets}&=63,\\
\mathrm{baseline}&=0.19821739130434785,\\
\mathrm{hit}_{>\mathrm{baseline}}&=0.9841269841269841,\\
\mathrm{hit}_{M-T}&=0.9365079365079365,\\
\mathrm{hit}_{M-T+\partial}&=0.9841269841269841.
\end{aligned}
$$
The chapter-specific readback is therefore narrow: when one code table is left out, the local codon-tile geometry and its boundary recover almost all of the target label behavior relative to the baseline.  The result is a code-read regularity, not a claim that the table is generated by geometry alone.

\paragraph{Reality-contact boundary.}
The only reality contact used here is the curated standard genetic-code bridge.  It observes that a standard-code table assigns codons to amino-acid or stop labels.  It can test code readback and local codon-tile label splits.  It cannot test translation realization, folded structure, structural order, physical admissibility, biological function, universal mechanism, or the claim that DNA must obey this BEDC coordinate geometry.  Amino-acid label-sequence readback is therefore not folded protein structure.  Any statement crossing from this chapter into translation, folding, admissibility, function, mechanism, or universality requires a separate contact supplying that higher-layer evidence.

\paragraph{Local verdict.}
The finite-row evidence supports the following bounded claim.  In the six-bit codon cube $Q_6$, the standard-code label table exposes a thirteen-codon punctured-cube boundary row with reproducible spectral and module structure.  Leave-one-code-out testing over $23$ tables and $63$ targets gives a hit rate of $0.9841269841269841$ above baseline, compared with a baseline of $0.19821739130434785$, and the module classification assigns all $13$ boundary codons without residue.  The result is ready as a code-read chapter because it is finite, reproducible, and locally scoped.  It remains deliberately blocked from any higher biological promotion unless a separate reality contact is supplied.
