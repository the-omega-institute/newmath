\subsection{Leave-One-Code-Out Readback on the Window-Six Codon Tile}\label{subsec:h1-leave-one-code-out}\origin{ai}

\paragraph{Finite object.} The carrier is the six-bit codon cube $Q_6$, read through the ordinary four-letter codon display. A vertex is a codon, and the only external biological contact used in this row is the curated standard-code table assigning each codon to an amino-acid label or to stop. The BEDC-side object is therefore a code-layer readback object: it contains coordinates, closure, spectrum, and relation data, but it does not contain translation realization, folded structure, physical admissibility, biological function, mechanism, or universality. Those higher-layer assertions remain outside the row unless a separate contact supplies them.

\paragraph{Boundary set.} The relation row $R$ has cardinality $13$. Its members are
$$\begin{aligned}
R=\{&AAA, AGA, AGG, AUA, CUA, CUC, CUG, CUU,\\
&UAA, UAG, UCA, UGA, UUA\}.
\end{aligned}$$
The same row is recorded as a punctured-cube boundary with $f$-vector $(13,16,4)$. Thus the finite certificate is not merely a count: it identifies a concrete codon list and a local tile shape in $Q_6$. The module partition is complete on this same $13$-codon set: the $agr$ module is $\{AGA,AGG\}$, $ar\_lys$ is $\{AAA\}$, $cun$ is $\{CUA,CUC,CUG,CUU\}$, $ile\_met$ is $\{AUA\}$, $ssn$ is $\{UCA,UUA\}$, and $stop\_trp$ is $\{UAA,UAG,UGA\}$. The module audit records $R$-size $13$, assigned count $13$, no missing codons, no extra module codons, and no unassigned codons.

\paragraph{Spectral row.} The associated spectrum records $\lambda_M=0.6752479308830213$, $\lambda_R=0.4758352843197002$, and $\mu=3.051487585298128$. The cubic or radial polynomial residual is $-6.224354365258478\cdot 10^{-12}$, so the row is numerically closed to the displayed tolerance rather than being a purely descriptive table. The finite spectral statement used here is only that the local codon-tile relation has this recorded spectrum; it is not a claim that spectra determine biological realization.

\paragraph{Shape comparison.} Against $N=2000$ sampled comparison rows with seed $42$, the exact-shape count is $1$, giving $p_{\mathrm{exact}}=0.0005$. The high-overlap count at Jaccard threshold $0.9$ is $25$, giving $p_{J\geq 0.9}=0.0125$. The same run reports $57377$ distinct-sample attempts and $55377$ duplicate retries. In the amino-acid-class-preserving null row, again with $N=2000$ and seed $42$, the observed $M$-size is $20$, the exact-match count is $0$, the lower-tail count at or below the observed $M$-size is $7$, the exact-shape value is $0.0$, and the lower-tail value is $0.0035$. These numbers support a finite local-code signal; they do not promote the row beyond code labels.

\paragraph{Leave-one-code-out row.} The leave-one-code-out experiment evaluates $23$ code tables and $63$ total targets. The baseline hit rate is $0.19821739130434785$. The observed hit rate above baseline is $0.9841269841269841$, the hit rate inside $M-T$ is $0.9365079365079365$, and the hit rate inside $M-T$ plus boundary is $0.9841269841269841$. By family, the hit rates above baseline are $1.0$ for bacterial-plastid, $1.0$ for candidate-other, $1.0$ for mitochondrial, $1.0$ for nuclear-ciliate, and $0.8333333333333334$ for alternative-yeast, across $5$ families. The chapter claim is therefore narrow: when a code table is held out, the local finite tile and its boundary recover most code-layer targets relative to the baseline row.

\paragraph{Leave-one-codon-out check.} The companion leave-one-codon-out row records anchor count $0$, boundary count $5$, predictable count $8$, high closure coverage $8$, and recovery rate $1.0$. The recovery-rate-above-threshold scalar is also $1.0$. This supports the same interpretation as the code-table row: the signal is concentrated in predictable or boundary positions of the local codon tile, not in an independent biological mechanism.

\paragraph{Reality contact and cannot-claim boundary.} The contact is the curated standard-code table, used only at codon-label resolution. It can test code readback and local codon-tile label split. It cannot test translation realization, folded protein structure, structural order, physical admissibility, biological function, universal mechanism, or the claim that DNA obeys BEDC geometry as biological necessity. Amino-acid label-sequence readback is therefore not folded protein structure. The correct local verdict is that window-six coordinates expose a finite boundary-tile pattern in the standard-code label table, while the labels themselves remain external reality contacts.
