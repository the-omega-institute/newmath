$\{AAA, AGA, AGG, AUA, CUA, CUC, CUG, CUU, UAA, UAG, UCA, UGA, UUA\}$.
The audit attached to experiment run \texttt{338799579a87} reports no missing codons and no extra codons relative to this motif union. Thus the row is not being used as a statistical extrapolation from an unlisted population: the certified object is the thirteen-entry set displayed above, together with the assertion that the computed representative set has empty discrepancy lists. In set-count form, the local record says
$$
\begin{aligned}

\mathrm{missing}(R) &= \varnothing,\\

$$
This is the biological content permitted at this row: a finite RNA-codon motif packet whose enumerated entries are known exactly within the experiment ledger.

The quotient data remain a finite mathematical signature until a separate biological observation is supplied for the biological layer under discussion.

\paragraph{ORF-eligibility interpretation.}
The associated ORF seed concerns eligibility from start and stop codon enumeration. In the stricter boundary form, a start/stop window is not counted as an observed open reading frame merely because a finite codon geometry supplies compatible endpoints. The start/stop-window row \texttt{h2.orf.start\_stop.window\_eligibility} records one loaded external window and one classified input window, with scope restricted to external window start/stop enumeration only. It also records the local guard \texttt{no\_geometry\_to\_translation\_promotion}$=\texttt{orf\_eligibility\_only}$. Thus the row permits only the following statement: in the supplied finite window manifest, the classifier assigned the single provided window at the ORF-eligibility layer. It does not license translation, folding, physical admissibility, biological function, mechanism, or universality.

\paragraph{Curated interval contact.}
A later curated interval contact supplies a stronger, still limited, biological test. The row \texttt{h2.orf\_eligibility.curated\_interval\_contact} records a dataset with $168$ rows from NCBI RefSeq GenBank XML CDS intervals for \emph{Escherichia coli} K-12 MG1655, accession NC\_000913.3. The input classes present are boundary cases, matched non-ORF decoys, and positive curated ORFs. The positive interval test accepts $72$ of $72$ positives, and the matched decoy test rejects $72$ of $72$ decoys. Both exact one-sided binomial computations report upper-tail $p=2.117582368135751\times 10^{-22}$. The paired baseline sign test likewise records $72$ wins in $72$ pairs with the same upper-tail value. The held-out stability check contains three splits, each with $24$ positives accepted at rate $1.0$ and $24$ decoys rejected at rate $1.0$. Boundary controls record $24$ cases with $0$ promoted cases and promoted fraction $0.0$.
$$
\begin{aligned}
\mathrm{rows} &= 168,\\
\mathrm{positives\ accepted} &= 72/72,\\
\mathrm{decoys\ rejected} &= 72/72,\\
\mathrm{paired\ wins} &= 72/72,\\
p &= 2.117582368135751\times 10^{-22},\\
\mathrm{boundary\ promotions} &= 0/24.
\end{aligned}
$$
This contact is external curated biological evidence for the interval-eligibility layer only. It improves the row from a contact-gap seed to a tested interval-eligibility record on the stated dataset, but the recorded no-promotion clauses remain part of the result.
