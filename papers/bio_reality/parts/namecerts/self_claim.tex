\subsection{Self-check boundary for open-reading-frame eligibility}\label{sec:bioreality-self-claim-orf-eligibility-boundary}
\origin{ai}

\paragraph{Finite claim being audited.}
The record named \texttt{self.claim} is a self-audit gate for the biological layer called \texttt{orf\_eligibility}. Its own checked value is deliberately negative in scope: the boundary field says that this is a BioReality self-check only, and that no codon geometry is carried across biological layers. The associated gate count is $3$. Thus the object under discussion is not an asserted open reading frame, not a translated product, and not a functional gene call. It is a finite audit packet whose role is to keep a start--stop enumeration claim from being mistaken for a biological observation.

\paragraph{Codon-level finite support.}
The row contains a verified motif-compression fact for the set $R$. In this contact, $R$ is exactly the following thirteen codons:
$$
\begin{aligned}
R=\{&AAA, AGA, AGG, AUA, CUA, CUC, CUG, CUU,\\
&UAA, UAG, UCA, UGA, UUA\}.
\end{aligned}
$$
The same finite packet verifies that there are no missing and no extra codons in this motif union. The larger finite set $M$ has cardinality $20$ and is identified with $WNR\cup CUN$:
$$
\begin{aligned}
M=\{&AAA, AAG, ACA, ACG, AGA, AGG, AUA, AUG,\\
&CUA, CUC, CUG, CUU, UAA, UAG, UCA, UCG,\\
&UGA, UGG, UUA, UUG\}.
\end{aligned}
$$
A saturation check supplies the intermediate list $\operatorname{Sat}_{f3}(R)$ as the eighteen-element set obtained from $M$ by excluding $ACA$ and $ACG$:
$$
\begin{aligned}
\operatorname{Sat}_{f3}(R)=\{&AAA, AAG, AGA, AGG, AUA, AUG, CUA, CUC,\\
&CUG, CUU, UAA, UAG, UCA, UCG, UGA, UGG,\\
&UUA, UUG\}.
\end{aligned}
$$
The same row states $M=\operatorname{Med}(R)\cup\{ACA\}$ and records an empty $f3$ boundary. These are finite combinatorial facts about codon lists. They do not by themselves identify a realized biological interval.

The finite row also records an empty product-decomposition witness field for this entry. The appropriate reading is therefore conservative: the quotient data organizes codon geometry, but it does not license a biological conclusion beyond the audited layer.

\paragraph{Start--stop window enumeration.}
One linked row supplies a start--stop window eligibility check. It loaded one external nucleotide-window dataset, with snapshot date $2026$-$05$-$25$, and classified all $1$ of $1$ input windows by start--stop enumeration. Its scope field is explicitly limited to external-window start--stop enumeration. The row also records the non-promotion value \texttt{orf\_eligibility\_only}. Consequently, this evidence can support only a statement about whether a supplied window passes the enumerated start--stop boundary test. It is not evidence that translation occurred, that a protein folded, that a molecule is physically admissible, or that any biological function is present.

\paragraph{Curated interval contact.}

The required input classes are present: boundary cases, matched non-ORF decoys, and positive curated ORFs. The protocol was frozen at $2026$-$05$-$29T19:11:13+00:00$, before the external evidence time $2026$-$05$-$30T10:29:31+00:00$; the row states that classifier rules, input classes, held-out splits, and baselines were frozen, and that no translation or structure input entered the analysis.

These figures strengthen the local interval-contact claim while keeping the biological scope narrow: they test the frozen start--stop interval classifier against curated interval rows and matched controls, not against translation, folding, mechanism, or function.

\paragraph{Local negative scope.}
The finite facts above allow the following bounded conclusion. Codon-set geometry, quotient rows, one loaded start--stop window, and the curated $168$-row interval contact can support only an open-reading-frame eligibility statement at the specified external interval-contact layer. No translation realization follows. No folded structure follows. No physical admissibility follows. No biological function follows. No molecular mechanism follows. No universal law about genomes, genes, or proteins follows. Any assertion at one of those stronger layers would need its own separately attached observation and its own audit row. In the absence of such a contact, the self-check record remains a guardrail: it records that the finite geometry is useful for eligibility testing only when an appropriate external interval is supplied, and that the geometry alone is not a biological realization.
