\subsection{Window-six boundary row for the standard code}
\label{subsec:bioreality-h0-r-cardinality-13}

\paragraph{Finite object.}
The record concerns the standard-code codon table viewed through a six-bit codon display. The carrier is the codon cube $Q_6$, with local distinctions given by a tile face, a singleton label corner, and a three-point label class. The biological contact is not a molecular mechanism; it is the curated table that assigns each codon to an amino-acid or stop label. The row therefore lives at the code-read layer: it tests whether a finite coordinate description singles out a local boundary pattern in the label table.

\paragraph{Observed boundary set.}
The verified boundary row $R$ has cardinality $13$. Its complete codon list is
$$
\begin{aligned}
R=\{&AAA, AGA, AGG, AUA, CUA, CUC, CUG, CUU,\\
&UAA, UAG, UCA, UGA, UUA\}.
\end{aligned}
$$
The same row was checked as a punctured-cube pattern with $f$-vector $(13,16,4)$. Thus the finite geometry is not merely a count: the thirteen selected codons form a local cubical boundary shape with sixteen one-dimensional incidences and four two-dimensional faces in the tested representation.

\paragraph{Spectral coordinates.}
The row also carries a numerical spectral readout. The reported values are $\lambda_M=0.6752479308830213$, $\lambda_R=0.4758352843197002$, and $\mu=3.051487585298128$. The radial cubic residual was $-6.224354365258478\times 10^{-12}$, the same value reported for the cubic-polynomial check, so the finite numerical test is at roundoff scale rather than a visible discrepancy. This supports using the row as a stable local coordinate feature of the code table, while still leaving biological interpretation to the separate contact data.

\paragraph{Randomization checks.}
Against the shape randomization with $N=2000$, the run found $1$ exact shape match and $25$ samples with Jaccard score at least $0.9$. The corresponding probabilities were $p_{\mathrm{exact}}=0.0005$ and $p_{J\geq 0.9}=0.0125$. The run required $57377$ distinct-sample attempts and recorded $55377$ duplicate retries under seed $42$, so the finite sampling was constrained by repeated candidates rather than by a large supply of independent alternatives. Under the amino-acid-class-preserving null model, again with $N=2000$ and seed $42$, the observed $M$-size was $20$, exact matches were $0$, and $7$ samples lay in the lower tail at or below the observed $M$-size. The corresponding values were $p_{\mathrm{class\ exact}}=0.0$ and $p_{\mathrm{lower\ tail}}=0.0035$.

\paragraph{Leave-out behavior.}
The code-table leave-one-code-out test evaluated $23$ tables and $63$ total targets. Its baseline hit rate was $0.19821739130434785$, while the observed rate above baseline was $0.9841269841269841$. The rate inside $M-T$ was $0.9365079365079365$, and the rate inside $M-T$ plus boundary was $0.9841269841269841$. In the leave-one-codon-out test, there were $0$ anchors, $5$ boundary cases, $8$ high-coverage closure cases, and $8$ predictable cases, with recovery rate $1.0$. These numbers say that the row is useful for code-label recovery inside this finite table experiment; they do not say that codons are physically produced, translated, folded, or selected by this geometry.

\paragraph{Family and module split.}
The family-level leave-one-code-out table contains $5$ families. The hit-rate-above-baseline values were $1.0$ for bacterial-plastid, $1.0$ for candidate-other, $1.0$ for mitochondrial, $1.0$ for nuclear-ciliate, and $0.8333333333333334$ for alternative-yeast. The module classification accounts for all thirteen row codons: $agr=\{AGA,AGG\}$, $ar\_lys=\{AAA\}$, $cun=\{CUA,CUC,CUG,CUU\}$, $ile\_met=\{AUA\}$, $ssn=\{UCA,UUA\}$, and $stop\_trp=\{UAA,UAG,UGA\}$. The reported completeness data are $R\_size=13$, assigned count $13$, no extra module codons, no missing row codons, and no unassigned codons. This is the strongest finite bookkeeping claim in the row: the boundary set is completely partitioned by the listed module names.

\paragraph{Local scientific interpretation.}
The contact supports one statement: the curated standard-code table can be read at the codon-label level, and within that table the thirteen-codon row behaves as a compact local boundary pattern. The singleton-corner and three-point-class language should be read as coordinate structure on the label table, not as an assertion about a cellular process. The finite evidence is compatible with a meaningful code-layer split, especially because the exact-shape and amino-acid-class-preserving checks give small empirical probabilities, and because leave-out recovery remains high across the reported tests.

\paragraph{Scope boundary.}
Nothing in this row establishes translation realization, folded protein structure, physical admissibility, biological function, a molecular mechanism, or a universal biological law. Amino-acid label readback is still label readback; it is not a structure contact. CUN or UUR module wording and tRNA-Leu notation, if used elsewhere, also remain outside this row unless paired with an organism-matched contact. To cross from the code-read layer to translation, folding, physical admissibility, function, mechanism, or universality, a separate observation would have to be supplied and tested on its own terms.
