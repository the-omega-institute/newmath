\paragraph{Finite codon carrier.}
The packet uses the six-coordinate codon cube $Q_6$ as its finite display space.  A codon is read as a six-bit window coordinate, while the biological label attached to that coordinate is not generated by the cube itself.  The observed code-layer contact is a curated standard-code table assigning each codon to an amino-acid or stop label.  Within that contact, the distinguished boundary row is the following thirteen-codon subset:
$$
\begin{aligned}
R=\{&\mathrm{AAA},\mathrm{AGA},\mathrm{AGG},\mathrm{AUA},\mathrm{CUA},\mathrm{CUC},\mathrm{CUG},\mathrm{CUU},\\
&\mathrm{UAA},\mathrm{UAG},\mathrm{UCA},\mathrm{UGA},\mathrm{UUA}\}.
\end{aligned}
$$
The row count is therefore $|R|=13$.  This is a finite code-read fact: it says that this window-six selection picks out exactly thirteen codon labels in the supplied table.  It does not by itself say that translation occurs, that a protein fold is realized, that a physical pathway is admissible, that a biological function follows, that a mechanism has been identified, or that the same local pattern is a universal biological law.

The tile may be used to ask whether the same boundary is visible under perturbations of code labels or codon omissions, but the tile remains a code-layer object unless another observation supplies a higher-level biological interpretation.

\paragraph{Scope of the claim.}
The claim supported here is exactly the following finite statement: in the supplied standard-code table, the window-six codon display contains a thirteen-element boundary row $R$ with the listed codons, punctured-cube face data $(13,16,4)$, the recorded spectral values, significant finite null comparisons, strong leave-one recovery statistics, and a complete six-module assignment.  The claim stops there.  Any assertion about translation, protein structure, physical admissibility, biological function, mechanism, or universal law must be backed by a separate contact that directly observes that higher layer.  Without such a contact, amino-acid label readback remains label readback, not folded structure or functional biology.
