\paragraph{Boundary interpretation for ORF eligibility.}
The biological object attached to this row is open-reading-frame eligibility from start/stop codon enumeration.  In the seed record, the biological carrier, distinctions, and readback are all marked as awaiting contact with observed biological material.  This is why the row must be read as a boundary certificate: it identifies a finite codon-side test that could participate in an ORF-eligibility assay, but it is not already an assay result.  The two probes state the operational failure condition.  The row would break its own scope if it asserted ORF eligibility, translation, folded structure, physical admissibility, biological function, or mechanism from the codon-window calculation alone.  It remains within scope only while those stronger biological claims are withheld unless a separate curated sequence, transcript, or annotation input supplies the interval to be tested.
