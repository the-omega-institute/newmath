\paragraph{Finite row.}
This record concerns the standard-code codon table only at the code-read layer. The carrier is the six-bit codon cube $Q_6$, read as a finite display of codon coordinates, local tile faces, singleton label corners, and three-point label classes. The distinguished row $R$ is the finite boundary set detected by the coordinate-and-closure construction. In the verified run with identifier $\mathrm{e19917012c60}$, the row is exactly
$$
\begin{aligned}
R = \{&\mathrm{AAA},\mathrm{AGA},\mathrm{AGG},\mathrm{AUA},\mathrm{CUA},\mathrm{CUC},\mathrm{CUG},\mathrm{CUU},\\
&\mathrm{UAA},\mathrm{UAG},\mathrm{UCA},\mathrm{UGA},\mathrm{UUA}\}.
\end{aligned}
$$
The independent cardinality check with run identifier $\mathrm{5d4a3459997d}$ gives $|R|=13$ and returns the same thirteen codons. The punctured-cube face count is
$$
\begin{aligned}
f(R)=(f_0,f_1,f_2)=(13,16,4),
\end{aligned}
$$
so the observed object is not merely a list of codons; it is a finite local cubical patch with thirteen vertices, sixteen one-dimensional incidences, and four tile faces. This is the concrete content of the phrase ``punctured cube'' in this chapter.

\paragraph{Label readback.}
The external contact is the curated standard genetic-code table. It assigns codons to amino-acid or stop labels, and in this record it is treated as a reality bridge for label readback. It is not treated as a mechanism for translation, a model of folding, a physical admissibility test, or a function assay. Under that contact, the thirteen codons in $R$ decompose into the six recorded modules
$$
\begin{aligned}
\mathrm{agr} &= \{\mathrm{AGA},\mathrm{AGG}\},\\
\mathrm{ar\_lys} &= \{\mathrm{AAA}\},\\
\mathrm{cun} &= \{\mathrm{CUA},\mathrm{CUC},\mathrm{CUG},\mathrm{CUU}\},\\

The module-classification check reports $R\_\mathrm{size}=13$, $\mathrm{assigned\_count}=13$, no missing codons, no extra module codons, and no unassigned codons. The local tile therefore has a complete code-read partition relative to the curated table.

\paragraph{Null comparison.}
The finite-row shape is not recorded as an unavoidable consequence of the table size alone. In the shape-significance run $\mathrm{0b267c88f8e4}$, with $N=2000$ samples and seed $42$, there is $1$ exact shape match, giving $p_{\mathrm{exact}}=0.0005$. The same run gives $25$ matches at Jaccard threshold $0.9$, giving $p_{\mathrm{Jaccard}\ge 0.9}=0.0125$. The sampler required $57377$ distinct-sample attempts and recorded $55377$ duplicate retries, which should be kept visible because the empirical null is a finite-row randomized comparison rather than a closed-form distribution. The amino-acid-class-preserving null run $\mathrm{e8d5de747a0c}$ has $N=2000$, seed $42$, observed $M$ size $20$, exact-match count $0$, and lower-tail count $7$. Its recorded values are
$$
\begin{aligned}
p_{\mathrm{aa\ class\ exact}} &= 0.0,\\
p_{\mathrm{aa\ class\ lower\ tail}} &= 0.0035,\\
|M|_{\mathrm{observed}} &= 20.
\end{aligned}
$$
These null comparisons support the statement that the observed local shape is statistically sharp under the recorded finite randomizations. They do not by themselves promote the row to a global biological principle.

\paragraph{Cannot-claim boundary.}
The chapter makes a deliberately narrow claim. The curated standard-code table supplies codon-to-label readback, and the six-bit coordinate construction supplies a finite local tile with a punctured-cube boundary. From these data one may state that the row $R$ is a thirteen-codon local code-read boundary whose module classification is complete and whose recorded null comparisons are sharp. One may not infer that DNA obeys this geometry as a biological necessity. One may not infer translation realization, folded protein structure, structural order, physical admissibility, biological function, universal mechanism, or universality of the code. Amino-acid label-sequence readback remains a code-layer observation. Any claim beyond the code-read layer requires a separate contact, such as an explicit translation experiment, structure contact, physical assay, or function assay. Without such a contact, the punctured cube is a local finite-row reality record, not a global biological law.
