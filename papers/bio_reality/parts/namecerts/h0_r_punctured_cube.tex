\paragraph{Finite carrier.}

$$
The finite-row audit records both $|R|=13$ and exact agreement between this set and the expected codon rows.  In the cubical display the same row set has face-vector
$$
\begin{aligned}

The word ``punctured'' is used only at this finite-coordinate level.  It names the observed cube-combinatorics of the row set and does not assert any global biological law.

\paragraph{Label contact.}
The only external contact used here is the curated standard genetic-code table, which assigns each codon to an amino-acid or stop label.  This contact has resolution ``codon label readback.''  It can test code-layer statements such as whether the thirteen displayed codons form the claimed local tile and how their labels split across singleton corners, local tile faces, and three-point label classes.  It cannot test translation realization, folded structure, physical admissibility, biological function, a universal mechanism, or any necessity claim about DNA geometry.  Thus the row set $R$ is a code-read object: it is a finite table pattern in $Q_6$ plus an external label readback, not a mechanism.

\paragraph{Null-model pressure.}
Against the shape null with $N=2000$ samples and seed $42$, the exact shape matched once.  The exact-shape probability is therefore
$$
\begin{aligned}
p_{\mathrm{exact}}=0.0005,
\end{aligned}
$$
with $25$ high-overlap cases at Jaccard threshold $0.9$, giving
$$
\begin{aligned}
p_{J\geq 0.9}=0.0125.
\end{aligned}
$$
The run required $57377$ distinct-sample attempts and recorded $55377$ duplicate retries, so the finite sampling space was highly constrained by the preservation rules.  A second amino-acid-class-preserving null, also with $N=2000$ and seed $42$, recorded observed $|M|=20$, exact-match count $0$, and lower-tail count $7$.  Its exact-shape probability is $0.0$, while the lower-tail probability is

\begin{aligned}

\end{aligned}
$$
These two null contacts say that the row shape is not a common artifact under the tested finite randomizations.  They do not say that the code must have arisen from the cube geometry.
