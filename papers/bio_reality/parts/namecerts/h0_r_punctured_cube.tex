\paragraph{Interpretation of the puncture.}
The phrase punctured cube is used here only for the finite row structure returned by the six-bit coordinate calculation. The row keeps four BEDC distinctions separate: the six-bit codon display supplies the carrier, the local tile face supplies the neighborhood structure, the singleton label corner records a one-codon corner phenomenon, and the three-point label class records a small label-coherent class inside the same finite display. \paragraph{Null-model pressure.}
Two finite null checks constrain how much of the row can be dismissed as a random label artifact.  In the shape-significance run with $N=2000$ sampled alternatives and seed $42$, the exact shape appeared once, giving $p_{exact}=0.0005$.  High-overlap alternatives with Jaccard score at least $0.9$ appeared twenty-five times, giving $p_{J\geq 0.9}=0.0125$.  The sampler records $57377$ distinct-sample attempts and $55377$ duplicate retries, so the rare-shape conclusion is tied to a concrete finite sampling process rather than to an asymptotic approximation.  A separate amino-acid-class-preserving null with $N=2000$ and seed $42$ reports no exact matches, hence $p=0.0$ for the exact-shape count in that finite run.  The same class-preserving run reports observed $M$ size $20$, lower-tail count $7$, and lower-tail value $0.0035$.  The supplied data do not include a codon-by-codon list for $M$, so this chapter uses only the recorded scalar $M$ size and the reported lower-tail statistic.
