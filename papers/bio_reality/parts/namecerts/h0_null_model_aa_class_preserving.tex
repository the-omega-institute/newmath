\paragraph{Finite carrier and readback.}
The record \texttt{h0.null.model.aa\_class\_preserving} concerns the standard-code codon table only at the code-read layer. The carrier is the six-bit codon cube $Q_6$, read through a curated standard-code label table. The internal BEDC distinctions used here are the six-bit codon display, local tile face, singleton label corner, and three-point label class; the internal structure is coordinate, closure, spectrum, and relation. The external contact is not a biochemical mechanism. It is the finite table assigning each codon to an amino-acid or stop label, and therefore it can test codon-label readback and local label splits but cannot test translation realization, folded structure, physical admissibility, function realization, mechanism, or universality.

\paragraph{Observed boundary set.}
The finite relation $R$ has cardinality $13$. Its listed codons are $AAA$, $AGA$, $AGG$, $AUA$, $CUA$, $CUC$, $CUG$, $CUU$, $UAA$, $UAG$, $UCA$, $UGA$, and $UUA$. The same set appears in the punctured-cube check, where the $f$-vector is $(13,16,4)$. Thus the local object is not merely a count: it is a finite codon subcomplex with $13$ vertices, $16$ one-dimensional incidences, and $4$ two-dimensional faces under the window-six coordinate rule. The module classification accounts for all $13$ listed codons, with $R\_size=13$, assigned count $13$, no extra module codons, no missing module codons, and no unassigned codons. The six modules are $agr=\{AGA,AGG\}$, $ar\_lys=\{AAA\}$, $cun=\{CUA,CUC,CUG,CUU\}$, $ile\_met=\{AUA\}$, $ssn=\{UCA,UUA\}$, and $stop\_trp=\{UAA,UAG,UGA\}$. Every module has a predictable-or-boundary classification, so the finite row closes at the local code-read boundary rather than by appeal to a global biological law.

\paragraph{Shape significance against an unrestricted finite null.}
A separate shape-significance run used $N=2000$ samples with seed $42$. It required $57377$ distinct sample attempts and incurred $55377$ duplicate retries. There was $1$ exact shape match, giving $p_{exact}=0.0005$, and $25$ samples had Jaccard similarity at least $0.9$, giving $p_{J\geq 0.9}=0.0125$. Thus the observed local shape is rare under that finite sampling scheme. This rarity is evidence for a nontrivial coordinate-and-label pattern in the curated code table; it is not evidence for a higher-layer biological realization.

\paragraph{Amino-acid-class-preserving null.}

In this row, $N=2000$, seed $42$, the observed $M$-size is $20$, the exact match count is $0$, and the lower-tail count at or below the observed $M$-size is $7$. Hence the exact-shape empirical value is $p=0.0$ at this sample resolution, and the lower-tail value is $7/2000=0.0035$. The reported significant values are therefore $p\_value\_aa\_class\_exact\_shape\_significant=0.0$ and $p\_value\_aa\_class\_lower\_tail\_significant=0.0035$. The point of this null is conservative: even after preserving amino-acid class, the exact local boundary tile is not reproduced in $2000$ sampled draws, and the lower-tail event occurs only $7$ times.

\paragraph{Boundary verdict.}
The finite evidence supports a chapter-level claim with narrow scope: window-six codon coordinates expose a local boundary-tile structure in the curated standard-code label table, and the amino-acid-class-preserving null model does not reproduce the exact observed shape in $2000$ samples. The contact remains the curated code table. Therefore the permissible conclusion is a code-read conclusion: the local tile is a finite label-readback phenomenon in $Q_6$. The following promotions are explicitly blocked here: no translation realization follows, no folded protein structure follows, no structural order follows, no physical admissibility follows, no biological function follows, no mechanism follows, and no universality follows unless a separate reality contact supplies that higher-layer evidence.
