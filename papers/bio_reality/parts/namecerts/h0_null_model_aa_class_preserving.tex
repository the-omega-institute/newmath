\paragraph{Amino-acid-class-preserving null.}
The null model preserves amino-acid label classes and asks whether the observed local shape is reproduced under that constraint.  In $N=2000$ sampled class-preserving trials with seed $42$, the observed tile size is $|M|=20$.  The exact observed shape appears zero times:
$$\begin{aligned}
N&=2000,\\
\#\{\hbox{exact shape matches}\}&=0,\\
p_{\mathrm{aa\ class, exact}}&=0.0.
\end{aligned}$$
The lower-tail count at or below the observed $M$ size is $7$, giving
$$\begin{aligned}
\#\{\hbox{null rows with size}\leq 20\}&=7,\\
p_{\mathrm{aa\ class, lower\ tail}}&=0.0035.
\end{aligned}$$
Both reported significance scalars are the same as their corresponding $p$-values: $p_{\mathrm{aa\ class, exact\ shape\ significant}}=0.0$ and $p_{\mathrm{aa\ class, lower\ tail\ significant}}=0.0035$.  The amino-acid-class-preserving control therefore does not explain the observed local tile shape as a routine consequence of preserving label classes alone.
