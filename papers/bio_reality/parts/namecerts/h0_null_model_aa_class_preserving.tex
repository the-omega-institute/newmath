\paragraph{Amino-acid-class-preserving null comparison.}
The null model keeps the amino-acid class bookkeeping fixed while testing whether the same shape arises by chance under the allowed class-preserving rearrangements.  In the run with seed $42$ and $N=2000$ samples, the observed module size was $20$.  The exact shape was never reproduced: the exact-match count was $0$, giving
$$
\begin{aligned}
p_{\mathrm{exact}}=0.0.
\end{aligned}
$$
The lower-tail count at or below the observed size was $7$ out of $2000$, hence
$$
\begin{aligned}
p_{\mathrm{lower\ tail}}=\frac{7}{2000}=0.0035.
\end{aligned}
$$
Thus the class-preserving null does not explain the observed tile as a typical class-respecting arrangement.  The result is only a finite-row significance statement for the code-label layer; it supplies no evidence for folding, physical realizability, function, or a universal biological law.

\paragraph{Comparison with the broader shape null.}
A separate shape-significance run, also with seed $42$ and $N=2000$, required $57377$ distinct-sample attempts because $55377$ attempts repeated previously seen samples.  It found one exact shape match and $25$ samples with Jaccard similarity at least $0.9$.  The corresponding empirical probabilities were
$$
\begin{aligned}
p_{\mathrm{shape}}&=\frac{1}{2000}=0.0005,\\
p_{J\geq 0.9}&=\frac{25}{2000}=0.0125.
\end{aligned}
$$
The amino-acid-class-preserving test is therefore sharper on exact reproduction, while the broader shape run records both a rare exact match and a small high-overlap tail.
