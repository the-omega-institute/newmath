\paragraph{Finite object under test.}
The object in this record is the standard-code codon label table, read only as a finite assignment from codons to amino-acid or stop labels.  The coordinate display is the six-bit codon cube $Q_6$, where each codon is treated as a point in a finite window-six representation.  The observed local boundary set is the thirteen-codon row
$R=\{\mathrm{AAA},\mathrm{AGA},\mathrm{AGG},\mathrm{AUA},\mathrm{CUA},\mathrm{CUC},\mathrm{CUG},\mathrm{CUU},\mathrm{UAA},\mathrm{UAG},\mathrm{UCA},\mathrm{UGA},\mathrm{UUA}\}$.
The verified cardinality check reports
$$
\begin{aligned}
|R|&=13.
\end{aligned}
$$
The same finite row is recognized as a punctured-cube pattern with face data
$$
\begin{aligned}
(f_0,f_1,f_2)&=(13,16,4).
\end{aligned}
$$
This is a statement about the local shape of a labeled codon tile.  It is not a claim that the coordinate geometry generates biology by itself.

\paragraph{Amino-acid-class preserving null.}
The null model preserves the amino-acid class information while randomizing the relevant codon-label arrangement within that constraint.  Against $N=2000$ sampled class-preserving null rows, the observed tile has $\mathrm{observed\_M\_size}=20$.  No sampled null row exactly reproduced the observed shape:
$$
\begin{aligned}
N&=2000,\\
\mathrm{exact\_match\_count}&=0,\\
p_{\mathrm{class,exact}}&=0.0.
\end{aligned}
$$
The lower-tail comparison counted only $7$ null samples with size at most the observed $M$ size, giving
$$
\begin{aligned}
\mathrm{lower\_tail\_count}_{\leq 20}&=7,\\
p_{\mathrm{class,lower\ tail}}&=0.0035.
\end{aligned}
$$
Thus the finite row is not merely an artifact of preserving amino-acid classes in this sampled null: exact shape recovery was absent in the run, and the observed size lies in a small lower-tail event.  The result remains a finite empirical comparison with seed $42$ and experiment run $\mathrm{e8d5de747a0c}$, not a universal theorem about all possible biological codes.

\paragraph{Unconstrained shape comparison.}
A second shape-significance run supplies a broader comparison scale.  In $2000$ sampled rows, there was $1$ exact shape match and $25$ rows with Jaccard similarity at least $0.9$.  The sampler required $57377$ distinct-sample attempts and recorded $55377$ duplicate retries.  The corresponding finite probabilities were
$$
\begin{aligned}
p_{\mathrm{exact}}&=0.0005,\\
p_{\mathrm{Jaccard}\geq 0.9}&=0.0125.
\end{aligned}
$$
This supports the same local reading from a second angle: the displayed codon tile is rare under the sampled comparison distribution, while near matches still occur often enough that the result should be described as a boundary-sensitive local pattern rather than as an all-or-nothing biological law.

\paragraph{Scope boundary.}
The contact used here is a curated standard genetic-code table.  It supports the code-layer statement that codons are assigned amino-acid or stop labels, and it permits local split checks against that finite assignment.  It does not observe ribosomal translation, molecular structure, energetic admissibility, organismal function, or evolutionary universality.  Therefore the chapter records a constrained result: window-six coordinates expose a rare and recoverable local boundary tile in the standard-code label table under the amino-acid-class preserving null, with exact class-preserving null recovery $0/2000$ and lower-tail probability $0.0035$.  No claim about translation, folding, physical admissibility, biological function, mechanism, or universality follows from this row unless an additional contact supplies that higher-layer observation.
