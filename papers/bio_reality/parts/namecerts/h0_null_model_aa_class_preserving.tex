\subsection{Amino-acid-class-preserving null model for the window-six codon tile}
\label{subsec:h0-null-model-aa-class-preserving}

\paragraph{Finite object under test.}
The BioReality object in this record is the standard-code codon table read only as a finite codon-label contact.  The BEDC-side carrier is the six-bit codon cube $Q_6$, equipped with a six-bit codon display, local tile faces, singleton label corners, and three-point label classes.  The biological readback is deliberately narrower: a curated standard-code table assigns each codon to an amino-acid or stop label.  This contact tests code-layer label readback and local codon-tile splits; it does not test translation realization, folded structure, physical admissibility, biological function, mechanism, or universality.

\paragraph{Observed boundary row.}
The boundary set $R$ has cardinality $13$.  Its observed codons are
$$
\begin{aligned}
R=\{&\mathrm{AAA},\mathrm{AGA},\mathrm{AGG},\mathrm{AUA},\mathrm{CUA},\mathrm{CUC},\mathrm{CUG},\mathrm{CUU},\\

$$
The same finite row is recorded as a punctured-cube pattern with $f$-vector $(13,16,4)$.  Thus the readback is not merely a list of codons: it carries a local face incidence profile inside $Q_6$.  The row-level claim is that this local incidence profile is visible in the code table; the claim is not that the code table is generated by a biological law of cubical geometry.

\paragraph{Amino-acid-class-preserving null.}
The relevant null model keeps amino-acid class structure fixed while resampling the label geometry.  In the finite run with seed $42$ and $N=2000$ sampled rows, the observed comparison set has $|M|=20$.  The exact-shape count under this class-preserving null is $0$, giving the recorded empirical exact-shape value
$$
\begin{aligned}
p_{\mathrm{class,exact}}=0.0.
\end{aligned}
$$
The lower-tail count at or below the observed $M$ size is $7$, hence
$$
\begin{aligned}
p_{\mathrm{class,lower}}=\frac{7}{2000}=0.0035.
\end{aligned}
$$
This is the finite-row content of the null result: preserving amino-acid class information does not by itself reproduce the observed local window-six tile.  The exact-shape statistic has no hits in this run, and the lower-tail statistic is small but nonzero.  The correct interpretation is empirical finite-sample separation from this null, not a proof of inevitability.

\paragraph{Local cannot-claim boundary.}
The contact is aligned with the code-read conjecture and blocks higher-layer promotion.  A local codon tile is not a global biological law. Any claim about translation realization, structural order, physical admissibility, biological function, or universal mechanism requires a separate reality contact beyond the curated standard-code table.
