\paragraph{Finite carrier and code-layer contact.}
This BioReality row concerns the six-bit codon display $Q_6$ with the standard-code label table used only as a curated code-layer contact. The finite carrier is the codon cube read through three-letter RNA codons, and the observed contact assigns each codon an amino-acid or stop label. The relevant distinction is local: a window-six coordinate can expose a tile face, a singleton label corner, and a three-point label class, but the label table is not itself a translation mechanism. In this row the boundary set $R$ is finite and explicit:
$$
\begin{aligned}
R=\{&\mathrm{AAA},\mathrm{AGA},\mathrm{AGG},\mathrm{AUA},\mathrm{CUA},\mathrm{CUC},\mathrm{CUG},\mathrm{CUU},\\
&\mathrm{UAA},\mathrm{UAG},\mathrm{UCA},\mathrm{UGA},\mathrm{UUA}\}.
\end{aligned}
$$
The verified cardinality is $|R|=13$. The same row identifies this set as a punctured-cube boundary with $f$-vector $(13,16,4)$, so the finite witness is not just a list of labels: it has $13$ vertices, $16$ one-dimensional incidences, and $4$ tile faces in the induced local combinatorics.

\paragraph{Amino-acid-class-preserving null row.}
The null model preserves amino-acid label classes while testing whether the observed window-six shape is typical under that constrained reshuffling. The experiment used seed $42$ and $N=2000$ samples. The observed local module size was $|M|=20$. Among the $2000$ amino-acid-class-preserving null samples, the exact observed shape occurred $0$ times, giving
$$
\begin{aligned}
N&=2000,\\
\#\{\text{exact shape matches}\}&=0,\\
p_{\mathrm{exact}}&=0.0.
\end{aligned}
$$
The lower-tail count at or below the observed module size was $7$, hence
$$
\begin{aligned}
p_{\mathrm{lower\ tail}}=\frac{7}{2000}=0.0035.
\end{aligned}
$$
Thus, within this finite null family, the local shape is rare even after preserving amino-acid class membership. The row does not say that the codon table is forced by geometry; it says that, after the label-class constraint is held fixed, the observed window-six tile has a small empirical lower-tail frequency and no exact null replicate in the sampled finite row.

\paragraph{Comparison with the unconstrained shape row.}
The broader shape-significance row used the same seed $42$ and $N=2000$ but required many distinct draws: $57377$ distinct-sample attempts with $55377$ duplicate retries. In that row there was $1$ exact shape match, so $p_{\mathrm{exact}}=0.0005$, and there were $25$ samples with Jaccard similarity at least $0.9$, so $p_{J\geq 0.9}=0.0125$. The amino-acid-class-preserving null is therefore not a weaker duplicate of the unconstrained row. It asks a stricter biological-label question: after preserving label class, the exact count drops to $0$, while the lower-tail event remains small at $0.0035$.
