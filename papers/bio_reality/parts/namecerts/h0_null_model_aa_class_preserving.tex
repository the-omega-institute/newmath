\paragraph{Null experiment.}
The amino-acid-class-preserving null keeps the amino-acid class inventory fixed while randomizing the class-compatible code readback.  In $N=2000$ sampled tables with seed $42$, the observed tile size was $|M|=20$.  No sampled table matched the exact observed shape: the exact match count was $0$, giving
$$
\begin{aligned}
p_{\mathrm{exact}} &= 0/2000=0.0.
\end{aligned}
$$
Only $7$ of the $2000$ sampled tables landed in the lower tail at or below the observed $M$-size, giving
$$
\begin{aligned}
p_{\mathrm{lower\ tail}} &= 7/2000=0.0035.
\end{aligned}
$$
This is the claim-specific finite-row result: under a null that preserves amino-acid classes, the exact local tile is absent from the sample, and the observed size lies in a small lower tail.  The conclusion is not that the codon table must arise from the window-six geometry.  The conclusion is only that the curated standard-code labels expose a sharply constrained local boundary pattern when tested against this class-preserving finite null.

\paragraph{Comparison with unconstrained shape sampling.}

The class-preserving exact-shape value is $0.0$, and the class-preserving lower-tail value is $0.0035$.

\paragraph{Boundary verdict.}
The amino-acid-class-preserving null rejects a casual explanation in which the observed local codon tile is typical among class-preserving relabelings.  The finite evidence consists of $0$ exact matches among $2000$ class-preserving samples, a lower-tail value $0.0035$, the thirteen-codon punctured-cube boundary with $f$-vector $(13,16,4)$, the spectral row with $\lambda_{M}=0.6752479308830213$ and $\lambda_{R}=0.4758352843197002$, and the deletion rows with hit rates near or at one.  The cannot-claim boundary is part of the result: no translation, folding, physical admissibility, function, mechanism, or universality follows from this chapter unless a separate higher-layer contact supplies it.  The curated standard-code table is a code-label bridge, not a mechanism and not a structure experiment.
