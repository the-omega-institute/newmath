\subsection{Amino-acid-class-preserving null model for the window-six codon tile}

\origin{ai}

\paragraph{Finite carrier and readback contact.}
This record concerns the six-bit codon display on the finite codon cube $Q_6$, read through a curated standard-code table.  The carrier is not a biochemical mechanism; it is a finite coordinate object whose observable contact is the assignment of each codon to an amino-acid or stop label.  The contact therefore tests code-layer readback and local label-split structure only.  It does not test translation realization, folded structure, physical admissibility, biological function, universal mechanism, or any claim that DNA is forced to obey this geometry as a biological necessity.

\paragraph{Boundary row.}
The distinguished boundary set $R$ has cardinality $13$.  In the finite row it is exactly
$$
\begin{aligned}
R=\{&AAA, AGA, AGG, AUA, CUA, CUC, CUG, CUU,\\
&UAA, UAG, UCA, UGA, UUA\}.
\end{aligned}
$$
The same row identifies this set as a punctured-cube boundary with face-vector
$$
\begin{aligned}
(f_0,f_1,f_2)=(13,16,4).
\end{aligned}
$$
Thus the local tile is not reported merely by a count: the finite data also record a specific incidence shape, with thirteen vertices, sixteen edges, and four two-faces inside the window-six coordinate presentation.

\paragraph{Unrestricted shape-null row.}
Against the broader sampled null row, with $N=2000$ and seed $42$, the computation used $57377$ distinct-sample attempts and saw $55377$ duplicate retries.  It found $1$ exact shape match and $25$ matches with Jaccard similarity at least $0.9$.  The corresponding values are
$$
\begin{aligned}
p_{\mathrm{exact}} &= 0.0005,\\
p_{\mathrm{Jaccard}\geq 0.9} &= 0.0125.
\end{aligned}
$$
This row says that the observed local tile is rare under that sampling design.  It does not by itself identify a mechanism; it only records that the observed finite shape is not typical in that null ensemble.

\paragraph{Amino-acid-class-preserving null row.}
The named null model preserves amino-acid classes and uses $N=2000$ samples with seed $42$.  In this row the observed $M$-size is $20$.  The exact-shape count is $0$, giving
$$
\begin{aligned}
p_{\mathrm{aa\ class,\ exact\ shape}} &= 0.0.
\end{aligned}
$$
The lower-tail count at or below the observed $M$-size is $7$, hence
$$
\begin{aligned}
p_{\mathrm{aa\ class,\ lower\ tail}} &= 0.0035.
\end{aligned}
$$
The significance fields repeat these same finite-row values: $0.0$ for the exact-shape row and $0.0035$ for the lower-tail row.  The interpretation is deliberately narrow.  Preserving amino-acid class labels in the null does not erase the local tile signal; however, the conclusion remains a code-read conclusion about a finite label table.

Every listed module has a predictable-or-boundary witness in the finite row.  The module row is therefore aligned with the punctured-cube boundary row rather than being an independent biological universality claim.

\paragraph{Local verdict.}
The finite evidence supports a chapter-specific verdict: the standard-code codon table contains a non-generic local boundary-tile pattern in the window-six coordinate display, and that pattern remains visible under an amino-acid-class-preserving null model.  The strongest null row here is the amino-acid-class-preserving test, where $0$ exact matches occur among $2000$ samples and the lower-tail value is $0.0035$ at observed $M$-size $20$.  The supporting rows give the same local reading through $|R|=13$, face-vector $(13,16,4)$, spectral values $\lambda_M=0.6752479308830213$ and $\lambda_R=0.4758352843197002$, leave-one-code-out hit rate $0.9841269841269841$ above baseline, leave-one-codon-out recovery $1.0$, and complete assignment of all thirteen boundary codons into the displayed modules.
