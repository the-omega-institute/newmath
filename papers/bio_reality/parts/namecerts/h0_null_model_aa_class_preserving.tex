\subsection{Amino-acid-class-preserving null model for the window-six codon tile}\label{subsec:h0-null-model-aa-class-preserving}
\origin{ai}

\paragraph{Finite carrier and readback contact.}
The finite carrier is the six-bit codon cube $Q_6$, read through the standard-code label table as an external code-layer contact.  The contact used here is only the curated codon-to-amino-acid-or-stop table.  It supplies label readback for finite rows; it does not supply translation realization, folded structure, physical admissibility, biological function, mechanism, or universality.  The tested object is therefore the code-read layer of the standard genetic code, not a claim about the biochemical implementation of translation.

The distinguished boundary row $R$ has cardinality $13$.  In the finite row the codons are
$$
\begin{aligned}
R=\{&\mathrm{AAA},\mathrm{AGA},\mathrm{AGG},\mathrm{AUA},\mathrm{CUA},\mathrm{CUC},\mathrm{CUG},\mathrm{CUU},\\
&\mathrm{UAA},\mathrm{UAG},\mathrm{UCA},\mathrm{UGA},\mathrm{UUA}\}.
\end{aligned}
$$
The same row is recorded as a punctured-cube pattern with $f$-vector $(13,16,4)$.  Thus the local tile is not only a list of exceptional codons: it has a finite face-incidence signature inside the window-six cube.  The observed companion size is $|M|=20$, and this size is the statistic used in the amino-acid-class-preserving null test below.

\paragraph{Amino-acid-class-preserving null.}
The null model preserves amino-acid label classes while randomizing the row geometry against the observed window-six tile.  With seed $42$ and $N=2000$ sampled tables, the exact observed shape occurs zero times:
$$
\begin{aligned}
N&=2000,\\
\#\{\hbox{exact matches}\}&=0,\\
p_{\mathrm{class,exact}}&=0.0.
\end{aligned}
$$
The lower-tail statistic compares sampled $M$-sizes against the observed value $|M|=20$.  There are $7$ sampled rows with size at most the observed size, giving
$$
\begin{aligned}
\#\{|M_{\mathrm{sample}}|\leq 20\}&=7,\\
p_{\mathrm{class,tail}}&=7/2000=0.0035.
\end{aligned}
$$
This is the chapter-specific null result: preserving amino-acid classes does not by itself reproduce the exact local tile, and it rarely produces a row at least as small as the observed $M$-size under this finite sampling protocol.  The result is a finite-row significance statement about a code-label geometry, not a mechanistic law of molecular biology.

\paragraph{Comparison with the unconstrained shape row.}
A separate shape-significance row uses the same sample size $N=2000$ and seed $42$.  It records $57{,}377$ distinct-sample attempts and $55{,}377$ duplicate retries.  In that row there is $1$ exact shape match, hence $p_{\mathrm{exact}}=0.0005$, and $25$ samples with Jaccard similarity at least $0.9$, hence $p_{\mathrm{Jaccard}\geq 0.9}=0.0125$.  The amino-acid-class-preserving null is therefore not weaker because it finds many exact copies; it finds none in $2000$ trials.  Its lower-tail value $0.0035$ records a different statistic, namely rarity of rows no larger than the observed $|M|=20$ while preserving label classes.

\paragraph{Boundary verdict.}
The amino-acid-class-preserving null supports a narrow finite claim.  In the standard-code readback, the window-six carrier exposes a local boundary tile whose exact shape is not reproduced in $2000$ amino-acid-class-preserving samples, and whose lower-tail statistic for the observed $M$-size is $0.0035$.  Together with the $13$-codon punctured-cube row, the spectral values, and the complete six-module classification, this is evidence for a code-layer local tile.  It is not evidence that DNA obeys this geometry as a biological necessity.  No translation, folding, physical admissibility, biological function, universal mechanism, or protein-structure claim follows from this chapter unless a separate reality contact supplies that higher-layer observation.
