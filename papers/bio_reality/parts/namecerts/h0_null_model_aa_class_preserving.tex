\paragraph{Amino-acid-class-preserving null.}
The null model preserves amino-acid classes and asks whether the observed local tile shape is reproduced under that constrained relabeling.  In the verified run $e8d5de747a0c$, the sample size was $N=2000$ with seed $42$.  The observed $M$-set size was $20$.  Among the $2000$ amino-acid-class-preserving null samples, the exact observed shape occurred zero times, and only seven samples had $M$-size at most the observed value:
$$
\begin{aligned}
N&=2000,\\
|M|_{\mathrm{obs}}&=20,\\
\#\{\mathrm{exact\ shape\ matches}\}&=0,\\
\#\{|M|\leq |M|_{\mathrm{obs}}\}&=7,\\
p_{\mathrm{exact}}&=0.0,\\
p_{\mathrm{lower\ tail}}&=0.0035.
\end{aligned}
$$
This is the finite-row reason the null is rejected at the shape level: preserving amino-acid class alone did not reproduce the exact local boundary-tile geometry in the sampled population.  The lower-tail statistic is also small, so the observed $M$-size is not typical for the same null ensemble.
