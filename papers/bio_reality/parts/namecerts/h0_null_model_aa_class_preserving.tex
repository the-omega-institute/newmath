\subsection{Amino-acid-class-preserving null model for the codon boundary tile}
\label{sec:biorreality-h0-null-model-aa-class-preserving}

\paragraph{Claim and finite display.}
This record tests the claim that the standard-code codon table has a window-six local boundary pattern that is not explained by preserving amino-acid class sizes alone.  The carrier is the six-bit codon cube $Q_6$, read as the $64$ codon positions obtained from three RNA bases with two bits per base.  The data row separates the finite coordinate display from the curated label contact: the coordinate side supplies local faces, singleton corners, three-point label classes, closure, spectrum, and relations; the curated standard-code table supplies the codon-to-amino-acid-or-stop labels.  The observed residual boundary set $R$ has cardinality $13$ and is exactly
$$
\begin{aligned}
R = \{&AAA, AGA, AGG, AUA, CUA, CUC, CUG, CUU, UAA, UAG, UCA, UGA, UUA\}.
\end{aligned}
$$
As a cubical subcomplex this row reports the punctured-cube face vector
$$
\begin{aligned}
(f_0,f_1,f_2)=(13,16,4).
\end{aligned}
$$
Thus the finite object being tested is not an unconstrained biological law.  It is a concrete codon-coordinate pattern with a fixed curated readout table.

\paragraph{Class-preserving null experiment.}
The null model keeps amino-acid class structure as the randomized comparison constraint and asks whether the same local shape appears under that constraint.  In the verified run with seed $42$, the experiment evaluated $N=2000$ class-preserving samples.  The observed $M$ size was $20$.  No sample matched the exact observed shape, so the exact-shape count was $0$ and the reported exact-shape Monte Carlo value was
$$
\begin{aligned}
p_{\mathrm{class},\mathrm{exact}} = 0.0.
\end{aligned}
$$
The lower-tail statistic counted samples with $M$ size at most the observed $20$; this count was $7$ out of $2000$, giving
$$
\begin{aligned}
p_{\mathrm{class},\leq M} = 7/2000 = 0.0035.
\end{aligned}
$$
The row therefore supports a finite anti-null statement at the code-label layer: preserving amino-acid class sizes does not, in this run, reproduce the observed local boundary tile at ordinary Monte Carlo resolution.  The result is about a finite randomized comparison against the curated standard-code labels, not about biochemical realization.

These numbers agree with the class-preserving row in direction: the observed coordinate tile is uncommon under the tested finite randomizations.

\paragraph{Local boundary of interpretation.}
The only biological contact used here is the curated standard genetic-code table.  That contact observes codon-to-label assignments, with stop labels treated as table labels alongside amino-acid labels.  It does not observe ribosomal translation dynamics, folded protein geometry, physical admissibility, cellular function, evolutionary mechanism, or universality across biological systems.  Consequently this chapter may say that the standard-code label table exhibits a finite window-six local boundary pattern that is rare under the tested amino-acid-class-preserving null model.  It may not infer that DNA or RNA must obey this geometry as a biological necessity, that an amino-acid label sequence has a folded structure, that the represented peptide is physically admissible, that it has function, that translation has been realized, or that a universal mechanism has been found.  Any such stronger claim requires an additional contact that directly measures the relevant higher layer.

The row is ready as a code-read result.  Its strongest supported statement is finite and local: in the standard-code table, the residual set $R$ has $13$ codons, the punctured-cube face vector is $(13,16,4)$, the module partition is complete, and a class-preserving null run with $2000$ samples gives exact-shape count $0$ and lower-tail value $0.0035$ for observed $M$ size $20$. The claim stops there.
