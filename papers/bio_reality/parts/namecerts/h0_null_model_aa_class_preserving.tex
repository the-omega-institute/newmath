\paragraph{Finite object.}
The finite carrier is the six-bit codon cube $Q_6$, read through the standard-code codon-label table.  The row under test is not a claim about translation dynamics or protein formation.  It is a code-read claim: after the standard table supplies codon labels, the window-six coordinate display exposes a local boundary tile whose exceptional set is small, structured, and stable under several finite probes.  The external contact is the curated standard-code table, which assigns each codon to an amino-acid or stop label.  This contact is sufficient for codon-label readback and local split checks, but it is not a mechanism, a physical admissibility test, or a universality theorem.

$$\begin{aligned}

Thus the data are not merely a list of exceptional codons: the row also carries a local face incidence profile.  The singleton and three-point label-corner distinctions are therefore read as finite boundary features inside $Q_6$, with the amino-acid labels supplied by the curated table rather than derived from geometry alone.

\paragraph{Shape null model.}
Against the shape null with seed $42$, the experiment used $N=2000$ samples.  It required $57377$ distinct-sample attempts and saw $55377$ duplicate retries.  There was $1$ exact shape match, giving
$$\begin{aligned}
p_{\mathrm{exact}}=0.0005,
\end{aligned}$$
and there were $25$ samples with Jaccard score at least $0.9$, giving
$$\begin{aligned}
p_{J\geq 0.9}=0.0125.
\end{aligned}$$
This supports the statement that the observed local tile is uncommon under the sampled background, while still staying within the finite code-read layer.

\paragraph{Amino-acid-class-preserving null.}
The claim-specific null model preserves amino-acid class structure before testing the same window-six tile.  With seed $42$ and $N=2000$, the observed $M$ size is $20$.  The exact-match count is $0$, so the reported exact-shape probability is
$$\begin{aligned}
p_{\mathrm{aa\ class,exact}}=0.0.
\end{aligned}$$
The lower-tail count at or below the observed $M$ size is $7$, giving
$$\begin{aligned}
p_{\mathrm{aa\ class,lower\ tail}}=\frac{7}{2000}=0.0035.
\end{aligned}$$
The null therefore does not explain the observed finite row by amino-acid class preservation alone.  The conclusion is deliberately narrow: the finite coordinate display and curated labels jointly identify an unusually constrained local tile; they do not identify a translation mechanism or a structural protein principle.

\paragraph{Cannot-claim boundary.}
The contact boundary is part of the claim.  The curated standard-code table supports codon-to-label readback and local label-split tests only.  No translation realization, folded structure, structural order, physical admissibility, biological function, universal mechanism, or law of genetic coding follows from the window-six geometry unless a separate reality contact supplies that higher-layer evidence.  In particular, an amino-acid label sequence is not a folded protein structure, and a local codon tile is not a global biological law.  The ready verdict for this row is therefore a finite BioReality statement about an amino-acid-class-preserving null failure inside $Q_6$, with explicit confinement to the code-read layer.
