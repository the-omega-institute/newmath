\subsection{Amino-Acid-Class-Preserving Null Model for the Window-Six Codon Tile}
\label{sec:bioreality-h0-null-model-aa-class-preserving}

\paragraph{Finite object and contact.}
The object studied here is the standard-code codon table read through a six-bit codon display.  The carrier is the codon cube $Q_6$, with local distinctions given by coordinate, closure, spectrum, and relation data.  The external contact is a curated standard-code table assigning each codon to an amino-acid or stop label.  This contact is sufficient for code-layer readback and local label-split checks.  It is not a contact with translation realization, folded structure, physical admissibility, biological function, mechanism, or universality.

\paragraph{Observed boundary row.}
The distinguished boundary row $R$ has cardinality $13$.  In the finite run it is exactly
$$
\begin{aligned}
R=\{&\mathrm{AAA},\mathrm{AGA},\mathrm{AGG},\mathrm{AUA},\mathrm{CUA},\mathrm{CUC},\mathrm{CUG},\mathrm{CUU},\\
&\mathrm{UAA},\mathrm{UAG},\mathrm{UCA},\mathrm{UGA},\mathrm{UUA}\}.
\end{aligned}
$$
The same row is reported as a punctured-cube pattern with $f$-vector
$$
\begin{aligned}
(f_0,f_1,f_2)=(13,16,4).
\end{aligned}
$$
Thus the finite row is not only a list of codons; it also carries a small local face structure inside the window-six coordinate display.  The row contains the $\mathrm{AGR}$ pair $\{\mathrm{AGA},\mathrm{AGG}\}$, the singleton $\mathrm{AAA}$ row, the four-codon $\mathrm{CUN}$ block, the $\mathrm{AUA}$ codon, the mixed $\{\mathrm{UCA},\mathrm{UUA}\}$ row, and the stop-related triple $\{\mathrm{UAA},\mathrm{UAG},\mathrm{UGA}\}$.

\paragraph{Amino-acid-class-preserving null.}
The null model preserves amino-acid label classes while randomizing the relevant finite structure.  In the reported run, $N=2000$ samples were drawn with seed $42$.  The observed marked set has size $|M|=20$.  The exact-shape match count under this amino-acid-class-preserving null was $0$, hence
$$
\begin{aligned}
 p_{\mathrm{aa\ class,exact}}=0.0.
\end{aligned}
$$
The lower-tail count at or below the observed size was $7$, giving
$$
\begin{aligned}
 p_{\mathrm{aa\ class,lower\ tail}}=\frac{7}{2000}=0.0035.
\end{aligned}
$$
This is the central finite-row fact for the present claim: when the amino-acid class partition is held fixed, the observed local window-six configuration remains unusually sharp.  The finite record supports a local code-read anomaly relative to this null model; it does not promote the anomaly into a biological mechanism.

\paragraph{Unrestricted shape comparison.}
A separate shape-significance run with $N=2000$ and seed $42$ required $57377$ distinct-sample attempts and recorded $55377$ duplicate retries.  It found $1$ exact shape match and $25$ matches at Jaccard score at least $0.9$.  The corresponding values are
$$
\begin{aligned}
 p_{\mathrm{exact}}&=0.0005,\\
 p_{\mathrm{Jaccard}\geq 0.9}&=0.0125.
\end{aligned}
$$
These numbers are consistent with the amino-acid-class-preserving null: the boundary tile is rare as an exact local shape, and remains rare after preserving the label-class information used by the code table contact.

\paragraph{Leave-one-code-out readback.}
The leave-one-code-out experiment evaluated $23$ code tables and $63$ targets.  The baseline hit rate was $0.19821739130434785$.  The hit rate above baseline was $0.9841269841269841$, the hit rate inside $M\setminus T$ was $0.9365079365079365$, and the hit rate inside $M\setminus T$ plus the boundary was again $0.9841269841269841$.  Family-level readback covered $5$ families: alternative yeast had hit rate $0.8333333333333334$, while bacterial plastid, candidate other, mitochondrial, and nuclear ciliate each had hit rate $1.0$.  These values say that the local code-read signal survives a leave-one-code-table stress test in the curated label universe.  They do not say that the same geometry implements translation or explains biological adaptation.

\paragraph{Leave-one-codon-out and modules.}
The leave-one-codon-out run reported anchor count $0$, boundary count $5$, high closure coverage $8$, predictable count $8$, and recovery rate $1.0$.  The module classification is complete: $R$ has size $13$, all $13$ codons are assigned, and there are no extra module codons, no missing codons, and no unassigned codons.  The assigned modules are
$$
\begin{aligned}
\mathrm{agr}&=\{\mathrm{AGA},\mathrm{AGG}\},\\
\mathrm{ar\_lys}&=\{\mathrm{AAA}\},\\

Every listed module has predictable or boundary status.  The module row therefore supplies a finite internal explanation for how the thirteen boundary codons are grouped in the local display.

\paragraph{Scope boundary.}
The admissible conclusion is deliberately narrow.  Window-six coordinates expose a local boundary-tile structure in the standard-code label table, and the amino-acid-class-preserving null gives a lower-tail value $0.0035$ with no exact matches among $2000$ samples.  The curated code table supports codon-to-label readback and nothing stronger.  No translation realization, folded protein structure, structural order, physical admissibility, biological function, universal mechanism, or claim that DNA must obey BEDC geometry follows from this chapter unless a separate higher-layer reality contact is supplied.
