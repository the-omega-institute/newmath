\subsection{Amino-acid-class preserving null model for the codon tile}
\label{sec:h0-null-model-aa-class-preserving}

\paragraph{Finite object under test.}
The object considered here is the standard-code codon table viewed through a six-coordinate codon display. The display is only a finite coordinate instrument: it arranges codons in the cube $Q_6$ and permits local face, corner, closure, spectrum, and relation calculations. The biological contact used in this section is narrower than the geometry. It is the curated standard-code assignment of codons to amino-acid or stop labels. Thus the observation layer is code readback: codon labels may be compared with a local tile pattern, but the comparison does not by itself observe translation chemistry, folded protein structure, physical admissibility, organismal function, or a universal mechanism.

\paragraph{Observed exceptional row set.}
The finite relation row $R$ contains thirteen codons. In the observed standard-code table the row is exactly
$$
\begin{aligned}
R=\{&\mathtt{AAA},\mathtt{AGA},\mathtt{AGG},\mathtt{AUA},\mathtt{CUA},\mathtt{CUC},\mathtt{CUG},\mathtt{CUU},\\
&\mathtt{UAA},\mathtt{UAG},\mathtt{UCA},\mathtt{UGA},\mathtt{UUA}\}.
\end{aligned}
$$
The recorded cardinality check gives $|R|=13$. The same row is identified as a punctured-cube pattern with face-vector data
$$
\begin{aligned}
f(R)=(13,16,4).
\end{aligned}
$$
The accompanying module classification is exhaustive for this row: the assigned module codons are $13$, the row size is $13$, the unassigned list is empty, the missing-from-modules list is empty, and the extra-module list is empty. The six modules are
$$
\begin{aligned}
\mathrm{agr}&=\{\mathtt{AGA},\mathtt{AGG}\},&
\mathrm{ar\_lys}&=\{\mathtt{AAA}\},\\
\mathrm{cun}&=\{\mathtt{CUA},\mathtt{CUC},\mathtt{CUG},\mathtt{CUU}\},&

\mathrm{ssn}&=\{\mathtt{UCA},\mathtt{UUA}\},&
\mathrm{stop\_trp}&=\{\mathtt{UAA},\mathtt{UAG},\mathtt{UGA}\}.
\end{aligned}
$$
Every listed module has a predictable-or-boundary classification in the finite record.

\paragraph{Class-preserving null comparison.}
The null model used for the present claim preserves amino-acid class information while randomizing the relevant finite arrangement. With seed $42$ and $N=2000$ sampled class-preserving alternatives, the observed local pattern has $|M|=20$. Among the $2000$ sampled alternatives there were no exact matches to the observed shape. The exact-shape empirical count and estimate are
$$
\begin{aligned}
\#\{\text{exact matches}\}&=0,\\
\widehat p_{\mathrm{class,exact}}&=0.0.
\end{aligned}
$$
The lower-tail comparison counted seven sampled alternatives with $M$-size at most the observed value, giving
$$
\begin{aligned}
\#\{|M_{\mathrm{sample}}|\leq 20\}&=7,\\
\widehat p_{\mathrm{class,lower}}&=0.0035.
\end{aligned}
$$
Within this finite run, the class-preserving null therefore treats the observed local tile as rare both by exact shape and by the lower-tail size statistic. The zero exact-match estimate is a sample estimate from $2000$ trials, not a mathematical assertion that the underlying probability is zero.

\paragraph{Independent shape and spectral checks.}
A separate shape-significance run, also with seed $42$ and $N=2000$, required $57377$ distinct-sample attempts and encountered $55377$ duplicate retries. It found one exact shape match and twenty-five samples with Jaccard score at least $0.9$, giving
$$
\begin{aligned}
\widehat p_{\mathrm{exact\ shape}}&=0.0005,\\
\widehat p_{\mathrm{Jaccard}\geq 0.9}&=0.0125.
\end{aligned}
$$
The spectral record supplies numerical invariants for the same finite arrangement. The recorded values are
$$
\begin{aligned}
\lambda_M&=0.6752479308830213,\\
\lambda_R&=0.4758352843197002,\\
\mu&=3.051487585298128.
\end{aligned}
$$
The cubic radial check evaluates to $-6.224354365258478\times 10^{-12}$, which is numerically at zero at the recorded tolerance. These spectral values do not create a biological mechanism; they only give a finite numerical signature of the local codon-label pattern.

\paragraph{Leave-one-out stability.}
The code-table leave-one-out run evaluated $23$ tables and $63$ targets. The baseline hit rate was $0.19821739130434785$, while the hit rate above baseline was $0.9841269841269841$. The hit rate inside $M-T$ was $0.9365079365079365$, and the hit rate inside $M-T$ together with the boundary was $0.9841269841269841$. Family-resolved rates over five families were
$$
\begin{aligned}
\mathrm{alternative\_yeast}&=0.8333333333333334,\\
\mathrm{bacterial\_plastid}&=1.0,\\
\mathrm{candidate\_other}&=1.0,\\
\mathrm{mitochondrial}&=1.0,\\
\mathrm{nuclear\_ciliate}&=1.0.
\end{aligned}
$$
The codon-level leave-one-out run recorded anchor count $0$, boundary count $5$, predictable count $8$, high closure coverage $8$, and recovery rate $1.0$. These figures support a code-layer statement: the local window is stable as a label-readback pattern under these finite omissions.

\paragraph{Local conclusion and boundary.}
The amino-acid-class preserving null model gives a finite negative-control test for the local codon tile. The observed standard-code labels produce a thirteen-codon punctured-cube row, a twenty-point observed $M$-size, no exact class-preserving null matches in $2000$ samples, a lower-tail estimate $0.0035$, and independent shape and leave-one-out support. The scientific conclusion is therefore restricted: the six-coordinate codon display exposes a non-generic local label pattern in the curated standard-code table. Nothing in this row establishes that DNA must obey this geometry as a biological law, that the code table realizes translation dynamics, that label sequences determine folds, that the pattern is physically admissible in a molecular system, that it has biological function, or that it is universal across all contexts. Any one of those stronger claims would require an additional contact measuring that higher layer directly.
