\subsection{Amino-Acid-Class-Preserving Null Model for the Window-Six Codon Tile}
\label{subsec:bioreality-h0-null-aa-class-preserving}

\paragraph{Finite object and scope.}
This record concerns the standard-code codon table only at the level of codon-label readback. The finite carrier is the six-coordinate codon cube $Q_6$, with each codon displayed as a six-bit window coordinate and then read against a curated standard-code label table. The observed boundary set $R$ has thirteen codons:
$$
\begin{aligned}
R=\{&AAA,AGA,AGG,AUA,CUA,CUC,CUG,CUU,\\
&UAA,UAG,UCA,UGA,UUA\}.
\end{aligned}
$$
The same row reports $|R|=13$ and the punctured-cube face vector $(13,16,4)$. Thus the codon tile being tested is not a whole genetic mechanism; it is a finite labelled subconfiguration inside $Q_6$ with thirteen boundary vertices, sixteen one-dimensional incidences, and four local faces. The amino-acid-class-preserving null model asks whether the observed local class geometry could be reproduced when labels are randomized subject to preservation of amino-acid class counts.

\paragraph{Observed statistic and amino-acid-class-preserving comparison.}
For the tested row, the observed module size is $|M|=20$. The class-preserving null experiment used $N=2000$ sampled tables with seed $42$. It found no exact reproduction of the observed shape, so the exact-shape count is $0$ and the empirical exact-shape probability is
$$
\begin{aligned}
p_{\mathrm{exact}}=0/2000=0.0.
\end{aligned}
$$
The lower-tail count for samples with statistic at most the observed $M$ size is $7$, giving
$$
\begin{aligned}
p_{\mathrm{lower}}=7/2000=0.0035.
\end{aligned}
$$
Both reported significance fields equal the same numerical values, namely $0.0$ for exact shape and $0.0035$ for the lower tail. This finite row therefore supports a narrow conclusion: after preserving amino-acid class composition, the observed window-six tile is still rare under the sampled null ensemble. The conclusion is statistical and code-layered; it does not assert a biochemical cause.

These recovery numbers are compatible with the null result: the rare amino-acid-class-preserving shape is not an isolated numerical accident, because deletion-style probes still recover the local code-label boundary in the finite readback task.

\paragraph{Scientific interpretation.}
The supported claim is deliberately small. Window-six codon coordinates expose a local labelled boundary tile in the standard code, and the amino-acid-class-preserving null model makes that tile look statistically unusual in the sampled finite ensemble. The curated table supplies the label contact: codons are assigned to amino-acid or stop labels, and the probes compare those labels against the internal coordinate geometry. Nothing in this row observes ribosomal translation, folded protein shape, thermodynamic admissibility, molecular function, a mechanism of code origin, or a universal biological law. Any one of those stronger claims would require its own independent contact and its own finite audit. In this chapter, amino-acid label readback remains a code-level observation, not evidence for structure or function.
