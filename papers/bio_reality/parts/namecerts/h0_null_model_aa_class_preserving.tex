\subsection{Amino-acid-class preserving null model for the window-six codon tile}
\label{sec:h0-null-model-aa-class-preserving}

\paragraph{Claim under test.}
This record studies the standard-code codon table through a six-coordinate codon display.  The finite object is the codon cube $Q_6$, read only as a code-level table whose rows are codons and whose labels are amino-acid or stop assignments.  The test asks whether the local tile detected in the window-six coordinate system survives a null model that preserves amino-acid label classes.  In this setting, the relevant distinction is not whether the genetic code is explained by the coordinate system.  The narrower question is whether the observed local boundary and its label split are unusually sharp when compared with class-preserving relabelings of the same finite table.

\paragraph{Observed punctured tile.}
The boundary row set $R$ has cardinality $13$.  In codon notation it is
$$
\begin{aligned}
R = \{&AAA, AGA, AGG, AUA, CUA, CUC, CUG, CUU,\\
     &UAA, UAG, UCA, UGA, UUA\}.
\end{aligned}
$$
The same row set is recorded as a punctured-cube object with $f$-vector
$$
\begin{aligned}
f(R)=(13,16,4).
\end{aligned}
$$
Thus the tile is not merely a list of exceptional labels: it has a finite face structure with $13$ vertices, $16$ one-dimensional incidences, and $4$ two-dimensional faces.  The present claim uses that finite geometry as a local code-read signature and not as a statement about translation chemistry or organismal function.

\paragraph{Class-preserving null result.}
The null experiment preserves amino-acid class structure and samples $N=2000$ comparison tables with seed $42$.  The observed marked set has size $20$.  Among the $2000$ samples, the exact shape reappears $0$ times, giving
$$
\begin{aligned}
p_{\mathrm{exact}} = 0.0.
\end{aligned}
$$
The lower-tail count at or below the observed marked-set size is $7$, giving
$$
\begin{aligned}
p_{\mathrm{lower\ tail}} = \frac{7}{2000}=0.0035.
\end{aligned}
$$
Both recorded significance fields agree with these values: the exact-shape significance is $0.0$, and the lower-tail significance is $0.0035$.  The interpretation is finite and local: after preserving amino-acid classes, the observed tile shape is not a routine outcome of the null sample.  This supports a code-layer anomaly in the row geometry, not a claim that the coordinate system determines biology.

\paragraph{Shape rarity under an unconstrained row-shape comparison.}
A separate shape-significance run also used $N=2000$ and seed $42$.  It required $57377$ distinct sample attempts and encountered $55377$ duplicate retries, so the comparison space was collision-heavy under the sampling rule.  In that run the exact observed shape occurred once, giving
$$
\begin{aligned}
p_{\mathrm{shape}} = 0.0005.
\end{aligned}
$$
The high-overlap threshold produced $25$ matches at Jaccard score at least $0.9$, giving
$$
\begin{aligned}
p_{J\geq 0.9}=0.0125.
\end{aligned}
$$
These numbers give a second finite indication that the window-six local tile is not a generic random-looking subset.  The result remains a statement about codon-label rows and their coordinate pattern.

\paragraph{Reality contact and scope.}
The only reality contact used here is the curated standard-code table: it assigns codons to amino-acid or stop labels.  That contact can test code-read claims and local codon-tile label splits.  It cannot by itself test translation realization, folded protein structure, structural order, physical admissibility, biological function, or a universal mechanism.  Consequently, the positive finite evidence above supports the narrower statement that the window-six coordinate system exposes a sharp local code-layer boundary.  It does not promote an amino-acid label sequence to a folded protein, nor does it turn the local tile into a global law of biological organization.

\paragraph{Local conclusion.}
For the claim $h0.null.model.aa\_class\_preserving$, the finite row data support a ready code-layer verdict.  The class-preserving null sample gives $0$ exact matches in $2000$ trials and a lower-tail value of $0.0035$ for the observed marked-set size $20$.  The independent shape run gives $p=0.0005$ for exact shape and $p=0.0125$ for high Jaccard overlap.  The boundary set $R$ has $13$ explicitly assigned codons, punctured-cube face vector $(13,16,4)$, complete six-module coverage, and strong leave-one-out behavior across both code-table and codon holdouts.  The record is therefore a finite, table-bound support for a local codon-label tile, with the higher biological layers deliberately left outside the conclusion.
