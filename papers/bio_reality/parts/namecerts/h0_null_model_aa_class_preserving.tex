\subsection{Amino-Acid-Class-Preserving Null Model for the Window-Six Codon Tile}
\label{sec:aa-class-preserving-null-model}
\origin{ai}

\paragraph{Finite object and contact layer.}
This record concerns the standard-code codon table only at the code-read layer. The carrier is the six-bit codon cube $Q_6$, read through a curated standard-code table that assigns each codon to an amino-acid or stop label. The finite distinctions used here are the six-bit codon display, a local tile face, a singleton label corner, and a three-point label class. These distinctions are not asserted to be biological laws. They are a finite coordinate system for asking whether a local boundary-tile split remains visible after the amino-acid labels are read back from the external code table.

\paragraph{Observed residual set.}
The finite residual row $R$ has cardinality $13$. Its observed codons are
$$
\begin{aligned}
R=\{&\mathrm{AAA},\mathrm{AGA},\mathrm{AGG},\mathrm{AUA},\mathrm{CUA},\mathrm{CUC},\mathrm{CUG},\mathrm{CUU},\\
&\mathrm{UAA},\mathrm{UAG},\mathrm{UCA},\mathrm{UGA},\mathrm{UUA}\}.
\end{aligned}
$$
The same row is certified as a punctured-cube pattern with finite face vector
$$
\begin{aligned}
(f_0,f_1,f_2)=(13,16,4).
\end{aligned}
$$
Thus the coordinate readback is not merely a list of exceptional codons: it has a small local cubical shape with $13$ vertices, $16$ one-dimensional edges, and $4$ square faces. The corresponding module partition is complete. The six named modules account for all $13$ residual codons, with no missing, extra, or unassigned codons: $\mathrm{agr}=\{\mathrm{AGA},\mathrm{AGG}\}$, $\mathrm{ar\_lys}=\{\mathrm{AAA}\}$, $\mathrm{cun}=\{\mathrm{CUA},\mathrm{CUC},\mathrm{CUG},\mathrm{CUU}\}$, $\mathrm{ile\_met}=\{\mathrm{AUA}\}$, $\mathrm{ssn}=\{\mathrm{UCA},\mathrm{UUA}\}$, and $\mathrm{stop\_trp}=\{\mathrm{UAA},\mathrm{UAG},\mathrm{UGA}\}$. Each module has either predictable or boundary status in the finite row.

\paragraph{Amino-acid-class-preserving null comparison.}
The null model preserves amino-acid classes and samples $N=2000$ finite alternatives with seed $42$. Against this class-preserving null, the observed main row has size $|M|=20$. The exact shape reappears zero times:
$$
\begin{aligned}
N&=2000,\\
\#\{\text{exact matches}\}&=0,\\
p_{\mathrm{exact}}&=0.0.
\end{aligned}
$$
The lower-tail count at or below the observed size is $7$, giving
$$
\begin{aligned}
\#\{|M'|\leq 20\}&=7,\\
p_{\mathrm{lower\ tail}}&=0.0035.
\end{aligned}
$$
In this finite-row audit the amino-acid-class-preserving null therefore does not explain the observed local shape as a common class-preserving accident. The exact-shape statistic is at zero in the sample, while the lower-tail statistic is $0.0035$.

\paragraph{Boundary statement.}
The supported claim is therefore narrow. Window-six codon coordinates expose a local boundary-tile structure in the curated standard-code label table, and the amino-acid-class-preserving null row gives $p_{\mathrm{exact}}=0.0$ with lower-tail value $0.0035$ in $2000$ samples. The contact does not test translation realization, folded protein structure, structural order, physical admissibility, biological function, a universal mechanism, or biological necessity. Amino-acid label-sequence readback remains a code-layer observation. Any promotion beyond this layer requires a separate external contact that directly measures the promoted layer.
