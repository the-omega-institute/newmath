\subsection{Amino-acid-class preserving null model for the window-six codon tile}
\label{sec:bioreality-aa-class-preserving-null}
\origin{ai}

\paragraph{Finite object and contact.}
The finite object considered here is the standard-code codon table viewed through the six-coordinate codon cube $Q_6$.  The biological contact is deliberately narrow: a curated standard-code table assigns each codon to an amino-acid label or to a stop label.  The chapter uses that contact only for code-layer label readback.  It does not treat the table as an observation of translation dynamics, folded structure, physical admissibility, biological function, mechanism, or universality.  Those higher claims would require separate observations with their own finite rows.

The coordinate display separates a local tile, a singleton label corner, and a three-point label class inside the codon cube.  The distinguished residual set is
$$
\begin{aligned}
R=\{&AAA, AGA, AGG, AUA, CUA, CUC, CUG, CUU,\\
   &UAA, UAG, UCA, UGA, UUA\}.
\end{aligned}
$$
The row records $|R|=13$, and the punctured-cube computation gives the face-vector
$$
\begin{aligned}
(f_0,f_1,f_2)=(13,16,4).
\end{aligned}
$$
Thus the object under test is not an informal visual similarity claim.  It is a finite subset of the codon cube with a recorded combinatorial profile and a fixed code-table label interpretation.

\paragraph{Amino-acid-class preserving null experiment.}
The targeted null model preserves amino-acid classes while randomizing the relevant label placement.  In the recorded run, the sample size was $N=2000$ with seed $42$.  The observed module size was $|M|=20$.  The exact-shape count under the amino-acid-class preserving null was zero, while the lower-tail count at or below the observed module size was seven.  Equivalently, the recorded empirical tail values are
$$
\begin{aligned}
 p_{\mathrm{exact}}&=0.0,\\
 p_{\mathrm{lower\ tail}}&=7/2000=0.0035.
\end{aligned}
$$
This is the central finite-row result for the present claim.  Within this null ensemble, preserving amino-acid classes does not by itself reproduce the observed local codon-tile shape.  The result supports a code-layer distinction between the curated standard-code labeling and the local tile profile, but it does not promote that profile into a law of molecular implementation.

A related shape-significance row, using $N=2000$ and the same seed $42$, records $57377$ distinct-sample attempts and $55377$ duplicate retries.  It found one exact shape match and twenty-five matches with Jaccard similarity at least $0.9$, giving
$$
\begin{aligned}
 p_{\mathrm{shape}}&=1/2000=0.0005,\\
 p_{J\geq 0.9}&=25/2000=0.0125.
\end{aligned}
$$
These auxiliary values are consistent with the class-preserving result: the observed configuration is rare in the sampled finite null spaces, but the rarity is a statement about this code-read geometry, not about protein folding or cellular causation.

The hit rate above baseline was $0.9841269841269841$, the hit rate inside $M-T$ was $0.9365079365079365$, and the hit rate inside $M-T$ together with the boundary was $0.9841269841269841$. The hit rates above baseline were $0.8333333333333334$ for alternative yeast, $1.0$ for bacterial plastid, $1.0$ for candidate other, $1.0$ for mitochondrial, and $1.0$ for nuclear ciliate.

The leave-one-codon-out row records no anchors, five boundary codons, eight high-coverage closure cases, eight predictable cases, and recovery rate $1.0$.  The thresholded recovery field is also $1.0$.  These values say that, after removing one codon at a time in the specified finite setting, the local code-read geometry still recovers the recorded targets through predictable or boundary behavior.  The check is again local: it probes robustness of a codon-label tile, not organismal translation or protein phenotype.

Each module has either predictable or boundary status in the finite row.  This is important because the null-model result is not resting on an untracked residue: the entire residual set is accounted for by the listed modules.

\paragraph{Interpretation and limits.}
The finite evidence supports the following code-layer statement.  In the six-coordinate codon cube, the standard-code labels define a local tile whose residual set has cardinality $13$, face-vector $(13,16,4)$, a recorded spectral signature with $\lambda_M=0.6752479308830213$, and rare occurrence under both the general shape-sampling row and the amino-acid-class preserving null row.  The class-preserving null is especially direct for the present claim: among $2000$ sampled class-preserving alternatives, no exact shape match occurred, and only seven samples landed in the lower tail at or below the observed module size.

The licensed conclusion is correspondingly modest.  The curated code table can be used to read codon labels, and the resulting finite geometry exposes a local split with strong sampled-null separation.  Nothing in these rows observes ribosomal realization, protein folding, structural order, physical admissibility, cellular function, a universal biological mechanism, or a global law governing DNA.  Any such stronger statement would have to be introduced only after a separate contact supplies data at that layer.
