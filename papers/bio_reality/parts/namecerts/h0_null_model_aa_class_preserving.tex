\subsection{Amino-acid-class-preserving null model for the window-six codon tile}
\label{sec:h0-null-model-aa-class-preserving}

\paragraph{Finite object and readback layer.}
This record concerns the standard-code codon table only at the code-read layer.  The carrier is the six-coordinate codon cube $Q_6$, with each codon displayed as a six-bit word and read back through the curated standard-code label table.  The local object isolated by the window-six construction is a finite tile with singleton label corners and three-point label classes; its interpretation is therefore a statement about codon labels, not about translation dynamics.  The observed boundary row is the thirteen-codon set
$$
\begin{aligned}
R=\{&\mathrm{AAA},\mathrm{AGA},\mathrm{AGG},\mathrm{AUA},\mathrm{CUA},\mathrm{CUC},\mathrm{CUG},\mathrm{CUU},\\
&\mathrm{UAA},\mathrm{UAG},\mathrm{UCA},\mathrm{UGA},\mathrm{UUA}\}.
\end{aligned}
$$
The finite audit gives $|R|=13$, and the same row is identified as a punctured-cube boundary with face-vector $(13,16,4)$.  The module decomposition is complete: the $13$ row codons are assigned with no extras, no omissions, and no unassigned codons.  The listed modules are $\{\mathrm{AGA},\mathrm{AGG}\}$ for $\mathrm{agr}$, $\{\mathrm{AAA}\}$ for $\mathrm{ar\_lys}$, $\{\mathrm{CUA},\mathrm{CUC},\mathrm{CUG},\mathrm{CUU}\}$ for $\mathrm{cun}$, $\{\mathrm{AUA}\}$ for $\mathrm{ile\_met}$, $\{\mathrm{UCA},\mathrm{UUA}\}$ for $\mathrm{ssn}$, and $\{\mathrm{UAA},\mathrm{UAG},\mathrm{UGA}\}$ for $\mathrm{stop\_trp}$.

\paragraph{Null model.}
The null tested here preserves amino-acid class information while randomizing the finite label placement subject to that preservation.  Against $N=2000$ samples with seed $42$, the observed window-six object has $|M|=20$.  The amino-acid-class-preserving null produced no exact shape match:
$$
\begin{aligned}
N&=2000,\\
\#\{\text{exact shape matches}\}&=0,\\
p_{\mathrm{exact}}&=0.0.
\end{aligned}
$$
The lower-tail statistic is also small: only $7$ of the $2000$ null samples had size at most the observed value $|M|=20$, giving
$$
\begin{aligned}
p_{\mathrm{lower\ tail}}=\frac{7}{2000}=0.0035.
\end{aligned}
$$
Thus the finite-row evidence rejects the explanation that the observed local tile is a typical outcome of an amino-acid-class-preserving shuffle, at the finite sampling resolution of this experiment.  This is a code-table statement: it says that the curated labels and the window-six coordinates interact nontrivially under this null, not that the codon cube is a biological law.

\paragraph{Coordinate and spectral checks.}
The same finite object has a spectral signature recorded by the row audit.  The measured values are $\lambda_M=0.6752479308830213$, $\lambda_R=0.4758352843197002$, and $\mu=3.051487585298128$.  The radial polynomial residual is $-6.224354365258478\cdot 10^{-12}$, equivalently recorded as the cubic-polynomial residual, and the spectral-radius value in tolerance is again $0.6752479308830213$.  These numbers provide an internal coordinate check on the finite tile: the row is not being read merely as a set of names, but as a structured codon-cube subset with a stable spectrum.  A separate shape-significance probe with $N=2000$ and seed $42$ found $1$ exact match after $57377$ distinct-sample attempts, with $55377$ duplicate retries, giving $p_{\mathrm{exact}}=0.0005$.  In the relaxed high-overlap count, $25$ samples had Jaccard score at least $0.9$, giving $p_{J\ge 0.9}=0.0125$.

\paragraph{Predictive readback.}
The leave-one-code-out probe evaluated $23$ tables and $63$ total targets.  Its baseline hit rate was $0.19821739130434785$, while the hit rate above baseline was $0.9841269841269841$.  The hit rate inside $M\setminus T$ was $0.9365079365079365$, and the hit rate inside $M\setminus T$ plus the boundary row was $0.9841269841269841$.  By family, the hit-rate-above-baseline values were $0.8333333333333334$ for alternative yeast, $1.0$ for bacterial plastid, $1.0$ for candidate other, $1.0$ for mitochondrial, and $1.0$ for nuclear ciliate.  The leave-one-codon-out probe records $0$ anchor codons, $5$ boundary codons, $8$ high-coverage closure codons, $8$ predictable codons, and recovery rate $1.0$.  These readbacks support the local boundary-tile interpretation: the code table supplies enough finite label contact to test held-out codon and held-out code rows.

\paragraph{Scope boundary.}
The result is deliberately local.  A curated standard genetic-code table assigns codons to amino-acid or stop labels, and this contact can test code readback and local codon-tile label splits.  It cannot test translation realization, folded protein structure, structural order, physical admissibility, biological function, mechanism, or universality.  In particular, amino-acid label-sequence readback is not folded protein structure, and a local codon-tile pattern is not a global biological law.  Any claim crossing from code labels to translation, folding, physical admissibility, function, mechanism, or universality requires a separate reality contact supplying that layer.  With only the present contact, the justified conclusion is that the window-six coordinate system exposes a statistically sharp local codon-label tile under an amino-acid-class-preserving null model.
