The object tested here is the standard-code codon table, read only as a finite code-layer assignment from codons to amino-acid or stop labels.  The BEDC carrier is the six-bit codon cube $Q_6$, equipped with four local distinctions: the six-bit codon display, a local tile face, singleton label corners, and three-point label classes. This contact supplies codon-label readback, but it does not supply translation dynamics, folded protein structure, physical admissibility, biological function, a universal mechanism, or any claim that DNA must obey the window-six geometry as a biological necessity.

\paragraph{Amino-acid-class-preserving null.}
The null model tested in this row preserves amino-acid class while randomizing the relevant finite labels.  With seed $42$ and $N=2000$ trials, the observed local set size is $|M|=20$.  The exact-shape match count is $0$, so the reported Monte Carlo exact-shape value is
$$
\begin{aligned}
p_{\mathrm{aa\ class,exact}}=0.0.
\end{aligned}
$$
For the lower-tail statistic, $7$ of the $2000$ null trials had size at most the observed $|M|=20$, giving
$$
\begin{aligned}
p_{\mathrm{aa\ class,lower\ tail}}=7/2000=0.0035.
\end{aligned}
$$
This is the chapter-specific null result: after preserving amino-acid class, the observed window-six tile remains in the lower tail of the null ensemble, and no exact shape match appears among the sampled null rows.  The value $0.0$ is the finite-row report for zero observed matches in $2000$ trials; it is not a theorem that the exact probability is mathematically zero.

\paragraph{Unrestricted shape comparison.}
The companion shape-significance run also used seed $42$ and $N=2000$ distinct sampled tables.  It required $57377$ sample attempts and recorded $55377$ duplicate retries.  In that run, the exact shape appeared once, giving
$$
\begin{aligned}
p_{\mathrm{exact\ shape}}=1/2000=0.0005.
\end{aligned}
$$
The high-overlap statistic recorded $25$ rows with Jaccard similarity at least $0.9$, hence
$$
\begin{aligned}
p_{J\geq 0.9}=25/2000=0.0125.
\end{aligned}
$$
Together with the amino-acid-class-preserving null, this shows that the local tile is not only a raw coordinate artifact: it remains finite-row exceptional under the stated null sampling regimes.  The conclusion is still code-layer only.  It says that the standard-code labels occupy an unusual local boundary pattern in the six-bit display; it does not say why the biological code arose, how translation is physically realized, or whether the same pattern controls protein structure or function.
