\paragraph{Finite carrier and readback contact.}
This record concerns the standard-code codon table read as finite row data on the six-bit codon cube $Q_6$.  The carrier is not a mechanism for translation; it is a coordinate display in which each codon is a point and each curated amino-acid or stop assignment is an external label contact.  The relevant readback contact is the curated standard-code table: it assigns each codon to an amino-acid or stop label, and it supports code-layer label readback only.  The distinguished boundary row $R$ has cardinality $13$ and is exactly
$$
\begin{aligned}
R = \{&\mathrm{AAA},\mathrm{AGA},\mathrm{AGG},\mathrm{AUA},\mathrm{CUA},\mathrm{CUC},\mathrm{CUG},\mathrm{CUU},\\
&\mathrm{UAA},\mathrm{UAG},\mathrm{UCA},\mathrm{UGA},\mathrm{UUA}\}.
\end{aligned}
$$
As a punctured-cube row, this same set has finite face-vector
$$
\begin{aligned}
(f_0,f_1,f_2)=(13,16,4).
\end{aligned}
$$
Thus the object under test is a local tile with $13$ vertices, $16$ one-dimensional incidences, and $4$ two-dimensional faces inside the window-six display.  The observed model row has size $|M|=20$; the explicit finite list of the $20$ codons in $M$ is not included in this row packet, so this chapter uses the supplied scalar size and the supplied tests against $M$ without inventing a missing list.

\paragraph{Null model.}
The null model preserves amino-acid class structure while randomizing the codon-label arrangement within that constraint.  Its target is narrow: it asks whether the observed local shape can be reproduced by an amino-acid-class-preserving reshuffling, not whether the code is biologically necessary or whether the geometry explains translation.  In $N=2000$ null samples with seed $42$, the exact observed shape appeared $0$ times.  The lower-tail count for samples with size at most the observed value $|M|=20$ was $7$.  Hence the finite-row estimates are
$$
\begin{aligned}
\widehat p_{\mathrm{exact}} &= 0/2000 = 0,\\
\widehat p_{\mathrm{lower}} &= 7/2000 = 0.0035.
\end{aligned}
$$
Within this finite run, the amino-acid-class-preserving null therefore does not reproduce the exact tile, and the lower-tail event is also sparse.  This is evidence for a code-layer local boundary pattern relative to this null family; it is not evidence for folding, function, physical admissibility, or a universal biological mechanism.

\paragraph{Independent shape check.}
A separate shape-significance row sampled $N=2000$ distinct comparison shapes under seed $42$.  The sampler required $57377$ distinct-sample attempts and encountered $55377$ duplicate retries.  It found $1$ exact shape match and $25$ matches with Jaccard similarity at least $0.9$, giving
$$
\begin{aligned}
\widehat p_{\mathrm{shape}} &= 1/2000 = 0.0005,\\
\widehat p_{J\ge 0.9} &= 25/2000 = 0.0125.
\end{aligned}
$$
This second finite-row check points in the same direction as the amino-acid-class-preserving null: the observed local tile is uncommon among the sampled alternatives.  The high duplicate count is part of the audit surface rather than a defect to hide; it says that the finite comparison space has substantial collision pressure under the sampling procedure.

\paragraph{Local verdict and boundary.}
The finite-row verdict is that the window-six codon coordinate exposes a local boundary-tile pattern for the standard-code label table, and this pattern is not reproduced by the amino-acid-class-preserving null in the reported $2000$-sample run.  The strongest supported statement remains code-layer readback: the curated table supplies labels, the coordinate system supplies local geometry, and the probes test the compatibility of those two finite objects.  Nothing in this row supplies translation realization, folded protein structure, structural order, physical admissibility, biological function, mechanism, or universality.  Any claim crossing one of those boundaries would require a separate contact designed for that layer.
