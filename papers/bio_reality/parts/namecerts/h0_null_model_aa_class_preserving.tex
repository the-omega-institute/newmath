\subsection{Amino-Acid-Class-Preserving Null Model for the Window-Six Codon Tile}
\label{sec:bior-reality-aa-class-preserving-null-model}

\paragraph{Finite carrier and readback.}
The finite carrier is the six-bit codon cube $Q_6$, read through a curated standard-code label table.  The coordinate layer sees a codon as a point of $Q_6$; the reality contact supplies only the amino-acid or stop label attached to that point.  In this record the distinguished boundary row is the thirteen-codon set
$$
\begin{aligned}
R=\{&\mathrm{AAA},\mathrm{AGA},\mathrm{AGG},\mathrm{AUA},\mathrm{CUA},\mathrm{CUC},\mathrm{CUG},\mathrm{CUU},\mathrm{UAA},\\
&\mathrm{UAG},\mathrm{UCA},\mathrm{UGA},\mathrm{UUA}\}.
\end{aligned}
$$
The row count is therefore $|R|=13$.  Its finite face data match the punctured-cube vector $(13,16,4)$, so the observed row is not treated as an informal drawing but as a finite cubical object with thirteen vertices, sixteen local edges, and four local two-faces.  The measured module decomposition assigns all thirteen row codons: $\mathrm{AGR}=\{\mathrm{AGA},\mathrm{AGG}\}$, $\mathrm{ar\_lys}=\{\mathrm{AAA}\}$, $\mathrm{CUN}=\{\mathrm{CUA},\mathrm{CUC},\mathrm{CUG},\mathrm{CUU}\}$, $\mathrm{ile\_met}=\{\mathrm{AUA}\}$, $\mathrm{SSN}=\{\mathrm{UCA},\mathrm{UUA}\}$, and $\mathrm{stop\_trp}=\{\mathrm{UAA},\mathrm{UAG},\mathrm{UGA}\}$.  The module audit has $R\_\mathrm{size}=13$, assigned count $13$, no missing module codons, no extra module codons, and no unassigned codons.

\paragraph{Observed tile size and amino-acid-class-preserving null.}
The observed local tile has size $|M|=20$.  Under the amino-acid-class-preserving randomization, $N=2000$ finite rows were sampled with seed $42$.  The exact observed shape occurred $0$ times, giving
$$
\begin{aligned}
p_{\mathrm{class},\mathrm{exact}}&=0.0,\\
p_{\mathrm{class},\mathrm{exact}\text{-}\mathrm{significant}}&=0.0.
\end{aligned}
$$
The lower-tail count for rows with size at most the observed value was $7$, hence
$$
\begin{aligned}
p_{\mathrm{class},\mathrm{lower\ tail}}&=\frac{7}{2000}=0.0035,\\
p_{\mathrm{class},\mathrm{lower\ tail}\text{-}\mathrm{significant}}&=0.0035.
\end{aligned}
$$
This is the main finite-row verdict: preserving amino-acid class does not by itself reproduce the observed local tile.  The null does not merely fail exact reconstruction; it also places the observed size in a small lower tail of the sampled class-preserving ensemble.

\paragraph{Shape-control comparison.}
A separate shape-control run used $N=2000$ samples and required $57377$ distinct-sample attempts because $55377$ proposed samples were duplicate retries.  In that ensemble there was $1$ exact shape match, giving $p_{\mathrm{exact}}=0.0005$, and there were $25$ matches with Jaccard similarity at least $0.9$, giving $p_{J\geq 0.9}=0.0125$.  Thus the observed window-six tile is visible both as an exact-shape rarity and as a high-Jaccard rarity.  The amino-acid-class-preserving null is stricter for the exact shape in this finite run, since its exact-match count is $0$ rather than $1$, while its lower-tail statistic supplies a separate size-sensitive check.

They do not say that the biological code is forced by a spectral law; they say that the observed row has a reproducible numerical signature inside the chosen six-bit carrier.

\paragraph{Leave-one contacts.}
The leave-one-code-out probe evaluated $23$ tables and $63$ total targets.  Its baseline hit rate was $0.19821739130434785$, while the hit rate above baseline was $0.9841269841269841$.  The hit rate inside $M-T$ was $0.9365079365079365$, and the hit rate inside $M-T$ plus boundary was $0.9841269841269841$.  By family, the hit-rate-above-baseline values were $1.0$ for bacterial-plastid, $1.0$ for mitochondrial, $1.0$ for nuclear-ciliate, $1.0$ for candidate-other, and $0.8333333333333334$ for alternative-yeast, across $5$ families.  The leave-one-codon-out probe had anchor count $0$, boundary count $5$, high closure coverage $8$, predictable count $8$, and recovery rate $1.0$.  These leave-one probes support local code-read stability: the finite tile is useful for reading held-out codon-label events at the code layer.

\paragraph{Boundary of the claim.}
The contact is a curated standard-code table, so it tests codon-label readback and local label splits.  It does not test translation realization, folded protein structure, structural order, physical admissibility, biological function, mechanism, or universality.  The finite row may expose a local boundary-tile structure in the window-six display, and the amino-acid-class-preserving null shows that this structure is not reproduced in the sampled null ensemble.  Nothing in these rows promotes the tile into a global biological law.  In particular, amino-acid label-sequence readback is not folded protein structure, and a codon-coordinate pattern is not evidence of function unless a separate higher-layer contact supplies that evidence.

\paragraph{Local verdict.}
The finite-row conclusion is therefore a scoped code-read statement.  The thirteen-codon row $R$, the twenty-point tile size $|M|=20$, the class-preserving exact-match count $0/2000$, the lower-tail value $7/2000=0.0035$, the punctured-cube vector $(13,16,4)$, the spectral signature $(\lambda_M,\lambda_R,\mu)$, and the leave-one recovery statistics jointly support a local window-six boundary pattern in the standard-code readback.  The result remains a boundary record rather than a biological mechanism: it is strong enough to reject this finite amino-acid-class-preserving null for the observed local tile, and deliberately too weak to claim translation, folding, physical admissibility, function, mechanism, or universality.
