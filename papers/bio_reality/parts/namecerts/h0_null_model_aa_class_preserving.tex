\paragraph{Amino-acid-class-preserving null.}
The null experiment preserving amino-acid class evaluated $N=2000$ finite-row samples with seed $42$.  Against the observed tile size $|M|=20$, it found no exact-shape matches:
$$
\begin{aligned}
\#\{\hbox{exact matches}\}&=0,\\
p_{\mathrm{aa\ class,exact}}&=0.0.
\end{aligned}
$$
The lower-tail statistic counted $7$ samples with size at most the observed $M$ size, giving
$$
\begin{aligned}
p_{\mathrm{aa\ class,lower\ tail}}=\frac{7}{2000}=0.0035.
\end{aligned}
$$
The row therefore survives a stricter label-aware null: preserving amino-acid class does not make the observed local tile typical.  This is evidence for a finite code-read anomaly, not evidence for a biochemical mechanism.
