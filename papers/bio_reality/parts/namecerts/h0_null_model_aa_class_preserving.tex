\paragraph{Amino-acid-class-preserving null model.}
The class-preserving null row is stricter: it tests the same local tile against finite samples that preserve amino-acid class information.  With $N=2000$ and seed $42$, the observed $M$ size is $20$.  The run reports $0$ exact matches and $7$ samples in the lower tail at or below the observed $M$ size.  Its empirical values are
$$
\begin{aligned}
