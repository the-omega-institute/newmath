\subsection{Amino-Acid-Class-Preserving Null Test for the Window-Six Codon Tile}\label{sec:bioreality-aa-class-preserving-null}\origin{ai}

\paragraph{Finite object and readout.}
The record concerns the standard-code codon table at the code-read layer. The finite carrier is the six-coordinate codon cube $Q_6$, equipped with the codon-label table as a curated contact. The observed residue set singled out by the window-six local tile has cardinality $13$ and is exactly
$\{\mathrm{AAA},\mathrm{AGA},\mathrm{AGG},\mathrm{AUA},\mathrm{CUA},\mathrm{CUC},\mathrm{CUG},\mathrm{CUU},\mathrm{UAA},\mathrm{UAG},\mathrm{UCA},\mathrm{UGA},\mathrm{UUA}\}$.
The same row is reported as a punctured-cube pattern with face data $(13,16,4)$, so the tested object is not merely a label count: it is a finite subset with vertex, edge, and two-face incidence. The observed complementary module size used by the null test is $20$.

\paragraph{Class-preserving null model.}
The null model preserves amino-acid label classes and asks whether class-respecting randomizations reproduce the same local tile shape. In the finite run with seed $42$ and $N=2000$ random trials, the exact-shape count was $0$. The lower-tail event, counted as samples with size at most the observed $M$-size $20$, occurred $7$ times. Thus the empirical row records
$$
\begin{aligned}
N&=2000,\\
\#\{\text{exact class-preserving matches}\}&=0,\\
\#\{\text{lower-tail samples with size}\leq 20\}&=7,\\
p_{\mathrm{exact}}&=0.0,\\
p_{\mathrm{lower}}&=0.0035.
\end{aligned}
$$
The exact-match value is an empirical finite-run value, not a proof that the probability is mathematically zero. What the row supports is sharper and more limited: under the sampled amino-acid-class-preserving null, the observed window-six local tile is rare enough that the exact-shape event is absent from $2000$ trials and the lower-tail event occurs at rate $0.0035$.

\paragraph{Comparison with the unrestricted shape test.}
A related shape-significance row used the same seed $42$ with $N=2000$ but required $57377$ distinct-sample attempts because $55377$ retries produced duplicates. In that row there was $1$ exact shape match and $25$ samples with Jaccard similarity at least $0.9$, giving $p_{\mathrm{exact}}=0.0005$ and $p_{J\geq 0.9}=0.0125$. The amino-acid-class-preserving test is therefore not a repetition of the same null: it keeps the class constraint and still reports no exact match in the finite run. The two rows jointly say that the local tile is visible both as a geometric subset of $Q_6$ and as a structure not easily reproduced after retaining only coarse amino-acid class information.

This gives a numerical fingerprint of the finite tile, but it remains a code-layer fingerprint of the codon table rather than a dynamical model of translation.

The statement that every module has a predictable or boundary representative is therefore a finite enumeration claim over this six-block partition, not an asymptotic biological law.

\paragraph{Scientific scope.}
The contact used here is a curated standard genetic-code table assigning codons to amino-acid or stop labels. That contact is sufficient for codon-label readback, local tile splitting, class-preserving randomization, finite module classification, and leave-one recovery checks. It does not observe ribosomal realization, folded protein geometry, molecular admissibility, biological activity, evolutionary mechanism, or universality across all organisms. Consequently, the supported claim is local and code-layered: the six-coordinate codon display exposes a rare and recoverable boundary-tile pattern in the standard-code label table. No conclusion about translation, folding, physical feasibility, function, mechanism, or universal biological necessity follows from these rows unless a separate contact supplies that higher-layer observation.
