\subsection{Amino-acid-class preserving null model for the window-six codon tile}
\label{sec:h0-null-model-aa-class-preserving}

\paragraph{Finite-row object.}
This record concerns the standard-code codon table at the code-read layer only.  The carrier is the six-bit codon cube $Q_6$, read through a curated table that assigns each codon to an amino-acid or stop label.  The BEDC distinctions used here are a six-bit codon display, a local tile face, a singleton label corner, and a three-point label class.  The tested null is class-preserving: the amino-acid label classes are kept as the admissible shuffle blocks, so the comparison does not destroy the observed label multiplicities before asking whether the same local boundary shape reappears.

\paragraph{Observed boundary set.}
The finite boundary row $R$ has cardinality $13$.  Its codons are
$$\begin{aligned}
R=\{&\mathrm{AAA},\mathrm{AGA},\mathrm{AGG},\mathrm{AUA},\mathrm{CUA},\mathrm{CUC},\mathrm{CUG},\mathrm{CUU},\\
&\mathrm{UAA},\mathrm{UAG},\mathrm{UCA},\mathrm{UGA},\mathrm{UUA}\}.
\end{aligned}$$
The same row is recorded as a punctured-cube shape with face-vector
$$\begin{aligned}
(f_0,f_1,f_2)=(13,16,4).
\end{aligned}$$
Thus the object being tested is not an unconstrained list of codons; it is a local cubical boundary pattern inside the six-coordinate codon display.  The associated module classification accounts for all $13$ codons, with $R\_size=13$, $assigned\_count=13$, no extra module codons, no missing module codons, and no unassigned codons.  The six recorded modules are
$$\begin{aligned}
\mathrm{agr}&=\{\mathrm{AGA},\mathrm{AGG}\},&
\mathrm{ar\_lys}&=\{\mathrm{AAA}\},&
\mathrm{cun}&=\{\mathrm{CUA},\mathrm{CUC},\mathrm{CUG},\mathrm{CUU}\},\\

\end{aligned}$$
Every listed module has either a predictable or boundary role in the finite row, so the local tile is fully assigned at the module level.

\paragraph{Class-preserving null result.}
For the amino-acid-class preserving null, $N=2000$ sampled class-preserving tables were evaluated with seed $42$.  The observed local object has $|M|=20$.  The exact-shape count under this null was $0$, giving the recorded empirical exact-shape value
$$\begin{aligned}
p\_{\mathrm{aa\ class,exact}}=0.0.
\end{aligned}$$
The lower-tail comparison counted $7$ samples with size at most the observed $M$ size, so
$$\begin{aligned}
p\_{\mathrm{aa\ class,lower\ tail}}={7\over 2000}=0.0035.
\end{aligned}$$
Within this finite row, the amino-acid-class preserving null therefore does not reproduce the exact observed tile shape, and only a small lower-tail fraction reaches an $M$ size no larger than the observed value.  This is the chapter's positive finite-row statement: the local six-coordinate codon geometry is not typical under the recorded class-preserving shuffle experiment.

\paragraph{Comparison with the unconstrained shape row.}
A separate shape-significance row used $N=2000$ samples and found $1$ exact shape match, with $p\_{\mathrm{exact}}=0.0005$.  The same run found $25$ samples at Jaccard similarity at least $0.9$, with $p\_{\mathrm{Jaccard}\ge 0.9}=0.0125$.  The sampler reported $57377$ distinct-sample attempts and $55377$ duplicate retries at seed $42$.  These figures give a second finite-row check: the local tile is rare not only under the amino-acid-class preserving null, but also under the recorded shape comparison.  The class-preserving null is the stricter row for the present claim because it keeps amino-acid class structure in the sampling frame.

\paragraph{Reality contact and boundary.}
The only reality contact used here is the curated standard-code table, whose observed fact is that codons are assigned to amino-acid or stop labels.  That contact tests code readback and local codon-tile label split.
