\subsection{Amino-acid-class-preserving null model for the codon tile}
\label{sec:h0-null-model-aa-class-preserving}

\paragraph{Finite setting.}
The object in this chapter is the standard-code codon table read as a finite labelled cube.  Each codon is displayed by six binary coordinates, with the curated standard-code label table supplying the amino-acid or stop label attached to each codon.  The construction is intentionally local: it asks whether the observed label split has a detectable shape inside the six-coordinate codon display, not whether the display explains translation or any subsequent biological process.  The distinguished boundary row is the thirteen-codon set
$$
\begin{aligned}
R=\{&\mathrm{AAA},\mathrm{AGA},\mathrm{AGG},\mathrm{AUA},\mathrm{CUA},\mathrm{CUC},\mathrm{CUG},\mathrm{CUU},\\
&\mathrm{UAA},\mathrm{UAG},\mathrm{UCA},\mathrm{UGA},\mathrm{UUA}\}.
\end{aligned}
$$
The row therefore has $|R|=13$.  Its recorded cubical profile is the punctured-cube vector $(13,16,4)$: thirteen vertices, sixteen local edges, and four local square faces in the induced tile.  The observed complementary or marked tile has recorded size $|M|=20$ in the amino-acid-class-preserving experiment.

\paragraph{Null model.}
The null model preserves amino-acid class information while resampling the codon placement of the local tile.  Thus the comparison is not against arbitrary relabellings of all sixty-four codons; it is against finite alternatives that keep the amino-acid-class constraint visible and test whether the same local shape is nevertheless common.  In the recorded run, the number of sampled alternatives was $N=2000$ with seed $42$.  Among those alternatives, the exact observed shape occurred zero times:
$$
\begin{aligned}
N&=2000,\\
\#\{\hbox{exact shape matches}\}&=0,\\
p_{\mathrm{exact}}&=0.0.
\end{aligned}
$$
The lower-tail comparison counted seven samples with size at most the observed value $|M|=20$, giving
$$
\begin{aligned}
\#\{|M_{\mathrm{sample}}|\leq 20\}&=7,\\
p_{\mathrm{lower\ tail}}&=0.0035.
\end{aligned}
$$
These numbers make the finite claim sharper than a visual pattern report: under this class-preserving null, the exact local tile was not seen in the sample, and the observed size lies in a lower tail of frequency $7/2000$.

This spectral datum is not used as a biological mechanism.  It is a finite diagnostic attached to the labelled cube, useful for distinguishing this tile from class-preserving random alternatives.

\paragraph{Shape significance against the broader sampled comparison.}
A separate finite run with $N=2000$ sampled distinct shapes with $57377$ distinct-sample attempts and $55377$ duplicate retries.  In that comparison, the exact shape count was $1$, giving $p_{\mathrm{exact}}=0.0005$, and the count with Jaccard similarity at least $0.9$ was $25$, giving $p_{\mathrm{Jaccard}\geq 0.9}=0.0125$.  This supports the same local conclusion from a different finite statistic: the row is not a generic codon-window artifact at the sampled resolution.

\subsection{Scope of the biological reading}
\label{sec:h0-null-model-aa-class-preserving-scope}

\paragraph{What the record supports.}
The supported conclusion is a code-read conclusion.  The curated standard-code table supplies an external codon-to-label contact, and the six-coordinate finite display exposes a statistically uncommon local tile under the amino-acid-class-preserving null model.  The exact class-preserving shape count is $0$ out of $2000$, the lower-tail count is $7$ out of $2000$, the row has $13$ codons with punctured-cube profile $(13,16,4)$, and the predictive checks recover held-out code labels at the recorded high rates.

\paragraph{What it does not support.}
No claim about translation realization follows from these rows.  No folded protein structure, structural ordering principle, physical admissibility condition, biological function, mechanism, or universal biological law is measured here.  Amino-acid label readback is not a structure assay, and a local codon tile is not a proof that organisms must implement this geometry.  Any statement crossing from codon labels into translation, folding, chemical feasibility, function, or universality requires a separate empirical contact of the appropriate kind.  Until such a contact is supplied, the present row remains a finite code-layer result: it identifies an uncommon, recoverable local pattern in the standard-code label table and keeps all higher biological interpretations outside the claim.
