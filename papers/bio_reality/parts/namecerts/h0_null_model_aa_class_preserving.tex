\subsection{Amino-acid-class preserving null model for the window-six codon tile}\label{sec:bioreality-h0-null-model-aa-class-preserving}
\origin{ai}

\paragraph{Finite object under test.}
The record concerns the standard-code codon table viewed through a six-bit codon display on the codon cube $Q_6$.  The finite data distinguish coordinate positions, closure behavior, spectral quantities, and the relation between codon corners and curated amino-acid or stop labels.  The biological contact is deliberately narrow: the table supplies codon-to-label readback for the standard code, not a biochemical account of translation and not a structure or function observation.  The distinguished boundary set $R$ has size $13$ and is exactly
$$
\begin{aligned}
R=\{&\mathrm{AAA},\mathrm{AGA},\mathrm{AGG},\mathrm{AUA},\mathrm{CUA},\mathrm{CUC},\mathrm{CUG},\mathrm{CUU},\\
&\mathrm{UAA},\mathrm{UAG},\mathrm{UCA},\mathrm{UGA},\mathrm{UUA}\}.
\end{aligned}
$$
The same finite row records $R$ as a punctured-cube pattern with face-count vector
$$
\begin{aligned}
(f_0,f_1,f_2)=(13,16,4).
\end{aligned}
$$
Thus the row is not merely a list of exceptional codons; it is a local cubical incidence pattern whose vertices, edges, and square faces can be counted without adding any mechanism-level biological hypothesis.

\paragraph{Null model.}
The null model preserves amino-acid label classes while randomizing the relevant finite arrangement.  In $N=2000$ sampled amino-acid-class preserving trials with seed $42$, the exact local shape was never reproduced: the exact-match count was $0$, giving the recorded finite-sample value
$$
\begin{aligned}
p_{\mathrm{class,exact}}=0.0.
\end{aligned}
$$
The observed $M$ side has size $20$.  Only $7$ of the $2000$ class-preserving trials landed in the lower tail at or below this observed size, so the row records
$$
\begin{aligned}
p_{\mathrm{class,lower\ tail}}=\frac{7}{2000}=0.0035.
\end{aligned}
$$
These two numbers are the central finite-row test for claim $\mathrm{h0.null.model.aa\_class\_preserving}$: preserving the amino-acid classes is not enough, in this sample, to make the observed local tile shape typical.  The statement is empirical and finite.  It says that the displayed codon-label arrangement is unusually sharp relative to this particular null ensemble; it does not say that the geometry forces the genetic code or that the code must arise by this geometry.

\paragraph{Independent shape and spectral checks.}
A separate shape-significance row used $N=2000$ trials with seed $42$.  It found $1$ exact shape match, giving $p_{\mathrm{exact}}=0.0005$, and $25$ matches with Jaccard similarity at least $0.9$, giving $p_{J\ge 0.9}=0.0125$.  The sampling process required $57377$ distinct-sample attempts and recorded $55377$ duplicate retries, so the reported probabilities should be read as finite Monte Carlo summaries for the stated generator rather than as closed-form population probabilities.

The spectral row gives three scalar quantities attached to the local pattern:
$$
\begin{aligned}
\lambda_M&=0.6752479308830213,\\
\lambda_R&=0.4758352843197002,\\
\mu&=3.051487585298128.
\end{aligned}
$$
The cubic residual is recorded as $-6.224354365258478\times 10^{-12}$, the same value listed for the radial polynomial evaluation.  This is numerically consistent with the asserted polynomial relation at the displayed precision.  The spectral radius in tolerance is $0.6752479308830213$, matching $\lambda_M$ in the finite row.

The hit rate inside $M-T$ was $0.9365079365079365$, and the rate inside $M-T$ together with the boundary was again $0.9841269841269841$. \paragraph{Local interpretation.}
The admissible conclusion is that the window-six coordinate display, when read against the curated standard-code labels, exposes a compact local tile boundary.  The amino-acid-class preserving null test strengthens this local observation: within the stated $2000$ trials, class preservation alone did not recreate the exact shape, and only $7$ trials fell at or below the observed $M$ size.  The punctured-cube counts, the spectral scalars, the leave-one recovery rows, and the module classification all point to the same code-layer object: a finite codon-label pattern around the listed $13$ boundary codons.

\paragraph{Local boundary on interpretation.}
Nothing in this row licenses promotion beyond the code-read layer.  The curated standard-code table observes label assignments, not ribosomal translation as a realized process, not folded protein geometry, not physical admissibility, not biological function, not a mechanism of origin, and not a universal law for all biological systems.  Amino-acid label readback is therefore kept separate from folded-structure claims.  Any assertion about translation realization, structural order, physical feasibility, function, organism-specific tRNA usage, or global biological necessity would require an additional contact whose measurements live at that higher layer.  Without such a separate contact, the present record remains a finite, code-layer codon-tile result.
