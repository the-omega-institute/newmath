\paragraph{Claim and finite object.}
The claim tests whether the window-six codon coordinate display is still exceptional after the amino-acid or stop-label classes of the standard code are preserved. The biological contact is deliberately narrow: a curated standard-code table assigns each codon to an amino-acid or stop label. The coordinate carrier is the six-bit codon cube, with local faces, singleton label corners, and three-point label classes used only as finite combinatorial data. The observed exceptional set $R$ has cardinality $13$ and is exactly
$$
\begin{aligned}
R=\{&\mathrm{AAA},\mathrm{AGA},\mathrm{AGG},\mathrm{AUA},\mathrm{CUA},\mathrm{CUC},\mathrm{CUG},\mathrm{CUU},\\
&\mathrm{UAA},\mathrm{UAG},\mathrm{UCA},\mathrm{UGA},\mathrm{UUA}\}.
\end{aligned}
$$
The row also records that this same set is a punctured-cube pattern with face vector $(13,16,4)$. The observed $M$-size is $20$; the finite row does not expose a per-codon $M$ enumeration, so the argument uses the recorded size and the associated sampling statistics rather than asserting an unavailable list.

\paragraph{Class-preserving null result.}
The amino-acid-class-preserving null run used $N=2000$ samples with seed $42$. Against this null, the exact shape was not observed at all: the exact-match count was $0$, giving
$$
\begin{aligned}
p_{\mathrm{aa\ class, exact}}&=0.0,\\
p_{\mathrm{aa\ class, exact\ significant}}&=0.0.
\end{aligned}
$$
The lower-tail count at or below the observed $M$-size was $7$ out of $2000$, so the lower-tail probability is
$$
\begin{aligned}
p_{\mathrm{aa\ class, lower\ tail}}&=\frac{7}{2000}=0.0035,\\
|M|_{\mathrm{observed}}&=20.
\end{aligned}
$$
Thus the local tile remains rare even after the null model preserves amino-acid or stop-label classes. This supports a code-layer statement only: the standard label table and the six-bit coordinate display jointly single out a finite local pattern that is not typical under the class-preserving randomization used here.
