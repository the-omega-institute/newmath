\paragraph{Boundary of the claim.}
The result is a BioReality code-read record: window-six coordinates expose a finite local tile, and the curated standard-code table supplies the codon labels used to test that tile.  Nothing in this chapter promotes the tile to translation realization, folded structure, structural order, physical admissibility, biological function, universal mechanism, or a global biological law.  Amino-acid label-sequence readback is not folded protein structure.  A local codon-tile split is not evidence that DNA obeys this geometry as a biological necessity.  Any claim above the code layer requires a separate contact that directly observes the higher layer.
