\paragraph{Finite carrier and contact layer.}
This record treats the standard-code codon table only as a finite code-read contact.  The carrier is the six-bit codon cube $Q_6$, read through the curated standard-code label table.  The distinctions used by the row are the six-bit codon display, a local tile face, a singleton label corner, and a three-point label class.  The internal BEDC-side structure is coordinate, closure, spectrum, and relation data; the external contact is only the codon-to-amino-acid-or-stop label readback.  Thus the biological object is the standard-code codon table at the code layer, not translation dynamics, folded protein structure, physical admissibility, biological function, mechanism, or universality.

\paragraph{Amino-acid-class preserving null.}
The null model tested in this row preserves amino-acid class information while resampling the label geometry.  In $N=2000$ null samples with seed $42$, the observed tile had $|M|=20$.  The exact shape was never reproduced: the exact-match count was $0$, giving the reported empirical exact-shape value $p=0.0$.  The lower-tail event $|M_{\mathrm{null}}|\leq 20$ occurred $7$ times, so the reported lower-tail probability was
$$
\begin{aligned}
p_{\mathrm{lower}}=\frac{7}{2000}=0.0035.
\end{aligned}
$$
The row therefore records both $p_{\mathrm{aa\ class\ exact\ shape}}=0.0$ and $p_{\mathrm{aa\ class\ lower\ tail}}=0.0035$.  This supports the narrow claim that the observed local code-read tile is not typical under this amino-acid-class preserving finite null.  It does not support any claim about why the genetic code is physically realized, how translation occurs, whether a peptide folds, or whether a biological function follows.

\paragraph{Cannot-claim boundary.}
The contact used here is the curated standard-code table: it supplies codon-label assignments and permits local split checks.  It does not observe ribosomal translation, peptide realization, folded structure, structural order, physical admissibility, biological function, universal mechanism, or biological necessity.  The admissible conclusion is therefore deliberately bounded: window-six codon coordinates expose a statistically unusual and recoverable local boundary-tile structure under the reported amino-acid-class preserving null, but the curated labels remain external reality contacts at the code-read layer.  Any promotion beyond that layer requires a separate contact designed for the higher-layer claim.
