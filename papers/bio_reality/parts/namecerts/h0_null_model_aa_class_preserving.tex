\subsection{Amino-acid-class preserving null model for the codon boundary tile}\label{subsec:h0-null-model-aa-class-preserving}\origin{ai}

\paragraph{Claim under test.}
The record \texttt{h0.null.model.aa\_class\_preserving} tests whether the observed window-six codon tile can be treated as a routine consequence of preserving amino-acid label classes in the standard code. The carrier is the six-coordinate codon cube $Q_6$, read only as a finite codon-label display. The biological contact is restricted to the curated standard-code assignment of codons to amino-acid or stop labels. Thus the datum is a code-table comparison: it may certify a local label split, but it does not by itself certify translation dynamics, folded structure, physical admissibility, biological function, a mechanism, or a universal law.

\paragraph{Observed boundary row.}
The distinguished row $R$ has cardinality $13$. In the verified run it is exactly
$$\begin{aligned}
R=\{&AAA, AGA, AGG, AUA, CUA, CUC, CUG, CUU,\\
&UAA, UAG, UCA, UGA, UUA\}.
\end{aligned}$$
The punctured-cube check gives the same row and reports the face-vector summary $(13,16,4)$. This is a finite geometric statement about the displayed codon cube: thirteen codons occupy a local boundary-shaped subset with sixteen one-dimensional incidences and four two-dimensional faces under the chosen window-six adjacency. No higher biological assertion is being made by that face count.

\paragraph{Amino-acid-class preserving null experiment.}
The null model keeps the amino-acid class constraint and asks how often a randomized table produces the same local shape. With seed $42$ and $N=2000$ sampled tables, the observed module size is $20$. The exact-shape count is $0$, while $7$ sampled tables fall in the lower tail at or below the observed $M$-size threshold. The finite-row summary is therefore
$$\begin{aligned}
N&=2000,\\
\#\{\hbox{exact shape matches}\}&=0,\\
\#\{\hbox{lower-tail samples at observed }M\hbox{ size}\}&=7,\\
|M|_{\mathrm{obs}}&=20,\\
p_{\mathrm{exact}}&=0.0,\\
p_{\mathrm{lower\ tail}}&=0.0035.
\end{aligned}$$
Within this finite simulation, preserving amino-acid label class alone did not reproduce the exact window-six boundary tile. The lower-tail value $0.0035$ says that the observed size also lies in a sparse part of this null ensemble. Since the experiment has $2000$ samples, the zero exact count is a sampled finite-row result, not a proof that the event has mathematical probability zero.

\paragraph{Comparison with the unconstrained shape null.}
A separate shape-significance row with the same seed and $N=2000$ reports $57377$ distinct-sample attempts, $55377$ duplicate retries, one exact shape match, and $25$ samples with Jaccard score at least $0.9$. Its reported values are
$$\begin{aligned}
p_{\mathrm{exact\ shape}}&=0.0005,\\
p_{\mathrm{Jaccard}\geq 0.9}&=0.0125.
\end{aligned}$$
The amino-acid-class preserving null is therefore not a weaker confirmation of the same event. It is a stricter label-aware comparison: the exact observed boundary shape appears in none of the $2000$ class-preserving draws, while the broader shape null saw one exact match and twenty-five near matches.

\paragraph{Spectral row.}
The spectrum row records three scalar values for the same finite construction:
$$\begin{aligned}
\lambda_M&=0.6752479308830213,\\
\lambda_R&=0.4758352843197002,\\
\mu&=3.051487585298128.
\end{aligned}$$
The radial cubic residual is $-6.224354365258478\cdot 10^{-12}$, and the spectral-radius value in tolerance is $0.6752479308830213$. These numbers are internal checks on the finite matrix or relation associated with the codon tile. They help identify the same object across probes, but they do not turn a code-label pattern into a statement about molecular folding or protein activity.

\paragraph{Leave-one-code-out and leave-one-codon-out checks.}
The code-table holdout row evaluates $23$ tables and $63$ targets. Its baseline hit rate is $0.19821739130434785$, while the reported hit rate above baseline is $0.9841269841269841$. The hit rate inside $M-T$ is $0.9365079365079365$, and the rate inside $M-T$ plus the boundary is $0.9841269841269841$. The codon holdout row reports zero anchors, five boundary codons, eight high-coverage closure cases, eight predictable cases, and recovery rate $1.0$. These rows show that the local boundary split is useful for code-layer recovery under the tested omissions. They still remain code-table diagnostics: they do not observe ribosomal implementation, structural order, biochemical viability, or organism-level function.

\paragraph{Family and module decomposition.}
The family-level holdout row groups the evaluated tables into five families. The hit rates above baseline are $1.0$ for bacterial-plastid, mitochondrial, nuclear-ciliate, and candidate-other families, and $0.8333333333333334$ for the alternative-yeast family. The module classification accounts for all thirteen codons in $R$: assigned count $13$, $R$-size $13$, no extra module codons, no missing row codons, and no unassigned codons. The module map is
$$\begin{aligned}
\mathrm{agr}&=\{AGA,AGG\},\\
\mathrm{ar\_lys}&=\{AAA\},\\
\mathrm{cun}&=\{CUA,CUC,CUG,CUU\},\\
\mathrm{ile\_met}&=\{AUA\},\\
\mathrm{ssn}&=\{UCA,UUA\},\\
\mathrm{stop\_trp}&=\{UAA,UAG,UGA\}.
\end{aligned}$$
This decomposition is complete at the row level and specific to the standard-code readback used here. It is not a claim that these modules are physically realized modules, folding domains, functional units, or conserved mechanisms across all organisms.

\paragraph{Local verdict.}
The finite evidence supports a narrow conclusion: in the tested window-six codon representation, the standard-code labels expose a thirteen-codon boundary row whose exact shape is not reproduced in $2000$ amino-acid-class preserving null draws, and whose holdout behavior is strong at the code-label level. The same evidence also fixes the stopping point. The curated code table supplies codon-to-label assignments only. A separate observation would be required before promoting the result to translation realization, folded protein structure, physical admissibility, biological function, mechanistic necessity, or universality.
