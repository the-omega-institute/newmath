\paragraph{Finite carrier and readback contact.}

The external contact is therefore narrow: the table supplies codon-to-label assignments, including amino-acid and stop labels, but it does not supply translation dynamics, folded structure, physical admissibility, function, mechanism, or universality.

\paragraph{Amino-acid-class-preserving null.}
The null model preserves amino-acid class structure while resampling the relevant finite code rows.  With seed $42$ and $N=2000$ sampled tables, the observed size of $M$ is $20$.  The exact-shape count is $0$, giving
$$
\begin{aligned}
 p_{\mathrm{aa\ class,exact}}=0.0 .
\end{aligned}
$$
The lower-tail count at or below the observed $M$ size is $7$, giving
$$
\begin{aligned}
 p_{\mathrm{aa\ class,lower\ tail}}=\frac{7}{2000}=0.0035 .
\end{aligned}
$$
The exact-shape statistic and the lower-tail statistic both remain small after amino-acid classes are preserved.  Consequently the row geometry is not explained away by a null model that merely keeps the amino-acid class inventory while otherwise randomizing the local code readback.

\paragraph{Comparison with the unconstrained shape null.}
The broader shape-significance run used the same seed $42$ and $N=2000$.  It required $57377$ distinct-sample attempts and encountered $55377$ duplicate retries.  In that run there was $1$ exact shape match and $25$ matches at Jaccard score at least $0.9$, so

\begin{aligned}

\end{aligned}
$$
The amino-acid-class-preserving null is therefore not the only finite-row check supporting a non-generic local tile.
