\subsection{Amino-Acid-Class-Preserving Null Model for the Window-Six Codon Tile}
\label{sec:h0-null-model-aa-class-preserving}

\paragraph{Carrier and contact.}
The carrier is the six-bit codon display $Q_6$, read as the finite cube of codon coordinates whose visible labels are supplied by a curated standard-code table.  The coordinate layer contains the cube positions, local tile faces, singleton label corners, and three-point label classes.  The biological contact is narrower: it is only the standard-code codon-to-label table, hence it tests code-layer readback and local label splitting, not translation as a realized process.  In this row the claimed layer is therefore $\mathrm{code\_read}$, and the finite assertion is that window-six coordinates expose a local boundary-tile pattern while the amino-acid and stop labels remain external reality contacts.

\paragraph{Boundary set.}
The distinguished boundary row is the thirteen-codon set
$$
\begin{aligned}
R=\{&\mathrm{AAA},\mathrm{AGA},\mathrm{AGG},\mathrm{AUA},\mathrm{CUA},\mathrm{CUC},\mathrm{CUG},\mathrm{CUU},\\
&\mathrm{UAA},\mathrm{UAG},\mathrm{UCA},\mathrm{UGA},\mathrm{UUA}\}.
\end{aligned}
$$
The recorded cardinality is $|R|=13$.  As a punctured-cube row the same set has finite face data
$$
\begin{aligned}
f(R)=(13,16,4),
\end{aligned}
$$
so the row is not being treated as an unconstrained list of labels: it is a local cubical object with thirteen vertices, sixteen one-dimensional adjacencies, and four two-dimensional faces in the displayed boundary complex.  The module classification covers all thirteen codons without residue: $\mathrm{ar\_lys}=\{\mathrm{AAA}\}$, $\mathrm{agr}=\{\mathrm{AGA},\mathrm{AGG}\}$, $\mathrm{ile\_met}=\{\mathrm{AUA}\}$, $\mathrm{cun}=\{\mathrm{CUA},\mathrm{CUC},\mathrm{CUG},\mathrm{CUU}\}$, $\mathrm{stop\_trp}=\{\mathrm{UAA},\mathrm{UAG},\mathrm{UGA}\}$, and $\mathrm{ssn}=\{\mathrm{UCA},\mathrm{UUA}\}$.  The finite audit reports $R\_\mathrm{size}=13$, $\mathrm{assigned\_count}=13$, and empty extra, missing, and unassigned lists.

\paragraph{Amino-acid-class-preserving null row.}
The null model keeps amino-acid class structure fixed and asks whether the observed local shape is reproduced by class-preserving randomizations.  With seed $42$ and $N=2000$ sampled rows, the observed comparison set has size $|M|=20$.  The exact-shape count under the amino-acid-class-preserving null is $0$, giving
$$
\begin{aligned}
p_{\mathrm{aa\ class,exact}}=0.0.
\end{aligned}
$$
The lower-tail count for rows with size at most the observed $M$ size is $7$, hence
$$
\begin{aligned}
p_{\mathrm{aa\ class,lower\ tail}}=\frac{7}{2000}=0.0035.
\end{aligned}
$$
Thus the finite row rejects the explanation that the displayed local tile is a generic consequence of preserving amino-acid class sizes alone.  This is a code-layer statement about the codon-label table, not a statement about protein realization.

\paragraph{Boundary of the claim.}
The only admitted contact is the curated standard-code table assigning codons to amino-acid or stop labels.  Consequently the row can claim a local boundary-tile split in window-six code coordinates and can compare that split against class-preserving null models.  It cannot claim that DNA obeys this geometry as biological necessity; it cannot claim translation realization; it cannot claim folded structure, structural order, physical admissibility, biological function, mechanism, or universality.  Amino-acid label-sequence readback remains at the code layer unless a separate higher-layer contact supplies the missing evidence.
