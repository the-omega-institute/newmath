\subsection{Amino-acid-class-preserving null model for the local codon tile}
\label{sec:biorreality-aa-class-preserving-null-model}

\paragraph{Finite carrier and readback contact.}
The finite carrier is the six-bit codon cube $Q_6$ equipped with the curated standard-code label table as its external readback contact.  The contact supplies only codon-to-label assignments: it says which codons carry amino-acid labels or stop labels in the standard code, and it does not supply translation dynamics, folded structure, physical admissibility, biological function, a mechanism, or a universality theorem.  Within this bounded code-read layer, the distinguished boundary row is the thirteen-codon set
$$
\begin{aligned}
R = \{&\mathrm{AAA},\mathrm{AGA},\mathrm{AGG},\mathrm{AUA},\mathrm{CUA},\mathrm{CUC},\mathrm{CUG},\mathrm{CUU},\\
&\mathrm{UAA},\mathrm{UAG},\mathrm{UCA},\mathrm{UGA},\mathrm{UUA}\}.
\end{aligned}
$$
The row count is therefore $|R|=13$.  The same finite computation identifies $R$ as a punctured-cube boundary row with face data
$$
\begin{aligned}
f(R)=(13,16,4),
\end{aligned}
$$
so the local object being tested is not an arbitrary collection of labels but a finite tile with thirteen vertices, sixteen edges, and four two-faces in the selected window-six display.

\paragraph{Observed shape and class-preserving null.}
The observed local label-support size is $|M|=20$.  Under the amino-acid-class-preserving null model with seed $42$ and $N=2000$ sampled tables, the exact observed shape occurred $0$ times.  The empirical exact-shape probability in this run is therefore
$$
\begin{aligned}
p_{\mathrm{aa\ class,exact}}=0.0.
\end{aligned}
$$
The lower-tail statistic counts null samples with $|M_{\mathrm{null}}|\leq 20$; this occurred $7$ times among $2000$ samples, giving
$$
\begin{aligned}
p_{\mathrm{aa\ class,lower\ tail}}=7/2000=0.0035.
\end{aligned}
$$
Thus the amino-acid-class-preserving null does not merely fail to reproduce the exact tile: it also places the observed support size in a small lower tail for this finite run.  The conclusion is deliberately finite-row: it is a statement about this sampled null model, this codon display, and this curated code-read table.

\paragraph{Shape significance against unrestricted finite samples.}
A separate shape run with seed $42$ and $N=2000$ distinct sampled tables recorded $57377$ distinct-sample attempts and $55377$ duplicate retries.  In that run there was $1$ exact shape match, hence
$$
\begin{aligned}
p_{\mathrm{exact\ shape}}=1/2000=0.0005.
\end{aligned}
$$
The high-overlap row counted $25$ samples with Jaccard similarity at least $0.9$, giving
$$
\begin{aligned}
p_{\mathrm{Jaccard}\geq 0.9}=25/2000=0.0125.
\end{aligned}
$$
These two rows support the same bounded reading as the class-preserving null: the observed tile is scarce among the finite null samples, including both exact reconstruction and near-reconstruction at the stated Jaccard threshold.

\paragraph{Leave-one-code and leave-one-codon checks.}
The code-level recovery rows test whether the local tile remains informative when finite readback entries are withheld.  The leave-one-code-out run evaluated $23$ tables and $63$ targets. The hit rate inside $M\setminus T$ was $0.9365079365079365$, and the hit rate inside $M\setminus T$ plus the boundary row was $0.9841269841269841$.  The leave-one-codon-out row recorded $0$ anchors, $5$ boundary cases, $8$ high-closure-coverage cases, $8$ predictable cases, and recovery rate $1.0$.  These numbers justify treating the local tile as a code-read predictor in the finite table sense, not as a biological mechanism.
