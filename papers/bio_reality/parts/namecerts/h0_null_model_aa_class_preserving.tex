\paragraph{Finite carrier and reality contact.}
This record concerns the standard-code codon table only at the code-read layer.  The carrier is the six-bit codon cube $Q_6$, read through a curated standard-code label table assigning each codon to an amino-acid or stop label.  The BEDC-side distinctions used here are a six-bit codon display, a local tile face, a singleton label corner, and a three-point label class.  The external contact is deliberately narrow: it supplies codon-label readback, not translation realization, folded structure, physical admissibility, biological function, a mechanism, or a universality principle.  Consequently, the tested object is a finite row pattern in the codon-label table, not a biological law.

The distinguished boundary row $R$ has cardinality $13$.  The verified row is

\paragraph{Amino-acid-class-preserving null.}
The null model preserves amino-acid class information while randomizing the tested local arrangement.  In the verified run $e8d5de747a0c$, the sample size was $N=2000$ with seed $42$.  The observed size of the comparison object was $|M|=20$.  Across the $2000$ amino-acid-class-preserving null draws, the exact shape match count was $0$, so the finite sampled exact-match rate was
$$
\begin{aligned}
\widehat p_{\mathrm{aa,class,exact}}=0/2000=0.0.
\end{aligned}
$$
The lower-tail count at or below the observed $M$ size was $7$, giving
$$
\begin{aligned}
\widehat p_{\mathrm{aa,class,lower\ tail}}=7/2000=0.0035.
\end{aligned}
$$
This is the chapter-specific null result: after preserving amino-acid class, the observed local tile remains rare in the finite randomized ensemble.  The conclusion is statistical and finite-row: it says that this coordinate-and-label arrangement is not well explained by this particular class-preserving null, not that the codon table is forced by BEDC geometry.

\paragraph{Cannot-claim boundary.}
The admissible conclusion is therefore narrow.  The standard-code label table supports codon-label readback and local split checks for the six-bit codon display.  It does not supply an observation of translation realization, folded protein structure, physical admissibility, biological function, universal mechanism, or biological necessity.  Amino-acid label-sequence readback is still only label readback.  A separate higher-layer contact would be required before any claim about protein realization, folding, structural order, physical admissibility, function, mechanism, or universality could be made.  Within that boundary, the amino-acid-class-preserving null result is ready: the observed local tile has exact null frequency $0/2000$ and lower-tail frequency $7/2000$ under the stated finite randomized test, while the concrete boundary row $R$ and its module decomposition remain fully specified at the code-read layer.
