\paragraph{Amino-acid-class-preserving null model.}
The null model preserves amino-acid label classes while randomizing the finite-row shape.  In the verified run with seed $42$ and $N=2000$ samples, the observed BEDC row size was $|M|=20$.  No amino-acid-class-preserving sample exactly matched the observed finite shape:
$$
\begin{aligned}
N&=2000,\\
\#\{\text{exact shape matches}\}&=0,\\
p_{\mathrm{exact}}&=0.0.
\end{aligned}
$$
The lower-tail statistic is also small: only $7$ of the $2000$ samples had row size at most the observed value $20$, giving
$$
\begin{aligned}
p_{\mathrm{lower\ tail}}=\frac{7}{2000}=0.0035.
\end{aligned}
$$
Consequently, the amino-acid-class-preserving null does not explain the observed local tile merely by retaining label classes.  The empirical claim is finite and narrow: within this sampled null, exact shape recurrence was absent, and lower-tail recurrence was rare.  This supports a local code-read anomaly relative to the tested null, not a claim about translation, folding, physical chemistry, fitness, or universality.

It does not say that the coordinate display causes the table, realizes translation, or determines protein structure.
