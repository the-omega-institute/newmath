\subsection{Amino-acid-class-preserving null model for the local codon boundary}
\label{subsec:h0-null-model-aa-class-preserving}

\paragraph{Record.}
This record tests whether the window-six codon display has a local boundary shape that survives a null model preserving amino-acid classes. The biological contact is deliberately narrow: a curated standard-code table assigns each codon to an amino-acid or stop label. The mathematical carrier is the six-coordinate codon cube, with local tile faces, singleton label corners, and three-point label classes used only as finite code-layer distinctions. The claim is therefore not that biology must obey this geometry, but that this coordinate system exposes a finite split whose label readback can be checked against the standard-code table.

\paragraph{Observed boundary set.}
The finite boundary set $R$ has cardinality $13$. The verified row lists it as
$$
\begin{aligned}
R = \{&\mathrm{AAA},\mathrm{AGA},\mathrm{AGG},\mathrm{AUA},\mathrm{CUA},\mathrm{CUC},\mathrm{CUG},\mathrm{CUU},\\
&\mathrm{UAA},\mathrm{UAG},\mathrm{UCA},\mathrm{UGA},\mathrm{UUA}\}.
\end{aligned}
$$
The same computation identifies this set with a punctured-cube shape whose face-vector is
$$
\begin{aligned}
f(R)=(13,16,4).
\end{aligned}
$$
Thus the code-layer object is not merely a count. It is a finite local shape with $13$ vertices, $16$ one-dimensional adjacencies, and $4$ two-dimensional faces inside the codon-coordinate cube.

No mechanistic claim is inferred from the eigenvalue data.

\paragraph{Unrestricted-shape comparison.}
A broader shape-significance run sampled $N=2000$ distinct null trials with seed $42$. Obtaining those distinct trials required $57377$ sample attempts and $55377$ duplicate retries. The exact local shape appeared once, so the recorded exact-shape probability is
$$
\begin{aligned}
p_{\mathrm{exact}}=0.0005.
\end{aligned}
$$
The high-overlap comparison counted $25$ samples with Jaccard score at least $0.9$, giving
$$
\begin{aligned}
p_{\mathrm{Jaccard}\ge 0.9}=0.0125.
\end{aligned}
$$
These numbers say that the observed tile is rare under the sampled coordinate null model. They do not say that the tile is a law of translation, folding, physical chemistry, or biological function.

\paragraph{Amino-acid-class-preserving null row.}
The claim-specific null row keeps amino-acid-class information fixed while testing the same boundary-shape question. In $N=2000$ trials with seed $42$, the observed module size is $20$, the exact-match count is $0$, and the lower-tail count at or below the observed $M$ size is $7$. The reported probabilities are
$$
\begin{aligned}
p_{\mathrm{class\ exact}} &= 0.0,\\
p_{\mathrm{class\ lower\ tail}} &= 0.0035.
\end{aligned}
$$
Consequently, within this finite experiment, preserving amino-acid class does not make the observed shape typical. The evidence is stronger than a free label shuffle because the null model retains a biologically meaningful label constraint while still failing to recover the same exact local boundary.

These rows support the narrower claim that the local boundary plus the module context tracks code-label behavior across the recorded table family split.

The leave-one-codon-out row has anchor count $0$, boundary count $5$, high closure coverage $8$, predictable count $8$, and recovery rate $1.0$. The completeness row records $R$-size $13$ and assigned count $13$. Hence every codon in the observed boundary is accounted for by one of the named modules, and each module has a predictable or boundary status in the finite classification.

\paragraph{Scope of the conclusion.}
The evidence supports a code-read statement: window-six coordinates expose a rare local codon-boundary pattern, and the curated standard-code table supplies the label contact needed to read that pattern as amino-acid or stop labels. The evidence does not support promotion to translation realization, folded structure, physical admissibility, biological function, mechanism, or universality. Those stronger conclusions would require separate contacts of the appropriate kind. In particular, amino-acid label readback is not protein structure, and a local codon tile is not a global biological principle. The present row is therefore ready only as a finite code-layer record with an explicit upper boundary on interpretation.
