\paragraph{Finite carrier and contact.}
The finite carrier is the six-bit codon display $Q_6$, read as the $4^3$ codon cube with one coordinate pair per nucleotide position.  The biological contact used here is only the curated standard-code label table: it assigns each codon to an amino-acid or stop label, and it supplies no independent observation of translation dynamics, folded structure, physical admissibility, biological function, mechanism, or universality.  Thus the experiment is a code-read experiment.  Its legitimate output is a local tile and label-boundary description inside $Q_6$, together with a readback to the standard-code label table.

The module classification is complete for $R$: it assigns all $13$ codons, with $R$-size $13$, assigned count $13$, no extra module codons, no missing codons, and no unassigned codons.  The modules are

The check named by this row is that every module has predictable-or-boundary support. The two-codon modules $\mathrm{agr}$ and $\mathrm{ssn}$, the three-point stop-tryptophan class, and the four-codon $\mathrm{cun}$ face together express the local tile, singleton-corner, and three-point label-class distinctions stated by the conjecture.

\paragraph{Boundary of the claim.}
The finite row supports the following statement: window-six codon coordinates expose a local boundary-tile structure whose codons are completely classified into predictable-or-boundary modules, and the curated standard-code table reads those codons back as code labels.  The row does not support the stronger statement that DNA, translation, protein folding, structural order, physical admissibility, biological function, mechanism, or universality is generated by this geometry.  Amino-acid label-sequence readback is still label readback.  A separate higher-layer contact would be required before any translation-realization, fold-realization, physical, functional, mechanistic, or universal biological claim could be made.
