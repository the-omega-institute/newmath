\subsection{Leave-one-codon-out module classification for the standard code}\label{sec:h1-leave-one-codon-out-module-classification}
\origin{ai}

\paragraph{Finite object under test.}
The object in this record is the standard-code codon table viewed only at the code-read layer.  The carrier is the six-coordinate codon cube $Q_6$, with the four nucleotides encoded as a two-bit coordinate at each codon position.  The observed row is not a claim about translation dynamics.  It is a finite table comparison: a curated standard-code contact assigns each codon to an amino-acid or stop label, and the coordinate construction is asked whether the exceptional local tile can be recovered after codons are withheld.

The distinguished residual set is the thirteen-codon row
$$
\begin{aligned}
R=\{&AAA,AGA,AGG,AUA,CUA,CUC,CUG,CUU,\\
&UAA,UAG,UCA,UGA,UUA\}.
\end{aligned}
$$
The finite audit reports $|R|=13$.  The same row is the punctured-cube support with $f$-vector $(13,16,4)$, so the coordinate trace contains thirteen vertices, sixteen one-dimensional adjacencies, and four square faces.  These counts are the complete local combinatorial object used here.

\paragraph{Module partition.}
The leave-one-codon-out classification splits the thirteen residual codons into six named modules.  The finite assignment is
$$
\begin{aligned}
R={}&\{AAA\}_{ar\_lys}\cup\{AGA,AGG\}_{agr}\cup\{AUA\}_{ile\_met}\\
&\cup\{CUA,CUC,CUG,CUU\}_{cun}\cup\{UCA,UUA\}_{ssn}\\
&\cup\{UAA,UAG,UGA\}_{stop\_trp}.
\end{aligned}
$$
The row classifier assigned all thirteen codons: $R\_size=13$, $assigned\_count=13$, with no missing module codons, no extra module codons, and no unassigned codons.  Every module contains a codon marked predictable or boundary-supported by the leave-one-codon-out test: $ar\_lys$ contributes $AAA$; $agr$ contributes $AGA,AGG$; $ile\_met$ contributes $AUA$; $cun$ contributes $CUA,CUC,CUG,CUU$; $ssn$ contributes $UCA,UUA$; and $stop\_trp$ contributes $UAA,UAG,UGA$.  Thus the module statement is not an asymptotic or qualitative summary; it is a complete finite classification of the residual thirteen-row.

$$
The reported radial cubic residual is $-6.224354365258478\cdot 10^{-12}$, which is also the recorded cubic-polynomial residual.  The value is close to zero at numerical precision, so the spectral calculation is internally consistent for this finite row.

The shape-significance run used $N=2000$ sampled tables with seed $42$.  It observed $57377$ distinct-sample attempts and $55377$ duplicate retries.  Exact shape agreement occurred once, giving $p_{exact}=0.0005$.  High-Jaccard agreement at threshold $0.9$ occurred $25$ times, giving $p_{J\geq 0.9}=0.0125$.  In the amino-acid-class-preserving null, again with $N=2000$ and seed $42$, the observed module size was $20$, the exact match count was $0$, the lower-tail count at or below the observed size was $7$, the exact-shape value was $0.0$, and the lower-tail value was $0.0035$.  These tests do not turn the tile into a biological mechanism; they only say that this finite code-layer pattern is uncommon under the stated table randomizations.

\paragraph{Leave-one-out behavior.}
For leave-one-code-out evaluation, $23$ tables and $63$ targets were evaluated.  The baseline hit rate was $0.19821739130434785$.  The hit rate above baseline was $0.9841269841269841$, the hit rate inside $M\setminus T$ was $0.9365079365079365$, and the hit rate inside $M\setminus T$ together with the boundary was $0.9841269841269841$.  Grouped by code family, the above-baseline hit rates were $1.0$ for bacterial-plastid, $1.0$ for candidate-other, $1.0$ for mitochondrial, $1.0$ for nuclear-ciliate, and $0.8333333333333334$ for alternative-yeast, across $5$ families.

For leave-one-codon-out evaluation, the row contains $0$ anchors, $5$ boundary cases, and $8$ predictable cases.  The high-closure coverage count is $8$, and the recovery rate is $1.0$, above the recorded threshold with value $1.0$.  The module classification therefore refines the leave-one-codon-out result: the row is not merely recovered as an unlabeled set, but distributed across six named code modules without residue.

\paragraph{Reality contact and local scope.}
The only external contact used by this record is the curated standard genetic-code table.  That contact has one job: it supplies codon-to-label readback at codon resolution.  It does not observe ribosomal realization, folding, thermodynamic admissibility, organismal function, or evolutionary universality.  Consequently the result supports a local code-layer statement: window-six coordinates expose a finite boundary-tile pattern whose residual row is completely covered by predictable or boundary modules.  It does not support any claim that DNA is forced by this geometry, that translation is implemented by the tile, that an amino-acid label sequence has a folded structure, that the peptide is physically admissible, that any function follows, or that the pattern is a universal biological law.  Those stronger statements would require separate contacts at their own observational layer.
