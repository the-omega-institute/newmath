\subsection{Leave-one-codon-out module classification for the window-six codon tile}

\paragraph{Finite carrier and readback contact.}
The carrier is the six-bit codon cube $Q_6$ equipped with the standard-code label table as an external readback contact.  The finite row studied here does not assert that the biological code is generated by the cube geometry.  It asks a narrower question: after the window-six coordinate display has identified a local tile boundary, do the codons in the distinguished boundary set fall into a complete finite module classification whose members are predictable or boundary-supported under leave-one-codon-out tests?  The standard-code table is used only as a codon-to-label readback bridge.  It supplies labels for code-layer comparison, not translation dynamics, folding, physical admissibility, function, mechanism, or universality.

\paragraph{Distinguished boundary set.}
The finite boundary set $R$ has cardinality $13$.  In codon notation the observed set is
$$
\begin{aligned}
R=\{&AAA, AGA, AGG, AUA, CUA, CUC, CUG, CUU,\\
&UAA, UAG, UCA, UGA, UUA\}.
\end{aligned}
$$
The punctured-cube check returns the same $13$ codons and has face-vector data
$$
\begin{aligned}
(f_0,f_1,f_2)=(13,16,4).
\end{aligned}
$$
Thus the row is not merely a label list: it carries a finite local cell signature with $13$ vertices, $16$ one-dimensional incidences, and $4$ two-dimensional faces.  This is the geometric object to which the module classification applies.

$$
\begin{aligned}

Consequently every codon in the finite boundary set is accounted for once at the module level, and no codon outside the row is silently imported into the classification.  The classification claim is therefore a finite exact statement about the observed $R$-row, not a statistical extrapolation beyond it.

\paragraph{Boundary of the claim.}
The conclusion is deliberately local.  The finite data support the following statement: in the window-six codon display, the distinguished $13$-codon boundary set admits a complete six-module classification, every module has predictable-or-boundary support under the leave-one-codon-out probe, and the associated code-read signal is high relative to the reported baselines. Any such higher-layer statement requires a separate contact supplying the missing translation, structure, physical, functional, mechanistic, or universality evidence.
