\paragraph{Finite carrier and contact layer.}
This record concerns the standard-code codon table only at the code-read layer.  The carrier is the six-bit codon cube $Q_6$, read as a finite coordinate display for the $64$ codons.  The external contact is a curated standard genetic-code table assigning each codon to an amino-acid label or a stop label.  That contact is sufficient for codon-label readback and for local split checks, but it is not a translation experiment, not a folding experiment, not a physical-admissibility witness, not a function assay, not a mechanism, and not a universality proof.  The object classified here is therefore a finite row of codon labels in a window-six coordinate system, not a global biological law.

In the amino-acid-class-preserving null, again with $N=2000$, the exact match count was $0$, so the reported exact-shape $p$-value was $0.0$ at this sample resolution.  The observed $M$ size was $20$, and the lower-tail count at or below that size was $7$, giving $p_{\mathrm{lower\ tail}}=0.0035$.  The finite row supplied here gives the size of $M$ and its significance statistics; it does not provide the full membership list of $M$.
