\subsection{Leave-One-Codon-Out Module Classification}
\label{sec:h1-leave-one-codon-out-module-classification}

\paragraph{Finite carrier and readback contact.}
The object in this record is the standard-code codon table read through a six-bit codon display.  The carrier is the codon cube $Q_6$, while the reality contact is only a curated codon-to-label table: each codon is assigned an amino-acid label or a stop label.  The contact has codon-label resolution and does not contain a translation event, a folded structure, a physical admissibility test, a function assay, a mechanism, or a universality theorem.  Thus the data may test a code-read split in local codon coordinates, but it cannot by itself promote that split beyond the code layer.

The finite exceptional row is the set $R$ of thirteen codons
$$
\begin{aligned}
R=\{&AAA, AGA, AGG, AUA, CUA, CUC, CUG, CUU,\\
&UAA, UAG, UCA, UGA, UUA\}.
\end{aligned}
$$
The row count is $|R|=13$.  The same row is recorded as a punctured-cube object with face-vector
$$
\begin{aligned}
f(R)=(13,16,4).
\end{aligned}
$$
These numbers are finite readback facts, not a mechanism for why the biological code has its labels.

\paragraph{Boundary of the claim.}
The claim stops at code readback.  A local codon tile is not a global biological law.  Window-six geometry alone is not translation, is not folded protein structure, is not physical admissibility, is not biological function, is not a mechanism, and is not universality.  Amino-acid label-sequence readback is also not folded protein structure.  Any movement from this module classification to translation realization, structural order, physical admissibility, function realization, or a universal biological mechanism would require a separate contact with the appropriate resolution.  In the present finite rows, the curated standard-code table supplies labels, the coordinate analysis supplies local tile structure, and the module classification supplies an exhaustive thirteen-codon partition; nothing in those rows supplies a higher-layer contact.
