\paragraph{Finite carrier and contact.}
The finite carrier is the six-bit codon display $Q_6$, read only at the code layer through a curated standard-code label table.  The observable object is not translation as a process, and not protein realization; it is the finite readback assigning each codon to an amino-acid or stop label.  In this row the distinguished residual set has cardinality $13$ and is exactly
$$
\begin{aligned}
R = \{&\mathrm{AAA},\mathrm{AGA},\mathrm{AGG},\mathrm{AUA},\mathrm{CUA},\mathrm{CUC},\mathrm{CUG},\mathrm{CUU},\\
&\mathrm{UAA},\mathrm{UAG},\mathrm{UCA},\mathrm{UGA},\mathrm{UUA}\}.
\end{aligned}
$$
The same row records the punctured-cube face data for $R$ with $f$-vector $(13,16,4)$.  Thus the local object being classified is a finite boundary-tile pattern inside $Q_6$, with singleton label corners and three-point label classes read against the external table.  The contact supports code-label readback and local split checks; it does not supply a physical or mechanistic account of why a cell translates a codon, why a polypeptide folds, or why a biological function is realized.

The module-classification row is complete: $R$ has size $13$, the assigned count is $13$, and the lists of extra module codons, missing module codons, and unassigned codons are all empty.  The six modules partition the row as

Every displayed module has at least one predictable or boundary member according to the row data.  The classification therefore supports the narrow statement that the window-six coordinates expose a local boundary-tile organization in the standard-code label table.  The result is best read as a code-read packet: coordinate, closure, spectrum, and relation data cohere on a finite set of codon labels.

\paragraph{Boundary of the claim.}
Nothing in this chapter promotes the local tile to a global biological law.  Window-six geometry alone is not translation realization, not folded protein structure, not structural order, not physical admissibility, not biological function, not a mechanism, and not a universality theorem.  Amino-acid label-sequence readback stays at the code layer unless a separate higher-layer contact is supplied.  A future structure or function claim would need its own contact, with its own observable, noise model, and failure condition; it cannot be inherited from the finite codon-table classification above.
