\subsection{Leave-one-codon-out module classification in the codon cube}
\label{sec:bioreality-h1-leave-one-codon-out-module-classification}
\origin{ai}

\paragraph{Carrier and reality contact.}
The finite carrier is the six-bit codon cube $Q_6$ equipped with a standard-code label readback.  The BEDC object being tested is not the biochemical realization of translation, but the code-layer table in which each codon has a curated amino-acid or stop label.  The contact is therefore deliberately narrow: it can test codon-label readback and local tile label splits, while it cannot test translation realization, folded structure, physical admissibility, biological function, mechanism, or universality.  In this chapter the local coordinate object is a finite row system with coordinates, closure, spectrum, and relation data; the biological object is only the standard-code codon table as a label table.

\paragraph{The distinguished row set.}
The observed distinguished row set $R$ has cardinality $13$.  Its codons are
$$
\begin{aligned}
R=\{&\mathrm{AAA},\mathrm{AGA},\mathrm{AGG},\mathrm{AUA},\mathrm{CUA},\mathrm{CUC},\mathrm{CUG},\mathrm{CUU},\\
&\mathrm{UAA},\mathrm{UAG},\mathrm{UCA},\mathrm{UGA},\mathrm{UUA}\}.
\end{aligned}
$$
The punctured-cube test returns the same set, with finite face-vector
$$
\begin{aligned}
f(R)=(13,16,4).
\end{aligned}
$$
Thus the row-level claim is not merely that thirteen codons were selected, but that the selected rows carry a local cubical boundary shape with $13$ vertices, $16$ one-faces, and $4$ two-faces.  This is the finite object classified below.

\paragraph{Spectral finite-row profile.}
The same finite row system has a measured spectral profile
$$
\begin{aligned}
\lambda_M &= 0.6752479308830213,\\
\lambda_R &= 0.4758352843197002,\\
\mu &= 3.051487585298128.
\end{aligned}
$$
The radial cubic residual is $-6.224354365258478\cdot 10^{-12}$, the same value reported for the cubic-polynomial check, so the spectral-radius computation is numerically stable at the displayed tolerance.  The point of this number is local and finite: it records the spectrum of the window-six row geometry.  It is not a biological law and it does not imply that genomic translation must implement this geometry.

\paragraph{Null-model pressure.}
Against the row-shape null with $N=2000$ trials and seed $42$, the exact shape match count is $1$, giving
$$
\begin{aligned}
p_{\mathrm{exact}} &= 0.0005,\\
p_{J\geq 0.9} &= 0.0125,
\end{aligned}
$$
with $25$ high-Jaccard matches among $57377$ distinct-sample attempts and $55377$ duplicate retries.  Against the amino-acid-class-preserving null, also with $N=2000$ and seed $42$, the observed $M$-size is $20$, the exact-shape count is $0$, and the lower-tail count at or below the observed size is $7$.  The corresponding values are
$$
\begin{aligned}
p_{\mathrm{aa\ class,exact}} &= 0.0,\\
p_{\mathrm{aa\ class,lower\ tail}} &= 0.0035.
\end{aligned}
$$
These finite-row tests support the claim that the observed local shape is not a typical draw from the tested null ensembles.  They do not identify a biochemical mechanism, and they do not promote a codon-coordinate observation into a statement about protein structure or function.

\paragraph{Leave-one-code-out behavior.}
For the leave-one-code-out probe, $23$ code tables and $63$ targets were evaluated.  The baseline hit rate is $0.19821739130434785$, while the observed hit rates are
$$
\begin{aligned}
h_{\mathrm{above\ baseline}} &= 0.9841269841269841,\\
h_{M-T} &= 0.9365079365079365,\\
h_{M-T+\partial} &= 0.9841269841269841.
\end{aligned}
$$
The by-family breakdown covers $5$ families: alternative yeast has hit rate $0.8333333333333334$, while bacterial plastid, candidate other, mitochondrial, and nuclear ciliate each have hit rate $1.0$.  This supports a code-read classification statement: when related code tables are treated as finite label contacts, most held-out targets land in the predicted or boundary part of the local row system.  The interpretation remains table-level; it does not assert a universal mechanism behind code evolution.

\paragraph{Leave-one-codon-out row classification.}
For the leave-one-codon-out probe, the recovery rate is $1.0$.  The finite counters are
$$
\begin{aligned}
\mathrm{predictable\ count} &= 8,\\
\mathrm{boundary\ count} &= 5,\\
\mathrm{anchor\ count} &= 0,\\
\mathrm{closure\ coverage\ high} &= 8.
\end{aligned}
$$
The module classification is complete: $R$ has size $13$, the assigned count is $13$, and there are no extra module codons, no missing module codons, and no unassigned codons.  The six modules are
$$
\begin{aligned}
\mathrm{agr} &= \{\mathrm{AGA},\mathrm{AGG}\},\\
\mathrm{ar\_lys} &= \{\mathrm{AAA}\},\\
\mathrm{cun} &= \{\mathrm{CUA},\mathrm{CUC},\mathrm{CUG},\mathrm{CUU}\},\\
\mathrm{ile\_met} &= \{\mathrm{AUA}\},\\
\mathrm{ssn} &= \{\mathrm{UCA},\mathrm{UUA}\},\\
\mathrm{stop\_trp} &= \{\mathrm{UAA},\mathrm{UAG},\mathrm{UGA}\}.
\end{aligned}
$$
Every displayed module has at least one predictable-or-boundary witness under the leave-one-codon-out rule, and the modules partition the thirteen codons in $R$ exactly.  This is the finite classification result for the chapter: the local tile does not merely recover isolated rows, but organizes all thirteen distinguished rows into named codon modules without leftovers.

\paragraph{Scope boundary.}
The result is a window-six local tile statement at the code-read layer.  The curated standard-code table supplies labels, and the finite row computations expose a boundary-tile structure in those labels.  Nothing in this evidence supplies translation dynamics, folded protein structure, physical admissibility, biological function, a molecular mechanism, or a universality theorem.  Any claim crossing those boundaries requires a separate reality contact with the appropriate resolution.  In particular, amino-acid label-sequence readback is not folded protein structure, and a local codon tile is not a global biological law.
