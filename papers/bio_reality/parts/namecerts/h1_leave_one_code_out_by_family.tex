\subsection{Leave-One-Code-Out Stability by Code Family}\label{subsec:h1-leave-one-code-out-by-family}
\origin{ai}

\paragraph{Finite object.}
This record concerns the window-six codon coordinate model on the six-bit codon cube $Q_6$, with readback supplied by a curated standard-code label table.  The carrier is finite: each codon is displayed as a six-bit coordinate, and the observable distinctions used here are local tile faces, singleton label corners, and three-point label classes.  The claim tested in this subsection is not that this geometry is a biological mechanism.  It is the narrower code-layer claim that the local boundary tile remains stable when whole code tables are left out and then grouped by code family.

The finite boundary row $R$ has cardinality $13$.  In the verified run it is exactly
$$
\begin{aligned}
R=\{&\mathrm{AAA},\mathrm{AGA},\mathrm{AGG},\mathrm{AUA},\mathrm{CUA},\mathrm{CUC},\mathrm{CUG},\mathrm{CUU},\\
&\mathrm{UAA},\mathrm{UAG},\mathrm{UCA},\mathrm{UGA},\mathrm{UUA}\}.
\end{aligned}
$$
The same finite row is verified as a punctured-cube object with face vector $(13,16,4)$.  Thus the row is not merely a list of exceptional labels: it is a local cubical residue with thirteen codon vertices, sixteen one-dimensional incidences, and four two-dimensional tile faces.

\paragraph{Spectral coordinate.}
The spectral coordinate attached to the observed tile separates the measured $M$ and $R$ components.  The verified numerical values are
$$
\begin{aligned}
\lambda_M &= 0.6752479308830213,\\
\lambda_R &= 0.4758352843197002,\\
\mu &= 3.051487585298128.
\end{aligned}
$$
The radial cubic check gives value $-6.224354365258478\times 10^{-12}$, so the displayed spectral coordinate is within numerical tolerance of its defining cubic relation.  This supplies an internal finite-row consistency check for the code-layer geometry, without promoting that geometry to translation, folding, physical admissibility, function, mechanism, or universality.

\paragraph{Null comparison.}
The shape significance run used $N=2000$ sampled tables with seed $42$.  It made $57377$ distinct-sample attempts and encountered $55377$ duplicate retries.  Only $1$ exact shape match occurred, giving $p_{\mathrm{exact}}=0.0005$.  At the looser threshold of Jaccard overlap at least $0.9$, there were $25$ matches, giving $p_{J\geq 0.9}=0.0125$.  A second amino-acid-class-preserving null model, also with $N=2000$ and seed $42$, found observed $M$ size $20$, exact-match count $0$, lower-tail count $7$, $p_{\mathrm{aa\ class, exact}}=0.0$, and lower-tail value $p_{\mathrm{aa\ class, lower}}=0.0035$.  These null rows support the statement that the observed local tile is not a typical accident under the sampled label-preserving controls.  They do not identify a physical cause.

\paragraph{Leave-one-code-out aggregate row.}
The leave-one-code-out run evaluated $23$ code tables and $63$ total held-out targets.  The baseline hit rate was $0.19821739130434785$.  The hit rate inside $M\setminus T$ was $0.9365079365079365$, and the hit rate inside $M\setminus T$ plus the boundary row was $0.9841269841269841$.  The reported hit rate above baseline was also $0.9841269841269841$.  In finite-row terms, the held-out labels mostly return to the same local code-layer region once the boundary residue is allowed.  This is the empirical content of the result: the window-six coordinate is predictive for local codon-label readback across the evaluated held-out code rows.

\paragraph{Family stratification.}
The family-stratified row has $5$ code families.  The hit-rate-above-baseline values by family are
$$
\begin{aligned}
\mathrm{alternative\ yeast} &= 0.8333333333333334,\\
\mathrm{bacterial\ plastid} &= 1.0,\\
\mathrm{candidate\ other} &= 1.0,\\
\mathrm{mitochondrial} &= 1.0,\\
\mathrm{nuclear\ ciliate} &= 1.0.
\end{aligned}
$$
Thus four of the five listed families attain perfect family-level recovery above the baseline criterion, while the alternative-yeast family remains high but not perfect.  The chapter-level conclusion is therefore family-specific rather than universal: the finite evidence supports a stable local code-read boundary across these five curated family rows, with the weakest displayed family value equal to $5/6$.

\paragraph{Codon-module closure.}
The leave-one-codon-out module classification accounts for all thirteen boundary codons.  The verified modules are
$$
\begin{aligned}
\mathrm{agr} &= \{\mathrm{AGA},\mathrm{AGG}\},\\
\mathrm{ar\_lys} &= \{\mathrm{AAA}\},\\
\mathrm{cun} &= \{\mathrm{CUA},\mathrm{CUC},\mathrm{CUG},\mathrm{CUU}\},\\
\mathrm{ile\_met} &= \{\mathrm{AUA}\},\\
\mathrm{ssn} &= \{\mathrm{UCA},\mathrm{UUA}\},\\
\mathrm{stop\_trp} &= \{\mathrm{UAA},\mathrm{UAG},\mathrm{UGA}\}.
\end{aligned}
$$
The classification row reports $R$ size $13$, assigned count $13$, no extra module codons, no missing codons from the modules, and no unassigned codons.  The separate leave-one-codon-out recovery row has anchor count $0$, boundary count $5$, predictable count $8$, closure coverage high $8$, and recovery rate $1.0$.  Hence every module has either predictable or boundary status in this finite test.

\paragraph{Reality contact and boundary.}
The only reality contact used here is the curated standard-code table as a codon-to-label bridge.  It tests code readback and local codon-tile label split.  It does not test translation realization, folded structure, physical admissibility, biological function, or a universal mechanism.  Consequently, the supported claim is deliberately narrow: window-six codon coordinates expose a finite local boundary-tile structure whose leave-one-code-out recovery remains high by family.  No statement about protein realization, folding, structural order, admissibility, function, mechanism, necessity, or universality follows unless a separate higher-layer contact supplies that evidence.
