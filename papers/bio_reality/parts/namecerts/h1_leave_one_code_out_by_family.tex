\paragraph{Finite carrier and code-layer contact.}
This record concerns the six-bit codon cube $Q_6$ read through a curated standard-code label table.  The carrier is finite: each codon is a vertex of $Q_6$, and the only external contact used here is the codon-to-label table assigning each codon to an amino-acid or stop label.  The contact is therefore a code-read contact, not a translation experiment, not a folding assay, not a physical-admissibility test, and not a function measurement.  The local tile under study is the exceptional row set
$$
\begin{aligned}
R=\{&AAA,AGA,AGG,AUA,CUA,CUC,CUG,CUU,\\
&UAA,UAG,UCA,UGA,UUA\},\qquad |R|=13 .
\end{aligned}
$$
The same row set is verified as a punctured-cube tile with $f$-vector $(13,16,4)$.  Thus the finite object is not a floating biological slogan: it is a concrete subcomplex of the codon cube, with thirteen vertices, sixteen edges, and four two-faces, read back only through curated code labels.

\paragraph{Interpretation boundary.}
The supported statement is deliberately narrow.  Window-six codon coordinates expose a local boundary-tile structure in the standard-code readback layer, and the leave-one-code-out family audit reports that this structure remains above baseline in all five named families.  The curated table supplies label readback only.  It does not observe translation realization, folded protein structure, structural order, physical admissibility, biochemical function, a universal mechanism, or a global biological law.  In particular, amino-acid label-sequence readback is not folded protein structure, and a codon-tile split is not evidence that DNA obeys this geometry as biological necessity.  Any claim crossing from code labels into translation, folding, physical admissibility, function, mechanism, or universality requires a separate higher-layer reality contact.  Without that additional contact, the result remains a finite code-layer boundary record on $Q_6$.
