\subsection{Leave-One-Code-Out Stability by Code Family}
\label{sec:bioreality-h1-leave-one-code-out-by-family}

\paragraph{Record.}
The packet \texttt{h1.leave\_one\_code\_out.by\_family} records a finite code-read test on the standard-code codon table.  The carrier is the six-coordinate codon cube $Q_6$, where a codon is read as a six-bit display and only local tile faces, singleton label corners, and three-point label classes are used internally.  The observed contact is not a biochemical mechanism.  It is the curated table assigning each codon to an amino-acid or stop label, and the computation asks whether the local window-six geometry continues to pick out the same unstable label region when one genetic-code table is withheld.

\paragraph{Boundary set.}
The finite boundary set $R$ has cardinality $13$.  Its codons are
$$
\begin{aligned}
R=\{&\mathrm{AAA},\mathrm{AGA},\mathrm{AGG},\mathrm{AUA},\mathrm{CUA},\mathrm{CUC},\mathrm{CUG},\mathrm{CUU},\\
&\mathrm{UAA},\mathrm{UAG},\mathrm{UCA},\mathrm{UGA},\mathrm{UUA}\}.
\end{aligned}
$$
The same row verifies that this set is a punctured-cube object with face-vector
$$
\begin{aligned}

$$
Thus the computation is not treating all codons symmetrically.  It isolates a concrete thirteen-codon local boundary pattern inside the $64$-codon display and then tests whether that pattern remains informative when code-table evidence is withheld.

\paragraph{Scientific reading.}
The result should be read as a finite code-table observation.  The curated standard-code contact supplies codon-to-label readback, and the computations show that the thirteen-codon boundary packet has stable finite behavior under the reported null models and leave-out procedures.  No claim about translation realization, folded protein structure, physical admissibility, biological function, mechanism, or universality follows from this packet.  Crossing any of those boundaries would require a separate contact that directly observes the higher layer.  In particular, amino-acid label sequences are still labels at the code layer; they are not structural measurements and they do not certify a folded state.
