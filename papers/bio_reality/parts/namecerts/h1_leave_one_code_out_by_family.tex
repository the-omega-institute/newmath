\subsection{Family-level leave-one-code-out readback}\label{sec:h1-leave-one-code-out-by-family}\origin{ai}

\paragraph{Finite carrier and reality contact.} The finite carrier is the six-coordinate codon display $Q_6$, read only at the code-label layer. A row is a codon together with its curated standard-code label, where the label is supplied by the external code table rather than by the window-six geometry itself. The contact used here is therefore narrow: it says that the curated standard code assigns codons to amino-acid or stop labels. It does not say that a local codon tile realizes translation, folding, physical admissibility, biological function, mechanism, or universality. Those higher layers remain outside the contact unless a separate experiment or source supplies them.

\paragraph{Boundary set.} The distinguished boundary set $R$ has cardinality $13$. In the finite row audit it is exactly
$$
\begin{aligned}
R=\{&\mathrm{AAA},\mathrm{AGA},\mathrm{AGG},\mathrm{AUA},\mathrm{CUA},\mathrm{CUC},\mathrm{CUG},\mathrm{CUU},\\
&\mathrm{UAA},\mathrm{UAG},\mathrm{UCA},\mathrm{UGA},\mathrm{UUA}\}.
\end{aligned}
$$
The same row also identifies $R$ as a punctured-cube boundary with $f$-vector $(13,16,4)$. Thus the local object being tested is not a free biological claim; it is a finite tile-shaped readback object with $13$ vertices, $16$ local adjacency edges, and $4$ two-dimensional faces in the supplied coordinate presentation.

\paragraph{Shape rarity.} Against the reported null sample of $N=2000$ distinct randomized tables, the exact local shape appeared once, giving $p_{\mathrm{exact}}=0.0005$. The high-overlap test counted $25$ tables with Jaccard overlap at least $0.9$, giving $p_{J\geq 0.9}=0.0125$. The audit also records $57377$ distinct-sample attempts and $55377$ duplicate retries, so the row is not merely reporting a raw sampling count; it is reporting a distinct-table null run with heavy rejection from duplicates. Under the amino-acid-class-preserving null, the observed $M$ size is $20$, the exact-shape count is $0$ out of $2000$, and the lower-tail count at or below the observed size is $7$, giving lower-tail probability $0.0035$. These finite counts support rarity of the local code-label pattern under the tested nulls, while still leaving all higher biological layers unclaimed.

\paragraph{Local verdict.} The family-level leave-one-code-out evidence supports a code-read claim: the window-six coordinate display exposes a stable local boundary-tile signature around the curated standard-code labels, and that signature remains above the baseline across the five reported code families. The evidence does not promote the tile to a global biological law. In particular, it does not establish translation realization, folded structure, structural order, physical admissibility, biological function, a universal mechanism, or necessity of the genetic code. The current contact is a curated label table, so the chapter-level result is a finite-row code-label readback with an explicit boundary against higher-layer biological interpretation.
