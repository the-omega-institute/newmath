\subsection{Leave-one-code-out recovery by code family}
\label{sec:h1-leave-one-code-out-by-family}

\paragraph{Finite carrier and contact.}
The finite carrier is the six-bit codon cube $Q_6$, read as a codon-coordinate display with a curated standard-code label table as its only external contact. The contact is deliberately narrow: it assigns each codon to an amino-acid or stop label, and therefore tests code-layer readback and local codon-tile label splits. It does not test translation realization, folded structure, structural order, physical admissibility, biological function, mechanism, or universality. The finite boundary set recovered in the coordinate analysis has cardinality $13$:
$$
\begin{aligned}
R=\{&\mathrm{AAA},\mathrm{AGA},\mathrm{AGG},\mathrm{AUA},\mathrm{CUA},\mathrm{CUC},\mathrm{CUG},\mathrm{CUU},\\
&\mathrm{UAA},\mathrm{UAG},\mathrm{UCA},\mathrm{UGA},\mathrm{UUA}\}.
\end{aligned}
$$
The same row reports that $R$ is a punctured-cube pattern with $f$-vector $(13,16,4)$. Thus the object under discussion is not an arbitrary biological mechanism; it is a finite code-read tile in which one singleton label corner and several three-point or multi-codon label classes can be compared against the curated table.

\paragraph{Boundary of the claim.}
The claim certified here is only that window-six codon coordinates expose a local boundary-tile structure whose leave-one-code-out recovery remains high when grouped by code family. The curated standard-code table supplies the external label contact, and the finite rows report a concrete recovery profile over $23$ tables, $63$ targets, $5$ families, and the $13$-codon punctured-cube boundary set $R$. Nothing in these rows promotes the local tile to translation realization, folded protein structure, physical admissibility, biological function, universal genetic-code mechanism, or biological necessity. Amino-acid label-sequence readback remains a code-layer readback. Any claim above that layer requires a separate contact that directly observes the higher-layer object.
