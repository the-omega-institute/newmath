\paragraph{Finite object under test.}
This record concerns the standard-code codon table as a finite code-read object on the six-coordinate codon cube $Q_6$.  The data used here are not a claim that biological translation is generated by the cube.  They are a finite comparison between a curated codon-to-label table and a window-six coordinate display with local tile faces, singleton label corners, and three-point label classes.  The row under audit is the family-stratified leave-one-code-out row, whose experiment id is $7dda978c6c97$ and whose verification timestamp is $2026$-$05$-$31T08{:}01{:}31+00{:}00$.

\paragraph{Exceptional codon set.}
The finite exceptional set $R$ has cardinality $13$.  Its members are
$$
\begin{aligned}

It says that, after the curated code labels are read as finite labels, this particular thirteen-codon set is the boundary object being compared across code variants and family bins.

\paragraph{Reality contact and scope.}
The only reality contact used in this chapter is the curated standard genetic-code table, treated as a codon-label contact.  It supplies assignments from codons to amino-acid or stop labels.  It does not supply a ribosomal translation experiment, a folded structure, a physical-admissibility measurement, a function assay, or a universality theorem.  Therefore the chapter's conclusion is intentionally narrow: the finite window-six display gives a reproducible local boundary pattern for code labels, and the family-stratified leave-one-code-out test preserves that pattern across five code-family bins with rates $0.8333333333333334,1.0,1.0,1.0,1.0$.

\paragraph{Blocked promotions.}
No statement in this chapter promotes the local codon tile to a global biological law.  No statement infers protein fold, structural order, physical admissibility, biological function, translation realization, or universal mechanism from codon-label readback.  Amino-acid label sequences are still labels at the code layer.  Crossing into higher biological layers requires a separate contact whose observed object is actually translation, structure, physics, function, or organism-level mechanism.  Until such a contact is supplied, the correct scientific status is a finite code-read boundary signal with strong row-wise support and explicitly bounded interpretation.
