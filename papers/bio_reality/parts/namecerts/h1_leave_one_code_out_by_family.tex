\subsection{Leave-one-code-out recovery by code family}\label{subsec:h1-leave-one-code-out-by-family}

\paragraph{Finite codon display.}
This record treats the standard-code codon table as a finite six-bit codon display with curated labels.  The biological contact is deliberately narrow: the table reports codon-to-amino-acid and codon-to-stop assignments, and it does not report translation dynamics, folded structure, physical admissibility, biological function, a mechanism, or a universal biological law.  The computation therefore asks a code-layer question only.  When one compares nonstandard code tables against the standard table, does the local window-six geometry mark the positions at which label changes are expected, and does this marking remain visible after grouping the changed tables into families?

The finite boundary set $R$ has cardinality $13$.  In the row, the members are

The same row identifies this set with a punctured-cube pattern and gives its face-count vector as $(13,16,4)$. The radial cubic residual is $-6.224354365258478\times 10^{-12}$, so the numeric check is at floating-point tolerance rather than a qualitative visual impression.  The row does not provide the explicit list of all codons in $M$; it does provide $|M|=20$ through the amino-acid-class-preserving null experiment.

\paragraph{Shape and null comparisons.}
The shape comparison used $N=2000$ sampled alternatives with seed $42$.  It encountered $57377$ distinct-sample attempts and $55377$ duplicate retries.  Exact shape agreement occurred once, so $p_{\mathrm{exact}}=0.0005$.  High-overlap comparison at Jaccard threshold $0.9$ occurred $25$ times, so $p_{J\geq 0.9}=0.0125$.  A second null comparison preserving amino-acid class structure also used $N=2000$ and seed $42$.  It found no exact match, giving exact-shape count $0$ and reported exact-shape probability $0.0$.  Its lower-tail count at or below the observed $M$ size was $7$, with $|M|=20$ and lower-tail probability $0.0035$.  These numbers support a finite code-layer anomaly test: the observed window-six region is rare under the stated randomizations, but rarity inside this coordinate experiment is not evidence for folding, function, or biochemical necessity.

\paragraph{Leave-one-code-out result.}
The leave-one-code-out experiment evaluated $23$ tables and $63$ changed targets.  The baseline hit rate was $0.19821739130434785$.  The row reports hit rate inside $M-T$ as $0.9365079365079365$, and hit rate inside $M-T$ together with the boundary as $0.9841269841269841$.  The same value, $0.9841269841269841$, is recorded as the hit rate above baseline.  Thus almost all observed target changes land inside the finite window-six region or its boundary expansion.  The statement is intentionally local: the computation recovers code-table label-change positions with high frequency, not a molecular process by which a ribosome realizes the table.

\paragraph{Scientific boundary.}
The supported claim is a code-read claim.  A curated standard-code table supplies labels, and the finite coordinate computation organizes those labels into a local tile-and-boundary pattern.  Nothing in this row observes a peptide being synthesized, a protein folding, a molecule satisfying a physical-energy constraint, an organism realizing a function, or a universal mechanism holding across biology.  Those are separate layers and would need their own contacts.  In particular, amino-acid label readback is not folded protein structure, CUN/UUR or tRNA-Leu wording is not structural evidence by itself, and a local codon tile is not a global biological law.  The result is ready as a self-contained BioReality chapter precisely because its positive statement and its boundary are both finite: $23$ tables, $63$ targets, $5$ families, $13$ boundary codons, the listed module partition, and the reported null probabilities all remain inside the code-label layer.
