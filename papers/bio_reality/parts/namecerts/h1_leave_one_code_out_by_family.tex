These numbers support a local code-layer boundary pattern, not a claim about translation chemistry or protein realization.

\paragraph{Boundary verdict.}
The ready claim is narrowly local.  Window-six codon coordinates expose a family-stable local boundary-tile pattern whose readback is mediated by the curated standard-code label table.  The row supports code-layer statements about singleton label corners, three-point label classes, local tile faces, relation size, module coverage, spectral consistency, and leave-one-code-out recovery.  It does not support any promotion to translation realization, folded structure, structural order, physical admissibility, biological function, universal mechanism, or biological necessity.  Amino-acid label-sequence readback is still only label readback.  Any claim about folding, physical admissibility, function, mechanism, or universality requires a separate higher-layer reality contact, not merely the six-bit codon geometry and not merely the curated code table.
