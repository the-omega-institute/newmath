\subsection{Leave-One-Code-Out Stability by Biological Code Family}\label{sec:h1-leave-one-code-out-by-family}
\origin{ai}

\paragraph{Finite object and scope.}
This record treats the standard-code codon table as a finite code-layer contact, not as a mechanism for translation or protein realization. The carrier is the six-bit codon cube $Q_6$, read through a curated codon-to-label table. The row under test is the leave-one-code-out family audit: variant code tables are grouped by biological code family, and each held-out table is tested against the local window-six tile recovered from the standard-code readback. The claim is deliberately local. It says that the finite codon coordinates expose a stable boundary-tile pattern across the evaluated families. It does not say that the tile is a global biological law, a physical admissibility constraint, a folding principle, a functional principle, or a universal mechanism.

\paragraph{Recovered boundary set.}
The finite row identifies a distinguished boundary set $R$ of size $13$. The row records both the cardinality and the explicit codons:
$$
\begin{aligned}

&\mathrm{UAA},\mathrm{UAG},\mathrm{UCA},\mathrm{UGA},\mathrm{UUA}\},\\

$$
The punctured-cube audit reports the same list and gives the face-vector signature

Thus the chapter is not using an informal visual analogy: the local tile is represented by an explicit finite set inside $Q_6$, together with its finite combinatorial boundary data.

\paragraph{Reality contact and cannot-claim boundary.}
The only external contact used here is the curated standard genetic-code bridge: a table assigning codons to amino-acid or stop labels. That contact tests code readback and local codon-tile label split. It does not test translation realization, folded structure, structural order, physical admissibility, biological function, mechanism, or universality. Therefore the correct reading of the row is narrow. The finite computation supports a local window-six code-layer regularity, calibrated by the displayed null rows and stable across the displayed families. It does not imply that DNA obeys this geometry by biological necessity, that a protein structure is determined by the tile, that the tile is physically admissible as a molecule-level constraint, or that it is a universal law of genetic codes. Any such promotion would require a separate higher-layer contact, not merely the curated codon-label table used in this chapter.
