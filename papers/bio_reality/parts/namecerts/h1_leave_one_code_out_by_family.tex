\paragraph{Finite carrier and contact.}
The finite carrier is the six-bit codon cube $Q_6$, read as the $64$ possible codon displays over the four-letter alphabet.  The reality contact used here is only the curated standard-code label table: each codon is assigned an amino-acid or stop label, and no translation apparatus, folded structure, physical admissibility test, biological function, mechanism, or universality principle is supplied by this contact.  The claim studied in this section is therefore a code-read claim.  It asks whether the window-six coordinate geometry exposes a stable local boundary-tile pattern when one compares the standard code with finite alternative code rows grouped by family.

\paragraph{Distinguished boundary row.}
The distinguished row $R$ has cardinality $13$.  In the observed finite table it is exactly
$$
\begin{aligned}
R=\{&AAA,AGA,AGG,AUA,CUA,CUC,CUG,CUU,\\
&UAA,UAG,UCA,UGA,UUA\}.
\end{aligned}
$$
The same row is verified as a punctured-cube boundary with $f$-vector
$$
\begin{aligned}
(f_0,f_1,f_2)=(13,16,4).
\end{aligned}
$$
This is the local tile object used by the leave-one-code-out test.  It is not treated as a global biological law.  It is a finite coordinate boundary inside $Q_6$ whose labels are read back through the curated table.

\paragraph{Amino-acid-class preserving null.}
A second null preserves amino-acid class structure.  In $2000$ samples with seed $42$, the exact-shape count is $0$, the lower-tail count at or below the observed $M$-size is $7$, and the observed $M$-size remains $20$.  Thus the recorded finite-row probabilities are

\begin{aligned}

p_{\mathrm{aa\ class\ lower\ tail}}&=0.0035.
\end{aligned}
$$
These numbers support a code-layer statement: the observed coordinate tile is not a generic draw under either sampled null.  They do not license a claim about molecular realization or function.

The recorded hit rate above baseline is also $0.9841269841269841$.  Therefore the finite-row evidence says that the local codon tile recovers almost all tested targets once the boundary contribution is admitted, whereas the naive baseline recovers about one fifth of them.

\paragraph{Module classification.}
The same finite row is completely partitioned into six modules:
$$

The module audit reports $R$-size $13$, assigned count $13$, no extra module codons, no missing row codons, and no unassigned codons.  Each module has a predictable-or-boundary witness.  Separately, the leave-one-codon-out row records boundary count $5$, predictable count $8$, closure coverage high $8$, anchor count $0$, and recovery rate $1.0$.  The module result is therefore a finite partition statement about the $13$ boundary codons, not a statement about biochemical causation.

\paragraph{Boundary verdict.}
The finite data support the following self-contained code-read verdict.  In the six-bit codon cube, the standard-code label table exposes a $13$-codon punctured-cube boundary whose spectral coordinates, null-model rarity, leave-one-code-out recovery, family-stratified recovery, and module partition are mutually consistent.  The strongest permitted claim is local: window-six geometry can organize codon-label readback and boundary splits for the curated tables supplied to the experiment.  The cannot-claim boundary is part of the result.  No translation realization, no folded protein structure, no physical admissibility, no biological function, no universal mechanism, and no claim that DNA obeys this geometry as biological necessity follows unless a separate reality contact supplies that higher-layer evidence.
