\subsection{Leave-one-code-out family stratification for the window-six codon tile}
\label{sec:bioreality-h1-leave-one-code-out-by-family}
\origin{ai}

\paragraph{Finite codon coordinate object.}
The object under test is the standard-code codon table read at the code-label layer through a six-coordinate codon display. Each codon is treated as a finite row in the codon cube $Q_6$, with local distinctions given by coordinate position, closure membership, spectral response, and label relation. The curated standard-code table supplies the amino-acid or stop label attached to each codon; it is used here only as a label contact. The distinguished residual set is
$$
\begin{aligned}
R=\{&AAA, AGA, AGG, AUA, CUA, CUC, CUG, CUU,\\
&UAA, UAG, UCA, UGA, UUA\}.
\end{aligned}
$$
It has cardinality $|R|=13$. The same finite row also matches the punctured-cube face count
$$
\begin{aligned}
f(R)=(13,16,4),
\end{aligned}
$$
so the local tile is not merely a list of labels: it carries a small cubical boundary signature inside $Q_6$.

\paragraph{Spectral and sampling controls.}
The companion tile $M$ has observed size $20$. Its shape was compared with $2000$ sampled controls generated with seed $42$. Among $57377$ distinct-sample attempts, $55377$ duplicate retries were needed; the exact shape reappeared once, giving $p_{\mathrm{exact}}=0.0005$. The high-overlap event with Jaccard score at least $0.9$ occurred $25$ times, giving $p_{J\geq 0.9}=0.0125$. Under the amino-acid-class-preserving null model with $N=2000$ and seed $42$, the exact-shape count was $0$, so $p_{\mathrm{class,exact}}=0.0$. The lower-tail count at or below the observed $M$ size was $7$, giving $p_{\mathrm{class,tail}}=0.0035$. The spectral row gives
$$
\begin{aligned}
\lambda_M&=0.6752479308830213,\\
\lambda_R&=0.4758352843197002,\\
\mu&=3.051487585298128,
\end{aligned}
$$
with radial cubic residual $-6.224354365258478\times 10^{-12}$. These numbers say that the coordinate tile is a finite, reproducible code-layer pattern; they do not say that the pattern is a biochemical cause.

\paragraph{Leave-one-code-out family row.}
The family-stratified leave-one-code-out test evaluates whether the local tile continues to recover altered code labels above a baseline when one code table is withheld. The global leave-one-code-out run used $23$ tables and $63$ targets. Its baseline hit rate was $0.19821739130434785$, while the tile-supported rate above baseline was $0.9841269841269841$. The hit rate inside $M\setminus T$ was $0.9365079365079365$, and the rate inside $M\setminus T$ plus the boundary row was again $0.9841269841269841$. Stratifying by code family gives five finite rows:
$$
\begin{aligned}
\mathrm{alternative\_yeast}&:0.8333333333333334,\\
\mathrm{bacterial\_plastid}&:1.0,\\
\mathrm{candidate\_other}&:1.0,\\
\mathrm{mitochondrial}&:1.0,\\
\mathrm{nuclear\_ciliate}&:1.0.
\end{aligned}
$$
Thus the family-level observation is not a single pooled artifact: four families reach full above-baseline recovery in this finite row, and the remaining family still exceeds the baseline by a large margin.

\paragraph{Codon-module bookkeeping.}
The leave-one-codon-out module classification accounts for all $13$ codons in $R$ with no missing or surplus module entries: $R_{\mathrm{size}}=13$, assigned count $13$, no extra module codons, no missing codons, and no unassigned codons. The partition is
$$
\begin{aligned}
\mathrm{agr}&=\{AGA,AGG\},\\
\mathrm{ar\_lys}&=\{AAA\},\\
\mathrm{cun}&=\{CUA,CUC,CUG,CUU\},\\
\mathrm{ile\_met}&=\{AUA\},\\
\mathrm{ssn}&=\{UCA,UUA\},\\
\mathrm{stop\_trp}&=\{UAA,UAG,UGA\}.
\end{aligned}
$$
Every listed module has either a predictable codon or a boundary codon. In the codon-level leave-one-out test, the recovery rate was $1.0$, the threshold comparison was also $1.0$, with $8$ predictable codons, $5$ boundary codons, $8$ high-coverage closure cases, and $0$ anchor codons.

\paragraph{Scientific reading.}
The result supports the following narrow statement. In the window-six codon coordinate system, the standard-code label table exposes a local boundary tile whose residual set, face counts, spectrum, null comparisons, leave-one-code-out behavior, and family stratification agree as a finite code-layer signal. The observation is about codon-to-label readback and local label splitting. It is not evidence for translation realization, folded protein structure, physical admissibility, biological function, a biochemical mechanism, or a universal biological law. Any statement across those boundaries requires a separate contact whose measured object is translation, structure, physical constraint, function, mechanism, or cross-organism universality rather than the curated codon-label table alone.
