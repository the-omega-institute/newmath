\subsection{Leave-One-Code-Out Stability by Code Family}\label{sec:h1-leave-one-code-out-by-family}

\paragraph{Finite codon window.}
This packet concerns a window-six codon coordinate system on the codon cube $Q_6$, with the curated standard genetic-code table used only as a codon-to-label contact.  The claim tested here is not that the local geometry explains translation, folding, biochemical admissibility, function, mechanism, or universality.  It is narrower: after one code table is omitted from a finite comparison panel, the local boundary tile still tends to recover the held-out code-level label change within the same family grouping.

The primary finite set is the distinguished codon set $R$.  Its verified cardinality is $13$, and the listed members are
$$
\begin{aligned}
R=\{&\mathrm{AAA},\mathrm{AGA},\mathrm{AGG},\mathrm{AUA},\mathrm{CUA},\mathrm{CUC},\mathrm{CUG},\mathrm{CUU},\\
&\mathrm{UAA},\mathrm{UAG},\mathrm{UCA},\mathrm{UGA},\mathrm{UUA}\}.
\end{aligned}
$$
The same run verifies that this set has the punctured-cube face-vector signature
$$
\begin{aligned}
(f_0,f_1,f_2)=(13,16,4).
\end{aligned}
$$
Thus the coordinate object is not merely a list of codons; it carries a small local cubical incidence shape with thirteen vertices, sixteen edges, and four two-faces.

This spectral row is used only as finite numerical structure attached to the codon window; it is not interpreted as a physical energy, kinetic rate, or molecular mechanism.

Two finite null checks give the shape a sharper reading.  In the shape-randomization run with $N=2000$ samples and seed $42$, there was exactly $1$ exact shape match, giving $p_{\mathrm{exact}}=0.0005$.  The high-overlap threshold produced $25$ matches with Jaccard score at least $0.9$, giving $p_{J\geq 0.9}=0.0125$.  The run reports $57377$ distinct-sample attempts and $55377$ duplicate retries.  In the amino-acid-class-preserving null run, again with $N=2000$ and seed $42$, the observed size was $20$, the exact match count was $0$, and the lower-tail count at or below the observed size was $7$.  Thus
$$
\begin{aligned}
p_{\mathrm{aa\ class,exact}}&=0.0,\\
p_{\mathrm{aa\ class,lower\ tail}}&=0.0035.
\end{aligned}
$$
These are finite empirical calibrations of the coordinate pattern against two restricted random backgrounds.  They do not by themselves promote the pattern beyond the code-label layer.

\paragraph{Leave-one-code-out result.}

In finite terms, the boundary-enlarged local tile recovers almost all held-out code-level targets in this panel, whereas the baseline is below one fifth:
$$
\begin{aligned}

Four families are complete at this resolution, while the alternative-yeast family remains high but not complete.  The finite conclusion is therefore family-stable rather than merely aggregate: the high aggregate recovery is not produced only by pooling unlike code families.

\paragraph{Claim boundary.}
The observed contact is a curated standard-code table assigning codons to amino-acid or stop labels.  That contact can test code readback and local codon-tile label splits.  It cannot test translation realization, folded structure, physical admissibility, biological function, a causal mechanism, or a universal biological law.  Consequently, the numerical rows above support a finite code-layer statement: the window-six local tile has strong leave-one-code-out stability by family, with verified family count $5$ and family hit rates $0.8333333333333334,1.0,1.0,1.0,1.0$ above baseline.  Any statement about actual translation, protein structure, molecular viability, organismal function, or universality would require a different contact carrying that information.
