\paragraph{Question fixed by the row.} This record tests whether the window-six codon coordinate picture keeps predictive force when the leave-one-code-out experiment is stratified by code family. The carrier is the six-bit codon cube $Q_6$, read through a curated standard genetic-code table. The distinction under test is local: a codon may lie in a boundary-facing tile, a singleton-label corner, or a three-point label class. The biological object is therefore only the codon-to-label table for the standard code and nearby code-family comparisons. The row does not assert a biochemical implementation of the coordinate geometry.

This contact can test the code-read layer and the local label split. It does not observe translation as a molecular process, folded protein structure, physical admissibility, biological function, or a universal mechanism.
