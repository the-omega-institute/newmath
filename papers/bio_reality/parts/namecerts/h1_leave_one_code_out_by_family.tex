\paragraph{Reality contact.}
The contact used here is a curated standard genetic-code table. It supplies codon labels: codons are assigned to amino-acid or stop symbols. That contact is sufficient for checking whether a six-bit codon display and a local tile boundary align with the standard-code label table. It is not an observation of ribosomal dynamics, tRNA abundance, molecular folding, energetic admissibility, organismal fitness, or universal biological necessity. The code table is therefore a label source for the code-read layer, not a certificate for any higher biological layer.

\paragraph{Local boundary.}
The claim supported by this packet is narrow. Window-six codon coordinates expose a reproducible local boundary-tile pattern, and the family-stratified leave-one-code-out row shows that the signal remains strong across five named table families. The strongest finite statement is that the curated code labels and the cube-derived residual structure agree unusually well under the listed null comparisons and ablation rows. No conclusion about translation, folded protein structure, physical admissibility, biochemical function, mechanism, or universality follows from these rows unless a separate contact directly measures that higher layer. Amino-acid label readback remains label readback; it is not a folded-structure observation.
