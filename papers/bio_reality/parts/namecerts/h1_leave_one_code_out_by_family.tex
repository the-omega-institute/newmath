\paragraph{Finite carrier and reality contact.}
The carrier in this record is the six-bit codon cube $Q_6$, read through the standard-code label table.  The biological contact is deliberately narrow: a curated standard-code table assigns each codon to an amino-acid label or to stop, and this contact tests only code-layer readback.  It does not test translation realization, folded structure, physical admissibility, biological function, a mechanism, or universality.  The row therefore treats the label table as an external reality bridge for codon labels, while the BEDC-side object is the local window-six coordinate geometry built from coordinates, closure, spectrum, and relations.

\paragraph{Boundary of the claim.}
The result is a finite code-read statement about a local codon tile.  It says that the window-six geometry, together with the curated standard-code label contact, exposes a stable boundary pattern across leave-one-code-out families.  It does not say that DNA obeys this geometry as a biological necessity.  It does not turn a local codon tile into translation, folded protein structure, physical admissibility, biological function, mechanism, or universality.  Amino-acid label-sequence readback is still only label readback.  Any higher-layer claim would require a separate contact that observes that layer directly.
