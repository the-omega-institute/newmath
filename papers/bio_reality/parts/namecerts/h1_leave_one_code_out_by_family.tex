\subsection{Family-stratified leave-one-code-out codon-window test}\label{sec:h1-leave-one-code-out-by-family}
\origin{ai}

\paragraph{Object and coordinate display.}
This record concerns the standard-code codon table viewed only at the code-reading layer.  The finite carrier is the six-coordinate codon cube $Q_6$, with one binary display for the first, second, and third nucleotide positions after choosing the usual two-bit nucleotide presentation.  The observable contact is a curated codon-to-label table: each of the $64$ codons receives an amino-acid label or a stop label.  The present test does not treat that table as a mechanism.  It uses the table as a finite label readback against which a local coordinate pattern can be checked.

\paragraph{Boundary set.}
The distinguished row set is the $13$-element set
$$
\begin{aligned}
R=\{&AAA, AGA, AGG, AUA, CUA, CUC, CUG, CUU,\\
&UAA, UAG, UCA, UGA, UUA\}.
\end{aligned}
$$
The cardinality audit gives $|R|=13$.  The same rows form the measured punctured-cube tile with face-vector
$$
\begin{aligned}
(f_0,f_1,f_2)=(13,16,4).
\end{aligned}
$$
Thus the local object being tested is not an arbitrary codon list: it has $13$ vertices, $16$ one-dimensional adjacencies, and $4$ square faces in the six-bit cube.  This supplies a finite local boundary surface for code-label readback, but by itself it is still only a coordinate pattern.

\paragraph{Spectral finite check.}
The associated spectral row gives
$$
\begin{aligned}
\lambda_M&=0.6752479308830213,\\
\lambda_R&=0.4758352843197002,\\
\mu&=3.051487585298128.
\end{aligned}
$$
The reported cubic residual is $-6.224354365258478\times 10^{-12}$, the same value recorded for the radial-polynomial residual.  The spectral-radius value in tolerance is $0.6752479308830213$.  These numbers are used here only as a finite consistency check on the local codon geometry.  They do not turn the tile into a biological law.

\paragraph{Module decomposition.}
The $13$ boundary rows are also exhausted by six named modules:
$$
\begin{aligned}
\mathrm{agr}&=\{AGA,AGG\},\\
\mathrm{ar\_lys}&=\{AAA\},\\
\mathrm{cun}&=\{CUA,CUC,CUG,CUU\},\\
\mathrm{ile\_met}&=\{AUA\},\\
\mathrm{ssn}&=\{UCA,UUA\},\\
\mathrm{stop\_trp}&=\{UAA,UAG,UGA\}.
\end{aligned}
$$
The module audit reports $R\_size=13$, $assigned\_count=13$, no extra module codons, no rows missing from the modules, and no unassigned codons.  In particular, every displayed boundary row is accounted for by one of the six finite module classes.  The same audit records that every module has a predictable-or-boundary status.  This is the chapter-specific reason that the family-stratified leave-one-code-out test is meaningful: the withheld-code rows are not scattered points but belong to an explicit finite partition of the local tile.

\paragraph{Leave-one-code-out evidence.}
Across the broader leave-one-code-out run, $23$ code tables were evaluated against $63$ total targets.  The baseline hit rate was $0.19821739130434785$.  The hit rate above baseline was $0.9841269841269841$, the hit rate inside $M-T$ was $0.9365079365079365$, and the hit rate inside $M-T$ plus the boundary was $0.9841269841269841$.  The family-stratified row keeps the same question but separates the withheld-code cases by biological-code family.  It reports $5$ families with hit rates above baseline
$$
\begin{aligned}
\mathrm{alternative\_yeast}&=0.8333333333333334,\\
\mathrm{bacterial\_plastid}&=1.0,\\
\mathrm{candidate\_other}&=1.0,\\
\mathrm{mitochondrial}&=1.0,\\
\mathrm{nuclear\_ciliate}&=1.0.
\end{aligned}
$$
The finite conclusion is therefore not merely that the aggregate leave-one-code-out score is high.  Four of the five families attain perfect above-baseline recovery in this row, and the remaining alternative-yeast family still attains $5/6$.  The stratification reduces the chance that the aggregate number is hiding one family that fails the local tile test.

\paragraph{Single-codon recovery.}
The leave-one-codon-out row supplies a finer sanity check.  It reports $anchor\_count=0$, $boundary\_count=5$, $closure\_coverage\_high=8$, $predictable\_count=8$, and $recovery\_rate=1.0$.  The value $recovery\_rate\_above\_threshold=1.0$ says that the local rule recovers all tested single-codon omissions at the declared threshold.  This single-codon result is not the same as the leave-one-code-out family result, but it supports the same interpretation: the window-six tile is detecting a local codon-label boundary rather than merely fitting one large table entry.

\paragraph{Null comparisons.}
Two finite null rows constrain the interpretation.  In the shape-significance run with $N=2000$ samples and seed $42$, there was $1$ exact shape match, giving $p\_exact=0.0005$, and $25$ matches with Jaccard score at least $0.9$, giving $p\_{\mathrm{Jaccard}\ge 0.9}=0.0125$.  The run required $57377$ distinct sample attempts and recorded $55377$ duplicate retries.  In the amino-acid-class-preserving null run, also with $N=2000$ and seed $42$, the observed $M$ size was $20$, the exact-match count was $0$, the lower-tail count at or below the observed $M$ size was $7$, the exact-shape value was $0.0$, and the lower-tail value was $0.0035$.  These rows make the family-stratified evidence stronger than a bare visual coincidence, while still leaving it at the level of finite code-label geometry.

\paragraph{What is established.}
The established claim is a finite, local one.  In the six-coordinate codon cube, the standard-code labels expose a $13$-row boundary tile with a complete six-module classification, nontrivial spectral consistency, strong aggregate leave-one-code-out recovery, and family-specific above-baseline rates across five code families.  The curated standard-code table is the required contact for label readback; the coordinate construction supplies the local tile and the boundary partition.

\paragraph{Scope boundary.}
No translation process is inferred from these numbers.  No folded protein structure, structural order, physical admissibility, biological function, universal mechanism, or necessity claim follows from the local codon-window geometry.  Amino-acid label sequences are still code labels in this chapter, not realized proteins.  Any claim crossing into translation, folding, physical chemistry, function, mechanism, or universality would require a separate empirical or mathematical contact supplying that layer.  The present row therefore supports a family-stratified local code-read pattern and blocks promotion beyond the code table itself.
