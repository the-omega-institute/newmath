\subsection{Leave-one-code-out behavior by code family}\label{sec:h1-leave-one-code-out-by-family}
\origin{ai}

\paragraph{Finite question.}
This record tests whether the window-six codon coordinate construction retains predictive signal when the changed labels are grouped by genetic-code family rather than pooled across all observed code variants. The carrier is the six-coordinate codon cube, and the observed readout is restricted to the curated codon-to-label table. A local face, corner, or three-point label class is therefore treated only as a code-layer pattern. It is not treated as an account of translation chemistry, folded structure, physical viability, biological activity, mechanism, or universality.

\paragraph{Boundary row.}
The distinguished boundary set has size $13$. In the finite row it is exactly
$$\begin{aligned}
R=\{&\mathrm{AAA},\mathrm{AGA},\mathrm{AGG},\mathrm{AUA},\mathrm{CUA},\mathrm{CUC},\mathrm{CUG},\mathrm{CUU},\\
&\mathrm{UAA},\mathrm{UAG},\mathrm{UCA},\mathrm{UGA},\mathrm{UUA}\}.
\end{aligned}$$
The same row identifies this set as a punctured-cube pattern with face-vector
$$\begin{aligned}
(f_0,f_1,f_2)=(13,16,4).
\end{aligned}$$
The module classification is complete: all $13$ elements of $R$ are assigned, with no missing, extra, or unassigned codons. The six modules are
$$\begin{aligned}
\mathrm{agr}&=\{\mathrm{AGA},\mathrm{AGG}\},\\
\mathrm{ar\_lys}&=\{\mathrm{AAA}\},\\
\mathrm{cun}&=\{\mathrm{CUA},\mathrm{CUC},\mathrm{CUG},\mathrm{CUU}\},\\
\mathrm{ile\_met}&=\{\mathrm{AUA}\},\\
\mathrm{ssn}&=\{\mathrm{UCA},\mathrm{UUA}\},\\
\mathrm{stop\_trp}&=\{\mathrm{UAA},\mathrm{UAG},\mathrm{UGA}\}.
\end{aligned}$$
This matters for the family-level test because every boundary module is already classified as either predictable or boundary-supported before code families are separated.

\paragraph{Spectral coordinate.}
The associated spectral row gives
$$\begin{aligned}
\lambda_M&=0.6752479308830213,\\
\lambda_R&=0.4758352843197002,\\
\mu&=3.051487585298128.
\end{aligned}$$
The radial polynomial residual is reported as $-6.224354365258478\times 10^{-12}$, the same value recorded for the cubic-polynomial check. This is small at the scale of the computation and is used here only as a finite coordinate consistency check, not as a biological mechanism.

\paragraph{Pooled leave-one-code-out row.}
Before splitting by family, the leave-one-code-out experiment evaluated $23$ tables and $63$ targets. The pooled baseline hit rate was $0.19821739130434785$. The observed hit rate inside $M\setminus T$ was $0.9365079365079365$, and the observed hit rate inside $M\setminus T$ together with the boundary was $0.9841269841269841$. The row also records the hit rate above baseline as $0.9841269841269841$. Thus the pooled finite table says that almost all held-out code changes land in the internally marked or boundary-marked region, while a naive baseline would hit about one fifth of the time.

\paragraph{Family split.}
The family-stratified row contains $5$ families. The observed above-baseline hit rates are
$$\begin{aligned}
\mathrm{alternative\_yeast}&=0.8333333333333334,\\
\mathrm{bacterial\_plastid}&=1.0,\\
\mathrm{candidate\_other}&=1.0,\\
\mathrm{mitochondrial}&=1.0,\\
\mathrm{nuclear\_ciliate}&=1.0.
\end{aligned}$$
The family split therefore does not show that the pooled result is carried by only one broad class of code tables. Four of the five families reach the maximal recorded rate, and the remaining family, alternative yeast, still records $5/6$ at the displayed precision. The finite conclusion is narrower than a law of genetic-code evolution: within this sampled row, when code variants are withheld by family, the same local codon-coordinate boundary continues to catch most label changes above the baseline rate.

\paragraph{Leave-one-codon-out compatibility.}
The companion leave-one-codon-out row records anchor count $0$, boundary count $5$, predictable count $8$, and high closure coverage $8$. Its recovery rate is $1.0$, equal to the reported threshold comparison value $1.0$. This supports the same local reading from the codon side: the thirteen codons in $R$ are not merely an unstructured list, because the module partition accounts for all of them and the held-out codon test recovers the classified cases through predictable or boundary status.

\paragraph{Null rows.}
The shape null row sampled $N=2000$ distinct randomized cases with seed $42$. It required $57377$ distinct-sample attempts and $55377$ duplicate retries. The exact shape was matched once, giving $p=0.0005$, while $25$ samples achieved Jaccard score at least $0.9$, giving $p=0.0125$. The amino-acid-class-preserving null row, also with $N=2000$ and seed $42$, observed $0$ exact matches and $7$ samples in the lower tail at or below the observed $M$ size. It records observed $M$ size $20$, exact-shape $p=0.0$, and lower-tail $p=0.0035$. These rows make the family split harder to dismiss as an immediate artifact of random label shuffling, but they do not identify an underlying biochemical cause.

\paragraph{Scope of the claim.}
The evidence supports a code-read statement: in the curated standard-code label table and its sampled code variants, the window-six local geometry marks a boundary region whose family-stratified leave-one-code-out performance remains high. The statement stops there. A codon-to-label assignment is not a ribosomal translation event, an amino-acid chain is not a folded protein, and a local codon-tile split is not evidence of physical admissibility, biological function, universal mechanism, or global necessity. Any such stronger assertion would require a separate empirical contact that directly observes the higher layer.
