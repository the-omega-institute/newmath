\paragraph{Finite carrier.}
The window-six carrier is the codon cube $Q_6$, read as a six-bit display of the $64$ codons in the standard-code table.  The carrier is not introduced as a biological mechanism.  It is a finite coordinate display on which codon labels can be read back from the curated standard-code contact.  The distinguished local boundary row is
$$
\begin{aligned}
R=\{&AAA,AGA,AGG,AUA,CUA,CUC,CUG,CUU,\\
&UAA,UAG,UCA,UGA,UUA\}.
\end{aligned}
$$
The verified cardinality is $|R|=13$.  The same row is also verified as the punctured-cube row with $f$-vector $(13,16,4)$, so the finite object has $13$ displayed vertices, $16$ displayed local edges, and $4$ displayed square faces in the row complex.  This is a code-read datum: it records a finite label-facing subset inside the codon cube, not a statement about translation dynamics.
