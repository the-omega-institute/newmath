\subsection{Finite Carrier And Code-Layer Contact}\label{sec:bioreality-h0-spectrum-carrier}

\paragraph{Carrier.}
This record uses the six-bit codon cube $Q_6$ as a finite display carrier for the standard-code codon table.  A codon is read only as a coordinate in a finite window and then as a label through the curated standard-code bridge.  The carrier therefore contains four relevant distinctions: the six-bit codon display, the local tile face, the singleton label corner, and the three-point label class.  The biological object contacted here is the standard-code codon table, and the claimed layer is only code readback.  No higher-layer biological conclusion is part of this record.

\paragraph{Reality contact.}
The only external contact used by the row is the curated standard genetic-code table: it assigns codons to amino-acid or stop labels.  This contact can test codon-label readback and a local codon-tile label split.  It cannot test translation realization, folded structure, structural order, physical admissibility, biological function, mechanism, or universality.  The code table is therefore a label bridge, not a mechanism bridge.

\subsection{Boundary Set And Punctured-Cube Shape}\label{sec:bioreality-h0-spectrum-boundary}

\paragraph{Observed boundary row.}
The finite boundary row $R$ has cardinality $13$.  The codons in the row are
$$
\begin{aligned}
R=\{&AAA, AGA, AGG, AUA, CUA, CUC, CUG, CUU,\\
&UAA, UAG, UCA, UGA, UUA\}.
\end{aligned}
$$
The same thirteen codons are reported by the punctured-cube check.  Its finite face vector is
$$
\begin{aligned}
(f_0,f_1,f_2)=(13,16,4).
\end{aligned}
$$
Thus the row is not merely a list of exceptional labels; it has a small local cubical profile in the six-bit window.  The row also decomposes into the following six module classes:
$$
\begin{aligned}
\mathrm{AGR}&=\{AGA,AGG\},&\mathrm{AR\_Lys}&=\{AAA\},\\
\mathrm{CUN}&=\{CUA,CUC,CUG,CUU\},&\mathrm{Ile\_Met}&=\{AUA\},\\
\mathrm{SSN}&=\{UCA,UUA\},&\mathrm{Stop\_Trp}&=\{UAA,UAG,UGA\}.
\end{aligned}
$$
The module audit assigns all $13$ row codons, leaves no unassigned codons, has no missing row codons, and introduces no extra module codons.

\paragraph{Shape significance.}
Against the reported finite null for the $M$-shape, the row has observed $M$-size $20$.  In $N=2000$ sampled trials with seed $42$, there was $1$ exact shape match after $57377$ distinct-sample attempts and $55377$ duplicate retries.  The exact-shape probability was $p=0.0005$.  The high-overlap test found $25$ samples with Jaccard overlap at least $0.9$, giving $p=0.0125$.  In the amino-acid-class-preserving null with the same sample size and seed, the exact match count was $0$, the lower-tail count at or below the observed $M$-size was $7$, the exact-shape probability was $0.0$, and the lower-tail probability was $0.0035$.  These figures support a code-layer local shape signal; they do not promote the signal to a biological law.

The interpretation is deliberately narrow: the six-bit coordinate display, the closure row, and the spectrum agree as a finite code-read pattern. \subsection{Boundary Verdict}\label{sec:bioreality-h0-spectrum-verdict}

\paragraph{Supported claim.}
The ready claim is that window-six codon coordinates expose a local boundary-tile structure in the standard-code label table.  The support is finite and row-specific: $R$ has $13$ listed codons, the punctured-cube face vector is $(13,16,4)$, all $13$ row codons are assigned to the six named modules, the spectral values satisfy the reported polynomial relation to numerical tolerance, and the finite null probes give small shape probabilities.  The curated code table supplies the external label contact needed for code readback.

\paragraph{Cannot-claim boundary.}
Nothing in this chapter says that DNA obeys this geometry as a biological necessity.  Nothing here establishes translation realization, folded protein structure, structural order, physical admissibility, biological function, mechanism, or universality.  Amino-acid label-sequence readback is not folded protein structure.  A local codon tile is not a global biological law.  Any move beyond the code-read layer requires a separate contact that actually observes the higher layer.
