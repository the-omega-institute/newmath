\subsection{Spectral row for the window-six codon tile}\label{sec:bioreality-h0-spectrum}
\origin{ai}

\paragraph{Finite carrier and contact.}
The finite carrier is the six-bit codon cube $Q_6$, read as a codon display rather than as a biochemical mechanism.  The only external contact used in this row is the curated standard-code label table: it assigns each codon to an amino-acid or stop label and therefore supports code-layer readback.  The contact has resolution ``codon label readback''.  It does not supply a translation apparatus, a folding observation, a physical-admissibility test, a function assay, a mechanism, or a universality theorem.  Thus the BioReality claim in this row is local and finite: window-six coordinates expose a boundary-tile pattern in the code table, while the labels themselves remain reality contacts supplied by the curated table.

\paragraph{Boundary row.}
The distinguished boundary row $R$ has cardinality $13$.  Its codons are
$$\begin{aligned}
R=\{&\mathrm{AAA},\mathrm{AGA},\mathrm{AGG},\mathrm{AUA},\mathrm{CUA},\mathrm{CUC},\mathrm{CUG},\mathrm{CUU},\\
&\mathrm{UAA},\mathrm{UAG},\mathrm{UCA},\mathrm{UGA},\mathrm{UUA}\}.
\end{aligned}$$
The punctured-cube check returns the same list and records the finite face data
$$\begin{aligned}
(f_0,f_1,f_2)=(13,16,4).
\end{aligned}$$
This is the local tile shape used by the row.  It is not promoted to a global biological law: it is a finite subconfiguration inside the codon cube, tied to the standard-code label table only at the code-read layer.

\paragraph{Spectral row.}

\mu&=3.051487585298128.

The reported cubic residual is $-6.224354365258478\times 10^{-12}$, and the radial polynomial value is the same number. These figures make the row a spectral certificate for the finite tile: the computation is about the relation between the observed local shape, its row boundary, and the coordinate-spectrum data, not about a biochemical realization outside the supplied contact.

\paragraph{Shape significance.}
Against the sampled coordinate null with seed $42$, the run used $N=2000$ samples.  There was $1$ exact shape match, giving $p_{\mathrm{exact}}=0.0005$, and there were $25$ matches at Jaccard threshold at least $0.9$, giving $p_{J\geq 0.9}=0.0125$.  The distinct-sample process records $57377$ attempts and $55377$ duplicate retries.  These row numbers support the statement that the observed tile is rare in that null sample.  They do not identify a mechanism for why biology uses the standard code, and they do not assert that the coordinate geometry causes the code.

\paragraph{Amino-acid-class preserving null.}
The amino-acid-class preserving null gives a separate check.  In $N=2000$ samples with seed $42$, the exact match count is $0$, so the reported exact-shape value is $p=0.0$ at this sample resolution.  The observed $M$ size is $20$, and the lower-tail count at or below that size is $7$, giving a lower-tail value $p=0.0035$.  This shows that preserving amino-acid class does not by itself erase the finite-row signal.  The explicit codon membership list for $M$ is not part of this finite row, so this chapter uses only the supplied size and spectral value for $M$.

\paragraph{Cannot-claim boundary.}
The row supports a finite claim at the code-read layer: the curated standard-code table, read through window-six coordinates, has a local tile boundary with the displayed spectrum, rarity checks, leave-one-code-out recovery, leave-one-codon-out recovery, and complete module classification.  Nothing in this row licenses a claim about translation, folded protein structure, physical admissibility, biological function, causal mechanism, or universality.  Any such claim requires a separate reality contact with the corresponding resolution.  Amino-acid label-sequence readback remains code-layer data; it is not folded structure and it is not an assay of function.
