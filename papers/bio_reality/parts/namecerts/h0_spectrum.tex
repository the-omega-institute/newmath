\begin{aligned}

$$

$$
\begin{aligned}

\end{aligned}
$$
The amino-acid-class-preserving null used the same sample count and seed. In that null, the observed finite row has $|M|=20$, exact-match count $0$, and lower-tail count $7$, so the reported values are

\begin{aligned}
