\paragraph{Finite carrier and contact.}
The finite carrier is the six-bit codon cube $Q_6$, read only through a curated standard-code table that assigns each codon to an amino-acid or stop label.  The BEDC row does not treat that table as a mechanism.  It is a reality contact for code-layer label readback: it can test the local codon-tile label split, but it cannot test translation realization, folded structure, physical admissibility, biological function, a universal mechanism, or any necessity claim about DNA geometry.  The local object exposed by the row is therefore a window-six boundary tile, not a global biological law.
