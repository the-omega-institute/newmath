\subsection{Leave-one-codon-out recovery in the window-six code layer}
\label{sec:h1-leave-one-codon-out}

\paragraph{Finite carrier and contact.}
The carrier is the six-bit codon cube $Q_6$ with the standard-code label table used only as an external contact for codon-to-label readback.  The local object under test is not a mechanism of translation.  It is a finite boundary tile inside the code layer: six-bit codon display, local tile face, singleton label corner, and three-point label class.  The distinguished recovered row set $R$ has cardinality $13$ and is exactly
$$
\begin{aligned}
R=\{&\mathrm{AAA},\mathrm{AGA},\mathrm{AGG},\mathrm{AUA},\mathrm{CUA},\mathrm{CUC},\mathrm{CUG},\mathrm{CUU},\\
&\mathrm{UAA},\mathrm{UAG},\mathrm{UCA},\mathrm{UGA},\mathrm{UUA}\}.
\end{aligned}
$$
The same row set is recorded as a punctured-cube pattern with face vector
$$
\begin{aligned}
(f_0,f_1,f_2)=(13,16,4).
\end{aligned}
$$
Thus the row-level datum is not merely a list of exceptional codons; it carries a finite cubical shape with $13$ vertices, $16$ one-faces, and $4$ two-faces in the induced local tile.

\paragraph{Leave-one-codon-out recovery.}
The direct leave-one-codon-out run for this chapter reports $8$ predictable codons, $5$ boundary codons, and $0$ anchor codons.  Its high-coverage closure count is $8$, and the observed recovery rate is
$$
\begin{aligned}
\mathrm{recovery\_rate}=1.0,
\qquad
\mathrm{recovery\_rate\_above\_threshold}=1.0.
\end{aligned}
$$
The interpretation is deliberately local.  With one codon withheld from the code-layer table, the window-six coordinate and its closure probes recover the withheld codon as predictable or place it on the boundary of the same finite tile.  The result does not say that the coordinate geometry causes the biological code; it says that this finite code-read contact is organized enough for the omitted row to remain visible to the local tile test.

Against the finite shape null with $N=2000$ sampled alternatives, there is $1$ exact shape match after $57377$ distinct-sample attempts and $55377$ duplicate retries, giving

\begin{aligned}

\qquad

with $25$ high-Jaccard matches.  Against the amino-acid-class-preserving null with $N=2000$, the observed $M$-size is $20$, the exact-match count is $0$, and the lower-tail count at or below the observed $M$-size is $7$, yielding

\begin{aligned}

$$
The finite rows therefore support a real code-layer signal: the local tile is rare under both a shape null and an amino-acid-class-preserving null.  The data supplied here include the scalar $M$-size and shape tests, but not a separate explicit list of the $20$ elements of $M$.

\paragraph{Cannot-claim boundary.}
The only reality contact used here is a curated standard genetic-code table assigning codons to amino-acid or stop labels.  Consequently the claim remains at $\mathrm{code\_read}$.  No translation realization, folded structure, structural order, physical admissibility, biological function, universal mechanism, or global biological law follows from these rows.  Amino-acid label-sequence readback is not folded protein structure.  A local codon tile is not a proof that DNA obeys BEDC geometry as biological necessity.  Any promotion beyond code-layer readback requires a separate higher-layer contact that observes the promoted layer directly.
