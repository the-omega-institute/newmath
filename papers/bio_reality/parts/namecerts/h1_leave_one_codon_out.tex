\subsection{Leave-One-Codon Local Boundary Test for the Standard Code}
\label{subsec:h1-leave-one-codon-out}

\paragraph{Finite codon display.}
This record treats the standard-code codon table as a finite labelled object placed on the six-coordinate codon cube $Q_6$.  The coordinates provide the internal display, while the amino-acid and stop labels come from the curated standard-code table.  The claim is deliberately local: the computation asks whether removing one codon label at a time still leaves enough nearby codon-tile information to recover the omitted label-class behavior at the code-read layer.  It does not assert that the cube geometry is a law of molecular biology, nor that the readback realizes translation, folding, physical feasibility, function, mechanism, or universality.

\paragraph{The recovered region.}
The finite exceptional region $R$ has cardinality $13$.  In codon notation it is
$$
\begin{aligned}
R=\{&\mathrm{AAA},\mathrm{AGA},\mathrm{AGG},\mathrm{AUA},\mathrm{CUA},\mathrm{CUC},\mathrm{CUG},\mathrm{CUU},\\
&\mathrm{UAA},\mathrm{UAG},\mathrm{UCA},\mathrm{UGA},\mathrm{UUA}\}.
\end{aligned}
$$
The same row identifies this set as a punctured-cube shaped region with face-vector data $(13,16,4)$.  Thus the codon subset is not introduced as a free biological category; it is the finite part of the coordinate display that is visible to the local boundary test.  The module assignment is complete on this region: $R$ has size $13$, the assigned count is $13$, and the lists of extra module codons, missing codons, and unassigned codons are all empty.

\paragraph{Module partition.}
The complete module classification is
$$
\begin{aligned}
\mathrm{agr}&=\{\mathrm{AGA},\mathrm{AGG}\},\\
\mathrm{ar\_lys}&=\{\mathrm{AAA}\},\\
\mathrm{cun}&=\{\mathrm{CUA},\mathrm{CUC},\mathrm{CUG},\mathrm{CUU}\},\\
\mathrm{ile\_met}&=\{\mathrm{AUA}\},\\
\mathrm{ssn}&=\{\mathrm{UCA},\mathrm{UUA}\},\\
\mathrm{stop\_trp}&=\{\mathrm{UAA},\mathrm{UAG},\mathrm{UGA}\}.
\end{aligned}
$$
Every listed module has either a predictable codon or a boundary codon in the leave-one-codon experiment.  This matters because the test is not merely recovering isolated names: it checks that the six-bit local display can organize all thirteen codons into the observed module classes without leaving residue outside the classification.

\paragraph{Leave-one-codon outcome.}
For the target row $\mathrm{h1.leave.one.codon.out}$, the experiment run reports $0$ anchors, $5$ boundary cases, $8$ high-closure-coverage cases, and $8$ predictable cases.  The recovery rate is $1.0$, and the recovery-rate-above-threshold value is also $1.0$.  In the finite interpretation used here, the result says that the omitted codon is always recovered at the intended local code-read level when the curated table supplies the labels and the coordinate construction supplies the neighborhood geometry.  The absence of anchor cases is also informative: the positive result is not being driven by a trivial singleton anchor already fixing the answer.  It is instead a boundary-and-closure result across the finite tile.

\paragraph{Shape and spectrum checks.}
The shape-significance row used $N=2000$ samples with seed $42$.  It required $57377$ distinct-sample attempts and encountered $55377$ duplicate retries.  It found $1$ exact shape match and $25$ matches at Jaccard score at least $0.9$, giving
$$
\begin{aligned}
p_{\mathrm{exact}}&=0.0005,\\
p_{\mathrm{Jaccard}\geq 0.9}&=0.0125.
\end{aligned}
$$
The amino-acid-class-preserving null row also used $N=2000$ and seed $42$.  It reports observed $M$-size $20$, exact-match count $0$, lower-tail count $7$, exact-shape probability $0.0$, and lower-tail probability $0.0035$.  The spectrum row records

The same numerical residual is reported for the cubic-polynomial check, and the spectral-radius-in-tolerance value equals $0.6752479308830213$.  These numerical witnesses strengthen the finite-row claim that the observed local tile is structured rather than a generic random subset of the same codon universe.

\paragraph{Reality contact and scope.}
The only reality contact used here is the curated standard-code table, understood as a codon-to-label source.  It can test code readback and local codon-tile label splitting.  It cannot test translation realization, folded structure, physical admissibility, biological function, or universal mechanism.  Therefore an amino-acid label sequence read from this table is still only a label sequence at the code layer.  A separate structure experiment would be required before any folded-structure statement; a separate biochemical or organism-level contact would be required before any functional, physical, or mechanistic statement.  The present chapter records a finite local boundary phenomenon in the standard-code label table and stops at that boundary.
