The row is intentionally code-layer only: the contact is a curated codon-to-label table, so the observable output is an amino-acid or stop label attached to each codon.  The BEDC distinctions used here are the six-bit codon display, a local tile face, a singleton label corner, and a three-point label class. The claim tested in this row is that leaving one codon out of the local code readback still exposes a finite boundary-tile signature; it is not a claim about translation realization, folding, physical admissibility, function, universal biological law, or mechanism.

The supported statement is that the six-bit codon-coordinate view exposes a local boundary-tile structure for the standard-code label table, with $13$ relation codons, face vector $(13,16,4)$, spectral values $\lambda_M=0.6752479308830213$ and $\lambda_R=0.4758352843197002$, exact-shape null probability $0.0005$, amino-acid-class-preserving lower-tail probability $0.0035$, leave-one-code hit rate above baseline $0.9841269841269841$, and leave-one-codon recovery rate $1.0$.
