\subsection{Finite Carrier and External Label Contact}
\label{sec:h1-leave-one-codon-out-carrier}

\paragraph{Carrier.}
The finite carrier is the six-bit codon display $Q_6$ equipped with the standard-code label readback supplied by a curated codon table.  The BEDC-side object is not translation itself; it is a coordinate record whose internal structure consists of coordinate, closure, spectrum, and relation data.  The biological contact is only the observed fact that the curated standard-code table assigns each codon to an amino-acid or stop label.  Thus the contact has codon-label resolution and tests the code-read layer, not protein realization, folding, physical admissibility, function, mechanism, or universality.

\paragraph{Observed boundary row.}
The distinguished finite row $R$ has cardinality $13$.  The verified codon list is
$$
\begin{aligned}

The row is therefore a concrete coordinate object inside the codon cube, not a verbal description of codon-table behavior.

$$

\subsection{Local Boundary Verdict}
\label{sec:h1-leave-one-codon-out-verdict}

\paragraph{Positive claim.}
The supported claim is that window-six codon coordinates expose a local boundary-tile structure in the standard-code label table.  The finite row $R$ has $13$ codons, punctured-cube $f$-vector $(13,16,4)$, stable spectral values, significant shape-null behavior, complete module assignment, and perfect leave-one-codon recovery under the recorded predictable-or-boundary criterion.  These facts justify a code-read statement: the curated standard-code label table contains a local tile whose boundary and closure behavior recover the held-out codon row in this finite probe.

\paragraph{Cannot-claim boundary.}
No translation, folded structure, structural order, physical admissibility, biological function, universal mechanism, or biological necessity follows from this chapter.  The curated code table supplies codon-to-label assignments only.  Amino-acid label readback is not folded protein structure, and a local codon tile is not a global biological law.  Any claim crossing from code-read geometry to translation realization, physical chemistry, structure, or function would require a separate reality contact with its own finite observations and its own null tests.

\paragraph{Interpretive rule.}
The safe interpretation is therefore deliberately narrow.  The BEDC coordinate object organizes a finite codon-label table into a local closure-boundary pattern; the external contact tells us that those labels are the curated standard-code labels; the probe verifies recovery and module coverage inside that code layer.  The chapter records a real finite-row signal at the code-read level and blocks promotion beyond that level.
