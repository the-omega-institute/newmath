\subsection{Window-six codon tile significance}\label{subsec:h0-m-shape-significance}\origin{ai}

\paragraph{Claim.}
The finite object studied here is the standard-code codon table viewed through a six-coordinate codon display.  The relevant observation is not that the genetic code is forced by this display, but that the displayed codons expose a sharply localized boundary pattern when the curated standard-code labels are read as amino-acid-or-stop labels.  The carrier is the six-bit codon cube $Q_6$, and the inspected distinctions are the local tile face, the singleton label corner, and the three-point label class.  The biological contact is deliberately narrow: the curated table assigns each codon to a label, and this chapter asks whether the observed local split has finite-row significance relative to label-preserving or shape-preserving null probes.

\paragraph{Boundary set.}
The boundary row $R$ has cardinality $13$.  The verified row is
$$\begin{aligned}
R=\{&\texttt{AAA},\texttt{AGA},\texttt{AGG},\texttt{AUA},\texttt{CUA},\texttt{CUC},\texttt{CUG},\\
&\texttt{CUU},\texttt{UAA},\texttt{UAG},\texttt{UCA},\texttt{UGA},\texttt{UUA}\}.
\end{aligned}$$
The same row is certified as a punctured-cube pattern with face-vector $(13,16,4)$.  Thus the codon boundary is not recorded merely as a list of exceptional names; it also has a finite cubical shape: $13$ vertices, $16$ one-dimensional incidences, and $4$ two-dimensional faces in the selected local geometry.  This is the row-level datum used below when the chapter compares the actual table with randomized alternatives.

\paragraph{Shape test.}
For the main shape-significance test, the finite run used seed $42$ and drew $N=2000$ distinct sampled shapes.  Constructing those distinct samples required $57377$ sample attempts, with $55377$ duplicate retries, so the empirical comparison is sampling from a heavily constrained finite space rather than from an unconstrained stream of independent names.  In that run there was $1$ exact shape match and $25$ matches with Jaccard similarity at least $0.9$.  The corresponding empirical probabilities were
$$\begin{aligned}
p_{\mathrm{exact}}&=0.0005,\\
p_{J\geq 0.9}&=0.0125.
\end{aligned}$$
Both reported significance fields agree with these values: the exact-shape significance value is $0.0005$, and the high-Jaccard significance value is $0.0125$.  The evidence therefore supports a specific finite statement: in this sampling regime, the observed local codon boundary is rare as an exact row shape and remains uncommon even after allowing high-overlap near misses.

\paragraph{Amino-acid-class preserving null.}
A separate null model preserved amino-acid class information while testing the size and shape behavior of the observed module.  This run also used $N=2000$ and seed $42$.  The observed $M$ size was $20$.  The exact-match count was $0$, giving reported exact-shape probability $0.0$ in the sampled row set.  The lower-tail count at or below the observed $M$ size was $7$, so the lower-tail probability was $0.0035$.  In symbols, the recorded row values are
$$\begin{aligned}
|M|_{\mathrm{obs}}&=20,\\
\#\{\text{exact class-preserving matches}\}&=0,\\
\#\{\text{rows with size}\leq 20\}&=7,\\
p_{\mathrm{lower\ tail}}&=0.0035.
\end{aligned}$$
This second comparison matters because it keeps more label-class structure in place.  The result still does not promote the codon tile into a mechanism.  It says only that, under the recorded class-preserving finite randomization, the observed row size and exact shape are not typical of the sampled alternatives.

\paragraph{Spectral row.}
The spectral check gives three scalar coordinates for the same finite pattern.  The verified values are $\lambda_M=0.6752479308830213$, $\lambda_R=0.4758352843197002$, and $\mu=3.051487585298128$.  The radial cubic residual is $-6.224354365258478\times 10^{-12}$, and the same value is recorded as the cubic-polynomial residual.  The spectral-radius value in tolerance is $0.6752479308830213$.  These numbers function as a consistency check on the finite geometry: the observed row is not only a named codon set and not only a sampled-significance event; it also occupies a stable spectral position within the coordinate calculation used by the packet.

The leave-one-code-out check evaluated $23$ tables and $63$ total targets.  The baseline hit rate was $0.19821739130434785$.  Against that baseline, the hit rate above baseline was $0.9841269841269841$.  The hit rate inside $M\setminus T$ was $0.9365079365079365$, and the hit rate inside $M\setminus T$ plus boundary was $0.9841269841269841$.  The family-level refinement used $5$ families.  The above-baseline family hit rates were $0.8333333333333334$ for alternative yeast, $1.0$ for bacterial-plastid, $1.0$ for candidate-other, $1.0$ for mitochondrial, and $1.0$ for nuclear-ciliate.  These finite rows support a code-layer generalization: when related code tables are held out, the local row is useful for recovering label changes in the tested target set.  They do not show that organisms implement the geometry as a biochemical rule.

\paragraph{Leave-one-codon behavior.}
The leave-one-codon-out row gives a complementary local check.  The anchor count was $0$, the boundary count was $5$, the high-closure coverage was $8$, and the predictable count was $8$.  The recovery rate was $1.0$, and the threshold comparison also recorded $1.0$.  Thus, within this finite evaluation, every tested held-out codon that the packet classifies as recoverable is recovered by the local closure-or-boundary criterion.  This is a strong code-read result, but it is still a statement about codon labels and finite local geometry.

Every listed module has at least one predictable or boundary-classified codon under the packet's finite criterion.  The assignment is therefore closed at the row-accounting level: the thirteen codons named by $R$ are exactly the codons named by the module decomposition.

\paragraph{Scientific interpretation.}
The result is best read as a finite code-layer claim.  A six-coordinate codon display, when paired with the curated standard-code label table, yields a local boundary row with $13$ codons, punctured-cube face-vector $(13,16,4)$, rare exact-shape frequency $0.0005$, high-overlap frequency $0.0125$, a class-preserving lower-tail value $0.0035$, stable spectral scalars, and strong leave-one-out recovery rows.  These facts make the local codon tile a meaningful target for further finite probes.

\paragraph{Local scope limit.}
Nothing in this chapter establishes translation realization, folded structure, physical admissibility, biological function, a biochemical mechanism, or a universal biological law.  The curated standard-code table supplies codon-to-label assignments only.  Amino-acid label readback is not a protein structure observation, and a local codon tile is not evidence that DNA must obey this geometry in living systems.  Any claim crossing from code labels to translation, folding, physical chemistry, organismal function, or universality requires an additional contact that directly measures that higher layer.
