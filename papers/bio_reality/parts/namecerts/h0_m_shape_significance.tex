\subsection{Window-Six Codon Shape Significance}\label{sec:h0-m-shape-significance}
\origin{ai}

\paragraph{Claim.}
The claim \texttt{h0.M.shape.significance} concerns a finite code-layer pattern in the standard genetic-code table.  The carrier is the six-coordinate codon cube $Q_6$, read as a window-six display of the $64$ codons.  The biological contact is only the curated standard-code assignment of codons to amino-acid or stop labels.  Within that contact, the relevant question is whether the observed local label split has an unusually specific shape when compared with finite randomized label arrangements that preserve the same ambient codon cube.  The result is a code-read statement: it says that the observed window-six shape is statistically rare under the stated finite sampling procedure, not that the geometry is a law of molecular biology.

\paragraph{Observed boundary row.}
The finite boundary set $R$ has cardinality $13$.  The verified row lists the codons exactly as
$$
\begin{aligned}
R=\{&\mathrm{AAA},\mathrm{AGA},\mathrm{AGG},\mathrm{AUA},\mathrm{CUA},\mathrm{CUC},\mathrm{CUG},\mathrm{CUU},\\
&\mathrm{UAA},\mathrm{UAG},\mathrm{UCA},\mathrm{UGA},\mathrm{UUA}\}.
\end{aligned}
$$
The same row is certified as a punctured-cube shape with face data
$$
\begin{aligned}
(f_0,f_1,f_2)=(13,16,4).
\end{aligned}
$$
Thus the local object is not merely a list of codons: it has a finite incidence profile inside $Q_6$, with $13$ vertices, $16$ one-dimensional adjacencies, and $4$ two-dimensional faces in the recorded tile.  The complementary label-side row $M$ is not enumerated in this data packet, but its observed size is recorded as $20$ in the amino-acid-class preserving null experiment.

\paragraph{Shape significance.}
The Monte Carlo row for \texttt{h0.M.shape.significance} used seed $42$ and $N=2000$ sampled alternatives.  To obtain the $2000$ distinct samples, the run recorded $57377$ distinct-sample attempts and $55377$ duplicate retries.  Among those $2000$ alternatives there was exactly $1$ exact shape match, giving
$$
\begin{aligned}
 p_{\mathrm{exact}}=0.0005.
\end{aligned}
$$
There were $25$ alternatives with Jaccard overlap at least $0.9$, giving
$$
\begin{aligned}
 p_{J\geq 0.9}=0.0125.
\end{aligned}
$$
Both reported significance fields repeat these same finite fractions: the exact-shape significance value is $0.0005$, and the high-Jaccard significance value is $0.0125$.  The evidence is therefore a finite-row rarity statement about the observed shape, with the duplicate-retry count included as an audit trace of the sampling constraint.

\paragraph{Class-preserving null.}
A second null row preserves amino-acid class information.  It again used seed $42$ and $N=2000$.  In this stricter comparison, the exact match count was $0$, so the recorded exact-shape value is
$$
\begin{aligned}
 p_{\mathrm{class},\mathrm{exact}}=0.0.
\end{aligned}
$$
The observed $M$ size is $20$.  The lower-tail count for alternatives with size at most the observed value is $7$, giving
$$
\begin{aligned}
 p_{\mathrm{class},\mathrm{tail}}=0.0035.
\end{aligned}
$$
This row supports the same interpretation as the shape-significance row but under a more biologically constrained randomization: preserving label class still does not make the observed local pattern typical.

\paragraph{Scope boundary.}
The curated standard-code table supplies codon-to-label readback and local split checks.  It does not observe ribosomal realization, folding order, physical admissibility, biological function, evolutionary necessity, or a universal generative mechanism.  Consequently, this chapter permits the statement that a window-six codon coordinate exposes a rare and reproducible local tile in the standard-code label table.  It does not permit promotion from that tile to translation, folded protein structure, molecular viability, organismal function, or universality unless an additional independent contact is supplied for that higher layer.
