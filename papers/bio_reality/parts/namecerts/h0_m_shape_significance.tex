\subsection{Window-Six Local Tile Significance in the Standard Code}\label{sec:bioreality-h0-m-shape-significance}\origin{ai}

\paragraph{Finite carrier.} This BioReality record concerns the standard-code codon table read through a six-bit codon display $Q_6$. The carrier is finite: each codon is treated as one vertex of the codon cube, and the admissible observations are only coordinate, closure, spectrum, and relation data extracted from the table. The external contact is the curated standard genetic-code table, used only as a codon-label readback bridge. Thus the biological object under test is not translation as a mechanism, not a folded protein, and not a physical or functional realization. It is the code-layer assignment of codons to amino-acid or stop labels.

\paragraph{Observed boundary row.} The distinguished boundary row $R$ has cardinality $13$. In the verified run it is exactly
$$
\begin{aligned}
R=\{&AAA, AGA, AGG, AUA, CUA, CUC, CUG, CUU,\\
&UAA, UAG, UCA, UGA, UUA\}.
\end{aligned}
$$
The same finite row is certified as a punctured-cube pattern with face-vector
$$
\begin{aligned}
(f_0,f_1,f_2)=(13,16,4).
\end{aligned}
$$
This gives the local tile a concrete shape: thirteen codon vertices, sixteen one-dimensional adjacencies, and four two-dimensional faces. The record therefore has enough internal geometry to support a local boundary statement, but the geometry is still only a codon-display geometry. It does not by itself identify any molecular dynamics, ribosomal process, folding pathway, or universal biological law.

\paragraph{Shape significance.} The main shape test used $N=2000$ distinct sampled comparison shapes with seed $42$. Because the sampler rejected duplicates, it made $57377$ distinct-sample attempts and recorded $55377$ duplicate retries. Against this null sample, the exact local shape occurred once, giving
$$
\begin{aligned}
\#\{\hbox{exact shape matches}\}&=1,\\
p_{\mathrm{exact}}&=0.0005.
\end{aligned}
$$
A high-overlap comparison using the threshold $J\geq 0.9$ produced $25$ matches among the same $2000$ samples, giving
$$
\begin{aligned}
\#\{J\geq 0.9\}&=25,\\
p_{J\geq 0.9}&=0.0125.
\end{aligned}
$$
The finite-row conclusion is therefore not that the codon table must obey this geometry, but that the observed local tile is rare under the sampled shape null at both the exact-shape and high-Jaccard thresholds.

\paragraph{Amino-acid-class-preserving null.} A second null model preserves amino-acid class information. In that test the observed $M$ row has size $20$. Among $2000$ sampled class-preserving comparisons, the exact shape count is $0$, so the reported exact-shape empirical value is $p=0.0$ at this sample resolution. The lower-tail count at or below the observed $M$ size is $7$, giving
$$
\begin{aligned}
|M|&=20,\\
\#\{\hbox{exact class-preserving matches}\}&=0,\\
\#\{\hbox{lower-tail samples}\leq |M|\}&=7,\\
p_{\mathrm{lower\ tail}}&=0.0035.
\end{aligned}
$$
The finite data therefore indicate that the observed tile is also unusual relative to this more constrained class-preserving background. No list of the individual $M$ codons is present in the finite row supplied here, so the chapter uses only the verified scalar $|M|=20$ for that row.

\paragraph{Spectral readback.} The spectral row separates the $M$ and $R$ components numerically. The verified values are
$$
\begin{aligned}
\lambda_M&=0.6752479308830213,\\
\lambda_R&=0.4758352843197002,\\
\mu&=3.051487585298128.
\end{aligned}
$$
The associated cubic or radial polynomial residual is approximately
$$
\begin{aligned}
-6.224354365258478\times 10^{-12},
\end{aligned}
$$
which is numerically consistent with the asserted spectral relation at the reported tolerance. This spectrum is an internal probe of the finite codon-coordinate object. It is not evidence for folding order, physical admissibility, function, mechanism, or universality.

\paragraph{Leave-one-code-out behavior.} The leave-one-code-out probe evaluated $23$ code tables and $63$ total targets. The baseline hit rate is $0.19821739130434785$. The observed hit rate above baseline is $0.9841269841269841$, the hit rate inside $M\setminus T$ is $0.9365079365079365$, and the hit rate inside $M\setminus T$ together with the boundary is $0.9841269841269841$. Thus the local tile is not merely a static shape statistic: it also carries strong code-read predictive structure under this finite deletion probe. The family-split row contains $5$ families, with hit rates above baseline

\begin{aligned}

These numbers support a code-layer robustness statement for the curated-table readback. They do not promote the result to a claim about translation realization or functional necessity.

\paragraph{Leave-one-codon-out and module classification.} The leave-one-codon-out row reports anchor count $0$, boundary count $5$, high closure coverage $8$, predictable count $8$, and recovery rate $1.0$. The module classification is complete: the module row has $R$ size $13$, assigned count $13$, no extra module codons, no missing codons, and no unassigned codons. The six verified modules are
$$
\begin{aligned}
\mathrm{agr}&=\{AGA,AGG\},\\
\mathrm{ar\_lys}&=\{AAA\},\\
\mathrm{cun}&=\{CUA,CUC,CUG,CUU\},\\
\mathrm{ile\_met}&=\{AUA\},\\
\mathrm{ssn}&=\{UCA,UUA\},\\
\mathrm{stop\_trp}&=\{UAA,UAG,UGA\}.
\end{aligned}
$$
Every module has predictable-or-boundary support, and the union of these modules accounts for the thirteen codons in $R$. This is the strongest local finite-row statement available here: the codon boundary row is not only rare under sampled nulls, but also decomposes into a complete module classification inside the supplied coordinate readback.

\paragraph{Local verdict and boundary.} The ready conclusion is a bounded significance claim. Window-six codon coordinates expose a rare local boundary-tile structure in the standard-code table, with exact-shape empirical value $0.0005$, high-Jaccard empirical value $0.0125$, class-preserving lower-tail value $0.0035$, a thirteen-codon punctured-cube boundary, and complete module coverage of $R$. The reality contact, however, is only the curated standard-code label table. Therefore the chapter cannot claim translation, folded structure, physical admissibility, biological function, mechanism, or universality. Any such promotion requires a separate higher-layer contact that is not present in this finite row.
